三星Galaxy Note 20+通过FCC认证

July 07, 2020

距离三星Galaxy Note20系列的正式发布还有不到一个月的时间，虽然三星还没有正式宣布发布日期，但最近三星Galaxy Note 20+已经通过了FCC认证。通常情况下，一款智能手机只有在临近上市的时候才会获得FCC的批准，我们也期待即将上市的Galaxy Note 20系列也会如此。换句话说，Note 20+在FCC认证网站上的上市，在某种程度上证实了传闻中8月5日的上市日期。

不过，FCC认证网站并没有提供太多关于三星Galaxy Note 20+的信息。我们得到的唯一有用信息是，Note 20+的型号将是SM-N986U，并且这是我们之前已经知道的。

据悉，三星Galaxy Note20系列和S20系列一样包括三款手机，分别是Galaxy Note20、Galaxy Note20+和Galaxy Note20 Ultra。Galaxy Note20 Ultra在设计上将与Note 10+有很多相似之处。Galaxy Note20 Ultra几乎囊括了所有最新的组件，正如你对2020年中期旗舰的期望，它搭载了骁龙865+，并采用120Hz LTPO显示屏，屏幕分辨率为2560×1440(QHD)。关于Note20 Ultra的另一个有趣的功能是，用户将可以同时开启QHD+120Hz，这是S20用户无法做到的。
