技嘉发布G242-Z11高性能计算服务器新品

August 07, 2020

技嘉今日发布了基于 AMD 双路霄龙（Epyc）Rome 处理器 + 四路英伟达 Ampere A100 GPU 的 G242-Z11 高性能计算（HPC）服务器。作为 G242 系列服务器产品线的新成员，其支持 PCIe 4.0 连接，专为人工智能、深度学习、数据分析和科学计算而设计。每个插槽上的二代霄龙（Epyc 7002 系列）处理器可提供最高 64 核 / 128 线程（Epyc 7H12），以及 128 条原生 PCIe 4.0 通道。

Epyc 7H12 的热设计功耗（TDP）为 280W（来自：Gigabyte）

技嘉 G242-Z11 高性能计算服务器最高支持 AMD 双路 Epyc 7002 系列处理器 + 四路 Nvidia PCIe Gen4 GPU 加速卡，并且提供了一个专用的 mLAN 远程管理端口。

存储方面，每颗二代霄龙 CPU 都支持八通道 RDIMM / LRDIMM DDR4 内存。每通道 1 DIMM 的配置下，最高支持 2TB 容量 @ 3200 频率。

此外服务器正面有四个可热插拔的 3.5 英寸 SATA HDD / SSD 驱动器位，背面则是双 NVMe / SAS / SATA 驱动器位。

技嘉 G242-Z11 采用了 2U 设计，提供了两个半高的 PCIe Gen4、以及一个 OCP 3.0 Gen4 x16 夹层插槽，整机由两个 80 Plus 钛金认证的 1600W 电源组成冗余。

与 PCIe 3.0 相比，PCIe 4.0 将速率带宽再次翻倍。每个通道 16 GT/s，总带宽 64 GB/s 。

通过专用的扩展槽托架，每个 GPU 都可以经由 PCIe 根连接到 CPU 进行大数量的并行处理，其余扩展插槽还可用于添加额外的网络 / 存储。

据悉，技嘉为 G242 系列高性能计算服务器提供了两个细分型号。除了上文介绍的 G242-Z11，还有一款 G242-Z10 。

两种系统配置均以性能为导向，控制将成本开销降至最低，追求总体成本控制（TCO）的客户可根据实际情况来选择合适的配置。
