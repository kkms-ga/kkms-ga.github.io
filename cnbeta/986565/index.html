德国汉堡市计划用开源软件替代微软产品

June 03, 2020

从微软商用产品迁移到开放源码的解决方案，可带来降低许可成本、以及更好地管控完整代码等益处。只是受历史数据和使用习惯等因素的影响，极少有大型机构愿意冒着风险去迁移。不过继慕尼黑之后，又一个德国城市表达了想要摆脱微软束缚的想法，它就是最近正在积极拥抱开源的汉堡市。

6·18活动已全面开启 大促活动入口汇总：

阿里云6·18上云年中大促 点击领取最高12000元红包

2020天猫6·18超级红包在此领取 6月1日追加40亿元消费券

京东6·18十七周年庆大促主会场入口 - 最高可领618元红包

报道称，在慕尼黑成为这方面的先驱之后，汉堡市也正考虑在本地计算机上使用开源软件，以减少对付费产品的依赖，并称之为“数字主权”的典范。

《Der Standard》的一份报告指出：尽管向开源过渡只是时间问题，但该项目仍处于初步阶段，因此目前所知的细节仍很少。

比如当地政府是考虑全面转向开源，还是借助 LibreOffice 等生产力套件向开源部分过渡。

如果是前者，那 Linux 操作系统很可能取代当前使用的微软 Windows 视窗系统。

至于是拿当前 Linux 社区中现成的发行版，还是自行开发衍生版本，仍有待时间去检验。

据悉，慕尼黑市已经创建了自己的 LiMux 项目，以推动向开源软件的过渡。同时当地政府还有在使用 OpenOffice（以及后来的 LibreOffice）来替代微软 Office 生产力套件。

无论怎样，对于 Linux 和开源生态来说，上述两个城市的支持，都有助于推动开源领域的快速发展。
