无缘高刷 三星Galaxy Note 20爆料：60Hz直屏/FHD+分辨率

June 18, 2020

6月18日消息，据外媒报道，三星预计将于8月5日在线上发布Galaxy Note 20系列。根据已经披露的信息，Galaxy Note 20系列至少有两款：Galaxy Note 20和Galaxy Note 20 Ultra，可能还会有Galaxy Note 20+。

外媒指出，Galaxy Note 20是Galaxy Note 10的小幅升级版，它采用的是直屏而非曲屏，分辨率为FHD+，刷新率只有60Hz。

至于Galaxy Note 20 Ultra，它采用的是QHD+分辨率、120Hz LTPO显示屏，无论是分辨率还是刷新率都远远高于Galaxy Note 20。

从Onleaks曝光的渲染图看出，Galaxy Note 20系列采用的是挖孔屏方案，前置摄像头居中排布，正面造型与Galaxy Note 10系列区别不大。

核心配置上，Galaxy Note 20、Galaxy Note 20 Ultra将搭载高通骁龙865旗舰平台，最高配备16GB内存，后置主摄最高为一亿像素（108MP），支持OIS光学防抖，支持超级快充。

此外，三星将在下半年推出Galaxy Fold 2和Galaxy Z Flip 5G。业内人士称由于UTG玻璃供货不足，Galaxy Fold 2、Galaxy Z Flip初期产能有限，预计在8月下旬或9月份登场。
