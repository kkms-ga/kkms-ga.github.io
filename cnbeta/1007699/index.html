回归塑料！三星Galaxy Note 20爆料：金属中框+塑料后壳

July 25, 2020

三星将于8月5日发布Galaxy Note 20系列新品。7月25日消息，爆料人@Ishan Agarwal推特透露，三星Galaxy Note 20几乎证实采用的是塑料后壳。有网友发现，Galaxy Note 20的中框上有断点，暗示Galaxy Note 20的中框为金属材质，后壳应该是塑料材质。

对比之下，Galaxy Note 20 Ultra的整机质感会好很多。它不仅采用的是玻璃材质，而且首发康宁新一代大猩猩玻璃，抗跌落能力相比第六代有了大幅提升。

值得注意的是，Galaxy Note 20 Ultra、Galaxy Note 20标配25W充电器，它支持45W快充。

核心配置上，Galaxy Note 20采用了6.7英寸直屏，分辨率为2400×1080，刷新率60Hz，搭载Exynos 990处理器，配备8GB内存+256GB存储，后置1200万像素主摄+1200万超广角+6400万长焦三摄，支持30倍数码变焦，前置1000万像素，电池容量为4300mAh，支持超声波屏幕指纹。

Galaxy Note 20 Ultra采用6.9英寸AMOLED柔性屏，分辨率为3200×1440，刷新率为120Hz，搭载Exynos 990处理器，配备12GB内存，提供256GB、512GB存储，电池容量为4500mAh。
