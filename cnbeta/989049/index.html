[图]六月更新已到：Windows 10 功能更新齐获累积更新

June 10, 2020

六月的补丁星期二活动日中，包括 Windows 10 May 2020（20H1/Version 2004）在内的功能更新都获得了累积更新。今天发布的本月累积更新并没有引入任何新的功能，只是对已发现的BUG进行修复，优化稳定性等。

Windows 10 May 2020 获得 KB4557957 累积更新，主要内容有

● 修复默认 Windows 语言为加拿大或者澳大利亚英语的时候，无法在 Windows Mixed Reality 使用某些语音命令的情况。

● 提高包括 Cortana 在内语音助手使用关键词的可靠性

● 更新改善在使用 IE 和 Microsoft Edge 浏览器时的安全性

● 更新在 Windows 执行基础操作的时候改善安全

● 更新改善使用 Microsoft Office 产品的安全性

● 更改使用外部设备（例如游戏手柄、打印机和网络摄像头）和输入设备（例如鼠标、键盘和手写笔）时的安全性

● 改善 Microsoft Xbox 和 Microsoft Store 的安全性

● 改善用户名和密码的更新验证

● 存储和管理文件更新

微软表示该累积更新中目前并没有发现任何已知问题，但这并不意味公司已经修复所有问题。根据 Windows 10 Health 面板，微软依然在调查 9 个特别问题，依然因为兼容性问题阻止部分设备升级。不过公司已经修复了部分设备使用 Always On、Always Connected 功能而出现的错误和预期之外的重启问题。

今天，Windows 10 Version 1909 和 Version 1903 也获得了相同的累积更新 KB4560960，在升级后版本号分别升至 Build 18363.900 和 Build 18362.900，主要更新内容如下

● 更新改善在使用 IE 和 Microsoft Edge 浏览器时的安全性

● 更新在 Windows 执行基础操作的时候改善安全

● 更新改善使用 Microsoft Office 产品的安全性

● 更改使用外部设备（例如游戏手柄、打印机和网络摄像头）和输入设备（例如鼠标、键盘和手写笔）时的安全性

● 改善 Microsoft Xbox 和 Microsoft Store 的安全性

● 改善用户名和密码的更新验证

Windows 10 Version 1809 也获得累积更新，版本号升至 Build 17763.1282，更新内容如下

● 更新改善在使用 IE 和 Microsoft Edge 浏览器时的安全性

● 更新在 Windows 执行基础操作的时候改善安全

● 更新改善使用 Microsoft Office 产品的安全性

● 更改使用外部设备（例如游戏手柄、打印机和网络摄像头）和输入设备（例如鼠标、键盘和手写笔）时的安全性

● 改善 Microsoft Xbox 和 Microsoft Store 的安全性

● 改善用户名和密码的更新验证

● 存储和管理文件更新

和往常一样，微软今天还将发布适用于Windows 10 Version 1803、1709、1703、1607和1507的新安全更新，不过这些更新只适用于Windows 10 企业和教育用户。
