FB批评苹果的应用商店政策 被迫在iOS上推出移除游戏功能的Gaming应用

August 29, 2020

据外媒Macrumors报道，Facebook周五和微软一起谴责苹果的应用商店政策，此前该公司被迫从周五在iOS上推出的 Facebook Gaming应用中移除游戏功能。Facebook在给The Verge的一份声明中表示，近几个月来，其Gaming应用多次被苹果拒绝，但苹果以其App Store指南为由拒绝，称Facebook Gaming应用的主要目的是玩游戏。

Facebook表示，它与苹果分享了其Android Facebook Gaming应用的使用数据，数据显示95%的活动涉及观看流媒体，但它无法改变苹果对此事的立场。

Facebook的首席运营官雪莉·桑德伯格表示，该公司选择继续在App Store中推出其应用，但与Android用户相比，用户面临着 “较差”的体验。

“不幸的是，为了获得苹果对独立的Facebook Gaming应用的批准，我们不得不完全删除游戏功能–这意味着iOS用户的体验不如使用Android的用户。”桑德伯格说。“我们将继续专注于为每月在Facebook上玩游戏的3.8亿多人建立社区–无论苹果是否允许在独立应用中使用。”

Facebook发言人补充说，其甚至根据苹果在WWDC上宣布的新应用审查程序对该准则提出上诉，但它没有得到回应。

苹果放宽App Store限制，遭到了竞争对手和游戏行业的猛烈批评。就在周四，微软表示，其与Xbox Game Pass配对的 “Project xCloud “流媒体游戏服务在今年9月推出时，将无法在iPhone和iPad上使用。

该公司抱怨说，它已经没有路径将其与Xbox Game Pass Ultimate的云游戏愿景带给iOS上的游戏玩家了，因为苹果“一贯对游戏应用区别对待，对非游戏应用适用更宽松的规则，即使它们包含互动内容”。
