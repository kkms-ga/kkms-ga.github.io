微软计划将Xbox All Access订阅服务推向更多市场区域

August 28, 2020

外媒报道称，随着 Xbox Series X 的正式到来，微软也希望在今年晚些时候，将 Xbox All Access 和 Xbox Game Pass Ultimate 等订阅服务推向更多市场区域。彭博社近日获悉，Xbox All Access 将于今秋登陆更多市场区域，且数量可能翻至三倍，包括美国、英国、澳大利亚、以及新西兰。

据悉，微软最初于 2018 年秋季在美国本土限时推出了 Xbox All Access 服务，一年后又带来了可升级至 Project Scarlett 的选项。消费者可选三档不同的套餐，起价为 19.99 美元，微软表示可帮助客户至少省下 100 美元。

时间快进到 2020 年，Xbox Game Pass 已覆盖 100 多款出色的游戏作品。在没有前期费用的情况下，月费依然相当具有吸引力。

此外微软希望通过与沃尔玛和塔吉特等零售伙伴合作，以促进 All Access 服务的销售，新主机买家将可选择月费 20~25 美元的 Game Pass 订阅（类似手机的包月费用制度）。
