Visual Studio 2019 for Mac 8.6 发布

May 21, 2020

Visual Studio 2019 for Mac 8.6 发布了，主要更新内容包括集成终端、对 Blazor WebAssembly 的支持、更新的身份库，以及对构建和运行 gRPC 项目的支持。

访问:

微软中国官方商城 - Visual Studio

更具生产力的集成终端

新的集成终端将有助于最大程度地减少上下文切换并提高生产率。它是具有丰富的 IDE 和 macOS 集成的功能齐全的终端，包括：

合理的默认值

搜索

与 Mac 终端集成

多实例支持

可配置的终端字体

构建 Blazor WebAssembly 应用程序

Visual Studio for Mac 8.6 带来了 Blazor WebAssembly 应用程序的模板，包括对构建 Progressive Web Apps 的支持。

使用 gRPC 创建高性能服务

此版本包含一个模板，用于创建具有 ASP.NET Core 托管支持以及对 .proto 文件的 C# 工具支持的 gRPC 服务项目。

可使用新的 gRPC 模板创建新的 gRPC 服务项目：

对 .proto 文件的支持使用户可以利用 gRPC 的合约优先（contract-first）方法进行 API 开发。可以在编辑器支持下创建或编辑 .proto 文件，.NET Core gRPC 工具将自动生成服务基类。

更强大的登录体验

随着新的库支持，Visual Studio for Mac 现在支持设备代码身份验证和系统浏览器。如有需要，可以在“首选项”（Preferences）中启用这两个选项。

拖放以设置下一条语句

调试器暂停时，空白处的箭头表示下一步将执行哪一行代码。现在可以单击并拖动箭头到另一行代码，以更改将执行的语句。

还可以通过右键单击一行代码并从上下文菜单中选择“设置下一条语句”（Set Next Statement）来实现同一目的。
