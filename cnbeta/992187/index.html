[图]并非BUG：微软有意从Windows 10 Version 2004中移除Fresh Start功能

June 17, 2020

几天前，有不少用户反馈称在升级 Windows 10 May 2020（20H1/Version 2004）功能更新之后，发现“Fresh Start”功能无法正常使用。现在可以确认微软有意移除这项功能，而不是BUG。微软虽然没有发布官方公告，不过在6月16日对 Fresh Start 功能支持文档进行了更新，解释了用户如何从 Windows 10 Version 2004 中从头开始。

微软在支持文档中写道：“Fresh Start 在 Windows 10 Version 2004 此前版本中可用，相关功能已经迁移到 Reset this PC 中。要重置电脑，请转到 “开始">"设置">"更新与安全">"恢复">"重置此电脑">"开始”。然后选择保留我的文件，选择云或本地，改变你的设置，并设置恢复预装的应用程序吗？”
