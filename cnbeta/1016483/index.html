[图]微软仍未放弃Store方式推进记事本更新 还带来了全新图标

August 17, 2020

记事本（Notepad）是深受 Windows 用户欢迎的工具。去年微软决定将适用于 Windows 10 系统的记事本作为独立应用使用，可以通过 Microsoft Store 进行更快的更新。此外这也意味着用户可直接从微软商城中卸载记事本并重新安装。

不过在2019年12月，也就是 Windows 10 May 2020（Version 2004）功能更新开发阶段进入尾声的时候，微软宣布表示包括记事本在内通过商城更新方式的应用不会进入 Windows 10 生产版本。

公司表示：“非常感谢你们对商城版记事本提供的所有反馈。现在，我们已经决定不会用户公开推送该功能”。目前我们仍不知道为何微软会搁置商城版记事本，不过显然微软并没有放弃这种更新方式。

目前微软仍在维护商城版记事本，在8月的首周获得了软件更新，从 10.1912.1.0 版本升级到了 10.2007.1.0 版本。目前商城版的系统要求是 Windows 10 Build 19541 及更高版本，目前，Build 19541或更新版本可在未来Windows 10功能更新的开发频道中供测试者使用。

在在预览版中，记事本也获得了一个新的图标，微软商店列表为我们提供了它的标志设计的早期外观。值得一提的是，记事本在最近得到了不少功能更新，比如支持环绕式搜索，以及有未保存内容时的指示器。
