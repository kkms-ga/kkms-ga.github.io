System76推出Bonobo WS高性能Linux笔记本 可选i9与RTX 2080 Super

August 21, 2020

曾推出 Bonobo WS 台式 PC 的 System76，现又推出了同样具有高性能配置选项的 Bonobo WS 系列 Linux 笔记本。除了十代酷睿 i5 / i7 / i9 处理器，消费者还可搭配英伟达 GeForce RTX 2060 / 2070 Super / 2080 Super 独显，17.3 英寸的雾面屏（4K UHD / 1080p @ 144Hz），128GB 的 DDR4-3200 内存，以及最高 26TB 的 M.2 固态存储（2×PCIe / SATA + 1×PCIe + 1×SATA）。

（来自：System76）

具体说来是，Bonobo WS Linux 笔记本提供了三款英特尔十代酷睿处理器的选项：

● Core i5-10600k：6 核心 / 12 线程，基础频率 4.1GHz、睿频可达 4.8GHz，具有 12MB 缓存。

● Core i7-10700k：8 核心 / 16 线程，基础频率 3.8GHz、睿频可达 5.1GHz，具有 16MB 缓存。

● Core i9-10900k：10 核心 / 20 线程，基础频率 3.7GHz、睿频可达 5.3GHz，具有 20MB 缓存。

软件方面，Bonobo WS Linux 笔记本可选预装自家的 Pop!_OS 20.04 LTS（64-bit）或 Ubuntu 20.04 LTS（64-bit）Linux 发行版，且开放了嵌入式控制器与系统固件。

该机还提供了全键无冲 RGB 背光键盘（Chiclet US QWERTY Keyboard）、多点触控板，千兆以太网、英特尔 Wi-Fi 6（AX 双频）无线网卡 + 蓝牙，720p 高清网络摄像头、带有低音炮的立体声扬声器、以及耳机 / 麦肯风 / SPDIF 符合音频插孔。

为顶住高端 CPU 和 GPU 产生的热量，Bonobo WS Linux 笔记本还配备了具有多条热管的双风扇散热系统。内置 97Wh 八芯锂电，以及 280W 的电源适配器。

该机的大小为 1.71×15.71×12.56 英寸（43.43×399.03×319.02 mm），重约 8.38 磅（3.8 公斤），取决于最终的配置。

System76 Bonobo WS Linux 笔记本的起价为 2399 美元（1.65 万 RMB），如果选择顶配的十代 Core i9 和 RTX 2020 Super 独显等配置，价格还可以冲到更高。

好消息是，目前该公司正在开展“Launch into Learning”教育促销，感兴趣的消费者可自选配置并享受 100 美元的优惠。
