[技巧]如何强制Windows 10始终选用指定的浏览器和搜索引擎

July 10, 2020

升级到 Windows 10 操作系统的一大烦恼，就是微软总会自作主张地向用户推荐自家的产品或服务。即便我们此前已经手动指定了第三方，也可能在某次不经意的软件或系统升级过后被替换，尤其在该公司全面向 Chromium Edge 转型之后。此外当你在搜索框中执行 Web 查询、或者向 Cortana 询问某些信息时，默认返回的结果也是来自微软旗下的必应搜索。

虽然 Chromium Edge 和必应搜索已经做得相当不错，但对于更喜欢使用其它厂商的产品和服务的用户来说，强扭的瓜还是不够甜。

如果你希望强制 Windows 10 始终选用指定的浏览器和搜索引擎，那不妨考虑下 GitHub 上的《Search Deflector》解决方案。

（来自：GitHub）

最近更新后，这款工具已支持强制 Windows 10 选用指定的默认浏览器和搜索引擎选项。你只需在首次运营该软件时完成偏好设置，然后就可以将它放到一旁。

需要指出的是，开发者提供了免费和付费两个版本的 Search Deflector 。尽管功能都是相同的，但免费版本未经签名，可能会让某些反病毒软件产生误报。
