[图]微软在OneDrive中启用巨大横幅广告：推荐用户使用Edge

June 05, 2020

基于 Chromium 的 Edge 浏览器于 2020 年 1 月正式发布，现在微软正努力说服 Chrome 用户体验尝试这个这款浏览器。从本周开始，当你使用非 Edge 浏览器访问 OneDrive 的时候，微软就会在显眼位置显示 Microsoft Edge 广告横幅。

例如现在你通过 Chrome 浏览器访问微软 OneDrive 的时候，会弹出“这是由微软推荐的全新浏览器。在Microsoft Edge上，你可以获得性能、兼容性、速度和易用等全方面升级体验。”

这个巨大的横幅还提供了新版Edge浏览器的下载链接，出现在 OneDrive 最近使用文件的上方。用户可以通过点击关闭按钮来删除这个广告。

事实上，微软并不是唯一这样做的科技企业。基于 Chromium 的 Edge 用户访问包括 Docs、Forms、Drive 以及 Gmail 等谷歌服务的时候，谷歌会弹出广告推荐用户使用 Chrome 浏览器。在弹出框中写道：“和大部分 Windows 用户一起使用 Chrome。Chrome是快速、简约和安全的浏览器，专为现代化网页而创建。”
