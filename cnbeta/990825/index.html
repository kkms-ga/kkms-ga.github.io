[图]Psystar在前：又一家公司尝试违反苹果EULA销售商用黑苹果电脑

June 14, 2020

在 Psystar 之后，一家名为“OpenCore Computer”的初创公司（和 OpenCore Bootloader 无任何关系）在本周推出了名为“Velociraptor”的商用黑苹果（Hackintosh）电脑，应该违反了苹果的 macOS 最终用户许可协议（EULA） 。

OpenCore Computer 在其产品官网上宣称，这款能够让这款采用 Mac Pro 风格的工作站变的更加容易使用。对于公司的产品线，宣称“零妥协的黑苹果”，预装了 macOS Catalina 和 Windows 10 专业版。

首款商用机型“Velociraptor”最高可配置 16 核处理器、64GB 内存和 Vega VII GPU，起价2199美元。OpenCore Computer打算在以后推出更多的机型，选装配置最高可以达到64核 CPU 和 256GB 内存。

黑苹果（Hackintoshes）是指在未经苹果授权的硬件设备上运行 macOS 系统。OpenCore 是一个免费的开源工具，用于准备启动 macOS 系统的引导。销售这些黑苹果设备的公司似乎挪用了开源启动程序的名称，实际上与 OpenCore 的开发者没有任何关系。

商业Hackintoshes有一段臭名昭著的法律历史。现在已经倒闭的Psystar公司从2008年开始销售所谓的 “Open Computers”，可以选择预装Mac OS X Leopard。苹果的EULA禁止第三方安装其软件，任何商业化的Mac克隆软件都违反了该协议以及数字千年版权法（DMCA）。

苹果公司在2009年起诉Psystar，并赢得了对该公司的永久禁令，美国最高法院在2012年拒绝复审此案。考虑到这一先例，OpenCore Computer选择销售Hackintosh尤其令人惊讶。

OpenCore Computer似乎试图绕过EULA，只接受比特币加密货币支付。为了证明该公司不是骗局，它提供了通过 “Bitrated “使用托管支付，该公司打算为加密货币交易带来消费者保护和防欺诈措施。
