[图]Edge Canary现支持标签页悬停预览功能

July 31, 2020

援引外媒 Aggiornamenti Lumia 最新报道，基于 Chromium 的 Edge 浏览器新增了一个实验性特性，在启用之后用户将鼠标悬停在标签页上之后能预览该标签页的快照，查看标题、页面URL和缩略图。这个功能并不是什么新功能，在 Opera 和经典版 Edge 上已经上线。

标签页预览对于基于 Chromium 的浏览器来说也并不新鲜，因为去年在 Chrome Canary 中就发现了对该功能的引用。除了 Canary 频道版本之外，所有其他版本的 Edge 浏览器都有了名为“Tab Hover Cards”的隐藏 Flag。现在，Edge Canary 也增加了“Tab Hover Card Images”提供标签页预览。

想要尝试该功能的用户，可以前往 edge://flags/ 页面，然后将“Tab Hover Cards”和“Tab Hover Card Images”两个 Flag 选项启用，然后重新启动浏览器。不过需要注意的是，该特性无法像经典版 Edge 那样永久预览，悬停一段时间之后就会自动隐藏。
