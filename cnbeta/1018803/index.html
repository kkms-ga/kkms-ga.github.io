Parallels Desktop 16 for Mac试用 支持ARM架构的苹果自研处理器

August 21, 2020

Parallels 是一款 Mac 端虚拟机软件，它可以让你在 Mac 电脑上运行 Windows、Linus 系统，去年我们就对 Parallels 15 这个版本进行过介绍。最近，新版本 Parallels Desktop 16 for Mac 正式发布，它全面集成了 macOS Big Sur，与 Windows 10 上的许多功能进行了整合。我们也对 Parallels 16 进行了一段时间的体验，下面就来介绍一下它的一些特性。

访问：

Parallels Desktop 官网

与 macOS Big Sur 全面集成

安装 Windows 10 的过程不再赘述了，可以参考之前的文章，总之速度和体验依然是非常顺畅。安装之后，Windows 10 基本是作为一个应用的形式显示在程序坞里，点击可以直接打开。在 macOS Big Sur 中程序坞、图标风格、窗口设计都有微小的变化，Parallels 16 与其进行了深度集成，不会违和。

Windows 10 能以全屏、窗口等方式运行，全屏方式下鼠标指针、分辨率等都会根据 Mac 电脑的参数进行优化，使用与 PC 并无区别。至于硬件驱动、网络设置等问题，Parallels 已经帮你解决好，可以直接“拎包入住”。

正在运行的 Windows 10 可以选择中止或暂停（关于状态的解释可以查看官方说明），释放一部分资源，再次点击可以快速恢复。这次还增加了旅行模式，可以更省电，续航时间延长 10%。

支持本机 macOS 系统扩展

这次 Parallels 16 最重要的一个更新，就是将传统的内核扩展替换为了系统扩展。一些应用需要通过内核扩展来实现一些功能（比如罗技的软件），这种方式的安全性和稳定性存在风险，苹果在 2019 年通知开发者更改为系统扩展的方式。Parallels 16 已经完成针对苹果虚拟机监控程序的迁移工作，WWDC 2020 上也有过展示，安全性不必担心。

Parallels 16 首次支持了 Metal 应用的 3D 功能（Metal 是苹果提供的硬件加速 API），这可以让 Parallels 运行 3D 软件、游戏时更流畅。

新支持的专业软件包括 DIALux evo 9、SAMSON Connect、ProPresenter 6 等。

虚拟机设置中新增了一个回收磁盘的选项，可以在虚拟机关闭时自动回收空间。性能方面相比上代版本，Parallels 16 的启动速度增加了一倍，从挂起到恢复速度加快 30%，在我这个 16 款的 MBP 上感觉还是比较明显的。

触控板操作是 Mac 电脑的优势之一，在 Parallels 16 上你能用多点触控手势在更多 Windows 应用中进行操作，比如查看地图。

支持 ARM 架构自研芯片

这次的 Parallels 16 更新其实以底层功能为主，因为 macOS Big Sur 是一次比较大的更新，一些旧的接口面临换代。至于很多人关心的 ARM 架构转移，其实 Parallels 已经发过一个新闻稿，表示未来将通过 Apple Silicon（苹果自研芯片）继续在 Mac 电脑上提供虚拟机支持，WWDC 2020 上已经有过展示。

在 Mac 电脑上运行 Windows 系统，这个使用场景还是相对小众，不过对于一些专业人士来说则是必需。这也是为什么苹果一直在 macOS 上支持双系统，在宣布更换架构后还专门提供虚拟机支持。如果你有这样的需求，Parallels 可能是最好的选择之一。
