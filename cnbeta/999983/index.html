[图]打开这个功能 能够让新版Edge浏览器的续航明显增加

July 07, 2020

相比较其他主流浏览器经典版 Edge 的优势之一，就是更低的功耗、更长的续航表现。不过得益于最新的 Chromium 改进，新版 Edge 浏览器也获得了续航方面的改进。

当你同时打开多个标签的时候，浏览器将自动节能 30%。具体来说，新设置会启动一个定时器，当某个标签页闲置超过 5 分钟之后就会限制 JavaScript 的使用，从而节省 CPU 的使用。

在 Flag 的标签页面中写道：

启用该功能之后，当标签页在后台运行超过 5 分钟以上，或者每次运行时间不超过 1 分钟，会为后台标签页启动 JavaScript 倒计时，从而更积极地节省功耗。这是一个符合Web标准的功能，但它可能会破坏某些网站的功能，因为它导致某些操作延迟了一分钟。然而，启用该功能后，可显著节省 CPU 和耗电量。该功能支持 Mac 和 Windows 平台。
