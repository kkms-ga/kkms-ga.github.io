AMD线程撕裂者PRO 3000系列CPU官方规格泄露 3995WX打头阵

July 11, 2020

外媒 VideoCardz 已经分享了，AMD 线程撕裂者 PRO 3000 系列 CPU 的官方规格，可知旗舰级的 3995WX 具有 64 核 / 128 线程、128 条 PCIe 通道、并且支持 8 通道 / 2TB DDR4 内存。与普通的线程撕裂者 SKU 相比，PRO 系列主要面向渴求高性能的工作站市场，提供了大量的 I/O 和内存等功能支持。

据悉，AMD 线程撕裂者 3000 PRO 系列 CPU 一共有四款 SKU 。其采用了 7nm Zen 2 内核架构，并在基板上集成了 14nm 制程的 I/O 芯片。

之所以将 PRO 产品线与面向 HEDT 发烧友的普通消费级线程撕裂者区分开来，是因为它们旨在服务于不同的市场，比如工程师、科学家、设计师等专业人士。

这四款型号分别是 PRO 3995WX、PRO 3975WX、PRO 3955WX、以及 PRO 3945WX，详细规格如下。

作为该系列的旗舰主打，3995WX 采用了 64 核 / 128 线程的设计、拥有总计 288MB 的缓存，基础频率 2.7GHz、动态可达 4.2GHz，热设计功耗 280W 。

从参数来看，PRO 系列线程撕裂者 CPU 可在同等功耗水平下，努力达成更高的持续时钟频率。而面向消费级 HEDT 平台的版本，被设计成可在给定 TDP 范围内飙高更高的频率。

PRO 3000 系列线程撕裂者 CPU 的其它功能包括 128 条 PCIe 4.0 通道（较普通线程撕裂者翻倍），以及备受追捧的 8 通道 ECC 内存支持（最高 2TB UDIMM / RDIMM / LRDIMM）。

次旗舰的 3975WX 采用了 32 核 / 64 线程的设计，拥有总计 144MB 的缓存。基础频率 3.5GHz、动态可达 4.2GHz，热设计功耗为 280W 。

当然，PRO 3000 系列线程撕裂者 SKU 都保留了对 8 通道 DDR4 内存 / 128 条 PCIe 4.0 通道的支持。

第三款 3955WX 采用了 16 核 / 32 线程的设计，拥有 72MB 的总缓存。基础频率 3.9GHz、动态可达 4.3GHz，热设计功耗为 280W 。

第四款 3945WX 采用了 12 核 / 24 线程的设计，拥有 64MB 的总缓存。基础频率 4.0GHz、动态可达 4.3GHz，热设计功耗为 280W 。

显然，AMD 希望通过 3000 PRO 系列线程撕裂者 SKU 与英特尔的 vPro 竞品一较高下，同时提供更加丰富的扩展选项，包括更多的核心 / 线程数和更好的内存功能支持，以期在工作站市场上处于领先地位。

遗憾的是，AMD 尚未披露上述新品的售价。感兴趣的朋友，还请留意 7 月 14 日的 AMD Ryzen Threadripper PRO 3000 系列 CPU 的首发公告。
