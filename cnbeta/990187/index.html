[图]六月累积更新又出问题：打印机故障 部分程序无法打开和卸载

June 12, 2020

日前的六月补丁星期二活动日中，微软面向所有 Windows 10 功能更新发布了累积更新，其中 Version 2004 获得 KB4557957，Version 1903/1909 获得 KB4560960，官方并没有罗列已知问题。现在，有用户反馈升级这些累积更新之后出现打印机故障，以及程序无法打开和卸载的问题。

打印机故障

根据 Reddit 社区和微软 Answer 网站以及其他论坛的用户反馈，KB4557957 和 KB4560960 均会导致打印机故障。

一位用户反馈道

有人在安装最新的 Windows 更新之后出现了打印机问题吗？我的理光打印机在升级之前是正常的，但升级后出现了好多问题。更改打印机驱动貌似能够解决这个问题，但是要在如此多的客户端上再部署驱动就非常痛苦了。

随后在这个帖子下，其他用户也确认存在这个问题。在微软 Answer 网站上有用户写道：“ 在安装这个更新之后，文档会在打印列表中闪现，然后就消失了。卸载这个更新之后就解决了这个问题。”

目前受影响的主要是理光的打印机，不过也有用户表示 Brother 和 Canon 品牌的打印机也出现了这样的情况。卸载累积更新后，一切都会恢复正常，你可以通过以下步骤来完成。

1.打开 “设置">"更新与安全">"Windows更新”。

2.点击’更新历史’链接。

3.在’卸载更新’选项上选择，并选择要删除的KB。

部分程序无法打开和卸载

值得注意的是，本次累积更新还出现了其他问题。在其他的帖子中，表示在升级 Windows 10 KB4560960 更新之后，部分应用程序无法打开或者卸载。系统会提示：“找不到*.exe “的错误。确保你输入的名称正确，稍后再试。”

修复这个问题的步骤是

1.打开注册表编辑器。

2.导航到以下位置。

ComputerHKEY_LOCAL_MACHINE/SOFTWARE/Microsoft/Windows NT/CurrentVersion/Image File Execution folder.

3.重命名不能使用的应用程序的文件夹。例如，将winword.exe重命名为backupwinword.exe。

如果你已经安装了累积更新，并且遇到了问题，那么最好的解决方法就是按照上述步骤卸载补丁。
