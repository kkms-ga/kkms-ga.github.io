微软：IE11之后再无IE

August 19, 2020

1995年8月16日，微软推出了第一代Internet Explore浏览器，在之后二十多年间IE一直陪伴着Windows用户。2013年IE11发布后，微软开发了浏览器Edge，并将其预装于Windows 10中。今年1月15日微软新版Edge浏览器正式发布，它基于Chromium内核，代替旧版Edge为用户提供更出色的浏览体验 。

微软日前宣布，到2021年8月17日，微软365办公软件应用和服务将全面停止对IE11的支持。这引发了大家对IE11命运的讨论，虽然微软表示这一计划并不影响IE11的正常使用，IE11也不会消失，“IE11是Windows操作系统的组件，遵循其所在Windows产品的生命周期策略。”但实际上，对IE的淘汰已经是毋庸置疑的了。

在微软官方IE和Edge生命周期问答文档中，有这样一条内容揭开了IE的命运：

IE11是最后一版Internet Explorer吗？

是的，IE11是Internet Explorer浏览器的最后一个重大版本。在安装了IE11的Windows版本的生命周期内，IE11将继续获得安全更新和技术支持。IE11可以通过Microsoft更新目录、Windows更新和Windows服务器更新服务(WSUS)获取。

根据官方文档，目前支持IE11的Windows系统包括：

Windows桌面系统：

Windows 8.1 Update

Windows 10

Windows服务器系统：

Windows Server 2012

Windows Server 2012 R2

Windows Server 2016和后续版本

Windows嵌入式系统：

Windows Embedded Standard 7

Windows Embedded POSReady 7

Windows Thin PC

Windows Embedded 8 Standard

Windows 8.1 Industry Update
