广颖电通推出US70 NVMe PCIe 4.0固态硬盘新品

June 17, 2020

固态硬盘市场在近年迎来了飞速的发展，M.2 接口的产品容量已可轻松达到 8TB 。此外随着具有 PCIe 4.0 功能的主板变得日渐流行，许多厂商也推出了具有更高性能参数的 NVMe SSD 。本文要为大家介绍的，就是来自广颖电通（Silicon Power）的 US70 系列 PCIe 4.0 NVMe SSD 。

与 PCIe 3.0 x4 接口相比，PCIe 4.0 x4 的理论带宽速度再次翻倍。在高性能 3D NAND 闪存和 NVMe 1.3 等功能协议的支持下，速度可达传统 SATA III 6Gbps SSD 的近 10 倍。

以 Silicon Power 的 US70 系列 PCIe 4.0 NVMe SSD 为例，其采用了标准的 M.2 2280 外形，顺序续写分别可达 5000 MB/s 和 4400 MB/s，可满足游戏发烧友、专业设计/编辑、高端企业客户的使用需求。

据悉，SU70 系列 PCIe 4.0 NVMe SSD 可选 1TB 和 2TB 容量的版本，具有 1500G / 0.5ms 的耐冲击特性、工作温度 0~70 ℃、平均无故障时间（MTBF）为 170 万小时。

该产品的宽高厚为 22×80×3.5 mm、重 8g，通过了 CE / FCC / BSMI / Green dot / WEEE / RoHS / KCC 等认证，厂家给予 5 年的有限质保。

遗憾的是，Silicon Power 尚未公布 US70 系列 PCIe 4.0 NVMe SSD 售价和上市日期，但我们还是希望 1TB / 2TB 版本能够很快在新蛋 / 亚马逊等平台上架。
