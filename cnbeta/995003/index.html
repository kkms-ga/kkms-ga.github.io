[图]适用于Linux的Microsoft Defender ATP 正式上线

June 24, 2020

在宣布适用于 Android 的 Microsoft Defender ATP 首个预览版的同时，微软今天还宣布适用于 Linux 的 Microsoft Defender ATP 正式上线。这是微软首次在 Linux 平台上提供完整的命令行体验。

以下是支持 Microsoft Defender ATP 的 GNU/Linux 发行版本:

● Red Hat Enterprise Linux 7.2 及更高版本

● CentOS 7.2 及更高版本

● Ubuntu 16.04 LTS 及更高版本

● Debian 9 及更高版本

● SUSE Linux Enterprise Server 12 及更高版本

● Oracle Linux 7.2 及更高版本

微软还承诺，将在未来几个月内为Linux上的Microsoft Defender ATP带来多项新功能。
