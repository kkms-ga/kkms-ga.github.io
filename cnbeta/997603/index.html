微软推进新版Edge：教育和企业用户已开始通过Windows Update升级

July 01, 2020

2018年10月，微软宣布重建 Edge 浏览器；基于 Chromium 的首个公开预览版于2019年4月上线；正式版在2020年1月正式和大家见面。如果想要体验新版 Edge，Windows 7/8/8.1/10 以及 macOS 用户需要手动下载安装。不过从本月开始，微软将开始通过 Windows Update 方式来推进更新。

微软此前宣布，将通过 Windows Update 让更多人使用新版 Edge 浏览器。在推广之初并不会涉及到运行 Windows 10 教育和企业版的设备，而今天开始这些商用设备也会通过 Windows Update 接收新浏览器更新。

然而，这一变化将不会适用于通过 Windows Update for Business（WUfB） 或 Windows Server Update Services（WSUS） 管理更新的企业。有趣的是，教育设备将会率先替换为新版 Edge，以“适应返校时间”。

该公司表示，它将提供一个详细的时间表，说明企业用户何时开始看到更新，以基于Chromium 的 Edge 取代经典版Edge。然而，它向用户保证，这一变化不会影响任何预先设置的偏好，如默认浏览器。
