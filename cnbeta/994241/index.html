华为正式发布HMS Core 5.0：新增六大服务

June 22, 2020

今天，华为悄然推送了HMS Core 5.0服务框架，新增六大服务能力，同时强化了定位服务、推送服务。HMS也就是HUAWEI Mobile Services，是华为用来替代Google GMS服务的自有服务框架。

详细更新内容：

－ 新增音频、视频、图像、图形引擎、计算机图形、增强现实引擎等服务能力；

－ 定位服务：支持网络定位众包及围栏管理能力；

－ 推送服务：支持LBS(基于位置服务)、情景化推送；

－ 优化部分服务体验。

HMS Core是华为终端云服务、移动服务开放能力的合集，是为应用开发提供基础服务的平台，可以帮助开发者快速开发、快速增长、灵活变现，实现一点接入、全球全场景全终端的智慧分发。

它汇聚了华为终端芯-端-云能力，包含一整套开放的HMS Apps、HMS Core、HMS Capabilities、HMS Connect，以及相应的开发、测试的IDE工具。

今年初，华为上线了HMS Core 4.0，新增大量的新功能，既有已经运营多年的华为帐号、支付、分析、云空间、游戏服务，已经上线4个月的广告、定位服务外，还新增了大量开发者期待已久的新能力，包括机器学习服务、情景感知服务、统一扫码服务、近距离通信服务、全景服务、安全检测服务、动态标签管理器服务、线上快速身份验证服务、位置服务、快应用服务、数字版权服务、运动健康服务、用户身份服务等等，其中不少服务还支持融合华为芯-端-云能力。

HMS Core的主要亮点：

1、全球分发：170多个国家和地区，覆盖6亿用户。

2、节省成本：免费开始、易于上手，从开发到上架，一站式服务体验。

3、安全可信：遵循GAPP、GDPR及当地法规。

4、精准触达：应用市场月活4亿用户，多种基于用户行为的推送方式，助力精准营销。

5、开发者生态：截止今年初，已有超过5.5万款APP已经使用HMS Core服务，全球注册开发者超过130万。
