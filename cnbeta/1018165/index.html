外媒实测新款27英寸iMac：散热表现不错 酷睿i9压得住

August 20, 2020

外媒 Apple Insider 于近日对苹果新款 27 英寸 iMac 一体机的性能和散热表现进行了检查，以了解其较旧款有多大的改进。测试机型配备了英特尔十代处理器（3.6GHz @ Core i9），8GB RAM，以及具有 8GB 显存的 AMD Radeon 550XT 独显。在 Blackmagic 磁盘基准测试中，其持续读写速率分别为 2365 MB/s 和 1871.3 MB/s，较入门机型有着较为显著的优势。

视频截图（来自：Apple Insider）

在 Geekbench 的 CPU 基准测试项目中，新款 27 英寸 iMac 的得分为单核 1256 / 多核 7958 。作为对比，入门机型的得分为 1141 / 5635 。

计算测试方面，配备 8GB RAM + AMD Radeon 5500XT 独显的 27 英寸 iMac 为 38318 。虽然较入门机型的提升没有想象中那么高，但还是有一定区别的。

在最能反映“生产力”的 Cinebench R20 渲染测试中，这台十代酷睿 i9 版 27 英寸 iMac 的成绩为 5424 分。

“梦幻单热管”强压 i9 处理器，这个梗可能要跟随 iMac 很长一段时间。

虽然各机型都撞到了所谓的“温度墙”（系统散热不足以支撑长时间高频工作），该机较入门机型的提升还是相当显著的。

据悉，取决于具体的目标市场，现代处理器被设计成能够在不同的热设计功耗（TDP）水平下运行。通常情况下，台式机能够承受更高的 TDP，而笔记本、平板、移动设备上的热设计功耗相对较低。

但在撞到温度墙后，处理器将被迫降频运行，而无法在标称加速频率上持续更长的时间，否则可能造成损伤。

10-core i9 27-inch iMac TESTED - Is There A Thermal Problem？（via）

Apple Insider 指出，其在 21 ℃ 的典型办公室环境中对这台 Core i9 版 27 英寸 iMac 开展了 Cinebench R20 测试，且中间丝毫不给任何喘息的机会。

在起初的测试中，散热系统的压力似乎并不大，连风扇噪音都没有。多次运行后，尽管温度升到了 38 ℃（风扇开始运作），处理器的频率仍能稳定在 4.3GHz 左右。

测试之前，大家最担心的就是 27 英寸 iMac 无法保障高负载下的性能，毕竟其散热系统难以与 iMac Pro 相提并论。

另一方面，这枚 Core i9 处理器的热设计功耗约为 98W，低于 iMac Pro 上 115W 及以上的至强芯片，更别提 iMac Pro 搭配的 Vega 56“大火炉”独显。

现在看来苹果之所以不在 27 英寸 iMac 上配备与 iMac Pro 同样的散热系统，也不过是觉得没有必要。
