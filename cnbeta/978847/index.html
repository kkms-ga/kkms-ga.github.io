Dynabook刷新Comet Lake vPro轻薄商务笔记本产品线

May 14, 2020

随着英特尔发布 Comet Lake vPro 处理器，Dynabook 也及时推出了刷新后的商用笔记本产品线。首先介绍下在 CES 2020 上亮相过的 Portégé X30L-G，其拥有超轻薄的机身、13.3 英寸 IGZO 显示屏、重仅 1.92 磅（870g），起价为 1373.99 美元（约 8743 RMB）。

Portégé X30L-G

Portégé 系列的其它产品也用上了 Comet Lake vPro 处理器，包括 X30-G、X40-G 和 X50-G 等机型。

其提供了网络摄像头隐私挡板、防窥屏、指纹传感器、红外摄像头等配置，辅以 vPro 自带的一系列硬件级安全特性。

Portégé X30-G

即便如此，新机依然维持着轻薄的体型，其中 X30-G 厚 15.9 mm、重仅 2.3 磅（约 1.04 kg）。14 英寸的 X40-G 为 2.76 磅（1.25 kg），15.6 英寸的 X50-G 则是 3 磅（1.36 kg）。

价格方面，X30-G / X40-G / X-50G 分别为 1329.99 / 1299.99 / 1399.99 美元（约 9431 / 9218 / 9927 RMB）。

Tecra A40-G

最后，Dynabook 也准备了入门级的 Tecra 系列。A30-G / A40-G 机型的起价分别为 449.99 / 399.99 美元（约 3191 / 2836 RMB）。

值得一提的是，A30-G 也是 Dynabook 迄今为止最轻薄的 A 系列笔记本，重仅 2.65 磅（1.2 kg）。
