[图]微软：新版Edge已成Windows系统重要部分 不再提供卸载选项

August 07, 2020

根据 Edge 支持网站更新的内容，微软确认如果用户是通过 Windows Update 系统更新方式，从经典版 Edge 迁移到基于 Chromium 的新版 Edge 之后，无法在应用页面卸载该浏览器程序。

微软已经于1月15日正式发布了基于 Chromium 的新版 Edge 浏览器，并通过更新的方式邀请那些使用经典版 Edge 的用户升级到该新版本。不过在升级之后，用户无法卸载该应用程序。

今天微软发布了一篇名为“Can’t uninstall Microsoft Edge”（无法卸载微软Edge）的支持文档，微软表示已经分阶段将 Windows 用户从经典版 Edge 迁移到新版本。

微软表示：“我们要确保所有的 Windows 客户都有最新的微软 Edge 浏览器的性能，隐私，安全，生产力。”用户可以从经典版 Edge 中导入数据到新版中，操作步骤为启动新的Edge > 设置 > 配置文件 > 导入浏览器数据 ，选择” Microsoft Edge Legacy”。

微软在文档中表示：“新版 Microsoft Edge 已经包含在 Windows 系统更新中，因此无法卸载或者继续使用经典版 Edge”。
