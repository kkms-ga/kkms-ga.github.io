华为3 Pro超能版vs.小天才Z5 - 千元档儿童手表试玩与对比

May 13, 2020

与传统的智能穿戴不同，儿童智能手表实际上同时承载了沟通、安全定位、寓教于乐等多重任务。伴随儿童智能手表市场的发展，许多厂商推出相应的新品，在售价与功能上不一而足。

访问:

华为商城

作为老牌智能手机/通信厂商，华为也在上月更新了自家儿童智能手表新款 —— 华为儿童手表 3 Pro 超能版，目前预约售价为 988 元，而作为在这一领域实力较强的品牌 —— 小天才，已经发布一段时间了的小天才 Z5 也是相当受欢迎的儿童手表之一，目前在京东等平台的优惠价格也进入了千元左右这个档位。作为两家头部厂商在千元价位具有竞争力的产品，且在各自的功能卖点，如全网通 4G 支持，九重定位等具体功能上也高度相同，让不少想要为自家孩子选购一款产品的家长面对这两款产品陷入纠结。

在本文中，笔者就针对这两款价位在千元左右目前关注度非常高的儿童智能手表，从安全控制、通话表现与内容服务上，进行日常使用“模拟”体验，来分析这两款手表的性能与优势异同。

舒适/便捷/屏显

首先从机身材质来看，华为儿童手表3Pro超能版和小天才Z5均采用亲肤的材质，手感上同样柔软舒适。不同的是，华为儿童手表的表带上多布置了两排额外的散热孔，在同样的表带尺寸下，尤其是夏季，孩子运动量加大，出汗量也大，更需要一款透气性良好的儿童手表，在舒适性上，两款产品都照顾得都比较到位。

（左为小天才Z5,右为华为儿童手表3Pro超能版）

除了佩戴舒适性，两款手表在外观上最大的区别，就是按键的设计：小天才 Z5 的操作主要以触屏操作为主，唯一一颗实体按键，置于机身底部，并非为日常使用而设计。对比之下，华为儿童手表 3 Pro 超能版则将两个实体按键设计在机身两侧。体验下来，长按五秒华为儿童手表3Pro超能版机身一侧的“Fun”按键就可以呼出 SOS 紧急呼叫功能，这样的硬件设计在情况危急的紧急时刻，要远比各种触屏操作来的可靠，而且这个功能还有一个比较契合实际使用的设计：在 SOS 呼叫模式下，手表会默认持续呼叫直到紧急联系人接通为止。

在两款手表的屏幕显示对比中，华为儿童手表 3 Pro 超能版所搭载的那片 1.4 英寸 TFT 显示屏，相较小天才 Z5 的1.41 英寸的 AMOLED 显示屏，屏幕尺寸略小0.01英寸，不过在实际观看体验中，两片屏幕的尺寸差异并不大。为了反映两款产品真实的对比结果，现在将两款手表同时放置在室内相同的自然光环境下，感受下他们的屏幕显示效果。

左边是小天才Z5，搭载了一片AMOLED 显示屏；而右边的华为儿童手表3 Pro超能版则采用TFT 显示屏。

在实际观感上，小天才 Z5 的 AMOLED 无论在色彩对比度还是观感上，都要比同代 TFT 屏幕来得更加柔和 ，而华为所采用的 TFT 屏幕材质虽然在目前的智能手机上逐渐退出主流，但在屏幕较小的手表屏幕下，也能维持不错的显示效果，且在实际使用中可视角度更广，也没有恼人的屏幕反光问题。

紧接着，在室外相同的自然光环境下，受强光照射影响，两款手表的亮度都有削弱，但明显小天才Z5的屏显亮度更低，阅读效果受影响更大。再看另一边的华为儿童手表3Pro屏幕显示仍较清晰，不影响正常的使用。当然，遇到这种强烈室外光影响时，小朋友可以选择手动调节亮度，纠正改善屏幕显示效果。

在防尘防水标准上，华为儿童手表 3 Pro 超能版为 IP68 认证，而小天才 Z5 则是防水等级的 IPX7 认证；这种区别具体到实际的使用场景上，就是手表能抵御的水压强度与时间的区别：IP68代表着华为儿童手表 3 Pro 超能版可以在 1.5m 深的水中正常运行至少 30 分钟，而 IPX7代表在 1m 的水中运行 30分钟。

不过，尽管产品都有一定的防水性能，但无论是华为还是小天才，官方都不建议小朋友在水中使用智能手表，但作为手表品质认证的一个重要标准，防尘防水等级代表着手表在更多环境下更高的适应性。更高的标准能够为小朋友带来更可靠的安全保障，在一些不可预测的环境下，能够与家人保持更畅通的联系。

沟通/安全功能

作为儿童智能手表最为重要的使用功能之一，优先保证高清流畅的通话质量，是家长在为自家小朋友挑选智能手表时非常关注的问题。两款儿童手表都支持国内三大运营商的全网通 4G 通话，且都将“全球漫游通话”作为一个卖点：不过在通信领域天生有着传统技能加成的华为在这方面就明显优秀得多，华为儿童手表 3 Pro 超能版共计支持了 5 模 15 频的网络频段；比小天才 Z5 的 5 模 14 频更为广泛。在实际通话体验中，分别选择了空旷的广场和信号较弱的电梯间进行对比测试。空旷的广场自然没话说，两款手表的信号均保持稳定水平。在电梯间的通话测试中，我们进行了多轮对比，小天才的通话网络波动情况稍微比华为儿童手表稍微多一些，总体体验差别不大。

在通话质量上，华为儿童手表 3 Pro 超能版采用了面向脸部正面发声的音腔设计，而小天才 Z5 则是将扬声器布置在了手表主体的右侧。在户外实际通话测试中，华为儿童手表 3 Pro 超能版通话音质以及最高音量都明显给人感觉更清晰，且在室外环境通话的降噪功能上也能更加清晰的分辨背景杂音与人声，在综合通话表现上要更胜一筹。此外，华为儿童手表 3 Pro 超能版还加入了一个别出心裁的“耳朵模式”：通话时，需要小朋友将手表贴近耳朵，这样通话的私密性就有更好保障，有防止重要隐私泄露的效果。

除了语音通话，视频通话也是家长与孩子沟通的高频使用功能。两款手表均搭载500 万像素摄像头，在实际的通话体验比对中，主要是视频通话显示效果的区别，清晰度上差别不大，小天才的视频显示较华为儿童手表更柔和。

此外，华为儿童手表 3 Pro 超能版还可以与搭载EMUI10.1及以上系统的华为手机/平板进行“畅连视频通话”。与小天才需要通过APP进行视频通话操作不同，华为手机开通畅连视频通话功能后，直接使用手机联系人拨号就能视频通话，体验更便捷。其次家里有华为平板的家庭，在平板上开通畅连通话后，也可以使用平板跟儿童手表进行畅连视频通话，效果要比手机视频通话看得更清晰。

在两款手表的主打功能 ——定位对比中，两款手表在实际外出测试中都有远超智能手机定位的不俗表现：无论华为儿童手表 3 Pro 超能版还是小天才 Z5，都宣称搭载了GPS、GLONASS、北斗、基站、Wi-Fi、加速度传感器、A-GPS定位、摄像头辅助和室内定位的九重定位功能并且支持在手机端设定常驻地点，当手表移动出预设范围之后，会自动通过手机推送提示；每次外出活动都会自动生成全部的移动记录，实际定位可供生成移动轨迹地图，在手机端 App 中复现小朋友全天的行动轨迹。且两款手表都支持全国大型公共场所的室内定位功能：比如在多层的大型购物中心中测试，两款手表的定位表现都非常出色，甚至都能定位到手表佩戴者所在的精确楼层位置。

内容服务生态

作为一种专门为不太适合使用智能手机的小朋友开发的智能产品，如何承担更多的兴趣培养以及寓教于乐任务，逐渐成为厂商新一轮角逐的主战场。

两个品牌都为旗下儿童手表开发了专属的儿童手表应用市场。内容丰富，涵盖日常生活类，如支付宝、语音助手等；知识学习类，如背单词、查汉字、听故事等。

左：华为手表应用市场  | 右：小天才手表应用市场

在手表系统适配的 App 数量与安装策略上，两个品牌的手表应用商店在实际使用体验中区别还是相当大的：当手表安装数量较多的应用之后，华为儿童手表 3 Pro 超能版自带 8G存储空间在使用流畅度上要好于小天才 Z5 的自带机身 4G 存储空间。在应用安装方式上，小天才在手表中内置了一个独立的应用商店 App，小朋友可以在商店中选择自己喜欢的 App 下载到手表上。而华为儿童手表 3 Pro 超能版则是将 App 商店功能完全集成在了手机端的“智能关怀”APP，只有家长才能选择下载，由手机推送到手表上。相对于小天才的手表自行安装而言，华为手表应用市场的这种应用推送安装的方式，更加方便家长管理小朋友手表上浏览的各种内容，更好地培养小朋友的各种兴趣爱好。

除了应用存储，小朋友还有一些照片、音频文件的存储需要。华为儿童手表 3 Pro 超能版搭载了 1GB + 8GB 的存储配置，而小天才 Z5 则是仅有 512MB+ 4GB 内存可用。明显的，华为儿童手表3Pro超能版更大的运行内存与存储空间，除了能保证手表安装更多 App使用不卡顿之外，对于喜爱拍照的小朋友们来讲，也不必担心因为照片过多导致手表存储空间不足了。

此外，华为儿童手表 3 Pro 超能版还支持与家长共享同一份华为云服务存储空间（华为手机默认云服务空间为 5GB）；除了可以自动备份手表拍摄的照片之外，还支持云备份足迹、联系人等更多信息以备不时之需。

电量/续航

在实际的模拟日常使用（移动定位）中，根据笔者自己戴着两块儿童手表的测试，两颗手表从早上 7 点- 晚上 8 点一直使用一天之后，华为儿童手表 3 Pro 超能版剩余 43% 电量，而小天才 Z5 剩余 29% 的电量。

这种续航上的差别，一大部分原因是华为儿童手表 3 Pro 超能版搭载了专为儿童智能手表研发的高通 Wear 2500 处理器，而小天才 Z5 则搭载的是传统智能手表处理器 Wear 2100 ；另一层更深的原因也可能在于在 Android 系统组件的定制上华为相比于小天才的定制更加简洁。这种区别，虽然在手表动画流畅度与功能丰富度上都有所体现，但在更加重要的手表续航上华为也能满足更多使用场景的需求。华为儿童手表3Pro超能版在典型使用场景中可以做到两天一充，超级省电模式下，仅开启定位与通话功能的情况下，三天一充。

在充电方式上，由于两款手表都没有相应的睡眠监测功能，所以最好的姿势就是在晚上睡觉前，将手表摘下充电。华为儿童手表 3 Pro 超能版搭载了 660mAh 电池，小天才 Z5 搭载了更大一些的 700mAh 电池，两款手表在数次正常充电测试中，均能在55-75分钟内充电完成；不过在充电问题上略有不同的是，小天才使用的是定制磁吸接口充电线，而华为则是传统的 Micro-USB 充电接口，后者在通用性上更好；在日常家用中，小天才的这种磁吸式充电设计也同样方便。具体用户认为哪种充电方式更加适合自己，这就见仁见智了。

总结

作为两家目前在儿童智能手表领域极具竞争力的厂商，小天才 Z5 与华为儿童手表 3 Pro 超能版各有优势。在儿童手表的核心场景——安全守护层面来看，两款产品实力相当。而其他方面，华为则凭借其，在手机/智能手表领域，多年积累的生产、设计经验，带来了更贴合儿童需求的产品构思，如实体 SOS 按键，IP68 级防水等，让人眼前一亮。此次华为儿童手表还带来手机与手表更加便捷的沟通体验以及颇具性价比的售价。华为儿童手表 3 Pro 超能版的上市，可以说为千元价位儿童智能手表市场，增添一个兼具性价比与服务品质的新选择。
