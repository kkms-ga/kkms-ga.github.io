浏览提速：微软宣布Edge Canary已支持预加载网页

May 30, 2020

基于 Chromium 内核打造的 Microsoft Edge 浏览器，已为 Canary 开发分支的用户们带来了新的功能。你将能够在“隐私和服务”设置菜单下，找到“预加载网页以加快浏览和搜索速度”的选项。启用后，基于 Chromium 内核的 Edge 浏览器便可查找当前网站页面上的所有链接，并预先加载接下来最有可能访问的页面（DNS 预取）。

访问:

微软中国官方商城 - Windows

其实在 Google Chrome 浏览器上，已经默认启用了“预加载”的操作，你可在“设置 -> 隐私”设置项下找到这项可加速浏览和搜索体验的功能。

如果启用此设置，则网站和任何预加载的嵌入式内容，都可以设置和读取它们的 cookie，就好像您曾经访问过它们一样（即使最后并没有访问）。

虽然早已有其它基于 Chromium 内核的第三方浏览器启用了该功能，但新版 Microsoft Edge 却是首次在 Canary 开发分支引入了该设置选项、并且默认启用。

需要指出的是，微软在其 Edge 浏览器中禁用了大多数 Google 服务。通常情况下，如果用户安装了某些隐私管理或广告屏蔽扩展，该设置项也会改成被禁用。

如果你在启用之后，又出于隐私顾虑想要禁用预加载，亦可参考如下方法来修改 Edge Canary 的设置：

（1）启动新版 Microsoft Edge 浏览器；

（2）点击‘…’图标，选择‘设置’；

（3）转到‘隐私和服务’->‘服务’；

（4）切换‘预加载网页以加快浏览和搜索速度’。

需要注意的是，如果您的 Edge 浏览器装有控制 / 管理扩展，还请在执行上述操作前临时禁用或移除。
