[图]三星Galaxy Z Flip 5G也会有黑色、灰色和铜色三种颜色

June 14, 2020

此前有消息称三星 Galaxy Note 20 将会拥有灰色、绿色和铜色三种颜色，而 Galaxy Note 20 Ultra 则只有黑色和铜色两种颜色。今天知名爆料人士 Ishan Agarwal 透露了有关于三星 2020 年下半年旗舰颜色的更多信息。

在今天发布的推文中，他写道

这是真的：

三星 Galaxy Z Flip 5G 会有黑色、灰色和铜色三种颜色。

三星 Galaxy Note 20 5G 会有灰色、绿色和铜色三种颜色。

三星 Galaxy Note 20+/Ultra 5G 会有铜色、黑色和白色三种颜色。

以上以上这些颜色之外三星可能会推出更多的颜色，这个我可以打包票。

在推文中表示，Galaxy Z Flip 5G 同样也会有黑色、灰色和铜色三种颜色。援引韩媒 The Dong-A Ilbo 报道，三星计划在2020年8月5日举行Galaxy Unpacked活动。该活动很可能会采取线上直播的方式进行，因为由于新型冠状病毒大流行的风险持续存在，很难邀请国内和海外的媒体参加现场的聚会。

根据国内认证机构曝光的型号数据，可知 Note 20 具有支持 5G 和 25W 快充等特点。屏幕为采用 LTPO 可变刷新率节能技术的 6.42 英寸的打孔屏，Note 20+ 机型则是 6.9 英寸 @ 120Hz AMOLED 屏。两款机型都有望配备 Exynos 992 芯片组，且 Note 20 的电池容量为 4000 mAh，较去年的 Note 10 机型（仅 3500 mAh）高了不少。

（图自：GalaxyClub）

大号的 Note 20+ 预计为 4500 / 5000 mAh，与上半年的 S20+ 或 S20 Ultra 看齐，同时可选支持更快的 45W SuperFast 快速充电器。拍照方面，预计 Note 20 会采用类似 Note 10 或 Galaxy S20 后置三摄组合，且只有 Note 20+ 会用上类似 S20 Ultra 的 1.08 亿像素传感器。尽管算不上是革命性，但 Note 20 系列智能机仍有望为消费者带来一定程度的可靠升级，比如更出色的屏幕、摄像头性能、以及 S Pen 手写笔支持。
