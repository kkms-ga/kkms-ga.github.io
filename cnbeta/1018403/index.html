传苹果已收购以色列拍照初创公司Camerai

August 21, 2020

根据Calcalist 报告，苹果一年半前已经默默收购了以色列拍照初创公司 Camerai，收购价格在几千万美元。Camerai 开发了各种拍照相关的技术，包括深度学习高级功能以及计算机视觉等。

Camerai 的技术已经成为每台苹果设备相机的重要组成部分，为想要在应用中集成 AR 功能的开发者带来了便利。

Camerai 的技术包括：检测图片中的不同物体，并精确地勾勒出它们的轮廓，改变它们的外观；能够在整个图像中勾勒并应用滤镜；“骨架追踪 “神经网络 API 可以实时检测并绘制人体关节，叠加在一张人类的图片上；选择性对焦，用于增强人像模式。

Camerai 之前的名字是 Tipit，是  Aaron Wetzler、 Erez Tal、Jonathan Rimon  和 Moty Kosharovsky 在 2014 年创立的一家公司。
