Windows应用程序今秋可通过Parallels在Chrome OS中运行

August 07, 2020

早在6月，Google 就预告了 Parallels 将于今年秋季登陆 Chrome OS，不过并没有透露更多信息。今天，Parallels 对这一改变进行了更详细的披露，用户将能够在 Chrome OS PC 上运行一个完整的 Windows 虚拟机，而不需要互联网连接。另外，作为一个虚拟化解决方案，可以不用出现双系统启动的情况。

“Parallels Desktop for Chromebook Enterprise将使IT管理员能够简单的为Chromebook提供Windows虚拟机和员工需要运行的全功能Windows应用程序，“Parallels工程与客户支持高级副总裁Nick Dobrovolsiky说。“IT 管理员将能够轻松管理熟悉的自动化企业Windows 映像部署，他们可以事先准备并自动提供给员工。管理员可以安装应用程序，远程管理它们，并使用熟悉的工具支持用户–就像他们使用物理Windows PC一样。”

不过目前，Parallels的这一新特性仅限于Chrome OS企业版，也就是说，这是专门针对企业客户开发的。看来，Parallels将成为谷歌为Chrome OS提供的虚拟化解决方案，类似于微软在Windows 10中的Hyper-V。在某些时候，这种合作关系可能会扩展到包括支持软件开发中使用的模拟器等。

Parallels for Chromebook Enterprise应该会在今年秋天到来。主要的价值主张是为企业运行完整的 Windows 应用程序的能力。毕竟，正如博文中所说，几乎每个企业都需要某种 Windows 应用程序。
