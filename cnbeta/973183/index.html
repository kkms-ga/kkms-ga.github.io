DisplayPort 2.0现已与USB4兼容 支持两台8K显示器或一台16K显示器

April 29, 2020

VESA今天宣布发布了DisplayPort Alt Mode 2.0，为未来USB4设备支持DisplayPort 2.0铺平了道路。2019年6月发布的DisplayPort 2.0规范最大有效带宽为77.4Gbps，是DisplayPort 1.4的近三倍。

新标准可支持高达16K分辨率的显示器、更高的刷新率、更高的分辨率下的HDR支持、对多种显示器配置的支持增强等。

USB4将Thunderbolt和USB协议融合在一起，这是英特尔的目标之一，即在无版权费用的基础上提供Thunderbolt，这将使Thunderbolt配件（如底座和eGPU）的供应范围更广、成本更低。

USB4采用了USB-C接口设计，并向后兼容USB3.2和USB2.0。

苹果可能会在未来的Mac中采用USB4，VESA预计2021年将有首批集成DisplayPort Alt Mode 2.0的产品出现在市场上。
