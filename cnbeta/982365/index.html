Windows 10大更新将引入Windows 10X的诸多功能

May 23, 2020

微软在去年于纽约举办的 Surface 硬件发布会上透露了 Windows 10X 项目，但此后一直没能进一步探讨该操作系统的细节。该公司曾在 4 月份证实，Windows 10X 将率先登陆单屏设备，原计划的 Surface Neo 等双屏设备的发布日期已被延后。据悉，Windows 10X 的内部代号为“Windows Lite”，目前尚不清楚该公司计划如何测试该操作系统。

微软目前正在为单屏设备商定 Windows 10X 的优先级，以便大家尽快展开测试体验。需要指出的是，面向台式机的 Windows 10 和面向双屏设备的 Windows 10X 并不会彻底取代对方，而是会和谐共存。

最新消息是，微软将尽快提供 Windows 10X 承诺带来的一些功能。但这些功能将不仅限于 Windows 10X，甚至可能通过 beta 公测的形式，而首先出现在 Windows 10 平台上。

消息人士称，软件巨头正在探索对 Windows 10 Win32 容器的支持，通过在沙盒环境中“虚拟”地运行现有的 Win32 应用程序，该桌面操作系统将首次迎来这项安全特性。

在早些时候举办的 Build 2020 线上开发者大会上，微软官方未披露这些细节。不过多个消息来源称，该公司正试图厘清将如何在 Windows 10 平台上引入 Windows 10X 的大量特性。

微软还计划测试应用程序的“信任系统”，以便具有良好口碑的签名代码和应用程序能够更安全地安装到桌面操作系统上。

来自用户的真实反馈，有助于该公司尽快修复 Windows 10X 上的各种问题，比如 Win32 和沙盒性能问题。

此外微软有望在 Windows 10 上测试新的 Windows Update 体验，据说更加方便快捷。至于上述改进将于何时引入，还请耐心等待。
