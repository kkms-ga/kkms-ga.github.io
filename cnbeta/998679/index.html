大量用户不满微软通过Windows更新强推Edge浏览器

July 03, 2020

近期的 Windows 10 更新政策，已经引发了许多用户的不满，比如在自作主张地在桌面和任务栏上放置 Edge 浏览器图标。尽管通过一系列的改进，新版 Edge 浏览器已经在许多方面受到了大家的好评，但并不是所有人都希望整天与它打交道。一些愤怒的网友甚至指责微软的行为与恶意软件无异，因为无论拒绝与否，微软都会试图说服用户从 Chrome 迁移至 Edge 。

这样的做法，不禁让我们联想到了 90 年代的一场反垄断诉讼。作为被告方的微软公司因强行在 Windows 操作系统中捆绑 IE 浏览器而遭到了巨额罚款。

对于长期以来购买实体安装介质来支持正版的 Windows 老用户来说（比如 Windows 7 和 Windows 8），实在难以想象这家软件巨头竟然会好了伤疤忘了疼。

当强制性的安全更新成为一种侵入式的营销工具，显然会引发用户强烈的抵触情绪。但我们还是很怀疑，Windows 10 和 Microsoft Edge 开发团队到底有没有仔细考虑过这么做的后果？

推特网友 @gothfundme 吐槽道：“一觉醒来，就发现电脑不仅被强制安装了 Edge 浏览器，还在未经我许可的情况下将图标固定到了任务栏上”。

推特网友 @im2blame4 更是嘲讽道：“微软啊，说好的‘我司很是关心用户的隐私’呢？我根本不想从 Windows 7 升级到新系统啊，但你咋又偷偷地在我电脑上安装了 Edge 浏览器呢？”

推特网友 @DjUchuu：“既然 Windows 7 已经终止支持，那为何昨晚我的工作电脑上又被自动安装 Microsoft Edge 浏览器？”

推特网友 @ paperichie：“我的电脑被强制关机后安装了 Microsoft Edge 浏览器，白白浪费了个把小时的绘图工作进度，‘真太感谢你**的微软了’。管你强制不强制，根本没人愿意用你家烂透了的浏览器”。

推特网友 @totccast：“Windows 显然在半夜悄悄地更新了，然后我为第 56 集节目制作的 Notes 文档全特么完蛋了！顺道感谢不请自来的 Microsoft Edge？”

推特网友 @Paul_Laris：“Windows 10 更新后被安装了默认的 Edge 浏览器，这广告打得也太 NB 了，你甚至没办法将它卸载。别犯傻了，建议跳过本次更新”。

TheVerge 的 Sean Hollister 从用户（而不是 IT 新闻从业者）的角度对此事进行了分析，然后认为微软强推 Chromium Edge 浏览器的行为弊大于利。

就算那些曾经对此感兴趣的人们，也可能被烦得去尝试 Mozilla Firefox 和 Opera 等第三方浏览器。
