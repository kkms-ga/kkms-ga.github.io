亚马逊开售Echo Flex数字时钟 支持Alexa智能语音交互

August 08, 2020

尽管亚马逊很早就推出了一款实用的智能时钟（Amazon Clock），但其采用了壁挂式 + 模拟显示的设计。对于喜欢小巧的数字时钟的用户来说，亚马逊新推出的 Echo Flex，或许更能满足你对 Alexa 智能语音交互的需求。得益于 Amazon Flex 最新迎来的 Smart Clock 配件，售价 24.99 美元（约 174 RMB）的 Echo Flex，或许是一种更加经济的功能扩展方式。

（来自：Amazon）

据悉，作为一款功能齐全的 Alexa 设备，它可以轻松地插入标准的市电插座。如果你讨厌在走廊、车库、浴室等场景布线，它 Echo Flex 显然可以帮你消除这些烦恼。

布置完成后，你可以通过 Alexa 平台来控制兼容的智能家居设备，或者查找你喜爱的球队的最新比赛得分和公告，Echo Flex 将通过内置的小扬声器来响应用户的提问。

Echo Flex 还提供了一个内置的 USB 端口，方便用户为手机或其它配件进行充电，比如智能夜灯（Smart Night Light）、运动传感器（Motion Sensor）等智能时钟配件（单价为 14.99 美元 / 约 104 RMB）。

只需将 Smart Clock 配件接入 Echo Flex 的 USB 端口，即可在小面板上显示当前时间或倒计时（最多支持 20 组计时器），并且能够基于环境光传感器来自动调节亮度。

感兴趣的朋友可通过 Echo Flex 为每个房间轻松扩展实惠的 Alexa 智能交互体验，目前亚马逊官网上标注的出货时间为 8 月 11 日。
