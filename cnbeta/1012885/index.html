[图]Ubuntu 20.04.1 LTS发布：安装镜像已开放下载

August 07, 2020

Canonical 近日发布了 Ubuntu 20.04.1 LTS 更新，这是该分支的首个维护版本更新。本次更新重点修复了存在于 20.04 版本中的诸多 BUG，并公布了新的安装镜像，用户可以直接安装升级。Canonical 还表示，仍运行 Ubuntu 18.04 LTS 的用户很快将通过 Update Manager 升级到 20.04.1 版本。

自然，基于 Ubuntu 20.04.1 LTS 的各种风味版本也已经发布，其中包括

Kubuntu 20.04.1 LTS

Ubuntu Budgie 20.04.1 LTS

Ubuntu MATE 20.04.1 LTS

Lubuntu 20.04.1 LTS

Ubuntu Kylin 20.04.1 LTS

Ubuntu Studio 20.04.1 LTS

Xubuntu 20.04.1 LTS

完整的更新日志可以访问这里。其中值得关注的改进包括，改善了 Update Manager 的体验，包括针对 snap 更新时出现的短暂卡顿等问题。Ubuntu 20.04.1 LTS还支持新的服务器英伟达驱动，英特尔Tiger Lake，另外还有针对Skylake处理器的额外微码修复。

本次更新还对某些预安装应用进行了更新，包括 LibreOffice、Thunderbird、Mesa、Firefox 和 Pulse 等等。
