Opera宣布基于浏览器加密钱包已迎来17万月活用户

July 25, 2020

Opera 刚刚宣布，其基于浏览器的加密钱包应用，现已迎来超过 17 万的月活跃用户（MAU）。据悉，这款 Crypto Wallet 支持跨 PC、Android 和 iOS 设备使用。尽管目前已经吸引了不少用户，但 Opera 坦言距离其奔向主流市场仍有很长一段距离。为增加吸引力，该公司现在英国推出了简易的加密钱包充值功能。

作为 Opera 的合作伙伴，Ramp 使用了欧洲开放银行 API，以便用户通过银行账户、或 Revolut 轻松实现低价值的加密钱包充值操作。简化的功能，有助于消除普通用户的接触壁垒。

此外 Opera 还与 Dapp.com 达成了合作，以便用户轻松发现托管在区块链上的去中心化应用，且 Opera 相信这会是帮助用户发现最佳 dApp 的一个重要方式。

据悉，Opera 将整个加密钱包生态系统称作“Web 3”。尽管目前尚无对于 Web 3.0 的官方定义和共识，但不难想象 Opera 希望给用户提供更大的数据控制权。
