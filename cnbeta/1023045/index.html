Manjaro社区版PinePhone Linux智能机即将上市 起售价149美元

September 01, 2020

尽管基于 Linux 底层的 Google Android 及其开源项目（AOSP）已经在移动市场取得了相当大的市场份额，但一些资深 Linux 用户仍觉得它不够纯粹。为满足这部分用户的需求，以 Purism Librem 5 和 Pine64 PinePhone 为代表的 Linux 智能机项目应运而生。说到后者，其社区用户将很快迎来一个正式的 Manjaro 版本！

（来自：Manjaro.org）

感兴趣的朋友，可购买预装了 Manjaro Linux 的 PinePhone，并且可从 2GB RAM + 16GB ROM / 3GB RAM + 32GB ROM 这两种存储组合中进行选择，售价为 149 / 199 美元（约 1019 / 1361 RMB）。

Manjaro 团队表示：“我们很高兴地宣布，PinePhone 的下一个社区版本（CE）将启用 Manjaro 。我们从一开始就为 Pine64 提供了技术支持，现已支持几乎所有 Pine64 单板机和相关设备”。

后续将有三个面向 PinePhone 的衍生版本（包括 Lomiri、Phosh 和 Plasma Mobile），有意向的用户可按需选择。

此外支持 Manjaro CE 的两款 PinePhone 带有 1.2a 版本的 PCBA，当前出货预装的还是 postmarketOS CE 。如果一切顺利，其有望于 9 月份开放预订。

最后，售价 199 美元的 Convergence Package 衍生版本还支持 USB-C 扩展坞，提供一个以太网端口（10 / 100Mbps 自适应）、两个 USB Type-A 和 HDMI 视频输出，搭售价仅为 10 美元。
