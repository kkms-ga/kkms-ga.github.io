谷歌将Pixel 4A智能机藏哪了？中端新机或推迟至7月发布

June 11, 2020

去年发布的 Pixel 3A 和 Pixel 3A XL，树立了谷歌对于 400 美元级别的智能机的期望。其因拥有在低光下更出色的拍照表现，而被该公司拿来与价格昂贵的苹果 iPhone 对标。如果按照正常的发布节奏，谷歌现在应该已经推出 Pixel 4A 智能机了。遗憾的是，受新冠病毒引发的 COVID-19 疫病大流行的影响，许多企业的新品发布节奏都被打乱了。

谷歌试图通过优秀的相机体验来推销 500 美元以下的智能机，Pixel 3A 和 Pixel 3A XL 的定位相当明确。与此前的 Pixel 3 和 Pixel 3 XL“旗舰设备”相比，消费者显然对“A”系列更买账。

可惜在 I/O 2020 线上活动期间，谷歌仅介绍了 Android 11，而没有提到 Pixel 4A 。新传闻称该机将于 7 月某个时候到来，距传统意义上的秋季（10 月）硬件发布会仅三个月的时间。

与此同时，Pixel 4A 的竞争对手也没有停下脚步，比如苹果 2020 款 iPhone SE、以及三星的 Galaxy A51，都在与之争夺本就很热闹的中端市场。

过去几个月，有关 Pixel 4A 原型设备的渲染图、规格、壁纸、甚至拍照样张都已流出。4 月份的时候，9to5Google 爆料称该机采用了 5.81 英寸 @ 2340×1080 分辨率的屏幕。

处理器为高通骁龙 730，辅以 6GB RAM + 64 / 128GB ROM，电池容量 3080mAh，采用 8MP 打孔前摄 + 12.2MP 后摄，支持与 Pixel 4 相同的视频录制功能（最高 4K @ 30fps）。

该机还预留了 3.5mm 耳机插孔，并且采用了后置指纹传感器。预计售价会与去年的 Pixel 3A 持平，即 399 美元（约 2800 RMB）起。
