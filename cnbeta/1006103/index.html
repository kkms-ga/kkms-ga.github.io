游戏本成电商销量榜常客 2020年它们的变化有多大？

July 22, 2020

在刚刚过去的年中大促中，不少新上市笔电都有不错的销量：除了远程办公、学习本身的需求之外，高性能、同时兼顾携带需求的游戏本，也在这期间成为了各大电商平台热销的笔电爆品。虽然年中的销售热潮已经告一段落，但对于今年来讲，无论是新游戏本产品、还是各大电商平台的打折力度，其实都还有很多值得期待。

同时在这期间也有不少新一线笔电厂商仍然在推出新的游戏本：无论是游戏电竞玩家、还是主打办公的同时，也想保证自己的笔电在未来数年中日常游戏性能不落伍的用户，都更倾向于选择性能更好的游戏本。

游戏本的强劲性能释放表现，成为本年度笔电中相当有看点的新系列 ， 搭载十代酷睿标压版的游戏本也成为主流用户在选购游戏本时的普遍选择。同时，在各家游戏笔电厂商今年推出的新款笔电产品上我们也能看到，得益于一线品牌设计优势，以及十代酷睿 CPU 深度软硬件优化，让今年不少游戏本在颇具竞争力的售价之外，还加入了不少提升游戏体验的特色功能。不少游戏玩家在预算充足的情况下，能够选到一台游戏体验更丰富的十代酷睿游戏本。

游戏体验决定性因素：性能

在今年上市的新款游戏本款式中，有不少凭借 CPU 新制程工艺与多线程让笔电跑分大幅提高的新品在销量排行榜上脱颖而出，这种纸面数据能显著反映出来的明显升级也足以让用户对于自己将要购买的笔电有一个大致的性能判断。

但归根结底，判断一台笔电性能是否适合自己，最直观的方式，就是通过让玩家自己上手游戏体验得出结论。新笔电上手之后，玩家从自己平时热衷的游戏实机感受到眼前的效果提升：高帧率、低延迟、乃至新世代 3A 大作普遍提升的游戏画质，才是构成玩家直观的感受到的实际体验决定因素。

同时，无论是日常娱乐的《英雄联盟》、《守望先锋》等主流网游、还是《GTA5》、《全面战争》等 3A 游戏大作，都对笔电 CPU 高频性能要求较高、从用户实际体验角度来讲、搭载更高主频 CPU 的笔电在游戏时，性能释放更加稳定、相比低主频 U 的笔电来讲无论是风扇噪音还是在游戏时，笔电的散热压力都更小，出现“风扇噪音奇大”与“电脑游戏自带键盘烫手”这样影响游戏体验的问题几率更小。

当然，以上体验提升是高主频 CPU 给游戏玩家带来的直观优势所在，优化实际游戏体验，还是需要游戏本从高性能内存到笔电散热系统的性能的共同提升，才能在游戏时保证笔电平稳应对高负载运行压力，不会出现“烤机”的尴尬。

虽然这些要求听起来都颇为苛刻，不过今年上架的主流 5000-8000 价位的游戏本，大都在保证游戏本性能释放的同时，还增加了自家独有的特殊卖点作为提升游戏体验的另外因素：比如外星人与戴尔定制功能颇受好评的 AWCC 智控中心应用、雷蛇的定制灯效、暗影精灵今年新品的设计风格转变，都还是让用户感到：随着十代酷睿与性能提升相结合的更多拓展性提升的加入，笔电的更多拓展特性也在发生着改变。

游戏优化

对应的，在游戏本优化这个命题上，不同的笔电厂商也根据自家的笔电产品特色推出了不同的性能调教方式：目前主流的优化方式，就是为笔电预装厂商研发好的调节软件，采用激进亦或是保守的性能输出模式，从系统 Bios 层面下手，给玩家更加灵活的性能调节区间。

同时，来自上游厂商的优化也是对于游戏优化有着巨大提升的一环：比如十代酷睿今年的新特征—— 最高主频 5.3GHz，就让游戏场景实时渲染、游戏 AI计算让主打算力的使用场景能更加游刃有余：具体来讲，就是让游戏内的各项数据加载进度更加“无感”。

在电竞场景中，尤其是在职业电竞赛事中，一般电竞选手的每分钟操作数（ActionsPer Minute）—— 也就是手速，都在每分钟 300 次乃至更高。在这种飞快的操作之下，无论是开放世界游戏不同场景切换、PVP 游戏多玩家建模场景；都更需要处理器有更高的运行频率，来减少系统加载速度跟不上玩家操作响应速度所产生的割裂感。

而此时上游厂商的开发优势就能体现出来：在十代酷睿的研发中，英特尔在开发过程中就与主流游戏开发厂商一道、优化十代酷睿新品的游戏表现。

比如十代酷睿新加入的睿频加速 MAX 3.0 技术，就能自动判断当前 CPU 运行状态，来调节核心之间的任务分配，让笔电提升至更高的运行频率来满足高负载游戏场景的性能需求；同时反过来讲，这项技术也能在笔电使用功耗较低的场景下自动降低 CPU 核心运行数量与运行频率，保证笔电的电池寿命；这项技术也有助于笔电厂商按照用户的性能调节习惯设定相应的游戏模式，让玩家能按照自己的使用习惯更自由地深度定制笔电的性能释放“不同风格”。

笔电设计/功能

除了硬件性能提升与软件优化之外，前文中也有提到，在直观的笔电性能释放上，CPU 高频多核的优势毫无疑问是本年度游戏笔电的升级重点，但随之而来的笔电其他方向上的性能提升，以及今年笔电新品在设计上改变，还是让今年的十代酷睿笔电有不少拓展功能与设计都比较亮眼的新品上市。

在 6000 元价位中，各家厂商新推出的游戏本新品基本上加入了对 Wi-Fi6 的支持，更适合网游与微软 XCloud 、英伟达 GeForce Now 这样的云游戏服务，对于硬核玩家来讲，好的网络环境还是相当重要的。

同时，对雷电 3 传输协议的支持也算是老生常谈的问题了：同样是得益于十代酷睿原生整合进雷电 3 的控制器，让搭载雷电 3 接口的笔电售价更加接近主流区间，让更多的玩家可以通过雷电 3 的高速率传输接口链接更多高性能外设：比如通过菊花链的方式链接多台 4K 显示器、连接外置独立 GPU,都能大幅提升游戏体验,，考虑到笔电往往是要伴随玩家超过 4 年的长期装备，这种能保证笔电更多性能拓展的功能，在小编看来，都已经成为了 2020年选购游戏本需要考虑的因素。

在拓展性之外，也有更多的游戏本新品，在设计上开始追求质感：比如今年不少游戏本新品所主打的窄边框、全高清屏幕，以及更接近传统轻薄本的外观设计、辅以游戏玩家热爱的 RGB 定制灯效、大面积散热风扇等设计，都让今年的游戏本在不牺牲游戏体验的同时，更符合“颜值党”的审美；同时部分笔电所采用的性冷淡商务风设计元素，对于日常有办公需求的游戏玩家来讲，也能让自己的笔电适应更多办公场景而不显突兀。

从上述的今年新款游戏本的普遍游戏性能提升：除了十代酷睿标压版的睿频升级、以及英特尔与游戏厂商的软件优化；还是笔电厂商为游戏本所作出的各种性能调教策略，都能直接反映在玩家上手游戏体验上，如果你也是一个希望购买一台高性能笔电来提升自己游戏体验的玩家的话，2020 年十代酷睿游戏本的各项游戏体验提升，值得作为购买笔电的参考标准。
