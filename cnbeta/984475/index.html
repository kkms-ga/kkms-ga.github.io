Galaxy Note 20系列首发 三星Exynos 992曝光

May 28, 2020

5月28日消息，据Android Authority报道，三星Galaxy Note 20系列将首发Exynos 992芯片。外媒援引消息人士称，三星Galaxy Note 20系列搭载的Exynos 992芯片会有大幅升级。报道称Exynos 992基于5nm工艺制程打造，采用ARM Cortex A78核心，GPU为Mali-G78。与7nm Cortex A77相比，采用5nm工艺制程的Cortex A78功耗降低50%，性能提升20%。

ARM还表示，采用5nm工艺的GPU比Exynos 990的Mali-G77性能提升25%。

需要注意的是，Exynos 992是否会采用5nm工艺及Cortex A78核心有待证实，因为此前爆料称Exynos 992采用的是6nm工艺制程。

而且三星通常是由Galaxy S系列首发ARM最新一代CPU，Cortex A78 8月份是否能量产商用还有待商榷（Galaxy Note 20系列预计在8月份登场）。

综上所述，Galaxy Note 20系列有望首发Exynos 992，该芯片有可能是Exynos 990的小幅升级版本。在此之前，三星Galaxy Note 10首发Exynos 9825，而Galaxy S10首发Exynos 9820。

三星有可能会延续去年的节奏，在Galaxy Note 20上使用Exynos 992芯片，它将于8月份发布，届时Galaxy Fold 2、Galaxy Tab S7系列平板电脑可能会同台亮相。
