金士顿发布KC2500 M.2 NVMe固态硬盘新品

April 28, 2020

金士顿刚刚推出了 KC2500 系列 M.2 2280 NVMe 固态硬盘新品，采用了支持 PCIe Gen3 x4 通道的慧荣（SMI）2262EB 主控 + 96 层 3D TLC 闪存方案，起售价低至 74 美元（约 524 RMB）。KC2500 最高可达成 3500 MB/s 和 2900 MB/s 的持续读写速度（1TB / 2TB 版本），兼顾出色性能和耐用性，能够极大地改善台式机、工作站等用户的使用体验。

KC2500 支持完整的 AES-XTS 256-bit 端到端硬件加密，允许通过赛门铁克、迈克菲、WinMagic 等支持 TCG Opal 2.0 规范的软件来管理数据安全。内置的 Microsoft eDrive 支持，还能够与 BitLocker 配合使用。

KC2500 系列可选 250GB / 500GB / 1TB / 2TB 容量，写入耐久度为 150 / 300 / 600 / 1200 TWB 。

250GB 版本的顺序读写性能为 3500 MB/s 和 1200 MB/s，500GB 版本则是 3500 MB/s 和 1200 MB/s 。

金士顿为 KC2500 系列 NVMe M.2 SSD 提供五年有限质保，250GB / 500GB / 1TB 版本的售价分别为 73.99 / 134.99 / 234.99 美元（约 524 / 956 / 1665 RMB），2TB 版本暂未公布。
