联想将开始提供预装Fedora Linux发行版的ThinkPad笔记本电脑

April 29, 2020

近年来，Linux 在 PC OEM 市场的合作已经不断壮大。比如作为 Canonical 的老伙计，戴尔长期在提供预装 Ubuntu Linux 发行版的系统选项，此外还有 System76、Purism、ZaReason 等厂商在提供预装了 Linux 操作系统的高性能硬件。最新消息是，联想也将开始提供预装 Fedora Linux 发行版的 ThinkPad 笔记本电脑。

访问购买页面:

ThinkPad自营旗舰店

（来自：Fedora Magazine）

作为与 Fedora Project 合作的一个项目，联想希望新手也能够轻松开启 Fedora Linux 发行版的体验之旅。

此外作为一个面向 Linux 社区的系列试验计划，初期提供 Fedora Edition 的机型包括 —— ThinkPad P1 Gen2、ThinkPad X1 Gen8、以及 ThinkPad P53 产品线。

这些机型将预装新发布的 Fedora 32 工作站版，想必联想也让它们通过了相关认证，能够在仅使用第一方资源库的情况下得到很好的利用。

如果消费者的反响比较强烈，那该公司还有望在后续提供对更多 ThinkPad 机型的支持。

虽然此举意味着 ThinkPad P53 和 ThinkPad P1 Gen2 不会默认安装英伟达显卡的驱动程序，但通过专有来源来下载和安装的操作，也是相对比较容易的。

鉴于戴尔过去几年已经在 XPS 13“开发者专版”（Developer Edition）上取得了相当大的成功，我们原本以为联想也会首选 Canonical 的 Ubuntu Linux 发行版，没想到最终选择的合作伙伴是 Fedora Project 。

当然，联想没有堵上可选装其它 Linux 发行版的可行性，毕竟 Canonical 那边已经提供了一个单独的长期支持（LTS）版本。

Fedora Lenovo Laptops（via）

除了每两年更新一次，各个 LTS 版本还可在发布后得到长达五年的支持期，相当于一台普通笔记本电脑平均寿命的两倍了。

此前，AMD 已经展示了基于锐龙 4000 系列移动处理器的诸多特性，很高兴联想能够宣布推出至少一款采用 Ryzen 4800H 的 ThinkPad 机型，为消费者提供了具有优秀的生产力的 APU 选项。

遗憾的是，联想尚未公布这些“功能强大”的 ThinkPad 机型的价格和上市日期。
