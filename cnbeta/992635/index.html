Google Assistant迎来了快速流畅和更准确的增强交互体验

June 18, 2020

谷歌表示，从今夏晚些时候开始，Google Assistant 智能显示屏将可以访问 AMP 移动加速网页上的文章。作为该公司力推的即时文章格式，其旨在移动设备上更快地加载网页。但是不久后，这类出色而又有深度的内容（主要是新闻资讯），也将登陆 Smart Display 设备。

资料图（来自：Google 官网）

据悉，当前谷歌已为 Smart Display 提供了大量新闻内容，但此前更倾向于呈现视频片段和文章摘录，而不是整篇 AMP 文章。

此外，谷歌宣布了面向智能语音助理的多项改进，以带来更快的交互、无缝的体验、以及更准确的查询响应。

针对家庭内多人使用的场景，Home Storage 功能允许开发者为多个用户保存不同的交互记录，比如某款益智游戏的多个进度。

Media API 允许用户从上次中断的地方继续，从而更轻松地收听长格式内容。连续匹配模式（Continuous Match Mode）则能够让用户与智能助理的交互更加流畅。

最后，谷歌介绍了新发布的 Actions Builder 网络工具，以便大家为 Google Assistant 开发新的技能。

通过与现有的 Actions Console 控制台集成，开发者能够在一个地方完成相关“动作”的构建、调试、测试、发布和分析。

结合更多的功能和更好的开发人员工具，每月五亿的 Google Assistant 将迎来体验上的进一步改善。
