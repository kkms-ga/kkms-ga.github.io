[图].NET 5.0预览版6发布：Windows ARM64设备支持Windows Forms

June 26, 2020

今天，微软宣布了第六个 .NET 5.0 的预览版，在改进性能的同时增加了一些新的功能。本次更新的主要功能是在 Windows ARM64 上支持 Windows Forms 应用程序。此前，Windows ARM64 上只支持控制台和 ASP.NET Core 应用程序。

通过这项支持，开发者可以在 Surface Pro X 等 Windows ARM64 设备上构建和运行 Windows Forms 应用。微软还透露，他们仍在努力为 Windows ARM64 设备提供 WPF支持。

Visual Studio .NET远程调试器对Windows ARM64的支持将在Visual Studio 16.7版本中提供。Visual Studio Code .NET远程调试器支持将在以后推出。
