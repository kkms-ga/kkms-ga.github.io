Realme真我8大新品发布会 真我X50 Pro玩家版售￥2999起

May 25, 2020

在预告 5 月 25 日发布新机的同时，爆料称该公司也将进一步拓展 IoT 设备产品线。下午 2 点整，为庆祝 Realme 征战国内市场一周年，徐起与虚拟主持人一同亮相直播间，并预告了本场发布会将有 8 款新品。本次发布会的主角是真我 X50 Pro 玩家版，6+128 版本售￥2999 起。其次是 Realme 真我 X50m 中端机型，两款 TWS 真无线耳机，两款快充充电宝，以及潮玩双肩包。

本次发布会带来的首款产品，就是 Realme 真我 X50 Pro 玩家版，具有独特的“光速银”、以及玩家们期待已久的“幻影黑”配色。

该机采用了 90Hz 高刷新率的三星 Super AMOLED 显示屏，辅以 180Hz 的触控采样率，峰值亮度 1100 nit，覆盖 100% DPI-P3 色域。

处理器方面，该机采用了高通骁龙 865 5G SoC，辅以 LPDDR5 运存 + UFS 3.1 存储，以及 65W SuperDart 闪充方案 —— 充电 3 分钟，游戏 1 小时。

在采用旗舰硬件的同时，真我 X50 Pro 玩家版还采用了 VC 液冷散热方案，以保障帧率的稳定。

为增强音效体验，带来全方位的沉浸感，真我 X50 Pro 玩家版还引入了 4D 游戏振感 2.0 技术。

网络方面，该机支持 2.4GHz + 5GHz 双 Wi-Fi 加速。

辅以 HyperBoost 3.0 全速电竞模式，性能最高提升 30%，功耗最高降低 12%。

而游戏防打扰功能可一键屏蔽电话、微信、QQ 等消息通知。

影像方面，该机采用了索尼 48MP 后置四摄方案，IMX 586 传感器、8MP 超广角 + 微距的组合，支持 AI 炫彩、极夜模式等体验。

在与 @sStarPro诺言 相约“王者峡谷见”之后，二次元产品经理爱酱（@绊爱China）接班介绍真我 X50m 机型。

X50m 具有星空白、银河蓝两种配色，采用高通骁龙 765G 双模全网通方案，配备了 120Hz 双摄畅速屏，以及 48MP 广角四摄。

内置 4200mAh 电池，支持 30W Dart 闪充。6GB RAM + 128GB 版本仅售 2099，线上线下热卖中。

说完两款手机，徐起又为我们带来了两款 TWS 真无线耳机，首先是真我 Buds Air Neo，然后是真我 Buds Q。

真我 Buds Air Neo 采用了 13mm 超大动圈单元、R1 真无线芯片，具有 119ms 游戏超低延迟。支持智能触控和 realme Link 自定义功能，首发特价￥269 。

通过与爱马设计师的合作，真我 Buds Q 采用了没有一条直线的设计，单只耳机仅 3.6g，轻过一张 A4 纸。20小时超长续航，蓝牙 5.0 秒连，辅以 IPX4 防水特性，首发特价 ￥129 。

说完两款耳机，徐起接着接招两款移动电源。

其中真我快充移动电源售 ￥109，30W Dart 闪充移动电源则是 ￥199 。

“不务正业”的真我潮玩双肩包，首发特价￥149 。

说完潮玩 IoT 配件，徐起正式公布了真我 X50 Pro 玩家版的售价，6+128 / 8+128 / 12+128 版本首发限时特价 ￥2999 / 3299 / 3599 RMB 。

细心的朋友可能已经发现，本场发布会只说了 7 款新品，那么第八款产品是什么呢？

不卖关子，第八款产品就是真我 X50 Pro 周年纪念版，但只有参与 Realme UI 创作的“百位潮玩制作人”才有机会获得。

以上就是本次真我 8 大新品发布会的全部内容，感兴趣的小伙伴可以去围观徐起和诺言的王者开黑直播了。
