[图]Firefox 78 Beta 1发布：可直接阅读下载的PDF文件

June 02, 2020

今天 Mozilla 发布了Firefox 78.0 Beta版本更新。本次更新幅度并不是很大，其中最大的改变就是下载PDF文件之后能够直接在Firefox浏览器中打开阅读。自 Firefox 78 版本开始，浏览器内置的PDF阅读器（PDF.js）可以设置为 Windows 系统中默认的PDF阅读器。

与Chrome和Microsoft Edge一样，Mozilla Firefox也内置了PDF浏览器，可以在浏览器内阅读PDF文件。当你安装了Edge或者其他基于Chromium的浏览器（例如Vivaldi、Opera和Brave等），它们都可以作为Windows 系统中默认的PDF阅读器。不过在Firefox浏览器中，如果没有安装插件的情况下并不支持阅读大部分PDF文件。

不过在最新的Firefox安装版本中解决了这个问题。该安装器能够在 Windows 上注册可支持的默认文件关联。想要在 Windows 10 系统中将Firefox作为默认PDF阅读器，操作步骤如下

1.确保运行Firefox 78或者更高版本，点击 “开始">"设置">"应用程序">"默认应用程序”。

2.向下滚动，选择 “按文件类型选择默认应用程序”

3.向下滚动找到.pdf，点击当前默认的程序图标。

4.在 “选择应用程序 “对话框中，选择Firefox

5.打开你的设备上的任何PDF文件，它现在会在Firefox浏览器中打开。

目前Firefox浏览器的PDF阅读器提供的功能包括：

● 支持打印

● 允许用户下载PDF文件并保存到本地

● 提供演示或者全屏模式

● 在侧边栏显示PDF文档的缩略图、索引目录或者附件

另外还调整了Linux的最低系统配置，需要GNU libc 2.17, libstdc++ 4.8.1和GTK+ 3.14及更高版本。针对开发者，在SpiderMonkey中引入了全新的RegExp引擎，添加了dotAll Flag、Unicode转义序列、lookbehind引用和命名捕获的支持。

此外还有一个细节地方，就是将“关闭右侧标签页”和“关闭其他标签页”整合到“关闭多个标签页 ”选项卡中。
