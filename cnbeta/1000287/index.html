三星Galaxy Note 20 Ultra通过FCC认证 确认骁龙865等关键细节

July 07, 2020

最近Galaxy Note 20 Ultra受到了很多媒体的关注。这是预料之中的，因为它将成为下个月推出的Galaxy Note 20系列当中的旗舰产品。我们已经看到了关于该设备的几乎所有信息，包括官方渲染图以及现场照片。因此，就设备的信息而言，已经所剩无几。

不过现在消息人士告诉我们，即将在美国上市的解锁版本现在正式称为SM-N986U。它将搭载骁龙865处理器，也有可能是865+，目前官方还没有公布。此外，Note 20 Ultra还将带来高通X55M调制解调器，将具备5G功能。此外，FCC认证还谈到Galaxy Note 20 Ultra将支持无线充电。另外，它附带的S-Pen以及LED观景盖也获得了认证。

在规格上据说Galaxy Note 20 Ultra配备了一块6.9英寸的显示屏，像素为3040 x 1440，用户还将在背面获得四摄像头设置，再加上正面的打孔自拍摄像头。你可以在下面查看整个规格列表。据说该设备零售价格在1400美元左右。整个Galaxy Note 20系列将在下个月的Unpacked活动中正式发布。
