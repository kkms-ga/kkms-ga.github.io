HyperX发布Predator和Fury系列DDR4 RGB内存新品

July 14, 2020

HyperX 刚刚发布了 Predator 和 Fury 系列 DDR4 RGB 内存新品，主打更高的频率和更高的延迟。对于 AMD 锐龙 3000 XT 系列处理器用户来说，高频内存可有效提升系统体验，而 Hyper Predator 和 Fury 系列正好提供了频率高达 4800MHz、以及最大 256GB 容量的套装容量。

首先介绍 HyperX Predator 系列 DDR4 内存模组，其提供了单条、双通道、四条、甚至八条的套装选项。

单根 Predator DDR4 内存的最大容量为 32GB，双通道版本最高 64GB，八条更是可以组成 256GB 。

值得一提的是，HyperX Predator 还提供了 RGB 灯条的选项，主要区别是没有 128GB 的套装版本。

此外 Predator 普条和 RGB 灯条之间的频率选项也有所区别，前者的最高频率仅为 4600MHz 。

HyperX Fury 系列 DDR4 内存模组亦提供了四个版本，最高频率为 3733MHz、最大容量 128GB 。

该系列 DRAM 产品主要面向需要大容量内存的高端计算机系统，此外也有支持 RGB 灯效的版本。

最后，HyperX Predator 系列 DDR4 普条 / 灯条套装的售价为 611 / 626 美元（约 4284 / 4389 RMB），Fury 系列 128GB 普条 / 灯条的售价则是 760 / 783 美元（约 5329 / 5490 RMB）。

所有这些 DDR4 DRAM 模组均在更大容量的情况下提供了较低的延迟（CL19），有助计算机达成更高效、轻松的信息访问，而无需担心导致处理器的延迟。
