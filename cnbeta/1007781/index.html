2020年度天文摄影师大赛入围名单公布 获奖照片将于今晚揭晓

July 26, 2020

格林威治皇家博物馆（RMG）刚刚宣布了 2020 年度天文摄影师大赛的入围名单，尽管获奖作品要到今年晚些时候才能揭晓，但这并不妨碍我们抢先感受下这些星空摄影作品的魅力。作为全球最顶级的天文摄影比赛之一，成立将近 400 年的 RMG 将为获胜者提供高达 10000 英镑的奖金。

日冕物质抛射（Łukasz Sujka / 摄）

本年度的入围作品依然多达数十张，内容涵盖了令人难以置信的太阳表面、到一系列宏伟的星系夜空。受限于文章篇幅，我们仅在此挑选出比较有代表性的几幅。

冰岛极光（Ben Bush / 摄）

上图为英国摄影师 Ben Bush 在冰岛著名的 Vestrahorn 上拍摄的极光，为了捕捉反射在水面上的美丽场景，其冒着 -6℃ 的寒意，在半夜踏入了冰冷的北大西洋海水中。

满月与摩天大楼（Mathew Browne / 摄）

为了捕获富有意义的镜头，Mathew Browne 耗费了许多心思和耐力来构图。经历了多次失败的尝试，他终于还是拍摄到了满月掠过伦敦 Shard 摩天大楼后方的转瞬即逝的镜头。

“大月小狼”（Kirsty Paton / 摄）

为了拍摄这个迷人的场景，摄影师 Kirsty Paton 用了三年的时间，期间最难的就是让她的狗狗安静地坐下来等拍照。

（图 via RMG）
