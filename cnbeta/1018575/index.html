[图]Edge Canary引入Link Format功能 复制粘贴URL链接更容易

August 21, 2020

今天，微软今天宣布了全新“Link Format”功能，从而在 Edge 地址栏上复制 URL 链接变得更加容易。该功能此前仅限于经典版 Edge 浏览器，现在微软将其迁移到 Edge Canary 频道版本中。

微软希望解决用户在 Edge 浏览器中复制和粘贴网址时的痛点。例如，该公司了解到，用户需要 “定期编辑他们复制的URL，从而在分享这些 URL 链接变得更容易理解或者移除一些额外的参数。

在启用全新的 Link Format 之后，通过新的右键菜单可以将 URL 链接作为链接或者纯文本进行粘贴。微软目前正在向Windows上的Edge Canary推出该功能，不久也将向Mac提供。截至目前，链接格式只支持地址栏中复制的URL，但该公司承诺在不久的将来也会将其扩展到 “页面内链接”。
