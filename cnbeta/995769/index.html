三星Galaxy Note 20 Ultra爆料：一亿像素主摄敲定

June 26, 2020

6月26日消息，知名爆料人士Roland Quandt推特表示，已经有确凿证据表明三星Galaxy Note 20 Ultra的主摄为108MP。

除了一亿像素主摄之外，Galaxy Note 20 Ultra还有可能会使用潜望式长焦镜头及超广角镜头。

有爆料称三星Galaxy Note 20 Ultra不支持100倍变焦，最高支持到50倍数字变焦。

核心配置上，Galaxy Note 20 Ultra搭载高通骁龙865旗舰平台，Exynos版本首发Exynos 992，最高配备16GB内存+512GB存储，电池容量为4500mAh，支持45W快充。

此外，Galaxy Note 20 Ultra采用了6.9英寸WQHD+ 120Hz LTPO显示屏，分辨率为3040×1440，这是业界第一款采用LTPO显示屏的旗舰手机。

最后是大家关心的发布时间，Galaxy Note 20 Ultra有望在8月初登场，爆料称发布时间是8月5日。受疫情影响，三星将在线上举行新品发布会。
