Das Keyboard推出新款4C机械键盘 售139美元

May 12, 2020

近年来，机械键盘已经成为了注重手感的文字工作者、以及游戏玩家等群体的首选。与薄膜键盘相比，其提供了触觉和听觉上的双重享受。今日，Das Keyboard 就宣布了刷新后的 4C 机械键盘产品。外形上，其取消了最右侧的数字键区，PBT 键帽和润滑后的 Cherry MX 茶轴可进一步提升平滑度和触觉体验。

作为一款紧凑型产品，Das Keyboard 认为 4C 很适合小型工作站和 SOHO 办公、以及注重便携性的专业人士选购。得益于持续的改进，用户可受益于未来几年的生产力提升。

Das Keyboard 为 4C 配备了独特的磁性可拆卸式脚垫（可兼作直尺使用），左侧有两个 USB 2.0 扩展坞，线缆长度为 6.5 英尺（约 2 米）。

在用户看不到的大键上，还添加了有助于减少噪音和增加平顺性的额外润滑，以及支持全键无冲（N-key Rollover）。

目前新款 4C 机械键盘已可通过 Das Keyboard 官网选购，价格为 139 美元（约 986 RMB）。
