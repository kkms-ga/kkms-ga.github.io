LibreOffice 6.4.6稳定版现已开放全平台下载

August 14, 2020

在发布 7.0 大版本更新的同时，文档基金会也提供了面向所有平台的 LibreOffice 6.4.6 版本，包括 Windows、Linux、以及 macOS 。作为 LibreOffice 6.4 的第六个修订版本，其旨在进一步完善该办公套件的体验、以及修复用户当前遇到的一些问题。本月早些时候发布的 7.0 大版本，对所有内置工具都引入了大量的改进。但是对于那些追求稳定体验的用户来说，目前还是 6.4.6 版本更合适。

下载地址：

● LibreOffice 7.0.0 – Windows 10 32-bit / 64-bit | macOS 64-bit | Linux 64-bit（deb / rpm）

● LibreOffice 6.4.6 – Windows 10 32-bit / 64-bit | macOS 64-bit | Linux 64-bit（deb / rpm）

官方表示：LibreOffice 6.4.6 已针对各种环境进行了优化，却包含了数月来的 bug 修复，即使是较保守的用户也可以放心体验，建议 6.3.6 及更早版本的老用户升级使用。
