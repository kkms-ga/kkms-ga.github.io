Chrome 85 Canary改进了标签页分组功能

May 18, 2020

谷歌本周为 Chrome 83 引入了标签页分组功能，但随着已打开的标签页数量的增长，用户也难以分辨和操作。好消息是，在 Chrome 85 Canary 中，谷歌已经带来了一项改进，允许通过点击标签或图标来折叠和进一步扩展分组。值得玩味的是，在谷歌对标签页分组功能产生浓厚兴趣的同时，Mozilla Firefox 这边早已不再流行。

最初引入这项功能时，谷歌允许 Chrome 用户手动打开该“标记”（flags），而后 Chromium 开发团队添加了可还原标签分组的功能。

如需创建分组，请在指定的标签页上右键点击，然后选择添加。用户可将同类标签页置于一个分组，然后为其分配一种方便区分的颜色。

通过标签页的情境菜单，还可选择移动到不同的分组、取消分组、或者关闭同一组的所有标签页。感兴趣的朋友可在最新的 Chrome Canary 85.0.4148.0（或更高版本）上尝试：

（1）通过地址栏访问 chrome://flags/#tab-groups-collapse；

（2）选择启用标签页分组，并重启 Chrome 浏览器；

（3）打开多个标签页，指定分配的标签页组别。

默认情况下，新版 Chrome 浏览器已经启用标签页分组功能，因此并不需要用户手动操作。如果意外关闭，还请手动开启这一“标记”。
