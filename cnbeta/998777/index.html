雷克沙推出SL200系列USB-C移动固态硬盘

July 03, 2020

随着固态存储技术的发展和成本的不断降低，轻便高速易携带的大容量固态移动硬盘开始更多地映入消费者的眼帘，比如 Sabrent 和三星近期展示的 8TB 版本。与传统机械式的移动硬盘相比，固态硬盘还具有节能、抗冲击等优点。不过本文要为大家介绍的，则是来自雷克沙的 SL200 系列 USB-C 移动固态硬盘。为了照顾只有 Type-A 接口的用户，包装内还随附了一条转接线。

SL200 系列移动固态硬盘采用了灰色的时尚外形，提供了 512GB / 1TB / 2TB 的容量，顺序读写可达 550MB/s 和 400MB/s 。此外具有出色的安全性，支持 256 位的 AES 加密功能。

如果想要在旅途中保护个人信息、财务记录或其它敏感文件，还可轻松创建一个受密码保护的文件保险箱，其中删除的文件将被安全擦除，并且无法恢复。

雷克沙为 SL200 系列 USB-C 移动固态硬盘选用了 USB 3.1 Type-C 接口，支持 macOS X 10.6 +、Windows 7 / 8 / 10、Android 4.4 以上版本的操作系统。

工作温度 0 ~ 70 °C，存放温度 -40 ~ +85 ℃ 。大小为 86×60×9.5 mm，重 40.6 g（不含数据线）。

Lexar SL200 系列移动固态已经上市，512GB / 1TB 版本售价分别为 89.99 / 159.99 美元（约 635 / 1130 RMB）。

该公司还计划推出 2TB 版本的 SL200 系列移动固态硬盘，但尚未公布价格和上市日期等信息。
