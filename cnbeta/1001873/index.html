谷歌或在今秋推出硬件配置差不多的Pixel 4a（5G）和Pixel 5机型

July 11, 2020

几个月来，外界一直在追寻谷歌 2020 系列 Pixel 智能机的相关代号。现在，已经有眼尖的人们在谷歌官方 App 中发现了三款产品代号，其中包括了传说中的“Pixel 5”和“Pixel 4a（5G）”。与此同时，外媒猜测即将于今年秋季发布的 Pixel 5 智能机，可能不具有 Pixel 设备粉丝们期待的“旗舰”规格。

传说中的 Pixel 5 / Pixel 5 XL 的代号或为“redfin”和“bramble”，辅以中端的高通骁龙 765G SoC 。

但在深入挖掘最新 Beta 版的 Google 搜索 App 的代码之后，Dylan Roussel 发现其中未提及 Pixel 5 XL，反而曝光了 2020 Pixel 智能机产品线中的另一个型号，即传说中的 Pixel 4a（5G）。

Pixel 4a（5G）机型的定位较 Pixel 5 更低，同时它可能抢走原以为是 Pixel 5 XL 的“bramble”这个代号。

然而到目前为止的线索表明，使用骁龙 765G 芯片组也会带来一些麻烦。虽然该 SoC 支持 5G 网络，但这也意味着 Pixel 4a（5G）和 Pixel 5 可能会在相同的主要硬件上运行。

此前每年，谷歌都会推出较小号的 Pixel 和较大号的 XL 机型。至于今年将如何在将如何在第三款 Pixel 4a（5G）和 Pixel 5 设备上作出区分，目前暂不得而知。

外媒 9to5Google 猜测，该公司或在 Pixel 4a（5G）机型身上使用塑料等更加低成本的材料，同时为 Pixel 5 选用铝金属+玻璃等具有更高级质感的外形设计。
