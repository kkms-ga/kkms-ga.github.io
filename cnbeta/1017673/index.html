Zorin OS：新线索表明有更多Windows用户拥抱Linux

August 19, 2020

根据最新分享的市场数据，可知 Linux 开源平台已于 2020 年收获了更多的份额，意味着许多人将之视作微软 Windows 平台的一个有价值替代。虽然想要确切统计有多少用户正从 Windows 切换到 Linux 有点难，但流行的 Linux 发行版提供商 Zorin OS 还是分享了一些值得关注的统计数据。

资料图（来自：Microsoft）

Zorin OS 于近日推出了最新的 15.2 发行版，可知许多用户都是在原先运行 Windows 和 macOS 的设备上下载和使用的。

自 9 个月前的 Zorin OS 15 大版本更新以来，其已迎来超过 90 万的下载量。这是 Zorin OS 有史以来提供的最强大、先进的发行版本。

值得一提的是，在每 3 个下载中、就有 2 个来自 Windows 和 macOS，表明最终用户对 Zorin OS 很是认可。

如果没有社区的帮助，Zorin OS 显然无法独自发展和广泛传播，感谢大家使这一 Linux 发行版变得如此强大。

（来自：Zorin OS）

从下载量数据来看，Zorin OS 显然算不上是最受欢迎的 Linux 发行版，更别提 Ubuntu、Linux Mint、乃至 Windows 拥有的庞大用户基数。

虽然目前尚不清楚有多少 Windows 用户正在迁移至 Linux，但事情显然正在发生。至于未来的发展，仍有待时间去检验。
