[图]微软现通过 Windows Update 邀请更多人使用新版 Edge 浏览器

June 04, 2020

今年 1 月，微软正式发布了基于 Chromium 的 Microsoft Edge 浏览器。目前用户可以手动下载和安装该浏览器，但伴随着 Windows 10 功能更新的发布微软显然希望通过 Windows Update 方式邀请更多人使用。

2020天猫6·18超级红包在此领取 6月1日追加40亿元消费券

微软中国官方商城 - Windows

近日，微软密集发布了KB4541301, KB4541302 和 KB4559309 三个累积更新，通知用户和管理员在安装该更新之后将替代经典版 Edge 浏览器，将会默认启用基于 Chromium 的新款 Edge 浏览器。

本批次累积更新目前已经在 Windows 10 Version 1803 以及更高版本中上线。通过 Windows Update 更新方式和通过手动安装方式更新有一点不同。虽然两种方式就会替代经典版 Edge 浏览器，但手动安装方式可能会移除访问经典版 Edge 的权限。

更新：

KB4541301 – 适用于Windows 10 version 1803 和 1809

KB4541302 - 适用于 Windows 10 version 1903 和 1909

KB4559309 – 适用于所有从Version 1803 到 2004 的 Windows 10 版本

KB4541301 和 KB4541302 两个累积更新有更新需求。其中 KB4541301 （适用于Version 1803和1809），微软推荐先安装 2019 年 11 月 12 日发布的KB4525237 和KB4523205。而对于KB4541302，微软推荐安装 2019 年 10 月 8 日发布的 KB4517389。

这些更新仅通过 Windows Update 方式引入新款 Edge 浏览器，并未出现在 Microsoft Update Catalog 网站上。目前尚不清楚微软后续是否会放出这些更新的手动下载地址。
