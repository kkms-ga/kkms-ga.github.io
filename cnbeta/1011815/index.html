[视频]无需昂贵外设也能畅玩Project xCloud 微软演示触控游戏方式

August 05, 2020

在宣布云游戏服务 Project xCloud 发布日期的同时，微软还公布了多款官方授权的 Xbox 移动游戏外设，从而在用户享受云游戏服务的时候获得更佳的游戏体验。除了这些外设之外，微软还在软件方面进行了优化，让玩家不需要定制的手柄就能在移动设备上玩 Xbox 游戏。

对于支持 Project xCloud 的游戏，开发者可以映射游戏的触控按钮到虚拟按键上。在 Microsoft Game Stack 的直播中，The Coalition 的高级游戏制作人 Jarret Bradley 展示了如何映射《Gears 5》游戏的触控按钮，从而让玩家在没有外设的情况下也能畅玩游戏。

在演示过程中，Bradley 解释道：“当我们第一次讨论这个问题的时候，我们想的有点复杂，因为我们不确定我们需要做什么。当我们开始深入研究时，我们意识到我们最好的做法是将其作为一种不同的输入设备，就像我们已经支持键盘和鼠标、控制器一样，我们有定制的控制和重新映射。”
