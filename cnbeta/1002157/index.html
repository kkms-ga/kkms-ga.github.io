开发人员可以使用Windows设备更轻松地开发苹果游戏

July 12, 2020

由于苹果发布了一套新的Metal开发工具，游戏开发人员现在可以使用Windows设备更轻松地为苹果设备开发游戏。苹果今年在 WWDC 上悄悄发布了一套 Windows 10 的 Metal 开发工具，使 Windows 上的 Metal Shading Language（MSL）编译成针对苹果平台的 Metal Library Objects。

下载地址：

Metal Developer Tools for Windows

这意味着游戏制作团队现在可以使用 PC 编译在 Windows 上针对 macOS 和其他苹果操作系统开发游戏。

许多游戏开发工作室已经建立了使用 Windows 基础结构的游戏或图形生产管道。创建过程中的关键最终步骤之一是编译图形和计算着色器。

由于苹果正在将 Mac 过渡到自家的定制芯片，因此将游戏引入 Metal 的开发人员将能够在 iOS，iPadOS，tvOS 和 macOS 上运行它们。苹果的 API 会自动将所有输入转换为任何可用的输入方法，例如触摸，控制器或键盘和鼠标。现在，面向苹果平台的开发人员可以同时为 iPad，iPhone，Mac 和 Apple TV 进行开发。这可能会使苹果平台对游戏开发商更具吸引力，因为他们将获得比仅在 macOS 或 iOS 上更大的市场份额。

尽管不可能完全在 Windows 上为苹果设备开发游戏，因为仍需要 Mac 来对软件进行签名和测试，但面向 Windows 游戏开发人员工具的可用性仍标志着在苹果设备上进行游戏的重要一步。

鉴于这些新的开发人员工具允许 Windows 游戏被编译为苹果平台的 Metal，而不是完全重建，因此开发人员现在应该可以更轻松地将本机 PC 游戏和 3A 游戏移植到 Mac 上。
