Chrome 将允许用户下载编辑后的 PDF 副本

June 06, 2020

不少人喜欢使用 Chrome 浏览器来查看 PDF 文档或对其进行编辑操作，但这些编辑后的内容迄今为止无法通过 Chrome 保存，人们只能下载到未经修改的原始 PDF 版本。当然目前也有一个方法，即借用 Chrome 的打印功能，将编辑后的版本另存为 PDF。不过，这算不上提供了最佳用户体验，Google 已着手解决此事。

6·18活动已全面开启 大促活动入口汇总：

阿里云6·18上云年中大促 点击领取最高12000元红包

2020天猫6·18超级红包在此领取 6月1日追加40亿元消费券

京东6·18十七周年庆大促主会场入口 - 最高可领618元红包

Chrome Story 近日发现了一项关于 UI 更改的代码更改请求，该更改将允许用户选择下载 PDF 文件的编辑版本或原始版本。

PDF 查看器：添加 UI，用以下载编辑后的 PDF

添加操作菜单，提供下载原始或已编辑的 PDF 的选项

仅在进行编辑并启用 “SaveEditedPDFForm” 功能时才显示菜单

在关于这项新功能的 bug 报告页面中，开发者提供了一张截图：

至于该 bug 报告显示的请求内容，是说该功能应在 PDF 文本可编辑且已编辑后，才显示下载编辑后版本的选项；若文档本身不可编辑，该选项不会出现。

这项新的功能大概会在即将发布的 Google Chrome Canary 版本中开始出现，稳定版还要再等上几个月。
