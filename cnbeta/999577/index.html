三星德国官网泄露Galaxy Tab S7系列：骁龙865 Plus+10090mAh

July 06, 2020

7月6日消息，据SamMobile报道，三星德国官网偷跑Galaxy Tab S7系列。三星德国官网显示，Galaxy Tab S7系列包含Galaxy Tab S7和Galaxy Tab S7+两款。据悉，Galaxy Tab S7采用11英寸Super AMOLED显示屏，Galaxy Tab S7+采用12.4英寸Super AMOLED显示屏，二者都是QHD+分辨率，刷新率达到了120Hz，搭载高通骁龙865 Plus旗舰平台。

此外，Galaxy Tab S7系列预装Android 10系统（One UI 2.5版本），后置双摄像头，支持S Pen。电池容量方面，Galaxy Tab S7为7760mAh，Galaxy Tab S7+为10090mAh，其中Galaxy Tab S7+支持5G连接。

报道称Galaxy Tab S7系列与iPad Pro直接竞争，这也是安卓阵营性能最强悍的平板电脑。

目前尚不确定Galaxy Tab S7系列发布时间，它有可能会与Galaxy Note 20系列同台亮相。消息称三星将于8月5日举行新品发布会，届时Galaxy Note 20系列将正式发布，不排除Galaxy Tab S7系列在此亮相的可能，我们拭目以待。
