押注疫情下网络攻击激增 日本一科技公司股价飙升逾670%

June 08, 2020

日本一家科技初创企业自上市以来股价已飙升逾670%。该公司首席执行官正确地押注新冠肺炎将引发大规模网络攻击，并加剧企业对未上市公司的偏见。此前该公司在3月中旬市场最动荡时期推进了上市计划。网络安全云（Cyber Security Cloud）股价的暴涨，使这家拥有45名员工的公司的市值达到了约8亿美元。

该公司于3月26日在东京证券交易所（Tokyo Stock Exchange）第一市场上市。此前几天，日经225指数（Nikkei 225）已累计下跌20%。

散户投资者对股票交易的兴趣意外回升，推动了股市上涨。自疫情爆发以来，散户投资者已开设了逾50万个在线经纪账户，并推动Mothers创业板指数接近12个月高点。

30岁的网络安全云（Cyber Security Cloud）创始人Hikaru Ono承认，该公司首次公开发行的时机并不吉利。这家利用人工智能（AI）增强客户网络防御能力的公司于今年2月底获准上市，而几天后，日本首相安倍晋三（Shinzo Abe）将下令关闭日本所有学校。

Hikaru Ono表示，从那时起，远程办公、视频会议以及其他突然改变正常工作生活的方式将在整个日本企业界出现，这一点变得很明显。他判断，这在很大程度上将严重依赖存储在云中的数据和应用程序——Hikaru Ono的公司专门从事这一领域，但日本企业在这一领域的起步比欧美同行晚了几年。

Hikaru Ono表示：“由此可以明显看出，如果在办公室实际工作的人减少，企业的网络安全就会变得越来越薄弱。”他补充称，日本本土网络安全公司相对较少，即使在形势最好的时候，对许多日本首席执行官来说，网络防御的重要性也低得危险。

对于包括日本在内的许多市场来说，CSC计划上市的前几天是自2008年全球金融危机以来最动荡的时期。有警告称，全球经济被迫关闭将使世界陷入经济萧条，这给股价的大幅下跌蒙上了沉重的阴影。

尽管如此，Hikaru Ono下定决心推进IPO，不仅是为了快速扩张筹集资金，而且是因为担心，无论日本企业多么迫切地需要CSC的服务，它们都会不愿聘用一家非上市公司从事如此敏感的工作。

事实证明，Hikaru Ono的预测是准确的：自3月份以来，日本和其它地区的网络安全专家警告称，网络攻击急剧增加，攻击对象往往是那些突然发现自己在不安全的家庭WiFi上使用公司笔记本电脑的员工。今年5月，日本专家告诉当地媒体，已经出现了1000多个模仿政府机构、银行和其他公司的虚假网站。

今年4月，英国国家网络安全中心（National Cyber Security Centre）与美国国土安全部（Department of Homeland Security）发布了一份联合咨询报告，警告称，国家或国家资助的组织正在“积极针对”参与应对新冠肺炎的国内和国际组织，包括医疗机构、制药公司、学术界和地方政府。
