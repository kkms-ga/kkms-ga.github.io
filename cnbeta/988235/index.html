[图]热门非官方Telegram客户端Unigram升至4.0版本

June 08, 2020

热门非官方 Telegram 客户端 Unigram，近日发布了适用于 Windows 的 4.0 版本更新。本次更新主要包括将聊天整合到Chat Folders中，创建灵活的自定义文件夹，在每个文件夹固定无限量个聊天， 右键单击聊天列表中的聊天内容，即可将其添加到文件夹中。
