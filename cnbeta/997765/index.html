三星Galaxy A01 Core入门新机后壳谍照曝光

July 01, 2020

Galaxy A01 Core 有望成为三星下一款 Android Go 智能机，部分规格信息已于上周被谷歌 Play 控制台曝光。现在，91Mobile 又从分销渠道商那里拿到了几张谍照，可知 Galaxy A01 Core 的背面和侧面采用了怎样的设计。比如位于背部左上角的后置单摄，以及底部的耳机和 microUSB 插孔。

第一张图片展示了 Galaxy A01 Core 的背部设计，可知左上角为后置单摄 + LED 闪光灯。

作为一款入门新机，其背部似乎还有明显的交叉纹理图案。

塑料框底部则是 microUSB 和 3.5 mm 耳机插孔，以及扬声器格栅。

机身右侧为电源和音量按键，虽然正面谍照尚未公布，但不难猜测该机的屏幕会采用水滴形刘海缺口。

根据谷歌 Play 控制台的早前爆料，Galaxy A01 Core 将采用入门级的联发科 MT6739WW 芯片组（集成 PowerVR GE 8100 核显）、辅以 1GB RAM 运存，屏幕分辨率为 HD+ 。

此外作为一款 Android Go 设备，其有望预装精简版的 Google Go 和三星原厂 App 。爆料称该机已处于生产阶段，如果一切顺利的话，想必还是可以赶在 7 月底或 8 月初发布的。
