三星Galaxy Note 20偷跑：120Hz屏 支持自适应刷新率

May 12, 2020

今年不少旗舰使用了高刷新率显示屏，Galaxy S20系列最高支持120Hz刷新率，但是它不支持2K+120Hz同时开启，外媒猜测原因之一可能是功耗高。5月12日消息，据XDA报道，三星Galaxy Note 20系列将会解决这一问题，它采用的是120Hz刷新率显示屏（预计是挖孔屏方案）。

SAMSUNG - 三星旗舰店

（资料图）

报道指出，三星Galaxy Note 20和Galaxy Note 20+将采用LTPO技术。LTPO即Low Temperature Polycrystalline Oxide（低温多晶氧化物）的简称，LTPO=LTPS+Oxide，同一个像素中集成了LTPS和Oxide两种TFT器件，Oxide是底栅结构，LTPS是顶栅结构。

这种新工艺结合了LTPS TFT工艺驱动能力强和Oxde TFT工艺漏电小功耗低的优点，它被苹果应用在Apple Watch Series 5型号中，也正是有了这项技术，Series 5做到了屏幕常显而且与Series 4一样具有长达18小时的电池续航。

不仅如此，Galaxy Note 20系列还支持自适应刷新率，外媒称通过自适应刷新率模式，三星Galaxy Note 20系列在不同场景下智能切换刷新率，从而改善功耗。

核心配置上，Galaxy Note 20采用6.42英寸显示屏，分辨率为2345×1084，Galaxy Note 20+采用6.87英寸显示屏，分辨率为3096×1444，搭载高通骁龙865旗舰平台。

另外，Galaxy Note 20系列不会有Ultra型号。
