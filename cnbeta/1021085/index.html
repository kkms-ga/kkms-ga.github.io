[图]Chrome OS Canary 86现已经启用圆形图标设计

August 27, 2020

去年，谷歌承诺 Chrome OS 中的应用图标将完全采用圆形设计。而在近日发布的 Chrome OS Canary 86 更新中，谷歌兑现了这个承诺。谷歌目前已经完成了 Chrome OS 所有应用图标的调整，包括 Stadia 和 Gmail 在内的应用背景中都有白色的圆圈。其他网络应用（如Photopea）则通过使用不同的图标来适应圆形设计。

总而言之，将应用图标做成圆形是一个非常微妙的变化，但这也说明了Android和Chrome OS之间设计的一致性。现在，Chrome OS和谷歌自家Pixel手机的默认主题之间的应用图标看起来几乎完全相同。随着时间的推移，或许Chrome OS可以引入一些Pixel手机上提供的其他图标形状，但目前来看，圆形很适合Chrome OS的整体设计。
