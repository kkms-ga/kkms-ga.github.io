为邮件打上蓝标：Gmail即将上线BIMI品牌认证Logo

July 22, 2020

谷歌本周透露，Gmail 将很快迎来“品牌认证 Logo”，以提升客户对该平台的信任度。在“生产力与协作周”活动期间，该公司在 Cloud Next ‘20：OnAir 环节宣布了这一公告。可知该系统将在邮件左上角的标准头像旁边显示一个“品牌徽标”。通过 Web 访问 Gmail 的用户，可在查看经过认证的官方发来的邮件时，于左上角看到此图标（比如 CNN）。

（来自：Google Blog）

谷歌将这套新系统称作“品牌消息指示器”（简称 BIMI），如果使用的是 Gmail 的 iOS 或 Android 客户端，亦可在收件箱的新电子邮件列表中看到它。

通过结合 BIMI 规范和基于域的消息身份验证、报告和一致性（DMARC）方案，谷歌希望此举能够更有效地阻止普通用户上了恶意电子邮件的当，继而终结网络钓鱼时代。

谷歌希望在 2020 年底前（未来几个月）完成 BIMI 系统的部署，并在接下来的几周内与有限数量的发件人、以及两个证书颁发机构（Datacard 和 DigiCert）进行试点合作。

在正式推出时，BIMI 与 DMARC 方案将默认启用。若能如预期般工作，那大家的体验将变得更加轻松和安全，因为谷歌会承担所有繁重的幕后工作。
