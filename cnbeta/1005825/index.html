[图]200美元的Linux笔记本PineBook Pro下月发货 并接受新订单

July 21, 2020

目前市场上可供用户挑选的 Linux 笔记本越来越多了，如果你想要购买一款价格亲民、隐私保护良好的 Linux 笔记本，那么 PINE64 去年推出的 PineBook Pro 是非常值得入手的。在宣布下月面向预订用户发货同时，PINE64 开始接受 2020 年的新订单。

在产品页上写道：

少量（1-3个）的残留像素（Stuck）或者坏点（Dead）是 LCD 屏幕的通病，这些都是正常的，不应该视为产品缺陷。在履行购买的时候，请记住我们是以为 PINE64、Linux 和 BSD 社区提供服务的心态提供如此低价格的 Pinebook Pro。我们没有从销售这些产品中获利。如果你不认可例如坏点等这些瑕疵，请提交 PayPal 退款并不要购买 Pinebook Pro。

据悉，Pinebook Pro 采用了中档的 Chromebook 配置，基于瑞芯微 RK3300 SoC 平台打造，CPU 部分为 2×Cortex A72 @ 1.8GHz + Cortex A53 @ 1.4GHz，辅以 Mali GPU 。

屏幕方面，该机采用了 14 英寸的全高清面板。存储方面，Pinebook Pro 提供了 4GB LPDDR4 RAM + 64GB eMMC（支持扩展或 microSD 启动）。接口方面，其提供了 USB-C（支持数据、充电和 4K / 60Hz 视频输出），USB 2.0 + USB 3.0，蓝牙 5.0 + 802.11ac Wi-Fi 。

值得一提的是，Pinebook Pro 在键盘上集成了控制摄像头、麦克风、以及无线网络连接的隐私开关，可选预装 Linux Debian、Chromium OS 或 Android 9 移动操作系统。
