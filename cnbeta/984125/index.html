Windows Server版本2004今日随Windows 10一同更新

May 28, 2020

今天早些时候，微软发布了Windows 10 2020年5月更新，也就是2004版。而它的服务器版本，也就是Windows Server 2004版本今天其实也紧随其后发布了，参加了半年度频道(SAC)的用户已经可以第一时间使用该版本。

访问官方发布页面：

https://techcommunity.microsoft.com/t5/containers/Windows-server-version-2004-now-available/ba-p/1419194

您可以使用下面的方法从MCR中获取Windows Server 2004版本的映像：

docker pull mcr.microsoft.com/windows/servercore:2004

docker pull mcr.microsoft.com/windows/nanoserver:2004

docker pull mcr.microsoft.com/windows:2004

本次升级操作系统的下载大小和操作系统本身安装后的尺寸都较小，下载量为1.865GB，磁盘占用为4.05GB。与1909版的2.257GB和4.97GB，以及1903版的2.311GB和5.1GB节约了不少空间。所做的改变之一是，Server Core不再对.NET Framework进行性能优化，因为现在应用是在.NET Framework运行时映像中完成的。微软正在优化的另一部分是Server Core容器，能够在不占用太多存储空间或带宽的情况下，对传统应用进行现代化改造。

事实上，对于Windows Server版本而言，这次更新似乎都是关于优化的。除了带来更小的操作系统外，还拥有更快的启动速度、优化性能等等。
