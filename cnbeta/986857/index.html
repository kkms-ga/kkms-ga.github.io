本田因燃油泵故障缺陷召回140万辆汽车

June 04, 2020

外媒报道称，本田（Honda）刚刚因燃油泵故障缺陷而在多个市场区域召回了 140 万辆汽车。若燃油泵发生故障，汽车可能遭遇失速或无法启动等问题。车主有多种途径来确认自己的车子是否在本次召回范围内，庆幸的是，目前尚未收到由此引发的任何事故或伤害报告。

在总计 140 万辆被召回的车辆中，美国地区占据了约 13.6 万辆，车主们可在 HSTSA 官网上查阅编号为 20V314000 的召回通告。

据悉，本次召回涉及许多新车型，包括 2018 / 2019 款的雅阁、讴歌 NSX、两厢思域 / Type-R / Type HR-V 。

另有 2019 款讴歌 RDX / RLX / RLX 运动混动车型（Sports Hybrid）、2019 / 2020 款飞度、以及 2020 款英赛特。

车主们也可以在本田官网上输入 VIN 编码来查阅是否在召回之列（传送门），或者等待预计 7 月底开始发出的通知邮件，受影响的车主们可联系经销商到店免费更换低压燃油泵。
