英伟达打算收购ARM 华为怎么又要莫名躺枪了？

August 22, 2020

最近有个传闻发酵的还挺厉害，说是核弹厂英伟达打算把芯片架构公司 ARM 给收购了。表面上来起来是一场简单的收购，但这件事儿要是真成了吧，它不仅会将整个芯片行业重新洗牌，还很有可能对国产芯片起到毁灭性打击。要解释这个问题，还得从 ARM 这家芯片公司说起。

和英特尔、AMD 这样的芯片公司不同，ARM 本质上是一家芯片架构公司，只提供芯片的架构图纸和方案，而不提供设计、生产，属于纯粹的上游公司。

ARM 架构的芯片目前运用的最多的地方，就是我们每天都在用的手机，不管你是苹果、高通、三星甚至华为，只要你做手机芯片，就一定离不了 ARM 架构。

如果把芯片比作房子的话，ARM 就像房地产开发商，他们家既

卖地

（ 架构授权 ），也卖

毛坯房

（ 内核授权 ）和

精装修

（ 使用授权 ）。

（ Yahoo ）

像苹果、高通、华为这样有芯片设计能力的厂商，一般会直接买地盖楼，想怎么发挥怎么发挥，ARM 只从每个芯片里抽取一定比例的架构授权费。

而像德州仪器、博通、富士通这样的半导体加工厂商，会选择毛坯房，可以对房子里面进行装修（ 架构外围设计 ），但不能违建（ 修改架构内核 ）。

最低一级的授权类似于精装修，基本可以拎包入住，不用做任何修改就能将芯片封装、生产出来，对于没有芯片设计能力的企业来说，是比较划算的一个选择。

（ 长芯半导体有限公司 ）

ARM 架构所主导的移动端芯片，从来都是一个巨大的香饽饽，未来的手机也好，AI 智能设备也好，都必须围绕着芯片性能来展开，而 ARM 架构所做出的芯片，目前来说无疑是性能最强劲的。

一旦英伟达吃掉了 ARM，所有的芯片制造、生产厂商都得看英伟达的脸色来做事，包括 ARM 自己，也会变得相当被动。

ARM 的联合创始人Hermann Hauser 前不久在接受 BBC 采访时曾表示，如果 ARM 被英伟达收购，将是一场灾难，将会损害 ARM 的中立性以及满足多家不同公司和供应商需求的能力。

（ Evening Standard ）

英伟达一家搞显卡的，怎么就想吞掉一家和自己八竿子打不着的移动端芯片的设计公司呢？是不是有点专业不对口啊？

其实并非如此，英伟达其实一直都在做移动端的芯片业务。

2014 年的时候，小米 3 就出过一款搭载了英伟达 Tegra 4 处理器版本的机器，奈何那时候除了 GPU 部分，不管是功耗、发热还是基带信号（ 外挂 Icera i500 基带 ）都被同时期的高通骁龙 800 吊打……

( 小米官网 )

毕竟自家是做显卡的，CPU 和信号这块儿确实也不擅长，所以后来他们干脆就不做手机芯片了，转向了游戏机芯片。

像是我们熟悉的 Nintendo Switch 游戏机，使用的就是英伟达定制过的

Tegra X1

芯片，图形性能自不必说，同时也满足了便携性的需求。

（ Linus Tech Tips ）

剩下的就是英伟达自家出的 Shield 游戏主机上有用到 Tegra 系列的芯片，除此之外乏善可陈。

（ Wccftech ）

所以，英伟达但凡想要在移动芯片端有所突破，就必须依赖 ARM 的资源，而相较于买授权，英伟达更倾向于把 ARM 买过来占为己有。

ARM 的身份就像是一家老牌的网络游戏公司，定时更新游戏版本，谁是版本的亲儿子，谁会大削，他说了算。

英伟达看上去则是一个不喜欢遵守游戏规则的进阶土豪玩家，氪再多金都不能得到自己想要的东西，那就干脆把这家游戏公司买下来，想怎么更新我自己说了算。

（ @ZORE _ Z _ Z ）

至于其他玩家（ 苹果、高通、三星等厂商 ）想要不吃屎，就必须乖乖交保护费，才能舒舒服服玩下去。

举例来说，如果英伟达想要高通的基带技术，就可以直接拿 ARM 架构来做要挟，进行技术置换，高通想要继续造手机芯片，就只能答应下来。

资本游戏从来都是不讲道理的，只要有一只鸡能下金蛋，就会有人买下整个养鸡场，杀鸡取卵。

而在这场芯片博弈战中，受美国技术禁令的影响，

最惨的应该要属中国芯片厂商了，华为首当其冲。

今年的 4 月份，英伟达以 69 亿美元收购了以色列的服务器供应商 Mellanox，为它的 AI 人工智能提供数据服务。

（ GOLOBES ）

Mellanox 最大的优势，正是它的芯片制造，英伟达的这一步，也被看作是进军芯片产业链的第一步。

以色列作为对中国出口芯片的第二大国，因为被排除在《 瓦森纳协定 》之外，（ 《 瓦森纳协定 》是一个由美国牵头制定的技术出口管制协定 ），芯片出口对中国一直都是开绿灯的。

( Strategic Security Analysis )

可英伟达本身是地地道道的美国企业， Mellanox 在注入了美国资本之后，难保不会被美国政府拿来做文章。

这个道理放到 ARM 上是一样的，作为一家英国公司， ARM 如果顺利被英伟达收购，同样也会因为美国的禁令，停止给中国企业 ARM 架构的授权。

尽管华为早已经买断了 ARM 的 V8 架构，通过魔改设计出了性能强劲的麒麟芯片，但往后 ARM 的新架构华为如果没办法获得，那就不光是芯片代工被卡脖子了，恐怕芯片的设计研发都会碰到大问题。

当然，华为也可以转到其他的架构上继续研发，因为 ARM 架构本身属于 RISC （ 精简指令集 ）的一种，类似的架构还有像 RISC-V、MIPS 等等。

( RISC-V )

比如之前阿里就发布了一款基于 RISC-V 的处理器——玄铁 910，但这块芯片主要应用在物联网、5G 以及人工智能领域。

( 3CEMS )

华为想要用 RISC-V 打破 ARM 在移动设备芯片上的绝对地位，几乎需要全部重头开始设计，谈何容易？

总而言之，不管英伟达这笔交易会不会成功，国产芯片往后的每一步，都只会越来越辛苦，芯片制造上下游的核心技术都掌握在别人手里，人家不卖，一点办法都没有。

自力更生，才能丰衣足食啊……

而对于英伟达来说，收购 ARM 明显是有野心的，或许是看到老冤家农企 AMD 如今 7nm 芯片做的起飞，一举把昔日的牙膏厂英特尔踩在脚下，他们也想跃跃欲试，想干翻高通吧……

( Wccftech )

反过来说，手机 Soc 要是能在图形处理方面补强的话，这活儿目前来看应该也就英伟达能干了。
