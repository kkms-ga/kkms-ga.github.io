Caviar推出四款限量版三星Galaxy Note 20 Ultra定制机型

August 24, 2020

三星在 8 月 5 日的 Unpacked 2020 活动期间发布了最新的 Galaxy Note 20 旗舰智能机产品线，而俄罗斯高端定制公司 Caviar 也迅速作出了跟进。据悉，Caviar 最新发布了四款限量版的三星 Galaxy Note 20 Ultra 定制机型。特点是覆有真大理石和玫瑰金材质的奢华外观，并且都属于该公司旗下的“永恒”（Eternity）系列产品线。

（来自：LetsGoDigital）

青铜版本成为了这四款 Galaxy Note 20 Ultra 限量定制机的入门选项，背部饰有大理石和玫瑰金等奢华材料。

该版本具有 256GB ROM，限量生产 99 分，建议零售价为 5830 美元（约 40298 RMB）。

（图 via LetsGoDigital）

第二款是 Galaxy Note 20 Ultra 的帕特农神庙版本，采用了白色 / 浅灰色的大理石材料，并且可以看到三个一系列玫瑰金色的圆柱。

该版本依然是 256GB ROM，限量 99 份，售价为 5750 美元（约 39745 RMB）。

（图 via LetsGoDigital）

第三款为故宫版，采用了带有绿色的大理石材质。具有 256GB ROM，限量 99 款，机身侧面带有唯一编号（辅以保真证书），售价为 6060 美元（约 41887 RMB）。

（图 via LetsGoDigital）

定价最高的是红褐色大理石版本，限量生产 99 份，售 6260 美元（约 43270 RMB），有意向的消费者均可享全球包邮和 Caviar 提供的一年质保。

（图 via LetsGoDigital）

配置方面，三星 Galaxy Note 20 Ultra 采用了 Exynos 990 芯片组，辅以 12GB RAM + 256 / 512GB ROM 。电池容量 4500 mAh，预装 Android 10 移动操作系统，以及延迟更低的新款 S-Pen 手写笔。
