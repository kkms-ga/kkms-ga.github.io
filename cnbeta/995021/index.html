想在Windows 7上体验新版Edge浏览器？还请了解这些区别

June 24, 2020

从 EdgeHTML 转投 Chromium 内核的新版 Microsoft Edge 浏览器已经吸引了许多人的目光，而且它不像经典版 Edge 浏览器那样局限在 Windows 10 平台上使用。现在，基于 Chromium 内核的 Microsoft Edge 浏览器亦可在 Windows 7 / 8 / 8.1 / 10、甚至 macOS 平台上使用。至于 Linux 版本，目前尚不清楚何时抵达。

今年早些时候，微软开放了 Windows 7 版 Microsoft Edge 浏览器的手动下载。不过最近，微软还开始通过 Windows Update 渠道进行投放。

对于不想从 Windows 7 升级到 Windows 10 的用户，Microsoft Edge 将支持其到 2021 年 7 月。感兴趣的朋友可手动检查并安装 KB4567409 更新。

需要指出的是，与 Windows 10 平台上的版本相比，Windows 7 平台的 Microsoft Edge 浏览器只针对消费者、而不面向企业客户。

微软解释称，该更新不适用于企业设备，而是 Windows 7 SP1 / Windows 8.1 或更高版本的操作系统（涵盖家庭 / 专业 / 旗舰 / 入门 / Core 版本）。

此外在 Active Directory 或 Azure Active Directory 域上运行的设备，也与本次自动更新无缘。

在安装了新版 Microsoft Edge 浏览器后，微软会在任务栏上固定一个图标、同时在桌面上提供一个快捷方式。

那些此前已经在 Windows 7 设备上手动安装了新版 Microsoft Edge 的用户，其快捷方式也会被替换和指向最近更新的版本。

需要指出的是，Windows 7 上的 Microsoft Edge 无法完全替代旧系统上默认自带的 Internet Explorer 浏览器。

如果设备上已经安装了 Google Chrome 或 Mozilla Firefox 等第三方浏览器，微软也不会对默认浏览器的设置做出更改。
