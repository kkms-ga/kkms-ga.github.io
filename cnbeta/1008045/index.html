[图]Android端微软远程桌面应用更新：全面支持Windows虚拟桌面

July 27, 2020

适用于 Android 平台的 Microsoft Remote Desktop 应用程序近日迎来重大更新，增加诸多新功能，包括全面支持 Windows 虚拟桌面、支持 Android TV 设备等等。

关于 Microsoft Remote Desktop 的介绍

可以使用 Microsoft 远程桌面客户端，通过几乎任何设备从几乎任意位置连接到远程电脑和你的工作资源。 你可以连接到你的工作电脑并访问你的所有应用、文件和网络资源，就像坐在办公桌前面一样。 通过使用 RD 客户端，即可在工作时使应用处于打开状态，然后在家里查看这些相同应用。

更新内容主要有：

● 实现了对 Windows 虚拟桌面的全面支持

● 基于和 iOS 和 macOS 客户端相同的底层 RDP 核心引擎来重写客户端

● 全新的 Connection Center 体验

● 全新的 Connection Progress UI

● 全新的会话 Connection Bar

● 添加对 Android TV 设备的支持

● 集成 Microsoft Authenticator，订阅 Windows 虚拟桌面源时启用条件访问。

● 启用了从 Remote Desktop 8 传输连接和设置的功能
