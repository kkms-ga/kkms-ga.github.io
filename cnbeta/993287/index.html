AMD新APU核显曝光：Tiger Lake Xe和MX350或迎来劲敌

June 19, 2020

@KOMACHI_ENSAKA 刚刚在 Twitter 上透露了即将到来的 AMD 新款入门级 APU 处理器，猜测它可能集成基于 RNDA2 架构的 Little Navi 核显。如果爆料靠谱，那这款传说中的 Van Gogh Lite GFX 1040 GPU，或对英特尔 Tiger Lake Xe 和英伟达 GeForce MX350 造成较大的竞争压力。

资料图（来自：AMD，via NotebookCheck）

据悉，爆料人是在 Linux 上的 AMD 驱动程序代码中发现“Van Gogh Lite”GFX 1040 核显的身影的。

外界普遍猜测 AMD 会在明年推出采用 Navi RDNA 核显的 APU，预计 GPU 性能将大幅提升。

不过 @KOMACHI_ENSAKA 认为，“Van Gogh Lite”指代的是 AMD 新 APU 家族中的入门级产品。

因为从 Linux 驱动代码来看，与标准款“Van Gogh”APU 集成的是 GFX 1033 核显，而“Van Gogh Lite”集成的是 GFX 1040 GPU 。

较早的泄露表明 GFX10 系列设备 ID 指的是 Navi 核显，因为 AMD Radeon RX 5700 XT 所采用的 Navi 10 GPU 代号就是 GFX 1010 。

不过另一方面，其它采用 Vega 核显的 APU（比如 Renoir 系列）所使用的 GPU 代号是 GFX 909，而 Ryzen 4000 系列 APU 则搭载了 RX Vega GPU 。

目前我们对 Van Gogh 的 GPU 规格知之甚少，而 Van Gogh Lite 的曝光度就更低了。

但是将上一代 APU 的设备 ID 拿出来一对比的话，两者之间的关系就不言自明了。比如较大的数字（GFX 1040，而不是 GFX 1033）可能与规模较小的 GPU 有关。

GFX 1020 或指第二代 RDNA2 Big Navi，而 GFX 1033（标准 Van Gogh GPU）则是 Little Navi —— 定位可能是 AMD Radeon RX 5300M 独显使用 Navi 14 芯片的继任者。

如果爆料靠谱，那英特尔即将面世的 Tiger Lake Xe 和英伟达 GeForce MX350 或面临强大的竞争对手。

至于 Van Gogh Lite 到底是不是尚未曝光的 Navi 移动 GPU，也只有等到 Van Gogh 家族（最早明年）亮相后才能知晓，除非 AMD 主动在未来几个月里分享更多细节。
