Wine 最新补丁：支持 64 位 POWER

August 19, 2020

André Hentschel 发出了最新的 22 个补丁，用于将 Wine 移植到 PPC64（64 位 POWER）。这些补丁来自于去年发送的早期 POWER 补丁的 cleaning、re-basing 和 splitting up。

Wine 本身不处理在其他体系结构上运行 Windows x86/x86_64程序的架构仿真，这项工作的一部分重点则在于让 Hangover 在其他体系结构上运行 Windows x86 程序。最新的 Hangover 代码主要适用于在 AArch64 上运行的 x86_64 Windows 程序，同时该团队也在努力将其扩展到 PowerPC64。

事实上，随着苹果将 macOS 转移到其 ARM64 内部芯片中，在未来几个月里，Hangover 将变得越来越重要。

André Hentschel 表示，“我们的目标是在 PowerPC64 上建立用于 Hangover 的 winelib 库，并且也有兴趣使用 winelib 进行交叉编译。”

最新的 Wine POWER 64 位补丁可以在 Wine 邮件列表中找到。
