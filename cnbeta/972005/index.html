Redmi Note 10？小米5G新机入网：支持33W快充

April 27, 2020

4月27日消息，型号为M2004J7AC、M2004J7BC的小米5G新机获得3C认证，它们配备的充电器型号分别为MDY-11-EM、MDY-11-EX，前者支持22.5W快充，后者支持33W快充。有报道称这是即将登场的Redmi Note 10系列新机，而不是传闻中的Redmi Note 9系列。

访问:

小米有品

报道指出，Redmi Note 9 Pro、Redmi Note 9 Pro Max已经在海外亮相，其型号分别为M2003J6A1I和M2003J6B1I，这意味上述入网的两款小米5G新机为全新系列机型。

联系到Redmi即将登场的千元新机命名为Redmi 10X而非Redmi 9X，因此这两款新机有可能会是Redmi Note 10系列，这也意味着国行版将会“跳过”Redmi Note 9系列。

在Redmi Note系列历史上，国行版也曾经“跳过”Redmi Note 6系列，直接推出Redmi Note 7系列，而Redmi Note 6、Redmi Note 6 Pro等机型在海外市场发售。

因此Redmi有可能会直接在中国市场发售Redmi Note 10系列，和上一代Redmi Note 8系列一样推出Redmi Note 10、Redmi Note 10 Pro两款。

之前小米集团中国区总裁、Redmi品牌总经理卢伟冰暗示，Redmi要用联发科天玑800处理器，这是一款支持SA、NSA的双模5G SOC。

有消息称Redmi Note 10搭载联发科天玑800，Redmi Note 10 Pro搭载联发科天玑800+，大核的主频速度相比天玑800提升到了2.6GHz。

值得注意的是，之前移动内部文件披露小米在6月份会有1599元的5G新机问世，所以应该是这次获得3C认证的Redmi Note 10，它有可能会成为价格最低的双模5G手机。

这也意味着5G手机市场竞争从中高端来到了千元市场，我们拭目以待。
