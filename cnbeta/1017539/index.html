[图]微软为Edge浏览器引入“安全DNS”功能

August 19, 2020

加密 DNS 能大大提升了互联网用户的隐私性，不仅可以让用户对网络流量的内容保密，而且还可以对目的地进行保密。该技术能够在常规 HTTPS 流量中隐藏 DNS 地址，现在微软计划将该功能引入到 Edge 浏览器中。

在 Canary 频道的最新 Edge 版本 Build 86.0.612.0 中，微软引入了名为“安全DNS”（Secure DNS）的功能，并提供了一系列备用的加密 DNS 提供商。
