[图]新版Edge将获“zero copy capture”，提升续航表现

August 26, 2020

在拥抱 Chromium 平台之后，微软逐渐将经典版 Edge 的诸多优秀特性迁移到新版 Edge 浏览器中，现在又一项优秀特性即将引入到新版 Edge 中。Media Foundation 是随 Windows Vista 引入的底层多媒体技术，微软已经确定该技术即将引入到新版 Edge 中。

基于 Media Foundation ，微软目前正在研究“零拷贝视频捕捉”（zero-copy video capture）功能，在 Windows 10 系统上利用 GPU 内存缓冲区减少过多的电力消耗。

在 WebRTC 视频会议场景中，基于 Chromium 的新版 Edge 耗电的主要因素是摄像头捕捉路径，而导致耗电过高的主要原因是复制和转换操作。在Windows 10上，NV12 是 YUV 4:2:0视频帧的首选格式。此外，在最近的硬件上，大多数相机驱动程序，帧最初是以GPU纹理（D3D11纹理）填充的。

目前在Chromium中的实现导致视频帧被复制/转换多次，当你使用Chrome流媒体视频时，这导致了过度的电力和性能问题。微软宣称可以通过利用GPU内存缓冲区在Windows上实现零拷贝视频捕获。

微软表示：“通过利用 Chromium 中现有的GPU内存缓冲基础设施，可以最大限度地减少副本数量”。通过这种方法，从NV12 D3D11纹理到VideoCaptureBufferPool NV12纹理只有一个拷贝，从而减少了过多的资源使用。
