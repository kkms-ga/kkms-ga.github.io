[图]Windows 10 Build 20170发布：改进搜索体验 发布新版微软Launcher

July 16, 2020

面向 Windows Insider 项目的 Dev 频道（此前称之为 Fast Ring）成员，今天微软发布了 Windows 10 21H1 分支的最新 Build 20170 预览版更新。该版本主要对搜索体验进行了优化改进，并引入全新的 Microsoft Launcher。

首先需要注意的是：由于“影响 PC 整体可用性的错误”，Windows 10 Build 20170 并不适用于那些运行 AMD 处理器的用户。微软表示，这个问题会在下个预览版更新中得到修复。

Windows Insider 发布推文称：

[更新]在开始之前…所有 Windows 10 v2004 已知问题已经被微软自己确认。

欢迎来自舰桥的 Windows Insider 成员们！我们在 Dev 频道上放出了新版本，请检查 Build 20170 更新。

需要注意的是，微软已经不再将新功能与操作系统的特定功能更新进行匹配。不过，目前正在测试的这些功能中，很多都有可能随着 Windows 10 21H1 的发布而推出。

以下是 Windows 10 2021 Build 20170 的更新日志

改善你的声音设置体验：

在上周的更新日志中，微软计划将更多的控制面板功能迁移到设置中。而现在微软又迁移了一项功能–将更新设置>系统>声音>管理声音设备，现在可以让你知道哪个设备是默认的，用户也可以将喜欢的设备设置为默认设备或默认通信设备。

微软优化了音量混合器，提供了跳转到每个应用的音频设置的链接，您可以使用它来重定向每个应用程序的音量。

面向 Insider 的其他功能

Microsoft Launcher v6 现在向公众推出

微软高兴地宣布，Microsoft Launcher v6 正式发布。该 Launcher 基于新代码构建，为用户带来了诸多新功能，包括个性化新闻、横屏模式、自定义应用图标、基于 Bing 的壁纸、深色模式，并改善了加载速度、内存占用、电池优化和流畅的过渡动画。

新功能

个性化新闻：

随时了解情况。个性化新闻源全天更新，提供热门新闻，让你轻松掌握任何与你相关的信息。

横向模式:

微软启动器支持垂直和水平方向，以提高你的观看偏好。

可定制的图标:

使用自定义图标包和自适应图标，让你的手机拥有一致的外观和感觉。

美丽的壁纸:

每天享受来自Bing的新鲜壁纸，或者选择自己的照片。

深色主题:

在夜间或弱光环境下使用手机时，减少眼睛疲劳。此功能与Android的默认黑暗模式设置兼容。

改进的性能

Microsoft Launcher现在加载速度更快，使用更少的内存，更节省电池，并提供流畅的动画。

新图标

Microsoft Launcher 要求

● 搭载 Android 7.0 或更高版本的手机

● 必须从 Play Store 下载 Microsoft Store。下载新版 Microsoft Launcher 之后将会取代默认的 Launcher。Microsoft Launcher 不会在 Android 手机上复制用户的 PC 主屏幕。用户仍然必须从 Google Play 购买和/或下载新的应用程序。

已知问题

● 原生 Android 10 导航手势可能不适用所有手机制造商和型号。

● Android 8.0 及以上版本的设备才支持系统深色主题。

● 仅在特定的 OEM 设备上支持将第三方启动程序设置为默认。

● 升级到 v6 后可能会出现便签同步问题。

● v6 升级后可能需要再次启用通知徽章。

改善 Windows 10 任务栏的搜索体验

在新版本中，微软为 Windows 中的搜索栏引入了两项改进，从而让你更容易找到你需要的东西。

点击任务栏的“搜索”或者按下 Windows+S 键组合，可以查看 Search Home 的设计变化。全新的双栏设计提高了易读性，快速访问快速搜索，在那里你可以看到当地的天气，热门新闻和更多有用的信息。要查找其他网页结果或直接导航到网站，只需像平常一样在搜索栏中键入即可。

新的搜索布局将会在美国、澳大利亚、加拿大、中国、法国、德国、印度、日本、墨西哥、西班牙和英国等国家和地区上线。

此外，Windows搜索栏现在还支持荷兰语客户的网页搜索。将Windows地区设置为荷兰，语言设置为荷兰语的用户现在可以从搜索栏搜索网页，而无需使用浏览器。此前，该功能仅限于该地区的英语用户。

针对开发者的改进

Windows SDK 和 Dev 频道将同步推进。每当有新的操作系统进入 Dev 频道，相应的 SDK 也会进入 Dev 频道。您可以随时从 aka.ms/InsiderSDK 安装最新的 Insider SDK。SDK 版本将和 OS 版本一起在 Flight Hub 存档。

常规改进

● 进一步推进 Windows 10 图标更换工程，在新版本中引入了全新的设置图标。新的设置图标与 Build 20161 中引入的主题识别磁贴在开始菜单上看起来很棒。

● 努力提高可靠性的同时，微软暂时关闭了记事本在重启和更新过程中持续打开窗口的能力。

●  从 Insider Preview Build 20170 开始，默认启用传输层安全（TLS）1.3的实验性实施。IIS/HTTP.SYS 默认启用了TLS 1.3。

SSPI 调用者在调用 AcquireCredentialsHanlde 时，可以通过传递新的加密代理SCH_CREDENTIALS 结构来使用 TLS 1.3。使用 TLS 1.3 的 SSPI 调用者需要确保他们的代码正确处理 SEC_I_RENEGOTIATE。

Windows 10 Build 20170 的常规修复

● 修复了在平板电脑模式下，调整某些应用程序的大小时，会出现闪烁的问题。

● 修复了使用IME时，按住鼠标左键最终会导致输入挂起的问题。

● 修复了导致声音设置中的主音量滑块在删除和重新配对某些蓝牙设备后无法工作的问题。

● 修正了更新通知不出现或连续多次出现的竞赛条件。

● 修复了导致在之前的版本中某些键盘上打字不起作用的问题。

● 修复了一个可能导致修饰键（如Shift和CTRL）有时在远程桌面连接中掉落的问题。

● 修复了导致少数 Insiders 在之前的版本中遇到了开始无法启动的问题。

● 修复了导致一些 Insiders 在最近几个版本中发生的注销时间增加的问题。

● 修复了导致一些 Insiders 发现他们的屏幕在升级到最后几个版本后变得更暗的问题。

Windows 10 Build 20170 的已知问题

● 正在努力修复一个问题，其中一些微软商店的游戏保护与Easy Anti-Cheat可能无法启动。

● 正在努力修复一个问题，导致一些系统崩溃与HYPERVISOR_ERROR错误检查。

● 正在研究有关更新过程在尝试安装新构建时长时间挂起的报告。

● 意识到一个问题，即任务管理器在性能选项卡中报告0.00 GHz的CPU使用率。

● 正在努力微调 “开始 “中的磁贴动画，以避免颜色闪烁。

● 正在调查一些Insiders的睡眠无法工作的报告。
