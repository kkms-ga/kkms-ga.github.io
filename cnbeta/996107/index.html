Linux Mint 20安装镜像已定 未来几天内发布

June 27, 2020

社区网站已经公布了 Linux Mint 20 的状态信息，可知稳定版已通过测试，稳定版 ISO 安装镜像将于未来数日内到来。虽然官网公告来得有点慢，但包括 Bytemark 在内的一些镜像站点，已经提供了 ISO 文件的访问，感兴趣的朋友可以抢鲜体验。

截图来自：LinuxMint 官网

功能方面，Linux Mint 20 主要包括了以下内容：

• 升级至 Linux 5.4 内核，加入对更多新硬件的支持；

• 自带 Warpinator 本地网络文件传输工具；

• 升级 LibreOffice 至 6.4 分支；

• 提供大量可选的新壁纸；

• 添加对系统任务栏图标的 HiDPI 支持；

• 改进 Cinnamon 4.6 对多显示器的支持；

• 改进对 NVIDIA Optimus 的支持。

Linux Mint 团队在近日的一篇博客文章中表示，不再依赖于快照（snap）的软件，将可从 Linux Mint 软件管理器中进行下载。

需要指出的是，尽管 Linux Mint 20 即将发布，但 Linux Mint 18.x 或 19.x 用户无需立即升级，两个版本将持续支持到明年 4 月 和 2023 年 4 月。

此外到 2025 年之前，Linux Mint 20 仍可接收到相关更新。
