三星发布Galaxy Note 20系列国行手机：12+512GB皇帝版9999元

August 13, 2020

8月13日晚，三星在国内正式推出了Galaxy Note 20系列，这次的发布会非常简短，反正最后的关键就是价格——Galaxy Note 20 8+256GB起价7399元，Note 20 Ultra12+256GB版起价9199元， 12+512GB皇帝版9999元。

访问:

三星 Galaxy Note 20

目前首发预定还有不少优惠，可以12期免息分期，赠送原装无线充，还可以优惠换屏，5折换电池，5折买S Pen，还有价值600元的Galaxy Bud Live无线耳机优惠券。

Galaxy Note 20系列手机之前已经发布，这次就简单说下吧。

Note 20系列整体设计延续了以往的家族式语言，都是居中挖孔全面屏，其中Note 20提供迷雾金、冰薄荷、冷山灰风格，机身厚度8.3毫米、重量192克，6.7英寸平面型AMOLED屏幕，分辨率2400×1080，刷新率60Hz，HDR10+认证。

Note 20 Ultra则是迷雾金、曜岩黑、初露白三种风格，机身厚度8.1毫米，重量208克，6.9英寸曲面屏，分辨率3200×1440，刷新率120Hz，HDR10+认证。

配置上，海外有Exynos、骁龙两个版本，国内应该都是骁龙865 Plus 5G平台，搭配8GB、12GB LPDDR4内存，以及128GB/256GB/512GB闪存。

拍照上，前置1000万像素自拍镜头，支持自动对焦，后置摄像头均为三摄。

其中Note 20 Ultra包括1.08亿像素主摄(F1.8)、1200万像素广角镜头(F2.2)、1200万像素长焦镜头(F3.0)，外加极光对焦传感器，支持5x光学变焦、50x超级分辨率数码变焦、OIS光学防抖。

Note 20则是1200万像素主摄、1200万像素超广角镜头、6400万像素长焦镜头，支持3x混合光学变焦、30x数码变焦。

电池容量分别为4300mAh、4500mAh，支持有线和无线快充，另有Wi-Fi 6(最大下载上传均为1.2Gbps)、蓝牙5.0、NFC、GPS/北斗/格洛纳斯/伽利略导航、USB-C接口。

除了手机之外，Note系列标配的S Pen这次变化也不小，适配了120Hz刷新率，并改进陀螺仪、加速计，响应时间缩短到8ms，还强化不同动作下的敏感度，另外压感级4096、笔尖直径0.7毫米、IP68防水防尘。

同时，隔空操作新增5种手势指令，还有更多遥控交互方式，比如用户只需做像翻手腕一样简单的动作，即可返回主屏幕或是截屏，无需点触手机即可拥有更多的操控体验。
