8月5日三星“Galaxy Unpacked”大会将发布5款新产品

July 21, 2020

三星电子总裁兼移动通讯部门主管卢泰文（TM Roh）今日透露，三星将在8月5日举行的“Galaxy Unpacked”发布会上发布五款新产品。受疫情的影响，今年的三星“Galaxy Unpack”大会将以线上形式举行。届时，预计三星将发布一系列新的移动设备，包括新一代旗舰手机Galaxy Note 20。

今日，三星电子总裁兼移动通讯部门主管卢泰文在一篇博客文章中称，在8月5日的“Galaxy Unpack”大会上，三星将发布五款新产品。卢泰文并未透露具体是哪五款新产品，但基于历史记录以及三星的暗示，新一代大屏旗舰智能手机Galaxy Note 20必不可少。

据媒体之前的报道，此次即将发布的Galaxy Note 20将至少包括两个型号的产品，分别为Galaxy Note20和Galaxy Note 20 Ultra。至于是否发布Galaxy Note 20 +，目前尚不确定。

其中，Galaxy Note 20配备60Hz、1080P屏幕，电池容量为4000mAh，支持25瓦快充；而Note 20 Ultra采用2K 120Hz屏幕，电池容量4500mAh，后置摄像头为108MP+12MP+13MP潜望长焦的组合。三个型号产品都将搭载高通骁龙865/Exynos 990处理器。

除了Galaxy Note 20，其他四款产品很可能为5G版Galaxy Z Flip折叠屏手机、新一代Galaxy Fold折叠屏手机、 Galaxy Watch 3智能手表和Galaxy Buds Live无线耳机。

报道称，Galaxy Watch 3包括1.2英寸和1.4英寸两种屏幕尺寸，其中41毫米的Galaxy Watch 3将采用1.2英寸屏幕，而45毫米版本采用1.4英寸屏幕，支持LTE和GPS连接，以及IP68防水。

根据三星的宣传视频，Galaxy Buds Live无线耳机外观看上去小巧精致，形似蚕豆，共有黑、白、金三种颜色供用户选择，呈现金属质感。
