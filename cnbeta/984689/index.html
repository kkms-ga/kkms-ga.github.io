Galaxy Note 20配色曝光：小号可选灰铜绿 大号可选铜与黑

May 29, 2020

GalaxyClub 爆料称，三星将为 Galaxy Note 20 系列智能机提供五种配色，其中标准版有灰、铜、绿三色，而 Note 20+ 机型只有黑、铜两色。鉴于 Note 10+ 已经用上充满活力的“彩虹色”玻璃背板，想必铜色版本的 Note 20+ 也不会差到哪里去。如果一切顺利，预计三星会在 8 月份正式发布 Galaxy Note 20 系列智能机。

访问购买页面:

SAMSUNG - 三星旗舰店

（图 via Windows United）

根据国内认证机构曝光的型号数据，可知 Note 20 具有支持 5G 和 25W 快充等特点。

屏幕为采用 LTPO 可变刷新率节能技术的 6.42 英寸的打孔屏，Note 20+ 机型则是 6.9 英寸 @ 120Hz AMOLED 屏。

两款机型都有望配备 Exynos 992 芯片组，且 Note 20 的电池容量为 4000 mAh，较去年的 Note 10 机型（仅 3500 mAh）高了不少。

（图自：GalaxyClub）

大号的 Note 20+ 预计为 4500 / 5000 mAh，与上半年的 S20+ 或 S20 Ultra 看齐，同时可选支持更快的 45W SuperFast 快速充电器。

拍照方面，预计 Note 20 会采用类似 Note 10 或 Galaxy S20 后置三摄组合，且只有 Note 20+ 会用上类似 S20 Ultra 的 1.08 亿像素传感器。

尽管算不上是革命性，但 Note 20 系列智能机仍有望为消费者带来一定程度的可靠升级，比如更出色的屏幕、摄像头性能、以及 S Pen 手写笔支持。
