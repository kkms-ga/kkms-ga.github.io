Google失去了其blogspot.in域名的所有权 这可能带来不少问题

July 08, 2020

谷歌似乎已经失去了其blogspot.in域名的所有权。该域名是谷歌拥有的博客发布服务Blogger的一部分，该公司在2003年收购了Blogger。由于在Blogger上，特定国家的域名并不是不同的博客地址，而是基于目前所在国家的域名重定向。因此在印度境内搜索时，Blogger指向的是username.blogspot.in，而不是username.blogspot.com。

由此，很多印度用户最终也会以.in域名作为文章的永久URL在网络上分享文章。但是现在从WHOIS查询来看，blogspot.in域名已经被某域名服务商捡到了。

Domain Name: blogspot.in

Registry Domain ID: DE2DC9C0E8E694C28ADEF0F444F121B45-IN

Registrar URL: www.domainming.com

Updated Date: 2020-06-29T20:00:06Z

Creation Date: 2020-06-24T20:00:05Z

Registry Expiry Date: 2021-06-24T20:00:05Z

Domain Status: inactive

2013年2月，Blogger推出了面向特定国家的URL。例如，在印度，用户名.blogspot.com将自动重定向到用户名.blogspot.in，在澳大利亚，用户名.blogspot.com.au。当时谷歌解释了这样做的意图：如果公司收到违反当地法律的内容的删除请求，那么在适用不同国别法律的当地域名上，读者可能无法再获得这些内容。

Blogger似乎早已不是谷歌的重点产品，然而，这次域名丢失事件会伤害到很多博客作者和读者，因为这样会产生大量无效链接。例如，在谷歌上搜索’site:blogspot.in’的结果，在撰写本文时，产生了超过450万个结果。
