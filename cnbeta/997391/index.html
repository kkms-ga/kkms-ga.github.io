[图]谷歌智能家居开发者活动将于7月8日线上举行 现已开放注册

June 30, 2020

谷歌今天宣布将于今年7月8日举办一场线上的智能家居开发者活动，以代替被取消的谷歌I/O大会，承诺“为智能家居开发者和用户提供新的功能”。本次线上活动包含 45 分钟的主题演讲、一系列开发者议程以及由谷歌和智能家居行业其他公司参加的小组讨论。

据悉，本次主题演讲由谷歌智能家居生态系统产品管理总监 Michele Turner 主持，将介绍谷歌最新的智能家居 API 功能，以及其他对开发者和用户的优势功能。谷歌表示，它将利用此次峰会 “分享我们最近的智能家居产品计划”。

自2019年 Nest 重塑品牌以来，谷歌的智能家居部门一直非常忙碌。去年，该公司发布了Nest Mini智能音箱、Nest Wifi网状路由器、一款名为 Nest Hub Max的新型智能屏幕。今年早些时候，它还对其智能家居安全摄像头的Nest Aware订阅进行了改造，并宣布对 Google Assistant 进行一系列改进。
