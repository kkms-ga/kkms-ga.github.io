文档基金会 2019 年度报告

August 14, 2020

LibreOffice 背后的文档基金会(The Document Foundation, TDF)公开发布了其 2019 年度报告。完整的报告总共有 54 页，内容包括项目历史、基金会 2019 重要事件、LibreOffice 2019 重要事件、财务与年度预算、市场行为，会议举办记录以及基金会在社交媒体的表现等。

总而言之，如果希望深入了解 LibreOffice 和 TDF，这份报告可以说是“必读材料”。

从这份年度报告中我们得知，TDF 作为一个非营利组织，其收入来源主要是捐款和会员费。TDF 在 2019 财年的总收入为 906,470 欧元。按照目前的货币汇率换算，大约为 106.3 万美元。与他们 2018 年的总收入相比有了可观的增长。从近两年的数据来看，TDF 每个财年总收入从 2017 的 743,111 欧元到 2018 的 855,847 欧元，再到 2109 的 906,470 欧元，期间拥有了显著的增长。这也说明 TDF 的总收入一直保持着健康的同比增长。

TDF 统计的数据显示，大约 68％ 的收入来自 PayPal 捐款渠道，另外 25％ 来自信用卡的直接捐赠，还有大约 6％ 的捐赠渠道是“其他”，最后 1％ 来自会员费。

至于支出方面，2019 的支出约为 679,080 欧元，其中 62% 是提供给雇主和自由职业者的资金、用于社区工作和活动、法律咨询和会计，以及基础设施成本等费用。

回到大家比较关注的代码方面的数据，在 2019 年，234 位开发者为 LibreOffice 项目贡献了 15,600 次 commit。
