新一代显卡危机：外媒分享《微软飞行模拟2020》基准测试数据

August 22, 2020

本周，玩家们期盼已久的《微软飞行模拟 2020》终于开售。为了对被誉为“新一代显卡危机”的该游戏进行测试，外媒 TechSpot 早早地就点击了预购。出于某些原因，微软并未选择通过 Steam 服务器进行游戏内容的完整分发。相反，你只会在 Steam 上找到一个 500MB 的小型安装包，然后费老大劲地加载近 100GB 的游戏数据。

尴尬的是，官方安装程序的性能难以让玩家们感到满意，并且会不断爆出错误。TechSpot 发现，许多人都遇到了同样的问题。

欣慰的是，这款游戏的内容和玩法体验着实值得等待。不仅视觉效果上令人叹为观止，技术层面也相当领先 —— 即便也对当前一代的 PC 硬件提出了相当高的要求。

据悉，《微软飞行模拟 2020》基于 Asobo 引擎构建。为模拟整个地球，其利用了微软云端必应地图中超过 2PB 的数据。同时借助 Azure 来分析地图数据，以生成道路、建筑、树木、草地、水面、地形等逼真的 3D 模型。

然而对于大部分玩家来说，他们更关心的是“需要多强劲的 CPU / GPU 硬件，才能让这款游戏跑得足够欢快”。为回答这一问题，TechSpot 找来了 28 款显卡，在预设的“极致 / 中等”画质下，以 1080p / 1440p / 4K 分辨率进行了跑分。

本次测试指定了澳大利亚悉尼航线的着陆挑战，并在加载后等待 30 秒时间。在适当的背景下，《微软飞行模拟 2020》将需要大约一分钟来加载。然后再过 30 秒，你才能看到帧率相对稳定下来。

在那之后，TechSpot 对该游戏在不同显卡和分辨率 / 画质设置下的表现进行了测试，持续时间为 60 秒。鉴于该作对系统硬件配置的需求相当高，其最终选择了基于 AMD 锐龙 9 3950X 的测试平台（而非英特尔）。

这枚处理器具有 16 核 / 32 线程，辅以 32GB DDR4-3200 C14 内存。显卡方面，TechSpot 安装了最新版的 AMD 和英伟达驱动程序，以满足对《微软飞行模拟 2020》的适当支持。

测试从 1080p / 极致预设模式开始，在游戏加载期间，即便不超过 30fps 的帧率也对体验影响不大。但在实际上手时，还是需要 60fps 才能保持畅快体验，最终测试结果如下：

1080p / 极致

由实测可知，想要尽情体验《微软飞行模拟 2020》的游戏魅力，建议至少选择 AMD Radeon RX 5600 XT 或 Nvidia GTX 1070 Ti / RTX 2060 这一档次的游戏显卡。

1440p / 极致

在 1080p 的主流分辨率下，高端 GPU 和 CPU 的表现或许会受到一定限制，2080Ti 也仅比 2070 Super / 1080 Ti 高那么几帧。
