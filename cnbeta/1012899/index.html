Reddit网友畅想Windows 10新UI设计 有望随20H2更新一同到来

August 07, 2020

与 Windows 8 时代的 Modern UI 相比，Windows 10 的用户界面着实有些过于“朴素”。即便微软正在努力实现 Fluent Design 设计语言的“大一统”，但与苹果在 WWDC 2020 上宣布的 macOS Big Sur 相比，老用户们还是希望微软能够将 Windows 10 打扮得更漂亮一些。

通过 Fluent 设计语言的循序渐进，微软正在为自家各个产品和操作系统带来全面改进，目前已可见到亚克力和阴影等桌面效果。本文展示的图片，便基于微软 Windows 10 的 Insider 预览编辑而来。

然而一些等不及的用户，还是忍不住在各个渠道分享他们对于 Windows 10 新 UI 的展望，比如 Reddit 网友分享的“开始”菜单截图。

从视觉效果上来看，不少网友应该会很喜欢加入了“主题感知”的图标背景。在此之前，Windows 10 只能给“开始”菜单中的图标加上纯色的大块背景。

此外 Reddit 网友对浅色主题和黑暗模式都进行了探讨，不过作为 Windows 10 20H2 更新的一部分，相信用不了多久，大家很快就能迎来官方正式版的系统 UI 改进。
