[图]本月Windows 10累积更新悄然删除记事本等预装应用程序

July 16, 2020

在本月的补丁星期二活动日上，微软面向 Windows 10 May 2020（20H1/Version 2004）功能更新发布了 KB4565503 累积更新。不过有用户反馈，在安装完该累积更新之后，部分预装的应用程序被悄然删除，而用户只能逐一重新安装。

从目前用户反馈的情况来看，本次更新默认卸载的应用程序包括记事本、写字板、画图三款，可能还有一些其他没有发现的应用程序。不过好消息是，如果用户想要重新安装这些应用程序，只需要进入到 Windows 10 系统的可选功能界面点击就能安装。

该累积更新首先对 Windows 组件和应用带来了安全改进，其中包括

Microsoft Scripting Engine, Windows App Platform and Frameworks, the Microsoft Store, Windows Graphics, Windows Input and Composition, Windows Media, Windows Shell, Windows Fundamentals, Windows Management, Windows Kernel, Windows Hybrid Cloud Networking, Windows Storage and Filesystems, Windows Update Stack, Windows MSXML, Windows File Server and Clustering, Windows Remote Desktop, Internet Explorer, Microsoft Edge Legacy 和 Microsoft JET Database Engine 获得了安全更新。

另外，该累积更新还引入了一些非安全的 BUG 修复，修复了部分应用程序在尝试打印含有某些图形的文档或者大文件时出现崩溃的问题。

该累积更新还改善了 OneDrive 桌面端的用户体验：

修复了可能阻止你使用 OneDrive 应用连接 OneDrive 的问题。这个问题发生在一些旧设备上，或者在使用旧应用的设备上，这些设备使用传统的文件系统过滤驱动程序。因此，这可能会阻止这些设备下载新文件或打开之前同步或下载的文件。

该累积更新只包含一个已知问题，影响了在某些应用中使用微软输入法的中文和日文用户：

在使用某些应用程序时，例如 Microsoft Excel，中文和日文的微软输入法编辑器(IME)可能会出现一个错误。或者在尝试使用鼠标拖动时，应用程序可能会停止响应或关闭。
