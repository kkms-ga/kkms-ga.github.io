奥林巴斯推出可将部分无反相机变成PC网络摄像头的软件

July 03, 2020

继富士、佳能、松下之后，奥林巴斯也宣布了可将部分无反相机变成高品质 PC 网络摄像头的软件，感兴趣的朋友可在官网下载《OM-D Webcam Beta》应用。奥林巴斯宣称该软件支持 OM-D E-M1X、OM-D E-M1、OM-D E-M1 Mark II、OM-D E-M1 Mark III 和 OM-D E-M5 Mark II 共五款机型，但目前尚无法在 Windows 之外的系统上运行。

（来自：Olympus）

相机的设置过程很是轻松，在安装了配套的《OM-D Webcam Beta》版软件之后，上述五款相机的用户只需通过 USB-C 端口连接 Windows PC，然后开机并启动软件。

正确识别之后，你就可以将奥林巴斯的无反相机当做网络视频会议的高品质 USB 摄像头来使用。如果不熟悉怎么操作，不妨参考下 Olympus 的官方演示视频。

Olympus OM-D Webcam Software Beta（via）

需要注意的是，《OM-D Webcam Beta》软件不会同时让相机采集音频，因此用户需要在使用网络视频会话时启用内置或外部 USB 麦克风。

至于这么做的原因，奥林巴斯解释称，其希望用户将相机摆放到最佳的位置，但又不至于因为距离太远而无法清晰地收音。
