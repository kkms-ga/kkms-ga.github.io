[图]PowerToys 0.19.1发布：修复内存泄漏在内的六个问题

July 08, 2020

在上周发布 PowerToys 重大版本更新 0.19 之后，今天微软又跟进发布了 0.19.1 维护版本更新。本次更新重点针对关键问题进行了修复，并提高了应用的整体稳定性。

本次增量更新主要修复了 FancyZones 和 PowerToys Run 工具的现有问题，并带来了一些可用性改进，以确保更容易通过搜索建议进行导航。

以下是 PowerToys 0.19.1 版本修复以下问题

#4578 - WinKey for PT Run 出现大量误报问题

#4646 - FancyZones：当 “按住Shfit键 “被停用时，Shift 键应停用 FancyZones。

#4625 - [Run] 上/下导航时搜索文本不更新。

#4432 - 缓解操作系统检测中的竞赛条件。

#2047 - PT 运行中的内存泄漏

#4647 - 修复重设的快捷键不能连续激活的问题。

除了上述问题外，微软还承认了另一个BUG，即目前当 PowerToys设 置在任务栏最小化时，用户无法打开它。该公司承诺将在 PowerToys v0.20.0 完整版中解决这一问题。

下载地址：https://github.com/microsoft/PowerToys/releases/tag/v0.19.1
