微软翻译现已支持印度24种官方语言之一的奥里亚语

August 15, 2020

人类历史上出现过许多种口头与书面语言，时至今日，还有不少语言在被积极使用。此外随着翻译器技术的不断发展和完善，使用不同语言的双方也能够更高效地沟通。比如微软翻译（Microsoft Translator）刚刚宣布，其现已支持印度官方认定的 6 种古典语之一的奥里亚语（Odia）方言。

（来自：Microsoft）

微软今日很高兴地宣布，其已将奥里亚语（Odia / 念做 oṛiā）的文本翻译功能添加到了 Microsoft Translator，并且能够通过翻译软件客户端、Office 生产力套件、必应搜索引擎、以及面向企业和开发者的 Azure 认知服务而提供。

起源于印度东部的奥里亚语属于印欧语系，它是奥里萨邦（Odisha）的官方语言。附近的西孟加拉邦、贾坎德邦、恰蒂斯加尔邦、中央邦和安得拉邦，也有在使用这种语言。

Microsoft Translator 体验传送门：

Google Play（Android）| App Store（iOS）| Microsoft Store（PC）

值得一提的是，奥里亚语还是印度政府官方认定的六种古典语言之一，其文学历史可追溯到 1000 年前。

据悉，目前在印度和世界各地，仍有约 3500 万人在使用奥里亚语。此外印度次大陆还有另外 11 种常用语言，包括孟加拉语、古吉拉特语、印地语、卡纳达语、马拉雅拉姆语、马拉地语、旁遮普语、泰米尔语、泰卢固语、乌尔都语、以及英语。
