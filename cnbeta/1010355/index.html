[图]Windows端Chrome浏览器即将支持Nearby Sharing功能

August 01, 2020

谷歌的 Nearby Sharing 功能类似于苹果的 AirDrop 功能，允许用户通过 Chrome 浏览器在设备之间分享文件。现在这项功能终于出现在 Windows 端的 Chrome 浏览器上了。

谷歌一直致力于将这项功能引入到 Windows、Mac、Linux 和 Chromebook 设备上的 Chrome 浏览器上。想要在 Windows 平台上通过 Chrome 浏览器来使用“Nearby Sharing”功能，需要

● 需要在 Chrome 浏览器中启用“Nearby Sharing” Flag，然后重启浏览器

● 使用支持蓝牙的 PC、Pixel 智能手机或者 ChromeBook。

● 想要使用该功能，设备要处于解锁状态，并在 1 英尺范围内，必须要启动蓝牙

● 如果你想通过使用该功能与 Chromebook 共享文件，请确保 Chromebook 在快速设置中启用“Nearby Sharing” 。
