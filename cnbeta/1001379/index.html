苹果推出新的教育课程 让每个人都可以用Swift进行编程开发

July 10, 2020

苹果公司今天宣布对其 “Develop in Swift “和 “Everyone Can Code “教育计划进行重大改进，提供免费课程和新的 “Develop in Swift “在线课程，以帮助教师掌握Swift应用开发教学的速度。

苹果市场、应用和服务副总裁Susan Prescott表示：“苹果与教育工作者一起合作了40年，我们特别自豪地看到Develop in Swift和Everyone Can Code帮助教师和学生在社区中产生影响。我们已经看到社区大学的学生为他们的校园建立食品安全应用程序，并看到中学教育工作者在暑假期间举办虚拟编码俱乐部。作为我们帮助扩大计算机科学教育机会承诺一部分，我们很高兴能够增加一个新的专业学习课程，以帮助更多的教育工作者，无论他们的经验如何，都有机会学习编码并教授下一代的开发人员和设计师。”

Develop in Swift面向高中及以上的学生，其课程完全重新设计，并包括四本通过苹果图书提供的免费书籍。“Develop in Swift Explorations”、“Develop in Swift AP CS Principles”、“Develop in Swift Fundamentals “和 “Develop in Swift Data Collections”，前三本现在就可以买到，第四本将在今年秋季推出。

对于年龄较小的学生和其他初学者，苹果还在扩展Everyone Can Code课程，推出了新的 “Everyone Can Code Adventures “一书，在之前的课程内容基础上，可以使用Swift Playgrounds进行更高级的编码练习。
