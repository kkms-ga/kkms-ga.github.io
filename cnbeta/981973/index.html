Windows 10引入诸多PWA改进 会是微软应用生态的突破口吗？

May 22, 2020

Microsoft Store自上线以来一直存在应用不足的问题，在UWP基本宣告失败后微软将更多精力倾注到渐进式网页应用（PWA）上，视为未来取得成功的关键。在近日召开的Build 2020开发者大会上，微软再次为PWA应用带来改进，能够让其像本地原生应用一样运行。

访问:

微软中国官方商城 - Windows

根据Build 2020开发者大会的一项议程，微软将允许用户通过Windows 设置应用来管理PWA应用程序，意味着能够像本地应用程序一样进行卸载。此外微软还允许PWA应用分享和接收内容。

此外针对PWA应用的改进还包括

● 和开始菜单更好的整合。你能够从开始菜单的PWA磁贴中执行多个右键操作

● 文件处理

● Windows Share UI

● 整合任务管理器

● 隐私设置

● 在设置中整合应用&功能页面

在即将推出的Microsoft Edge版本中，PWA将被允许直接写入文件系统。同样，如果你安装了照片编辑器，你可以在.jpg或.png文件上右键单击这些文件，用基于PWA的照片编辑器编辑图片。通过这些改变，微软表示，Web应用和原生应用之间的差距应该会缩小。
