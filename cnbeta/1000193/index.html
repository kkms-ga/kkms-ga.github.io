Poco M2 Pro正式登陆印度市场 13999卢比起售

July 07, 2020

早在 2018 年，小米就推出了基于 POCO 子品牌的首款主打性价比的 F1 旗舰智能机。但在那之后，Poco 的产品线却并没有变得多丰富。好消息是，继今年早些时候的 Poco X2 之后，该公司又于本月推出了 Poco M2 Pro 中端新品。从规格来看，它很像是 3 月份登陆印度市场的红米 Note Pro 的小改马甲版本，比如采用了高通骁龙 720G 芯片组，辅以后置四摄的组合。

Poco M2 Pro 采用了 6.67 英寸 @ FHD+（1080×2400 分辨率）的 LCD 屏，长宽比为 20:9，并有康宁第五代大猩猩玻璃的保护。

高通骁龙 720G 芯片组集成了 Adreno 618G GPU，存储可选 4 / 6 GB LPDDR4X RAM + 64 / 128GB UFS 2.1 ROM，且支持 512GB microSD 扩展。

前置摄像头为支持夜拍模式的 16MP 单摄，后置则是 48MP 主摄 + 8MP 超广角 + 2MP 深度 + 5MP 微距镜头的组合。内置 5000 mAh 电池，支持通过 USB-C 端口实现 33W 快充。

连接方面，Poco M2 Pro 支持 4G VoLTE、Wi-Fi 5（802.11ac）、蓝牙 5.0、GPS / A-GPS，且保留了 3.5 mm 耳机插孔。机身大小为 165.75×76.68×8.8 mm，重 209g 。

颜色方面，消费者可选蓝、绿、黑三种。该机将于 7 月 14 日通过 Flipkart 在线上独家销售，4+64 / 6+64 / 6+128 版本分别为 13999 / 14999 / 16999 卢比（约 1312 / 1405 / 1593 RMB）。
