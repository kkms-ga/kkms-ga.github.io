三星Galaxy Note 20保护壳曝光：矩阵四摄加持 8月发布

June 26, 2020

6月26日消息，博主@i冰宇宙曝光了三星Galaxy Note 20的保护壳。如图所示，三星Galaxy Note 20相机为矩阵造型，五个开孔暗示Galaxy Note 20拥有四颗摄像头，另一枚开孔为闪光灯。值得注意的是，保护壳底部开孔表明S Pen手写笔被移至左侧位置，没有3.5mm耳机孔。

爆料指出，三星Galaxy Note 20系列包含Galaxy Note 20标准版和Galaxy Note 20 Ultra至少两款。

二者均采用挖孔屏方案，前置摄像头居中。其中Galaxy Note 20为直屏，刷新率似乎是60Hz，分辨率可能是FHD+。

Galaxy Note 20 Ultra为双曲面屏幕，刷新率为120Hz，分辨率为QHD+，而且采用LTPO显示屏（业界首款）。

核心配置上，Galaxy Note 20系列均搭载高通骁龙865旗舰平台（也可能会使用骁龙865 Plus），最高配备16GB内存+512GB存储。其中Galaxy Note 20 Ultra的主摄为一亿像素（108MP），成像值得期待。
