三星Galaxy Note 20相机规格已在发布会前曝光

August 05, 2020

三星即将在 8 月 5 日的 Unpacked 活动期间揭晓 Galaxy Note 20 系列智能机，但知名爆料人 Evan Blass（@evLeaks）已经抢先曝光了 Note 20 和 Note 20 Ultra 强大的相机规格。据悉，Galaxy Note 20 Ultra 将配备 1.18 亿像素 @ f/2.2 光圈的主摄、12MP @ f/2.2 的超广角副摄、以及 12MP @ f/3.0 的长焦镜头。

为帮助更好地对焦，三星还为 Galaxy Note 20 Ultra 引入了激光 AF 传感器。至于 Galaxy Note 20 机型，其采用了 12MP @ f/1.8 光圈的主摄、12MP @ f/2.2 的副摄、以及 64MP @ f/2.0 的长焦镜头。

此外 Note 20 / Note 20 Ultra 都在机身正面采用了 Infinity-O 打孔屏 + 支持 4K 视频录制的 10MP @ f2.2 的双像素前摄。
