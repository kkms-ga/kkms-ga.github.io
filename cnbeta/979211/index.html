业内人士透露Galaxy Note 20和Galaxy Note 20+规格细节

May 15, 2020

有业内人士透露了Galaxy Note 20和Galaxy Note 20+的规格细节，与Galaxy S20系列保持一致。三星Galaxy Note 20预计将在8月初推出，并将采用类似S20系列的高端硬件。不过，与S20不同的是，Galaxy Note 20不会有Ultra版本。根据消息人士透露，三星准备让Galaxy Fold 2将充当Ultra版本，Galaxy Fold 2是第2代Fold手机，它将与Note 20系列同时推出。据称Note 20系列当中2个版本都将采用120Hz的OLED屏幕，分辨率略有提升。Galaxy Note 20将采用6.3英寸的屏幕，而Note 20+的显示屏尺寸将接近6.9英寸。

Note 20和Note 20+都将采用全新的屏内指纹传感技术，显示屏内集成高通在12月份发布的全新传感器-3D Sonic Max指纹识别器。3D Sonic Max技术将支持两根手指。Note 20手机将标配16GB内存，并表示这两款手机将配备4000毫安时至5000毫安时的电池。这两款手机将采用与Galaxy Note 10和S20类似的设计，其特点是在中央位置放置了一个打孔自拍摄像头。

这两款手机将于8月初亮相，几周后到店销售。这两款手机的定价尚未公布。目前三星S20和Note 20不仅在设计路线上共享，而且在主要规格上也将共享。与S20手机相比，Note 20将采用内置S Pen手写笔和更大的屏内指纹传感器。
