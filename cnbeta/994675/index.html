[图]你怎么看？微软悄然在Windows 10搜索UI中引入天气磁贴

June 23, 2020

在剥离语音助手 Cortana 之后，Windows Search 在很多地方进行了改进。援引外媒 SoftPedia 报道，近日微软悄然为搜索功能带来了细节方面的调整，在用户社区引起了不同的反应。

本周开始，微软在 Search UI 中启用了小型的天气磁贴，当点击任务栏中的搜索图标时，会在左上角显示当前的天气状况和温度。而这是服务器端的改进，因此并不捆绑特定的 Windows 10 版本。也就是说，运行 Windows 10 Version 1903 或者 2004 功能更新的用户都会看到这个磁贴。

而且看起来，整个事情是分阶段进行的，因为不是每个人现在都能获得天气磁贴–正如你在顶部的截图中所看到的那样，当启动搜索UI时，外媒编辑的设备上没有显示任何天气数据，尽管一切配置正确。

正如所说，这项功能受到的反响不一。有人说这是一个很酷的改进，让搜索UI更加有用，尤其是有些用户真的会搜索天气数据，但也有人认为这多少是一个多余的更新，因为你可以直接看窗外的天气信息。
