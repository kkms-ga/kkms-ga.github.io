微软宣布对WSL2子系统内核进行更改 将通过系统更新推送新版

June 11, 2020

据微软官方博客消息，在微软最新推出的 Windows 10 vNEXT Build 19645 快速测试版中，微软已经对 Windows Subsystem for Linux 2 (WSL2) 中的 Linux 内核服务模型进行调整。

在 Windows 10 Version 2004 及早期版本中子系统的 Linux 内核是预装在 Windows 映像中的，不过从最新测试版开始微软从 Windows 系统映像中删除 Linux 内核。

删除 Linux 内核的原因是因为没有必要再进行预装，因为从新版本开始 Linux 内核以及周边软件都会通过 Windows Update 提供，也就是说微软可以随时通过 Windows Update 向用户推送新版内核。当然不只是 Linux 内核，周边软件包括第三方软件例如驱动程序、触摸板驱动等等也会通过 Windows Update 提供。这样做的好处是用户可以随时获得最新的内核和驱动程序确保 Linux 子系统可以稳定运行。

当然推送归推送是否升级新版本内核或驱动程序还是用户自己决定的，现阶段微软还无法直接同步更新 Linux 官方内核，因为这些内核要经过微软工程师的适配后才可支持 Linux 子系统。微软也发布 WSL 内核更新页面供开发者们查看。

最新提供的预发行版本内核为 4.19.121 版，添加对 GPU 计算的支持，具体可以参考该页面：https://docs.microsoft.com/en-us/windows/wsl/kernel-release-notes
