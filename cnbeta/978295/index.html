爆料称三星Galaxy Note 20仅顶配版有120Hz高刷

May 13, 2020

5月13日消息，据外媒报道，三星Galaxy Note 20系列支持120Hz高刷新率，但仅限于顶配版本。报道指出，三星Galaxy Note 20的刷新率为60Hz，顶配版才拥有120Hz高刷新率。

原因是三星Galaxy Note 20系列要采用的是LTPO技术成本很高，因此这项技术仅限于顶配版使用，标准版则无缘LTPO技术，屏幕刷新率为60Hz。

据悉，LTPO即Low Temperature Polycrystalline Oxide（低温多晶氧化物）的简称，LTPO=LTPS+Oxide，同一个像素中集成了LTPS和Oxide两种TFT器件，Oxide是底栅结构，LTPS是顶栅结构。

这种新工艺结合了LTPS TFT工艺驱动能力强和Oxde TFT工艺漏电小功耗低的优点，它被苹果应用在Apple Watch Series 5型号中，也正是有了这项技术，Series 5做到了屏幕常显而且与Series 4一样具有长达18小时的电池续航。

外媒指出，Galaxy Note 20标准版不支持120Hz刷新率有点遗憾，毕竟Galaxy S20系列全系支持120Hz，尽管它只能在FHD+分辨率下才能开启。

核心配置上，Galaxy Note 20采用6.42英寸显示屏，分辨率为2345×1084，Galaxy Note 20+采用6.87英寸显示屏，分辨率为3096×1444，搭载高通骁龙865旗舰平台。

另外，Galaxy Note 20系列不会有Ultra型号。该机预计在8月份前后亮相，值得期待。
