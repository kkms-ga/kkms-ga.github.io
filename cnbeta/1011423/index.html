Xbox游戏串流功能更新 xCloud触屏控件曝光

August 04, 2020

早前有报道称，作为 Xbox Game Pass Ultimate 的一部分，微软正准备于下月推出 xCloud 游戏串流服务。现在，我们又知晓了有关 xCloud 触屏控件的详情。其实早在去年，软件巨头就已经介绍过该功能。只是现在，Xbox 游戏工作室的开发人员（比如 The Coalition 和 Ninja Theory）正准备联手将之引入 Xbox 游戏。

xCloud 触屏控件示意

在今日的 Microsoft Game Stack 直播期间，The Coalition 资深游戏制作人 Jarret Bradley 向观众展示了《战争机器 5》中的触屏控件。

你会发现 The Coalition 已经为其引入了自定义布局，以便玩家更好地执行瞄准、设计、装弹、以及驾驶车辆等操作。

《战争机器 5》操作演示

为了更加直观地执行切换操作，玩家还可在游戏视图上层看到图形 UI 的示意，且 D-Pad 并不总是显示在屏幕上。在无需交互的过场动画等环节，整个触屏 UI 也都会自动隐藏。

对于 FPS 游戏爱好者来说，The Coalition 还引入了陀螺仪瞄准控件，以帮助玩家在手机或平板上获得更加出色的游戏体验。

Game Stack Live August 2020（via）

Jarret Bradley 表示，其不希望触屏控件变得太过复杂，并且意识到不能将之视作键鼠或手柄的简单替代。因为它就是一种截然不同的输入方式，所以必须对相关操作进行重新映射。

据悉，The Coalition 已经对《战争机器 5》的触屏控件实施了精心的优化。但对于第三方开发者来说，微软还是希望他们在设计之初就考虑到这一点。
