[图]Fitbit Versa 2即将获得Google Assistant语音支持

June 12, 2020

去年11月，在收购 Fitbit 之后谷歌将其定位为“用可穿戴设备帮助更多人”。由于该交易仍有待监管部门批准，两家公司尚未整合或宣布任何联合计划。不过现在这种情况似乎正在发生改变，Fitbit 正努力为自家设备带来 Google Assistant 的支持。

在今天获得的更新中，Fitbit Versa 2 支持麦克风和亚马逊 Alexa 语音助手的整合。你可以查看天气、设置倒计时、控制兼容的智能家居设备等等。但这些体验需要用户按住 Versa 2 的物理按钮或者在用户界面滑动启用，并不支持热词启动。而且由于手表并没有扬声器，因此无法提供音频方面的反馈，只能让用户观看阅读。

援引英国连线（Wired UK）报道，Fitbit曾与谷歌接触，希望在Versa 2上提供Assistant。报道称，谷歌“的意愿要比亚马逊还低”，必要的Assistant API 无法获得。

不过现在看起来谷歌已经改变了自己的态度，在 Fitbit for Android 最新版本中开始准备添加对 Google Assistant 的支持。在应用反向编译后发现代码中出现了“Activate Assistant”，有点类似于现在 Versa 2 启用 Alexa 的步骤。

Activate Assistant

Unable to process request to activate Assistant

Error deactivating Assistant

三组XML文件清楚地提到了谷歌助手和如何引入它，以及欢迎过程。

res/layout/a_google_assistant_on_boarding.xml。

res/layout/f_google_assistant_landing.xml。

res/layout/f_google_assistant_teaser.xml

应用串还提到 “你一次只能拥有一个活跃的助手，你可以随时切换到另一个供应商”。这个选择过程表明，亚马逊Alexa会坚持下去，当Assistant推出后，这项语音服务不会立即消失。
