微软与脸书紧密合作 以改善Windows和macOS上的React Native体验

July 24, 2020

Facebook 的 React Native 框架，允许开发人员仅使用 JavaScript 来构建移动应用程序，以及借助声明性组件来编写丰富的移动用户界面。在去年的开发者大会（Build 2019）上，微软也宣布了一个性能取向的“React Native for Windows”实施方案。自那时以来，软件巨头便一直与 Facebook 的 React Native 团队紧密合作，以便在 Windows 和 macOS 上为 React Native 提供更好的支持。

（来自：React Native / GitHub）

昨日，Facebook 宣布与微软达成了更进一步的合作，通过创建一个新的子团队，来改善 Windows 和 macOS 上的 React Native 体验。

据悉，React Native 团队将与 Messenger Desktop（在 Windows 和 macOS 均基于 React Native 的基础上）同微软紧密合作。

通过借助 Reactor 无法使用的系统功能来改进 React Native，从而在桌面上实现更优的原生体验。

Facebook 和微软表示：双方表示很高兴将 React Native 带入更多平台，其为桌面平台构建的功能，还针对移动和 Web 方面的体验进行了改进。

下一步的内容将更加激动人心，两家公司希望在 Windows 和 macOS 平台上具有丰富经验的开发者来帮助共同实现这一点，且 Facebook 当前正在招募新的团队成员。
