[图]Edge Dev新版86.0.608.2发布：改进深色主题

August 20, 2020

微软刚发布了 Edge Dev 频道新版本，最新版本号为 86.0.608.2。新版本中引入了三项功能，首先是默认页面的控件支持深色主题，意味着每当你打开网站上的下拉菜单或日历时，它们也应该使用深色主题。

接下来，微软为 Kiosk 模式增加了一个新的管理策略，允许在浏览器关闭时删除下载。最后但并非最不重要的是，这个版本启用了对Chromium管理策略的支持，允许配置用户数据快照保留限制。

微软表示，在新版本中还修复了几个应用崩溃问题，包括在切换配置文件，以及在旧版 Windows 上关闭浏览器发生的问题。
