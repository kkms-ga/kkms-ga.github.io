[图]Edge无法卸载 引发诸多用户强烈不满

August 18, 2020

在拥抱 Chromium 的同时，新版 Edge 浏览器也在积极融入经典版 Edge 的诸多优秀特性（例如滚动、续航等等）。诸多优秀特性赢得了不少用户的青睐，但某些做法也让一些用户无法接受，其中一条就是无法卸载。

本月初，微软发布了一篇名为“Can’t uninstall Microsoft Edge”（无法卸载微软Edge）的支持文档，微软表示已经分阶段将 Windows 用户从经典版 Edge 迁移到新版本。

微软表示：“我们要确保所有的 Windows 客户都有最新的微软 Edge 浏览器的性能，隐私，安全，生产力。”用户可以从经典版 Edge 中导入数据到新版中，操作步骤为启动新的Edge > 设置 > 配置文件 > 导入浏览器数据 ，选择” Microsoft Edge Legacy”。微软在文档中表示：“新版 Microsoft Edge 已经包含在 Windows 系统更新中，因此无法卸载或者继续使用经典版 Edge”。

无法卸载 Edge 让不少用户不满，在谷歌趋势中“uninstall Microsoft Edge”（卸载微软 Edge）这个关键词的变化趋势如下

不过，方法总比苦难多，德国人就找到了强制卸载新Edge的办法。

1、打开资源管理器，定位到Edge安装目录，一般是C:Program Files (x86)MicrosoftEdgeApplication

2、进入对应当前Edge版本号的文件夹，并进入“Installer”文件夹内找到setup.exe。

3、选中setup.exe后单击窗口“文件”中的“以管理员身份打开PowerShell”

4、输入以下命令并回车即可：

setup.exe –uninstall –system-level –verbose-logging –force-uninstall
