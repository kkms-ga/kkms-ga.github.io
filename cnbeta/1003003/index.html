记者Joanna Stern呼吁告别Chrome并转投Safari或Edge浏览器

July 14, 2020

作为一名长期的技术爱好者，记者 Joanna Stern 近日在《华尔街日报》上发表了一篇专栏文章，呼吁人们告别 Chrome，并转投 Safari 或 Edge 浏览器。尽管文章有些冗长，但其中还是传达了一些有趣的观点。比如即便是 Windows 的老用户，也正在使用系统自带之外的新 Web 浏览器。

Joanna Stern 写道，她已将 Google Chrome 从自己的 MacBook 中扫地出门，并投入了 Safari 和 Edge 的环保。

这么做的动机，旨在了解互联网新手的早期浏览器选择，是否会对计算机的完整体验造成若干潜在的问题。

Joanna Stern 对 Chrome 的抱怨，主要集中在内存资源的消耗、电池性能的影响、以及对隐私的滥用上。

为了避免被 Chrome“拖慢”计算机的运行速度，她建议读者应立即停用谷歌浏览器。

Joanna Stern 承认 Chrome 在市场上占据主导地位，这点也证明了该浏览器确实适合大多数人。但微软 Edge 浏览器的技术实力也不差，大家无需对兼容性问题有过多质疑。

此外，Edge 和 Safari 浏览器在 MacBook 上的性能也在稳步提升，两者的内存占用分别减少了 5% 和 10%，同时让设备的电池续航增加了近两个小时。

如果实在不想从 Chrome 迁移到其它浏览器，Joanna Stern 也提供了三项改进体验的小技巧，比如卸载不必要的扩展程序、关掉消耗过多内存资源的标签页、以及限制基于浏览器的视频聊天。
