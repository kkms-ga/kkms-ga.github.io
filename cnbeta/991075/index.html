[图]Windows 10功能体验包暗示微软有意剥离更多系统功能

June 15, 2020

作为 Windows 10 May 2020（20H1/Version 2004）更新的一部分，微软悄然在系统中安装了神秘的“Windows Feature Experience Pack”。微软官方并没有公布这个体验包具体提供了哪些内容，不过根据外媒掌握的信息，该功能允许科技巨头在不发布重大操作系统更新的情况下修改一些功能。

Windows Feature Experience Pack 中包含了一些隐藏的组件，预示着在未来部分功能将会从 Windows UI 中剥离，可以通过 Windows Update 分发的独立 Shell 包进行更新。

在去年发布的预览版 Build 18917（20H1）中，就包含了一个“Shell Update Agent”，它被设计成可以在需要的时候获取和更新 shell 或 Action Center等功能。

像开始菜单和 Action Center 这样的外壳组件，可以根据从微软商店获取的软件包，分别进行更新和修改。在预览版中，微软曾测试过这种方式来获取 Windows 10 Action Center 的 shell 包。

对于大部分用户来说，“Shell Update Agent”和“Windows Feature Experience Pack” 并没有太大的意义，不过对于微软 Windows Core OS 来说是有趣的发展。

微软已经通知PC厂商，“Windows Feature Experience Pack"是Windows 10的 “功能需求"的一部分，它不应该被删除，因为它"包括对Windows功能至关重要的功能”。Windows Feature Experience Pack 的大小只有44.15MB，包括一小捆应用，可以在不进行重大操作系统更新的情况下进行改进。该包目前支持文本输入面板、Windows Snipping工具和shell建议。
