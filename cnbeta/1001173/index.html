本田：Civic Type R限量版成为日本铃鹿赛道上最快前驱车

July 09, 2020

据外媒报道，上个月，本田发布了一个展示其Civic Type R限量版在日本铃鹿赛道上的跑圈视频，不过当时这家公司没有就此给出任何的背景说明。日前，本田为车迷们带来了关于这款新车的更多细节信息。据其介绍，Civic Type R限量版正式成为铃鹿赛道上跑得最快的前驱汽车。

Super GT赛车手Takuya Izawa于今年2月携手Civic Type R限量版在这个赛道上创造了2分23.993秒的漂亮成绩。据悉，当时使用的这款车为一辆欧洲规格的量产车。据悉，该成绩比之前的纪录保持者–雷诺Megane RS Trophy-R快了约1.5秒。

本田非正式地表示，限量版是为了成为一款时间攻击车，最终目标是夺回R型的冠军头衔，成为德国纽博格林-诺德施莱夫赛道上最快的前轮驱动车。Megane RS Trophy-R是目前Nordschleife FWD的冠军，所以我们很好奇R型是否能在世界重新开放的时候完成类似的壮举。

本田将生产1000辆限量版Type R，全车将被漆成凤凰黄，其中有600辆将投放美国市场。虽然306马力、2.0L的涡轮增压I4引擎没有改变，但这款车将配置经过特别调优的减震器、重新校准的转向系统和性能更高的轮胎。其中，美国版将比标准版轻46磅，这一切要归功于轻型的20英寸BBS车轮、更少的隔音材料以及对尾门雨刷、后方加热器管道和座后置物板的取消。欧洲版则将去掉其音响和空调系统，从而额外减轻30磅。

2021款本田Civic Type R限量版将在未来几个月上市，预计价格在4万美元左右。
