[图]PhotoGIMP补丁：让GIMP交互体验更接近Photoshop

June 25, 2020

不仅仅在 Linux 平台，GIMP 在 Windows 和 macOS 平台上一直是资深图片编辑的首选应用程序，被很多人认为是 Adobe Photoshop 的最佳替代品。不过由于 GIMP 采用不同的用户界面和交互逻辑，因此那些从 Photoshop 迁移过来的用户需要一段时间的适应。

虽然有一些方法能够让 GIMP 的使用体验接近于 Photoshop，但是操作步骤有点复杂。现在有一个名为 PhotoGIMP 的全新补丁，可以让 Linux 用户快速找到熟悉的感觉。

PhotoGIMP 适用于包括 Windows 在内的很多平台。hotoGIMP所做的就是调整GIMP，使其模仿Adobe Photoshop的界面，同时还增加了一堆额外的功能。例如，数百种新的字体、全新的应用图标和启动界面和全新的 Python 滤镜。

此外，PhotoGIMP 还设置了几个快捷键，能够让你像 Photoshop 那样工作。而这些快捷键都是基于 Adobe 分享的文档，所以总体来说，对于之前使用Adobe软件的人来说，一切都应该感觉比较熟悉。
