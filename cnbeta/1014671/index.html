三星显示宣布Note 20 Ultra为首个采用可变刷新率面板的移动设备

August 12, 2020

近年来，许多移动设备的屏幕刷新率已经从 60Hz 增强至 90Hz / 120Hz 及以上。尽管市面上已经有不少支持高刷新率的机型，但它们几乎无法实现刷新率的可变切换，因而无法更好地减少耗电量 / 续航损失。不过今日，三星显示器（Samsung Display）公司宣布，它们首次为移动设备提供了支持可变刷新率（VRR）技术的新一代显示面板。

更让人感到欣喜的是，本月发布的 Galaxy Note 20 Ultra 成为了首批采用 VRR 移动显示屏的设备，感兴趣的消费者可在两周内上手体验。

当前的移动显示屏，只能固定在 60、90 或 120Hz 的帧率，而无法根据实际显示的内容而自动连续地调节屏幕刷新率。

即便显示静态的画面，屏幕也只能固定在较高的刷新率，结果不可避免地需要面对功耗增加 / 续航减少的考验。

好消息是，Galaxy Note 20 Ultra 旗舰新品配备的 VRR 移动显示面板，能够以 120 / 60 / 30 / 10Hz 的模式运行。在此之前，从未有一款智能机支持如此低的屏幕刷新率。

除了节省能源消耗，新一代 VRR 显示面板还可消除较低刷新率时的亮度差异而引起的图像闪烁。

虽然该公司没有披露更多细节，但有传闻称其是在 LTPO 技术之上开发的（使之具有更高的开关性能和更低的能耗）。

为让新面板能够在 10-120Hz 范围内具有较 30 / 60Hz 固定刷新率更好的细粒度，该公司还需要在系统 / GPU 驱动 / 软件等层面进行深度优化。

至于实际能取得怎样的效果，还请耐心等待 Galaxy Note 20 Ultra 机型的上市与后续评测。
