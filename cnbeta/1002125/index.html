CDPR表示《赛博朋克2077》不会在首发时加入Xbox Game Pass

July 12, 2020

近日外媒Venturebeat编辑Jeff Grubb在预测7月Xbox游戏发布会内的文章中，提到了微软可能会将某款重磅第三方作品在首发时加入Xbox Game Pass服务并以此对抗PS5上的第三方独占，其中《赛博朋克2077》会是一个很好的选择。

此事随后引起了CDPR公关主管Radek Grabowski的注意，他在推文评论中表示“Jeff你需要另选个游戏来举例，《赛博朋克2077》没有加入Xbox Game Pass的计划”。另外早在今年5月微软发布的一条推文中也曾提到《赛博朋克2077》会进入XGP，但随后官方删除了这条并表示他们本意是想宣传《赛博朋克2077》Xbox One X限定主机，发布的信息有误。

《赛博朋克2077》将于11月19日登陆PS4、Xbox One与PC，包含中文语音与字幕。本作将于2021年内登陆PS5/XSX，本世代版支持免费升级至次世代版本。
