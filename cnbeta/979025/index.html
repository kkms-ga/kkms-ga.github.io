[技巧]如何启用Windows 10的DNS over HTTPS功能？

May 14, 2020

在近日的 Windows 10 预览版本中，微软让 Insider 测试者们有机会对通过安全超文本传输协议（HTPPS）传输的域名解析服务（DNS）进行测试。感兴趣的朋友可安装 Windows 10 build 19628 以上的编译版本（Fast Ring / 快速更新通道），然后手动启用 DNS over HTTPS（以下简称 DoH） 。如果不清楚自己的系统版本号，可按 WinKey + R 组合键，输入“winver”并查询。

访问:

微软中国官方商城 - Windows

确认系统版本号后，便可通过注册表编辑器等工具激活 DoH 客户端：

● 首先以管理员身份，在开始菜单中运行‘regedit’；

● 导航 HKEY_LOCAL_MACHINESYSTEMCurrentControlSetServicesDnscacheParameters；

● 新建名为‘EnableAutoDoh’的 DWORD 值、并设定为‘2’；

● 重启计算机以正式激活 DoH 功能。

需要指出的是，目前只有 Cloudflare、Google 和 Quad9 等大型网络服务商支持 DoH，下面是外媒 Softpedia 提供的一些参考。

如需更改 Windows 10 操作系统中的 DoH 域名解析服务器的设置，可参考如下方法：

网络与互联网 -> 网络与共享中心 -> 更改适配器设置 -> 邮件连接 -> 选择属性 -> 互联网协议 IPv4 / IPv6（请按实际情况选择）-> 属性 -> 手动指定 DNS 地址

本月晚些时候，Windows 10 Version 2004（又称 2020 五月更新）将向广大用户开始推送。

不过预计到今年下半年的时候，微软才会正式将 DoH 域名解析服务引入 Windows 10 秋季更新。
