OpenAI开始销售其文本生成技术 Reddit等企业为其首批客户

June 12, 2020

作为一家非盈利组织，OpenAI 致力于减轻人工智能的潜在危害。经过长久的努力，该公司已经发布了首款商业产品，一个此前被警告过可能有些危险、而未能向大家分享的“AI 文本生成系统”。其实早在去年 2 月，OpenAI 实验室就已经发布过 GPT-2 文本生成器，且这项工作赢得了广泛的好评，被认为是该领域向前迈出的重要一步。

用户可先在 GPT-2 中输入任何文本提要，比如几句歌词、一个简短的故事、甚至科学论文的片段。然后该软件会遵从某种匹配的样式和内容，设法继续编写下去。感兴趣的朋友可在此体验 GPT-2 的网页版本。

此前因担心该系统被用于恶意的目的（比如大量生成假新闻或垃圾邮件），OpenAI 不得不限制了 GPT-2 的发布。经过代码上的完善，且未发现其有被滥用的有力证据，OpenAI 才发布了该系统的更高级版本。

GPT-3 的规模是前者的百倍，现已成为 OpenAI 的首款商用产品。目前该文本生成 API 仍处于受邀访问阶段，售价也不得而知。猜测其有助于提升企业聊天机器人的会话流畅度，以及改进游戏体验等。

据悉，此类文本生成 AI 通常需要分析遍历庞大的文本语料资源库，并学会预测字母和单词。尽管听起来很是简单，但成品软件的种类相当繁多，针对的应用场景也非常灵活。

比如 GPT-2 已被用于创建从聊天机器人、到基于本文的地牢游戏生成器等一系列工具。由于其学会了通过查找以往的模式来生成数据，因此也可经过适当的调整，来应对下棋和解决数据问题。

目前 OpenAI 已经为 GPT-API 找到了大约十几名客户，其中包括搜索服务提供商 Algolia（借助该 API 来更好地理解自然语言搜索查询）、精神健康平台 Koko（分析用户何时处于“危机”之中）、Replika（构建了“AI 伴侣”）、以及社交媒体平台 Reddit（探索如何使用 GPT-3 API 来帮助其自动执行内容审核）。
