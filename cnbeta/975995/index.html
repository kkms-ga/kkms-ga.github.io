GitHub趣味项目让你重温打字机时代的按键噪声

May 07, 2020

机械键盘因独特的手感和声音而受到许多玩家的喜爱，但在追求安静的场合，其他人可能就对你有意见了。然而近期宅家无聊的个别人，似乎觉得机械按键的声音还不够响，于是在 GitHub 上发起了一个名叫“键盘打字机”（keyboard_typewriter）的项目。顾名思义，它可以借助 Arduino 开发板来重现打字机时代的噪音。

视频截图

这套系统的组成很是简单，任何人都可以参考 GitHub 上的指令代码来 DIY 。（Arduino 和 Python 代码传送门，请戳这里）

首先，它会通过 Python 代码来读取按键输入，接着通过串行连接将数据发送到 Arduino 开发板。

OK now I am really feeling like typing on typewriter（via）

Arduino 本身又是通过 USB 连接到 PC 的，然后通过一段代码来读取串行数据，并将之传送到螺线管，以做出相应的动作和发出声音。

虽然是视频看上去趣味十足，Arduino 开发套件也只需 20 美元就能买到，但我们并不建议大家轻易模仿，尤其当你和邻居只隔着薄薄的一堵墙的时候。
