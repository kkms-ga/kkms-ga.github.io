Apache 软件基金会 2020 财年报告：总代码价值超 200 亿美元

July 31, 2020

Apache 软件基金会近日公开发布了其 2020 财年（即 2019.05.01-2020.04.30）的年度报告。报告开篇提到，“作为世界上最大的开源基金会，Apache 软件基金会管理着 2 亿多行代码，并向公众 100% 免费提供价值超过 200 亿美元的软件。”

今年是 Apache 软件基金会成立的第 21 个年头，从最初的仅 21 名创始人发展至今，Apache 在报告中强调了其 “Apache 之道”（Apache Way）。报告中表示，正是这样一种社区驱动的 Apache 之道，促成了成千上万个开发者合作，使数百个项目走向成熟。报告还称，“地球上每个接入了互联网的国家都在使用 Apache 软件。”

2020 财年，Apache 软件基金会达成了以下主要成就：

添加了约 800 万行 Apache 代码，价值约 6 亿美元；总代码价值超过 200 亿美元；

Apache 资料库中共管理着超过 2.27 亿行代码；

选举了 34 名新的 ASF 个人成员，目前总计 813 名；

超过 7,700 名代码提交者；

206 个顶级社区，监督 339 个以上的 Apache 项目，以及数十个子项目和计划；

9 个最新毕业的顶级项目；

Apache 孵化器中目前有 45 个项目正在开发；

apache.org 每周页面浏览量超过 3500 万；

从 Apache 镜像下载的源代码大约有 2 PB；

最活跃的 Apache 项目前五名分别是：Kafka、Hadoop、Lucene、POI、ZooKeeper；

提交次数排名前五的 Apache 项目分别是：Camel、Flink、Beam、HBase、Lucene Solr；

按代码行排列的前五名 Apache 项目分别是：NetBeans、OpenOffice、Flex (combined)、Mynewt (combined)、Trafodion；

2,892 个提交者在 174,889 次提交中更改了 60,132,710 行代码；

12,413 人创建了 63,172 个新 issue；2,868 人关闭了 54,633 个 issue；

19,396 位作者在 1,417 个邮件列表中针对 907,870 个主题发送了 2,137,560 封电子邮件；

前 5 个最活跃的邮件列表：Flink、Tomcat、Royale、Beam、Lucene Solr；

2,045 个 git 存储库，包含约 250GB 代码和存储库历史记录；

Apache HTTP Server 成立 25 周年（ASF 成立 21 周年）；

签署了 748 份个人贡献者许可协议（ICLA）；

签署了 33 份企业贡献者许可协议；

签署了 40 项软件授权协议；以及

ASF 连续 15 年担任 Google Summer of Code 的指导。

完整报告可在此查看：https://www.apache.org/foundation/docs/FY2020AnnualReport.pdf
