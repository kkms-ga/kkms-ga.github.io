Alpine Linux 3.12.0发布 面向安全的轻量级Linux发行版

May 31, 2020

Alpine Linux 3.12.0 现已发布，这是 v3.12 稳定系列的首个版本。Alpine Linux 是一个面向安全的轻量级 Linux 发行版，该发行版以安全为理念，面向 x86 路由器、防火墙、虚拟专用网、IP 电话盒及服务器而设计。

另外，不同于常见的 Linux 发行版，Alpine Linux 采用 musl libc 和 busybox 以减小系统的体积和运行时资源消耗。在保持瘦身的同时，Alpine Linux 还提供了自己的包管理工具 apk，可以在其网站上查询软件包，或直接通过 apk 命令进行查询和安装。

3.12.0 更新内容如下：

新特性和值得关注的新软件包

对 mips64（big endian）的最初支持

对 D 编程语言的最初支持

重要软件升级

Linux 5.4.43

GCC 9.3.0

LLVM 10.0.0

Git 2.24.3

Node.js 12.16.3

Nextcloud 18.0.3

PostgreSQL 12.3

QEMU 5.0.0

Zabbix 5.0.0

升级说明

升级到 OpenSSH >= 8.2_p1 后，服务器将不再接受新连接，直到重新启动

下载地址：https://alpinelinux.org/downloads/
