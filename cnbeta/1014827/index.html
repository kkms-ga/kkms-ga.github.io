报道称三星正与ARM和AMD合作以打败新一代高通骁龙

August 12, 2020

近期有许多报道称，三星自家的新一代 Exynos 芯片组难以与高通骁龙 875 一战，但该公司似乎并不会轻易放弃。最新消息是，三星正与 ARM 合作开发定制的 Cortex-X 处理器核心、且下一代 Exynos 芯片组有望配备来自 AMD 的定制 GPU 。

去年 11 月，三星传出了将停止为 Exynos 芯片组开发定制 CPU 内核的消息。当时许多人猜测，2021 上半年的 Galaxy S30 系列可能会直接采用基于 ARM 公版方案的新一代芯片组。

不过《韩国商业新闻》的最新报告称，三星计划与 ARM 和 ARM 合作，以期成为第一大的 Android 应用处理器（AP）制造商（目前这个位置被高通把持着）。

换言之，就算当前一代的 Exynos 990 在性能方面无法与高通骁龙 865 / 865+ 一战，但下一代 Exynos 芯片组仍有望在多个方面取得巨大的进步。

CPU 方面，报告称三星正与 ARM 合作，在 Cortex-X 内核的基础上作定制开发。

今年 5 月份的时候，ARM 刚刚宣布了 Cortex-X1 内核设计，理论峰值性能较 Cortex-A77 提升 30%。

与更新的 Cortex-A78 相比，其单线程性能亦有 22% 的领先。

GPU 方面，当前的 Exynos 同样落后高通骁龙竞品不少。

为补足这项弱点，三星计划在 2021 年的 Exynos 旗舰芯片组上使用 AMD 定制 GPU 。

此外未来的 Exynos 芯片有望在神经处理单元（NPU）和调制解调器（Modem）性能上都较高通竞品扳回一局。
