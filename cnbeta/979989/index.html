三星Galaxy Note 20曝光：首发高通新一代3D超声波指纹

May 17, 2020

据SamMobile报道，三星Galaxy Note 20系列将首发高通新一代3D超声波指纹识别，比Galaxy S20系列更好用。报道指出，三星Galaxy Note 20系列有望是第一台支持同时读取两个指纹的旗舰，它将率先使用高通新一代3D超声波指纹识别，其面积比Galaxy S20大17倍。

SAMSUNG - 三星旗舰店

SamMobile称，这一代高通3D超声波指纹原计划是在Galaxy S20系列上率先使用，不过因为某种原因被放弃，而Galaxy Note 20系列成为这一技术的首发旗舰。

不仅如此，在高通新一代3D超声波屏幕指纹加持下，Galaxy Note 20系列的解锁速度更快、更安全。

此外，爆料称三星Galaxy Note 20系列包括Galaxy Note 20和Galaxy Note 20+两款，其中Galaxy Note 20屏幕为60Hz刷新率，Galaxy Note 20+屏幕为120Hz刷新率，二者都是挖孔屏方案，前者屏幕分辨率为FHD+，后者屏幕分辨率为QHD+。

核心配置上，Galaxy Note 20系列将搭载高通骁龙865旗舰平台，消息称全系标配16GB内存，这将是业界第一款全系标配16GB内存的旗舰手机。

在此之前，16GB内存仅在高配版本上出现，比如努比亚红魔5G、三星Galaxy S20 Ultra等等。

最后是大家关心的发布时间，爆料称Galaxy Note 20系列将在8月份亮相，届时Galaxy Fold 2有可能会同台发布，我们拭目以待。
