[图]微软承认六月累积更新导致Windows 10强制重新启动

June 24, 2020

适用于 Windows 10 May 2020（20H1/Version 2004）以及 Version 1909 功能更新的六月累积更新又出新问题了，它能迫使你重新启动系统。Local Security Authority Process 是 WinLogon 服务中负责验证用户身份的进程，而安装六月累积更新之后会破坏该进程并导致设备强制重启。

微软表示，Local Security Authority Process 问题将导致设备强制重启，并在桌面上显示“你的 PC 将在一分钟内自动重启”的警告。根据多名用户提供的报告，错误信息中写道：“一个关键的系统进程C:Windowssystem32lsass.exe失败，状态码为c0000008。现在必须重新启动机器”。

微软随后承认 Windows 10 Version 1809/1903/1909/2004 功能更新均存在这个问题。不过好消息是微软已经在研究解决方案，计划在即将到来的新版本中提供修复，预估将会在7月14日发布。
