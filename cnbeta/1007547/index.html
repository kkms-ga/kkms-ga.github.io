网友分享Fluent设计风格的Edge浏览器UI 赋予养眼的半透明标题栏

July 25, 2020

近年来，Windows 10 用户一直希望微软能够为 Edge 浏览器引入完整的 Fluent Design 设计语言，比如用户界面（UI）中的半透明标题栏。随着 Chromium Edge 的快速发展，以及在 Windows 和 Mac 上的普及（另有 Linux 版本），等不及的网友还是分享了自己对于这一 UI 设计里面的观点。

Reddit 网友 u/saucojulian 构想了上图所示的这种 Edge 浏览器的 UI 设计风格，可知其带有养眼的标题栏、选项卡、以及圆角边框，与微软提倡的“Modern”设计理念不谋而合。

遗憾的是，微软已于早些时候证实，Microsoft Edge 中的 Fluent Design 已经定型，意味着短期内不会有任何重大的更改。
