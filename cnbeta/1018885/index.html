[图]UWP Remote Desktop应用更新：支持深色/浅色模式 支持ARM64

August 22, 2020

Microsoft Remote Desktop UWP 最新预览版中获得了重大更新，新增支持浅色/深色模式，并添加了对 ARM64 的支持。该应用程序还使用了和 macOS 以及移动端相同的 RDP 核心引擎。

以下是完整的更新日志：

● 重写客户端，使用了和 iOS、macOS 以及 Android 客户端相同的底层 RDP 核心引擎。

● 增加了对 Azure Resource Manager 版 Windows Virtual Desktop 的支持

● 增加了对 x64 和 ARM64 的支持

● 将侧板设计为全屏

● 增加对浅色和深色模式的支持

● 增加了订阅和连接到主权云部署的功能

● 添加了发布到生产 (RTM) 中启用工作空间 (书签) 的备份和恢复。

● 更新了在订阅过程中使用现有 Azure Active Directory (Azure AD) 令牌的功能，以减少用户必须登录的次数。

● 更新的订阅现在可以检测你是在使用Windows虚拟桌面还是Windows虚拟桌面（经典）。

● 修正了将文件复制到远程电脑的问题。

● 修正了通常报告的按钮的可访问性问题。
