WoA项目日趋成熟：Lumia 950移植Windows 10变得更加稳定

June 19, 2020

2018年，开发者已经成功将 Windows 10 移植到 Lumia 950/950 XL上；去年，开发者在一加6、Galaxy S8、小米Mix等设备上成功移植 Windows 10 on ARM。不过最初的移植只是成功运行，很多功能无法调用，性能表现也非常糟糕，也非常的不稳定。

随后在开发者的不懈努力下，已经成功在 Lumia 手机上移植 Windows 10 Build 19041（20H1）版本。现在，Lumia 950/950 XL 可以运行 ARM64、ARM 和 x86 桌面应用，可以调用诸如光线传感器、加速计陀螺仪等大部分手机传感器。

开发人员还创建了一个自适应的 Shell，在模仿 Windows 10 Mobile 的外观同时具备桌面功能。目前“MobileShell”已经在 Microsoft Store 上架，可以让运行 Windows 10 的 Lumia 用户使用。而且当你切换到平板模式时候，该 Shell 还能提供返回导航、状态栏、通知等功能。

热心的开发者还开发了“WOA Dialer”，能够让你在设备上拨打和管理通话。不过蜂窝网络支持还在开发中，可能不会支持所有通信运营商和网络。还有一款名为 “Chat “的应用，你可以用它来收发短信。目前，还不支持彩信消息。

开发人员还测试了Windows 10预览版构建19628.1，似乎运行良好。
