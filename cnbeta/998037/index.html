[图]Mozilla发布Firefox 78.0.1紧急维护版本： 修复搜索引擎故障

July 02, 2020

昨天 Mozilla 面向所有支持的桌面平台发布 Firefox 78 稳定版更新，仅过去一天时间，该公司又推出了新的维护版本更新。Firefox 78 更新幅度较大，很多用户在升级之后反馈浏览器的搜索功能无法正常使用。

下载地址：https://ftp.mozilla.org/pub/firefox/releases/78.0.1/

由于这个问题在很多设备上都能复现，因此 Mozilla 已经决定暂停该版本的升级，转而直接推送 Firefox 78.0.1 版本更新。在 Firefox 78.0.1 的更新日志中写道：“修复了从此前版本升级之后搜索引擎不可见的错误”。

根据外媒 TechDows 从 Bugzilla 帖子中汇总的信息，Firefox 78 稳定版主要遇到了三个搜索方面的问题

● 所有的搜索引擎都消失了，一键式搜索引擎是空的

● 地址栏的自动完成无法工作

● 开始页面的搜索功能无法执行搜索

目前，主要是 Windows 平台的用户反馈这个问题，但不排除其他桌面平台也存在这个问题。对于导致搜索功能无法正常工作的原因，一位 Mozilla 工程师解释说：“在 Firefox 78 中搜索服务现在使用远程设置，如果索引DB无法工作，那么就会导致其他很多东西无法正常工作。不幸的是，我们的搜索服务刚刚切换到使用那个本地数据库，因此无法正常工作。”
