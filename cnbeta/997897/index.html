荣耀X10 Max 5G价格曝光：2399元起步

July 01, 2020

荣耀明天将发布荣耀X10 Max 5G 和荣耀30 Lite（青春版）5G 智能手机。昨天，荣耀X10 Max 的清单已经出现在中国电信的数据库中。今天微博则曝光了新的图片，其中提到了荣耀X10 Max 5G 在线下市场的定价，而且该设备的实机图片也出现在微博平台上。

来自@官方的暗示曝光的照片中提到了荣耀X10 Max 5G 屏幕的一些重要信息。包括采用 7.09 英寸支持 RGBW 技术的全面屏。以及支持 P3 电影级色域，780nits 亮度和 HDR10。还获得了德国莱茵 TUV 的低蓝光认证。

荣耀X10 Max 拥有 5,000mAh 电池。除了图中提到的 8GB RAM + 128GB 存储版本之外，X10 Max 还有其它版本。它提供黑色，银色和蓝色。此外，该设备还支持 NFC 连接。

另一张来自@五月数码科技的图片显示，荣耀X10 Max 5G 的 6GB RAM + 64GB 存储版本定价为 2399 元。而较高的 6GB RAM + 128GB 存储价格为 2599 元。目前没有提到 8GB RAM + 128GB 存储的价格。

以往的报道表明，荣耀X10 Max 的前置摄像头为 800 万像素，背面为 4800 万像素+ 200 万像素的双摄像头系统。该手机搭载天玑800 5G 芯片，并装有基 于Magic UI 3.1.1 的 Android 10 OS。
