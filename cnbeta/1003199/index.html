OWC推出带千兆以太网端口的Thunderbolt 3 Mini Dock

July 14, 2020

OWC新发布了一款小型Thunderbolt 3扩展坞，专为随身携带而设计，让MacBook Pro用户可以在旅途中连接多台显示器和设备。这是继贝尔金为MacBook Pro推出双供电的Thunderbolt 3 Dock Core之后，OWC现在又推出的一款可以驱动4K显示器的迷你扩展坞。

OWC Thunderbolt 3 Mini Dock带有两个4K HDMI连接、两个USB 3.0接口和千兆以太网，可以更有效利用计算机的Thunderbolt 3 40Gb/s能力，为两台显示器提供支持，最高可达4096 x 2160 4K @ 60Hz，而不会影响计算机本身的性能。

OWC表示，该设备将允许用户连接两台显示器，并具有HDCP 2.2兼容性，可 “毫不费力地串流4K UHD内容”。除了支持显示器，迷你扩展坞还提供 “HDMI精密多声道音频，如LCPM、DTS和杜比数字”。

Thunderbolt 3扩展坞旨在减少MacBook、MacBook Air或MacBook Pro机型在端口有限的情况下插拔设备的需求，同时实现足够小的体积，方便在办公室或旅途中使用。

此外，扩展坞包装里还附赠了一条 7.2 英寸的 Thunderbolt 3 线缆，因此用户无需购买单独的线缆。

OWC Thunderbolt 3 Mini Dock的售价为119美元，现在就可以订购，7月27日起发货。
