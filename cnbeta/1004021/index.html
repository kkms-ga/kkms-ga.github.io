亚马逊推出IVS互动视频服务 采用与Twitch直播平台相同的技术

July 16, 2020

亚马逊云服务（AWS）今日宣布了一项“互动视频服务”（简称 IVS），借助这项完全托管的新功能，客户可快速为 Web 或移动 App 配置可实时交互的流媒体视频。如果你留意过亚马逊旗下的流媒体直播平台（Twitch），就会发现两者基于相同相同的幕后技术。

（来自：AWS）

据悉，Amazon IVS 会提取内容，然后自动对其进行代码转换和优化，以使之可借助与 Twitch 相同的技术来传输互动视频。只需几秒钟的时间，即可在遍布世界各地的 AWS 基础架构中完成实时交付。

通过专为流媒体视频用户而量身定制的基础设施，开箱即用的 Amazon IVS 可将延迟控制到三秒左右。AWS 还宣称其易于设置，仅需几秒即可启动并运行。

借助捆绑的软件开发套件（SDK）和应用程序接口（API），主播还可与观众进行实时互动，包括虚拟聊天室、民意调查、直播问答、以及同步促销等元素。

（Amazon IVS 定价标准）

亚马逊表示，用户无需为使用 Amazon IVS 而顾虑任何前期或额外费用，意味着它采用了随用随附的定价模型。

目前 Amazon IVS 已提供了基础和标准两个版本的套餐，主播们可根据视频时长或观众数量来付费。

按照直播源（输入内容）来算的话，前者的定价为 0.20 美元 / 小时，最大支持 480p @ 1.5Mbps 的标清码率。

如需升级到最大支持 1080p 全高清的 8.5 Mbps 码率，则需按照 2.00 美元 / 小时的标准付费。

至于内容输出的成本，也因市场区域的不同而异，详情请移步至 AWS 官网查看。
