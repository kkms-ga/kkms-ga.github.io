三星Galaxy Note 20为何要用60Hz屏？行家揭秘

July 24, 2020

7月24日消息，随着三星Galaxy Note 20系列发布时间的临近，有关Galaxy Note 20系列的参数陆续揭秘。根据爆料的信息，三星Galaxy Note 20标准版采用的是60Hz AMOLED显示屏（直屏），分辨率为FHD+，甚至传闻它采用的是塑料后盖。

另一方面，三星Galaxy S20 Fan Edition却采用了120Hz AMOLED显示屏，传闻其价格比60Hz的Galaxy Note 20更低。

为什么三星Galaxy Note 20标准版用60Hz AMOLED屏反而比120Hz的三星Galaxy S20 Fan Edition更贵呢？

分析师Ross Young表示，Galaxy S20 Fan Edition使用的是硬（刚）性OLED屏，而三星Galaxy Note 20使用的是柔性OLED屏，前者的不足是更厚、更重而且能显著降低成本。

据悉，三星Galaxy Note 20采用了6.7英寸直屏，搭载Exynos 990处理器，配备8GB内存+256GB存储，后置1200万像素主摄+1200万超广角+6400万长焦三摄，支持30倍数码变焦，前置1000万像素，电池容量为4300mAh。

该机将于8月5日发布。
