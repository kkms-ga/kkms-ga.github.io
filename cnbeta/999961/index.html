[图]Android端谷歌Sheets, Docs和Slides获得深色模式

July 07, 2020

适用于 Android 端的 Sheets, Docs 和 Slides 近日陆续获得深色模式（Dark Mode）。谷歌表示，深色模式将“智能调整产品界面和用户生成的内容”，能够在光线不足的较暗环境中提供良好的阅读体验。

这些应用程序可以跟随系统设置来启动深色模式。不过，用户也可以前往菜单>设置>主题>深色，切换到浅色主题。此外，用户还可以通过“…”菜单，在“以浅色模式浏览”选项进行切换。内容中的颜色不会受到模式变换的影响。
