《微软飞行模拟2020》已推出首个公共SDK

August 22, 2020

上周，开发人员回答了《微软飞行模拟》社区关心的一些主要问题。随着该游戏于 8 月 18 日在 PC 上正式发布，陆续到来的玩家报告，也丰富了该作的已知问题列表。同一天，《微软飞行模拟 2020》还加入了 Xbox Game Pass for PC 的订阅全家桶。现在，软件巨头又宣布了该游戏的首个公共 SDK 。

（图自：Xbox Wire）

除了首个公共 SDK，微软还宣布了新的合作伙伴系列更新，主要展示必应地图为环游世界系列带来的精彩画面。

为了解决最近上报的一些问题，幕后团队也在努力提供补丁更新。有关此问题的更多信息，将在下周的常规更新中提供。此外在支持页面上，微软也给出了针对某些已知问题的解决方案。

（图自：Flubberzwans）

在一段视频中，微软还介绍了 Asobo Studios 与必应地图服务的合作关系，即如何利用现代地图技术和大规模空间数据。然后微软概述了新版 SDK 的一些改进：

“自 Closed Beta 以来，我们已在多个方面带来了改进，比如滑行道标志、线条绘制、以及灯光和导航数据”。

此外《微软飞行模拟 2020》解决了开发人员模式下的许多问题和 bug 修复，现在可选择立即构建整个项目，而无需开发人员逐包构建。

对于复杂机型，WebAssembly 仍在积极开发中。团队正在研究一些急需的功能，为移植现有的 C++ 代码而提供帮助，且致力于映射 SIMConnect API 的功能。

Microsoft Flight Simulator - Oceania - Around the World Tour（via）

最后是题为《环游世界》的一个新系列，首集已于本周早些时候公布。除大洋洲外，该系列还将带玩家领略亚洲、非洲、欧洲，北美、南美、中美洲，以及加勒比海等地区的风光。

下一集的内容将聚焦亚洲，并深入探索东西半球、大洋洲、美拉尼西亚、密克罗尼西亚、以及波利尼西亚等地区。如有任何问题，可通过 @MSFS_Support 渠道提交报告。
