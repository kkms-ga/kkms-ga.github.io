[技巧]从Windows 10中删除Cortana已较此前更加容易

August 11, 2020

尽管微软努力将 Cortana 打造成一款有助于生产力提升的智能语音交互工具，但并非所有 Windows 10 用户都希望看到它在系统中到处刷存在感。庆幸的是，在 Windows 10 20H1（Version 2004）更新后，用户已能够较以往更加轻松地删除 Cortana 功能组件。

过去几年，以微软 Cortana、亚马逊 Alexa、苹果 Siri 和 Google Assistant 为代表的智能语音助理已经变得相当受欢迎，且其在许多设备上都能如用户期待的那样工作。

不过现在，微软已将 Cortana 从消费者为导向，转换成专注于生产力的提升，因而更适合作为 Microsoft 365 订阅服务的一个增强补充。

据悉，更新后的 Cortana 迎来了基于聊天式的新界面、改进了键盘支持、并且能够在桌面上拖动调整窗口位置。

不过在 Windows 10 2004 的 Microsoft Store 应用商店中将它卸载之后，你将无法利用 Cortana 来朗读电子邮件、在 Groove Music 召唤其播放音乐、以及无法使用聊天 / 基于语音命令启动的应用程序。

如果你实在用不到 Cortana，可参考以下操作：

（1）打开 Windows 搜索；

（2）以管理员权限运行 PowerShell；

（3）输入以下命令并回车 ——

Get-AppxPackage -allusers Microsoft.549981C3F5F10 | Remove-AppxPackage

（4）关闭PowerShell；

（5）重启系统。

完成上述操作之后，你将无法在 Windows 10 已安装程序或系统设置中找到 Cortana 的身影。

即便如此，这番操作并不会彻底消除 Cortana 的所有痕迹，比如任务栏中仍可看到 Cortana 按钮（可通过鼠标右键菜单来取消显示）。

此外在删除 Cortana 之后，WinKey + C 组合键将失效，但你仍可借助 PowerToys 的键盘重映射（Keyboard Remapper）功能来指定触发其它操作。

如果日后还想继续使用 Cortana，微软也提供了后悔药选项，只需前往 Microsoft Store 并重新下载即可。
