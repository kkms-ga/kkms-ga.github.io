新核显加入 十代酷睿轻薄本如何提供多样选择？

May 14, 2020

自十代酷睿低压 CPU 阵容上市至今，搭载两种制程的十代酷睿低压 CPU ——新10nm 制程的“Ice Lake”产品线与14nm++ 制程的“Comet Lake”的众多笔电产品已经陆续上市。在性能上，不同制程的十代酷睿低压 CPU 系列并无太大区别，但因为这种 CPU 系列上的调整，用户能在市场上看到更多细分的选择：

比如在今年，用户还能看到不少厂家的笔电在同样的型号/模具之下，同时发布搭载 Comet Lake 与 Ice Lake CPU 的两款版本。

在同样的 TDP（Thermal Design Power，CPU 设计散热功耗）之下，首发新 10nm 制程工艺的 Ice Lake 所搭载的 Iris Plus 锐炬 为笔电图形处理性能带来了不小的升级；而 Comet Lake 在睿频提升且无需给核显留出空余功耗设计，在 CPU 单核/多核主频与睿频都能更高；计算性能释放更强。

简而言之，十代酷睿 Comet Lake 制程的低压 CPU 在计算性能上要比同代 Ice Lake CPU 更强；而 Ice Lake CPU 在读取/存储速度与游戏性能上会有更好的原生优势；两系列虽然在性能提升与新增功能上相同，但却针对不同的使用场景下各有侧重；在实际上市的笔电设计上，OEM 笔电厂商也会针对某款笔电的目标用户群体，进行更加针对性的硬件搭配设计；这种设计理念在 CPU 性能侧重更加细化的十代酷睿低压 CPU 系列上更加明显。

十代酷睿低压 CPU 阵容变化，已经非常明显地体现在已上市的笔电产品设计上；而对于用户而言，则更需要从 CPU 、笔电综合性能以及售价上来评估，更加多元化的考量笔电是否能更加适合自己的需求；在目前上市的十代酷睿低压笔电新品中，选择出一款适合自己日常使用需求的轻薄本。

高性价比独显轻薄本：颜值/性能/拓展性，三维选择

在目前已经发布的产品阵容中，不乏兼顾颜值/性能/售价的笔记本产品，戴尔新灵越5000 就是这样一款典型的轻薄本，外型精巧简洁，机身轻盈便携，搭载了 i5-1035G1 的同时，还有一块 MX230 2G 独显。完美地平衡了日常学习、办公和轻娱乐的使用需求。

戴尔DELL灵越5000 14英寸酷睿i5 (十代i5-1035G1 8G512G MX230 2G独显)

同时，灵越 5000更多的接口以及双硬盘位带来的更强拓展性；以及窄边框 IPS 防眩光屏幕，同样能打动更多轻薄本用户。

同样具有代表性的还有联想ideapad系列推出入门级轻薄本ideapad 14s：与灵越 5000 不同，联想在ideapad 14s上选择搭载了 Comet-Lake 架构的十代酷睿 i5，但同时也搭载了性能更强的独显—— MX330 实现了更强的图形性能提升，十代酷睿低压CPU 的睿频提升也让性能释放更加稳定；相比于同代核显笔电，采用 MX330 的联想ideapad 14s图形性能能得到翻倍的提高，在应付游戏与图形处理工作时更加游刃有余，更适合需要轻薄笔电来承担更多不同场景工作的用户群体。

高性能专业核显笔电：极致轻薄生产力

之前的文章中，笔者曾介绍过，十代酷睿低压 CPU 系列的产品线之下，有着不少值得细讲的“支线升级”，比如核显性能的升级与新制程带来的读取/存储速度提升，从各家媒体的测试中也可以看出，带着新一代锐炬核显的 Ice Lake CPU 系列无论是在游戏性还是拓展使用场景上，都比前代低压处理器的UHD 核显轻薄笔电来的更加游刃有余。

虽然相比于独立显卡，目前的低压处理器核显性能提升还不能与之相提并论，但对于传统意义上的轻薄笔记本 —— 不过分强调性能、主打拓展性与便携续航这个定义下的轻薄本产品，通过提升核显图形处理性能，来改善笔电拓展性，也的确是十代酷睿低压笔电的一个重要的升级趋势。

在传统轻薄本笔电设计上，笔电厂商往往采用了无风扇的散热内部结构设计，来让笔电更加轻薄，这种形态的笔电往往更加需要高性能的核显性能而非性能更高的独立显卡：保障独显性能发挥需要更加强劲地散热性能，这与追求极致轻薄的商务笔电设计要求背道而驰；所以核显性能的升级，就是轻薄笔电形容升级的优先需求之一。

比如在Surface Laptop 3 上，我们就能看到这种设计思路的存在，SurfaceLaptop 3 通过搭载 i7-1065G7 这样英特尔阵营顶级核显 CPU，来让传统轻薄本拥有可以进行简单的图形处理以及运行主流网游的能力；不过作为传统形态笔电，Surface Laptop 3 仍然选择了单风扇+单热管来保证更好的性能发挥；与之相对的，更加追求轻薄机制的二合一笔电 —— Surface Pro 在第七代在搭载了 i5-1035G4 的同时采用了无风扇设计，来做到类似平板电脑的轻薄机身下，释放出更多笔电的“生产力”。

Surface Pro 7 二合一笔电 第十代酷睿i5 8G 128G SSD

当然，在这个轻薄本/二合一笔电的设计理念之下，我们也能在 2020 年看到采用 Comet Lake 架构的笔电：前面笔者也曾提到，Comet Lake 架构下的十代酷睿低压 CPU ，在同样的 TDP 标准之下，能有更好的运算性能释放；对于追求极致运算性能＋轻薄机身的商务人群来讲，这种笔电在日常工作场景中更“趁手”：目前在十代酷睿低压笔电中，也已经有如联想 Yoga C740 这样二合一笔电上市 —— 当然，仅依靠 UHD 核显也没有采用任何独显的这种轻薄本，在拓展性上就要比采用了独显与新一代 Iris-Plus 核显的十代酷睿低压轻薄本要差很多了。

结语：更加细分的轻薄本市场

虽然十代酷睿在制程/核显差异变化下，让传统的轻薄本产品线增加一定的复杂性，让用户产生了一定的混乱感；但通过上文的梳理，相信你也能区分不同系列的十代酷睿低压 CPU 轻薄本究竟是为哪些群体而“设计”，而在不同的价位之下，不同的用户群体也能找到更多主打差异化功能的笔电上市。

同时，在英特尔 10nm 低压 CPU 搭配Iris-Plus 核显带来的图形处理性能提升，虽然在目前还无法完全取代独显，但同样给了追求极致轻薄的笔电更高性能的商务办公人群更多选择；而十代酷睿低压 14nm++ 产品线运算性能的整体提升，也让追求“高性能轻薄本”这样的厂商，除了能推出更加极致的轻薄高性能笔电之外，也能在性价比的框架之下，做出更具有性价比的独显轻薄本。

除了十代酷睿低压版的性能提升之外，笔电厂商还能通过搭配更加强力的独显，在轻薄本范畴之内来满足更多的创作需求与拓展使用场景：这也是 2020 年轻薄本在发展上的新方向之一。

目前，凭借着十代酷睿覆盖宽泛的产品线，在各种用户的不同各类细分需求之内，相应的笔电产品推出：在新核显与主频/核心数的提升下，无论是续航、性能；还是轻薄外观、屏幕素质与拓展性，都能找到可以满足自己实际日常使用需求的那款十代酷睿轻薄本。
