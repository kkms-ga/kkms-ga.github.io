速度堪比SSD 索尼推世界首款CEA-G存储卡：160GB售价4199

July 29, 2020

今晚索尼发布了全画幅微单Alpha 7S III，除了带来了全新的4K 120p视频之外，这次还使用了双卡槽设计，索尼也一道推出了TOUGH三防CFexpress Type A存储卡CEA-G系列，及配套的读卡器，读取速度达到了800MB/s，比常见的SSD还要快，不过售价也不便宜，160GB要4199元了。

Alpha 7S III的拍照及视频要求都很高，它具有两个CFexpress Type A卡槽，还支持UHS-I和UHS-II SDXC/SDHC存储卡，用于同时记录或扩展连续记录，所以存储卡也不能拖后腿。

这次索尼推出的CFexpress Type A存储卡具备高速度、耐用性和可靠性的TOUGH三防规格，有2种容量，分别是80GB的CEA-G80T和160GB的CEA-G160T。

性能方面，CEA-G系列存储卡的读取速度为800MB/s，写入速度700MB/s，应付4K视频录制毫无压力，同时还有先进的散热性能，并且防尘防水，还支持File Scan Utility存储卡检测软件和Memory Card File Rescue数据恢复软件。

除了两款CEA-G系列存储卡之外，索尼还推出了配套的MRW-G2读卡器，支持SD卡及CFexpress Type A存储卡，USB 3.2接口，速率10Gbps，可以提供更快的数据传输。

最后是售价，80GB的CEA-G80T为2199元，160GB的CEA-G160T存储卡则要4199元，MRW-G2读卡器也要1399元，也是在9月下旬上市。
