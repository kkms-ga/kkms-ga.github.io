稳步推进普及：postmarketOS现已基于Alpine Linux 3.12稳定版

June 01, 2020

作为一个雄心勃勃的项目，postmarketOS 开发团队希望让 Linux 操作系统能够在移动设备上变得更加普及。经过持续 3 年的努力，其已经抵达几个重要的里程碑，包括在不少于 200 种设备上启动。最新消息是，postmarketOS 现已基于 Alpine Linux 3.12 稳定版打造。

PinePhone 上运行的 postmarketOS / Plasma Mobile

开发团队解释称，postmarketOS 现已基于 Alpine Linux 3.12 稳定版打造。尽管主渠道仅支持 QEMU 虚拟设备，但目前已可在超过 200 部手机上启动。

取决于正在运行的功能和移植的维护状况，所有设备都已经迁至测试通道（testing）。可选支持社区（community）或主分支（main），不过只有主分支的设备才属于特挑稳定。

另有一些设备正等着从测试通道转移至社区通道，包括诺基亚 N900、红米 4X、Moto G4 Play、三星 Galaxy A3 / A5 / S4 Mini、以及 Wileyfox Swift 等机型。

采用 MSM8916 芯片平台的设备已支持主线 Linux 内核，采用 MSM8974 芯片平台的设备也属于候选，比如 Nexus 5、Fairphone 2、一加 1 和三星 Galaxy S5 。
