垄断者 Google Chrome：曾经的屠龙者如今也变成了龙

June 05, 2020

都 2020 年了，桌面浏览器还重要吗？答案是：重要，而且是非常重要。毕竟，只要用电脑上网，就无法避免使用桌面浏览器，实际上，浏览器是如此不可或缺，以至于它就像空气一样常常被人忽略。不仅如此，在桌面浏览器领域，依旧是硝烟弥漫，余波不断，还在引起新的担忧。

毕竟，整个桌面浏览器领域，基本上成为 Google Chrome 的天下了。

Google 已主宰桌面浏览器

近日，根据市场调研机构 NetMarketShare 的最新调研数据，2020 年 5 月，在全球桌面浏览器市场，Google Chrome 的占比高达 69.81%，接近 70%。

这就意味者，全球范围内，10 个电脑用户差不多有 7 个都在使用 Google Chrome。

在 Google Chrome 之外，第二名是微软的 Edge 浏览器，占比 7.86%，第三名是 Firefox 占比 7.23%。

在这三者之外，其余的占比都在 5% 以下，甚至包括苹果 macOS 中自带的 Safari 浏览器，占比不过是 3.9%。

可见，Google Chrome 已经成为桌面浏览器领域无可争议的王者。

不仅如此，如果从底层技术来看，Chrome 作为一款商业浏览器，它本身是基于 Chromium 开源浏览器内核平台的，同时，微软 Edge 浏览器、QQ 浏览器、搜狗浏览器等也是基于 Chromium 内核。

从这个角度来看，全球范围内，基于 Chromium 内核的桌面浏览器，其市场份额加起来已经超过 80%。更要命的是，Chromium 本身也是 Google 主导开发并对全世界开源的。

这就意味着，整个人类桌面浏览器的底层技术和标准，都已经被一家美国商业公司所主宰，而这家公司就是 Google。

不仅如此，由于桌面浏览器与 Web 网页的密切关联，所以实际上整个网页的技术发展方向，也不得不受制于 Google 的技术和标准选择——在某种程度上，Google 也在主导 Web 的未来。

这对 Google 来说，无疑是一种巨大的荣耀，也是其强大技术实力和技术影响力的重要支撑。

作为 “屠龙者” 的 Chrome

在获取今天的地位之前，Google Chrome 曾经是一个 “屠龙者”。

这条龙，就是微软公司曾经显赫一时、主宰天下的 IE 浏览器；它在上个世纪 90 年代曾经有过一场激烈的浏览器大战，并借用各种不光彩的手段获得主宰地位，其中一项，就是 Windows 操作系统的捆绑。

那么，IE 当时有多辉煌呢？

从 1998 年底战胜网景浏览器开始，IE 开始统领并主宰整个浏览器市场。根据 OneStat.com 的数据显示，到 2002 年 4 月份，IE 的市场份额高达 96.6%——这是 IE 的顶峰。

后来，IE 在自身的问题和 Firefox 等竞争对手的冲击之下，开出现明显的市场份额下滑，在此后四五年间，Firefox 开始侵蚀 IE 的市场份额，一度超过 10%。

等到 Chrome 开始公测的 2008 年， Firefox 的份额停留在 15% 到 20% 之间，但 IE 依旧占据将近 80% 的份额，虽然是持续下滑之势，但依旧堪称主宰。

这时候，Google Chrome 横空出世，成为 IE 的真正大敌。

起初，在 Google 内部，浏览器的开发是被当时的 Google CEO Eric Schmidt 所反对的，但他最终听从了 Google 两位创始人 Larry Page 和 Sergey Brin 的建议。

值得一提的是，当年 Chrome 浏览器的开发，就是如今的 Alphabet CEO 兼 Google CEO Sundar Pichai 主导的。

2008 年 9 月，Chrome 浏览器正式开始公测，在 100 多个国家开放下载——当时，IE 8 Beta 2 版本刚刚发布几天，Firefox 3.0 也已经发布近 3 个月且获得了吉尼斯世界纪录。

但很快，Chrome 就展现出了挑战对手的实力。

浏览器领域的 “三国杀”

是的，桌面浏览器领域，曾经上演过一出 “三国杀”，参与者分别是 Google Chrome、Microsoft 的 IE 和 Firefox。

但其中的亮眼角色，就是 Google Chrome。

站在今天的眼光来看，Chrome 几乎可以说是重新发明了浏览器；当下网页浏览器的许多人性化功能几乎都来自于 Chrome。

比如说标签栏置顶，将搜索栏和地址栏合二为一，高度简约的界面设计，面向开发者的扩展程序商店，书签可以通过 Google 账号在云端同步……这一系列功能的出现划时代地提升了人们的网页浏览体验。

而且更为重要的是，在保持快速迭代速度的同时，Google 还设计出了一种极为有效的开发模式，它将 Chrome 分为稳定版和开发版，前者用来满足普通用户，后者方便开发者进行功能更新和测试，从而满足不同类型用户的需求。

因此，在 Google 深厚的互联网基因和强大的技术背景的支撑下，Chrome 大受欢迎，尤其是对于喜欢追求新技术的开发者而言。

2010 年底，Chrome 在推出两年多之后夺取了 10% 的市场份额。与此同时，IE 的市场份额继续下跌，低于 60%，而 Firefox 则维持在 20% 左右。

整个浏览器市场初步形成了由 IE、Firefox 和 Chrome “三分天下” 的局面。

当然 IE 和 FireFox 都在进步，在很多方面也向竞争对手尤其是 Google Chrome 学习，但 Chrome 就像是一个年轻的跑步健将，在快速前行中跟上并逐步超过对手。

但 Chrome 更加幸运的一点在于，它遇见了移动互联网的大潮。

连微软也选择了 Chromium

Chrome 是幸运的，因为它的背后是 Google，而 Google 拥有 Android。

2010 年底到 2016 年初的那段时间里，是 Android 飞速成长并占据移动操作系统主宰权的时候，也是 Chrome 借势成长的时候。

在这段时间里，Chrome 的市场份额在原有的基础之上稳步提升，IE 浏览器则继续稳步下降，至于 Firefox，则是在 10% 和 20% 之间徘徊，后来又呈现出下降的趋势。

根据 Net Applications 的数据显示，在 2015 年 12 月全球浏览器市场上，IE 份额跌至 48.57%；Chrome 则持续上涨到 32.33 %；而 Firefox 已经下降到为 12.13%。

很难具体阐明，在这五年时间里，移动互联网大潮究竟如何以及在何种程度上影响了桌面浏览器市场的发展，但 Chrome 无疑是巨大的受益者。

简单来说，Chrome 继续凭借跨越桌面、平板和智能手机等多终端的统一体验，吸引着更多的用户。而微软却在移动互联网大潮中严重缺位，IE 孤立无援，并在 2015 随着 Windows 10 和 Edge 浏览器的发布而走向边缘化。

此后，Chrome 在市场份额的成长道路上，可以说是一马平川了。

根据 NetMarketShare 的数据显示，2016 年 6 月到 2017 年 6 月，Google Chrome 的市场份额由 48.65% 增长到了 59.36%，而 IE 由 31.65% 减少到 17.55%，Edge 浏览器的占比都只不过是个位数。

这时候，Chrome 就已经占据半壁江山，而微软家的 IE 早已是明日黄花了——但微软并不想完全放弃，它还寄予希望于 Edge。

然而，到了 2018 年 12 月，微软实在没有耐心，宣布未来将基于 Google 此前推出的开源项目 Chromium 打造 Edge 浏览器——这意味着，微软基本上在桌面浏览器领域放弃了与 Google 的抗衡，并由此承认了 Google Chrome 的浏览器老大地位。

此后一年多时间里，基于 Chromium 的 Edge 浏览器正式发布，它的市场份额比以往略有上升，但也在个位数。

发展到现在，Google Chrome 的市场份额大概是 Microsoft Edge 的 9 倍。

Chrome 引起了新的担忧

曾经的屠龙者，如今也变成了龙。

Google Chrome 就是这样一条 “龙”，它目前依旧是最好用的桌面浏览器之一，但它也逐渐有了 “吃内存” 、追踪隐私等恶名，功能上也显得有些臃肿。

在桌面浏览器大战早已结束的情况下，大概不会有人去挑战 Google Chrome 在桌面浏览器领域的统治地位，但人们会有担忧。

比如说，在 Edge 浏览器宣布采用 Chromium 之后，Firefox 背后的运营机构 Mozilla 就认为，Goolge Chrome 一统江湖并不是好事，这只会增加 Mozilla 作为唯一独立选择的重要性——它还声称，并不觉得 Google 在 Web 上的作为应当成为消费者的唯一选项。

Mozilla 还宣称，它将通过 Firefox 为一个真正的开放 Web 而战。

而技术作者 Alex Danco 也认为，Chrome 是 Google 旗下的一种应该被分拆的垄断性业务，它让 Google 有能力影响所有网页浏览的工作方式，这远远超出了大多数人能意识到的范围。

作者称：

Chrome 已经悄悄地改变了 Web 工作方式的重要部分，它几乎无限接近真正意义上的技术垄断，而且这种垄断并非像我们想象中的 Google 搜索广告那样遥不可及。

另有作者 Blair 也发表观点称，最大的技术垄断可能是现代互联网体验中最司空见惯的东西之一，也就是——你的网络浏览器，以及它可能性最大的来源 Google。

Chrome 的确是一个垄断性产品，而且在现有的 Google 产品政策下，你很难说明它是好事还是坏事。

另外，Google 还有一个特殊之处在于：它是 Google 旗下少见的如今依然能够在中国市场无障碍地下载和使用（虽然功能上也有部分阉割）的产品，而且在市场地位上也有很大的优势。

考虑到 Google 主体业务在中国市场的长期缺位，Chrome 也算是一道奇观了。
