C++ 之父 Bjarne Stroustrup 被冻结 reddit 帐号

August 17, 2020

C++ 开发者 Bryce 在推特发表了 C++ 之父 Bjarne Stroustrup 被冻结 reddit 帐号的消息。不过刚刚打开 Bjarne Stroustrup 的 reddit 主页时发现已经可以正常访问，而根据推文中的评论所述，此前访问显示的消息是"This account has been suspended”。

按照 Bryce 的说法，Bjarne 帐号被冻结的原因是 reddit 不相信这个是本人使用的帐号，而且冻结帐号不会是系统自动执行的，所以 Bjarne 帐号是被 reddit 管理员手动冻结了。从 Bjarne 重新开启的 reddit 主页看到，他的全部讨论都是围绕 C++ 或者技术相关，即便如此正常的一个帐号，但被冻结后 Bjarne 也无法申诉，因为帐号已被冻结，所以不能登录 reddit 提出申诉。

无奈之下，Bryce 只能通过推特曝光此事，希望 reddit 的工作人员看到后能帮助解决问题。

经过推特的传播和 reddit 中关于该事件的讨论，Bjarne 的帐号现在终于可以正常使用了，不过具体的冻结原因暂未看到 Bryce 的说明。
