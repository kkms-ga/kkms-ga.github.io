官方渲染图流出：Galaxy Tab S7也支持Xbox Game Pass服务

July 31, 2020

三星即将于 8 月 5 日（下周三）举办 Unpacked 硬件新品发布会，除了 Galaxy Note 20 系列智能机和配件产品线，Galaxy Tab S7 / S7+ 平板电脑也有望一同亮相。早些时候，诸多泄露消息已经曝光了该平板的许多信息，比如配置规格。不过在 Evan Blass（@evLeaks）分享的一组官方渲染图中，我们欣喜地发现它还支持 Xbox Game Pass Ultimate 服务。

鉴于 @evLeaks 的爆料纪录一直很是靠谱，这组官方渲染图的真实性也是相当有保证的。

三星在渲染图中展示了 Galaxy Tab S7 平板电脑的许多功能，可知其支持键盘保护套和手写笔。

不过最让我们感兴趣的，还是对微软 Xbox Game Pass Ultimate 游戏串流服务的支持。（早前报道称 Galaxy Note 20 系列智能机也支持 Project xCloud 和 Xbox Game Pass）

虽然 Android 平板用户的数量较少，但大屏幕还是可以带来相当不错的使用体验。

配置方面，据说 Galaxy Tab S7 / S7+ 配备了高通骁龙 865+ 芯片组，以及支持 120Hz 高刷新率的显示屏。
