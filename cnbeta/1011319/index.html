德国零售商：AMD Radeon 5000系列显卡故障率高于英伟达Turing竞品

August 04, 2020

德国零售商 Mindfactory.de 刚刚分享了一批显卡售后数据，结果表明 AMD Radeon 5000 系列 GPU 的故障率高于英伟达 Turing 竞品，且投诉主要围绕在 Powercolor 品牌。鉴于数据集中包括了大约 4.4 万块 AMD 显卡和 7.6 万块英伟达显卡，其在统计上的意义还是相当值得参考的。

资料图（来自：Powercolor 官网）

以 AMD RX 5700 显卡为例，其它制造商的故障率和英伟达一样徘徊在 2~4% 之间，但 PowerColor 却高达 6% 。至于售后率最高的 RX 5700 XT 显卡，也存在同样的情况。

● GPU SKU 故障率垫底型号：AMD RX 5700（13.18%）。

● 同系列故障率最高的 GPU 产品：英伟达 RTX 2080 Ti（5%）。

● 独立硬件制造商（IHV）中表现垫底的：AMD Radeon（3.29％）。

（来自：Mindfactory.de）

需要指出的是，故障率也反映了 GPU 裸片和封装 / 组装工艺问题的部分指标。让人感到惊讶的是，甜品级的英伟达 GeForce 1660 Ti 的返修率（RMA）最低，而发烧级的 RTX 2080 Ti 却高达 5% 。

（来自：Mindfactory.de）

结合完整数据来看，PowerColor 的表现确实最为奇怪，甚至凭一己之力拉高了 AMD Radeon GPU 的总体故障率。作为参考，AMD GPU 的市场份额仅为 36.6%，而英伟达则是 63.3% 。
