Vivaldi浏览器3.1更新 改进笔记管理器的特色功能

June 11, 2020

Vivaldi 3.1 更新带来了进一步完善后的笔记管理器（Notes Manager）功能，以及针对 Windows、Mac 和 Linux 桌面平台的其它改进。作为 Vivaldi 浏览器的一项特色功能，其允许用户在侧边栏上随时留下自己的想法和笔记。本次更新后，Vivaldi 3.1 扩展了笔记的基础功能，现已包含一个功能更加完善的笔记编辑器，里面有提供可视化编辑的文本格式工具栏。

笔记管理器可方便用户在 Vivaldi 浏览器中随时添加研究笔记，而无需依赖第三方应用或扩展程序。

注视功能现已支持 Markdown 标记语法，且能够将屏幕截图和正在查看的网页地址自动附加到注释中。

得益于一个所见即所得的编辑器，用户还可借助 Ctrl + F 快捷键进行快速查找注释中的文本、字符、单词等内容。

有需要的用户可借助端到端的加密传输，在台式机和 Android 设备上同步相关笔记内容。

此外新版引入了可配置菜单，允许用户将常用操作挪动到菜单栏顶部，以便根据自己的使用习惯而自定义。

最后，Vivaldi 宣称新版浏览器的速度有显著增强，支持更快的启动、且优化了标签页的处理。
