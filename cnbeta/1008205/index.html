[图]苹果官网上架雷雳 3 Pro 连接线 零售价949元

July 27, 2020

苹果中国商城近日上线了高端的 ThunderBolt 3 线缆，零售价格为 949 元，预估发货日期为 1 个工作日，表明货源充足。根据苹果官网介绍这款 2 米的雷雳 3 Pro 连接线采用黑色编织设计，卷起时不易缠结。

这款线缆支持雷雳 3 数据传输 (速率最高可达 40Gb/s)、USB 3.1 第二代数据传输 (速率最高可达 10Gb/s)、DisplayPort 视频输出 (HBR3) 以及最高可达 100W 的充电能力。

你可以使用这条连接线将配备雷雳 3 (USB-C) 端口的 Mac 连接至雷雳 3 设备，如 Pro Display XDR、基座和硬盘。

这款线缆的亮点有

● 数据传输速率最高可达 40Gb/s

● USB 3.1 第二代数据传输速率最高可达 10Gb/s

● DisplayPort 视频输出 (HBR3)

● 连接至支持雷雳 3 或 USB-C 的设备和显示屏

● 电力供应最高可达 100W

● 采用编织设计，卷绕时不易缠结

● 雷雳标识易于辨识，不会与其他连接线混淆

● 能以菊花链方式连接最多达六部雷雳 3 设备
