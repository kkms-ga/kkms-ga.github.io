[图]三星Galaxy A01 Core真机谍照再曝光 已开始在印度量产

July 10, 2020

继上周放出实拍图之后，今天印度科技媒体 91Mobile 再次放出了三星 Galaxy A01 Core 真机谍照。不过展示的这些图片中，Galaxy A01 Core 并未完成，因此可以清晰的看到设备的内部结构。此外该手机已经在印度地区开始量产，表明即将会在印度地区上市发售。

从曝光的真机图片中可以看到，三星 Galaxy A01 Core 的额头和下巴部分边框都非常宽，屏幕长宽比例可能为 18：9。该设计似乎与爆料人Evan Blass泄露的渲染图一致。手机的右侧为电源键和音量摇杆。

正如此前报道，三星 Galaxy A01 Core 采用塑料材质，单颗后置摄像头、LED 闪光灯，Micro USB 接口、3.5mm 耳机插孔和底部边缘的单扬声器格栅。

根据谷歌 Play 控制台的早前爆料，Galaxy A01 Core 将采用入门级的联发科 MT6739WW 芯片组（集成 PowerVR GE 8100 核显）、辅以 1GB RAM 运存，屏幕分辨率为 HD+ 。
