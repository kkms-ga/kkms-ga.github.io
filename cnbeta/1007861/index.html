Edge浏览器现已支持通过鼠标来删除不需要的地址栏建议

July 26, 2020

早在 2019 下半年，谷歌工程师就已考虑为 Chrome 浏览器引入“可使用鼠标删除”地址栏建议的功能。2020 年 5 月，谷歌又在 Chrome 83 版本中提供了可将鼠标悬停、然后使用 Shift + Delete 组合键来删除地址栏建议的选项。好消息是，采用 Chromium 内核的新版 Microsoft Edge 浏览器现在也已经作出了跟进，允许用户在多功能地址栏（Omnibox）中移除不需要的建议。

从 Canary 86.0.587.0 版本开始，所有用户都可以体验到 Microsoft Edge 浏览器的这项功能改进。

虽然新版 Edge 浏览器可以记住用户在地址栏中输入的内容，并在下次给出贴心的填充提示。

但由于有时会出现错别字或涉及隐私的搜索关键词，针对特定条目的删除功能仍不可或缺。

此前，Microsoft Edge 和 Google Chrome 都已经能够使用 Shift + Delete 组合键来删除多功能地址栏（Omnibox）中的特定条目。

可惜对于许多用户来说，缺乏宣传这项特性，无异一个需要丰富经验的‘隐藏功能’。

庆幸的是，谷歌已经率先在 Chrome 上解决了该问题，现已能够同时在地址栏建议旁边，通过鼠标悬停来显示一个直观易懂的‘x’删除图标。

现在，基于 Chromium 内核的新版 Microsoft Edge 浏览器也接受了这项变动，并且率先向 Canary 开发分支进行了推送。

感兴趣的朋友可以启动 Edge 浏览器，在地址栏输入相关内容，然后将鼠标悬停在建议条目的上方，然后通过浮现的‘x’按钮来剔除特定条目。

最后，根据 Microsoft Edge 的发布计划，Edge 86 稳定版将于 10 月第二周向广大用户推送。
