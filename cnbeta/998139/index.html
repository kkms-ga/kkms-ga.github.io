[图]Edge Canary新版允许用户查看和验证PDF文件中的数字签名

July 02, 2020

Canary 通道的最新 Edge 版本现在允许用户查看和验证 PDF 文件中的数字签名了。尽管目前基于 Chromium 的新版 Edge 浏览器可以作为 Windows 10 的默认 PDF 阅读器，但是很多经典版 Edge 的功能并没有迁移过来。

目前在新版 Edge 浏览器中，对 PDF 文件支持的主要功能包括

● Read Aloud

● 允许以不同颜色高亮文本

● PDF上手写输入以及深色模式

● 填写 PDF 表格

近日，微软承诺将表单内容、文本注释、平滑滚动体验、高亮模式以及数字签名等功能迁移到新版 Edge 浏览器中。在最新发布的 Canary 通道版本 85.0.562.0 中，已经引入了数字签名功能。

想要在新版 Edge 浏览器中启用数字签名功能

  1. 启动 Microsoft Edge 浏览器

  2. 访问 edge://flags 页面

  3. 搜索 “Enable Digital Signature for PDF”

  1. 从下拉菜单中选择"启用"并重新启动浏览器。

现在在 Edge 浏览器中就可以在包含数字签名的 PDF 文件了。你会注意到 Edge PDF Viewer 在工具栏上显示新的 “数字签名"选项。它还提示您 “查看签名 “与消息"此文件包含数字签名…"。
