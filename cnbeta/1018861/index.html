[图]Windows 10 Build 20197发布：磁盘管理迁移到设置应用中

August 22, 2020

面向 Dev 频道，今天微软发布了 Windows 10 Build 20197 预览版更新。本次版本更新中，主要将磁盘管理从控制面板迁移到设置应用中。

Windows 10 Build 20197 更新内容

将磁盘管理引入到设置应用中

微软表示现在用户可以在设置应用中管理磁盘和卷了。这包括查看磁盘信息、创建和格式化卷以及分配驱动器字母等任务。与现有的磁盘管理MMC snap-in不同，这个现代化的体验从头到尾都是为了方便用户使用。它还具有更好的集成功能，如存储空间和存储分解页面。

用户可以前往 “设置">"系统">"存储”，然后点击 “管理磁盘和卷 “开始使用。

面向开发者的更新

目前 Windows SDK 已经和 Dev 频道同步更近。每当有新的操作系统版本到 Dev 频道时，相应的SDK也会被传送到 Dev 频道。您可以随时从 aka.ms/InsiderSDK 安装最新的 Insider SDK。

常规改进

● 对 Alt+ Tab 行为改进，能够默认显示 5 个 Edge 浏览器标签页，而不是最近访问的标签页。如果你想要进行更改，可以在设置>系统>多任务下进行调整。

● 在 “设置 “标题中加入 “网页浏览 “部分，内部人员可能会开始看到这一点。

● 今后，只有当前活动的个人资料图片才会在 “设置 “中的 “你的信息 “下显示。

● 根据反馈，我们将关闭ESENT警告事件ID 642。

● 现在可以在 “指定评估 “中选择新的Microsoft Edge作为您所需的应用程序。

● 我们更新了Narrator，现在使用微软拼音IME时，Narrator能够通过提供详细的阅读信息来区分候选字符/单词。

修复的内容包括

● 修复当所有应用程序列表设置为隐藏时，“开始 “菜单的所有应用程序列表中的字母标题会出现不必要的左侧填充。

● 修复即ALT + Tab键到浏览器标签有时会将之前活动的浏览器标签也移动到Alt + Tab列表的前面。

● 修复如果您将一个网站钉在任务栏上，然后将打开该网站的 Microsoft Edge 窗口移动到不同的虚拟桌面上，钉住的网站不会在新桌面上显示相应的打开标签。

● 修复即在最近的几个构建中，sysprep命令与错误0x80073CFA失败。

● 修复组策略编辑器在最后几个航班中显示一个关于WnsEndpoint_Policy的弹出错误。

● 修复即Get-Volume PowerShell命令无法识别exFAT格式的卷。

● 修复首次启动和登录时影响explorer.exe启动性能的问题。

● 修复开始菜单中被钉住的文件资源管理器文件夹在浅色模式下会以比其他磁贴更深的背景显示。

● 修复了一个可能导致字体设置页面崩溃的竞赛条件。

● 修复即字体设置和主题设置显示的是旧样式的微软商店图标。

● 修复了一个可能导致 “设置 “中的 “网络状态 “页面显示旧的网络名称，而不是您当前连接的网络，并且与网络飞出显示的内容不一致的问题。

● 修复导致某些VPN在启用 “自动使用我的Windows登录名和密码（和域，如果有的话）“选项后不再自动连接。

● 修复导致一些Insiders在最近的构建中遇到了一个错误检查，引用了一个超护卫违规。

● 修复在文件资源管理器中使用搜索框时，按下回车键会导致键盘焦点跳转到导航窗格的快速访问部分而不是搜索结果的问题。

● 修复微软拼音IME的一个问题，即输入特定的短语会导致候选窗格意外关闭。

● 修正了微软拼音IME中的一个问题，当使用双拼音风格时，当输入类似 “goldb;du “这样的词时，第一个候选词可能是一个英文单词，但转换不会成功。

● 修正了日语IME输入时，有时无法使用Shift+<数字键>在密码栏中输入符号的问题。

● 修复当使用中国繁体大义IME输入错误的组合时，可能会导致某些应用程序崩溃。

● 修复了微软Bopomofo IME的一个问题，当它们导致IME模式切换动作时，会消耗Shift键向上事件。

● 修复了在使用IME打字时将焦点切换到某些应用程序（如Outlook）时可能导致CTFMON.exe的问题。

● 修复了ImmGetCompositionString没有为YOMI文本字段返回正确的IME组成字符串的问题。

● 修复在最近的版本中，全触摸键盘布局上的Windows键在点击后不会突出显示。

● 修复使用叙述者时，扫描模式在Microsoft Teams中不会保持关闭的问题。

● 修复导致微软飞行模拟器在前两次飞行时无法启动，并出现错误提示。“文件名，目录名，或卷标语法不正确”。
