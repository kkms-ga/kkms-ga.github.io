[图]微软指责苹果：将自己孤立成拒绝云游戏和订阅服务的平台

August 07, 2020

对于苹果拒绝 Project xCloud 登陆 App Store，微软今天发布声明中指责了苹果的做法，并表示：“苹果将自己孤立起来，成为唯一一个拒绝消费者使用云游戏服务和 Xbox Game Pass 等订阅服务的平台”。

在今天早些时候苹果发布的声明中表示：“App Store 的初衷是为用户发现和下载应用程序提供安全、值得信赖的地方，同时也为所有开发者提供巨大商机。在这些应用登陆我们的商店之前，所有的应用都会根据同一套准则进行审查，目的是更好的保护客户，并为开发者提供一个公平和公正的竞争环境”。

根据 App Store 的相关政策，应用程序不能依赖来自云端的流媒体，这就意味着 Stadia、xCloud 和 Nvidia GeForce 这样的云端游戏服务不能托管在iOS应用商店。苹果对自己的要求与其他开发商相同，因为苹果商场的游戏不提供流媒体服务–用户必须从App Store的Arcade标签内下载游戏。苹果的平台上欢迎竞争性游戏服务，但前提是它们必须符合规则。

对此微软在声明中表示

Project xCloud 预览应用在 iOS 平台上测试期已经结束。遗憾的是，我们没有办法通过苹果应用商店将我们的 Xbox Game Pass Ultimate 云游戏愿景带给 iOS 上的玩家。

苹果将自己孤立起来，成为唯一一个拒绝消费者使用云游戏服务和 Xbox Game Pass 等订阅服务的平台。而且，它一贯区别对待游戏应用，对非游戏应用适用更宽松的规则，即使它们包含互动内容。

Xbox Game Pass目录中的所有游戏都由独立的行业评级机构（如ESRB和地区性同等机构）进行内容评级。我们致力于寻找一条将云游戏与Xbox Game Pass Ultimate带到iOS平台的方式。

我们相信，客户应该是游戏体验的核心，玩家告诉我们，他们希望在任何地方玩、连接和分享，无论他们在哪里。我们也深感认同。
