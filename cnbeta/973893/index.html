Xbox Game Pass今年晚些时候将迎来云流媒体服务

May 01, 2020

据外媒报道，本周早些时候，微软透露Xbox Game Pass已经拥有超1000万名用户。Xbox Game Pass通常被称为“游戏界的Netflix”，它是一项服务，允许用户访问每一款新第一方Xbox游戏以及从第三方工作室轮流选择游戏，每月则收取9.99美元的固定费用。

访问:

微软中国官方商城 - Xbox

对于微软来说，Game Pass已经取得了绝对的成功，今年晚些时候，它将会有一个巨大的升级。当地时间周四，Phil Spencer在Xbox Wire上的一篇博客文章中宣布，xCloud项目将于今年晚些时候加入Xbox Game Pass中。

这可能会比标准Game Pass订阅服务价格更高，但它将为用户提供100多款游戏的访问权限，这些游戏几乎可以在他们拥有的任何设备上玩。你不需要拥有Xbox Series X、Xbox One或是一台强大的PC就能玩上最新的游戏–玩家需要的只是一个智能设备、一个网络连接和Xbox Game Pass。所以就目前来看，撇开硬件不谈，Xbox的未来一片光明。
