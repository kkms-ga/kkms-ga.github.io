十铨发布MP33 Pro系列NVMe M.2 SSD新品 512GB售59.99美元

August 28, 2020

虽然 PCIe Gen 4 正在成为一个卖点，但采用 PCIe 3.0 x4 接口的 M.2 NVMe 固态硬盘，已能够满足大多数普通消费者的使用需求。本文要为大家介绍的，就是来自十铨（TeamGroup）的 MP33 Pro 系列 M.2 2280 SSD 新品。其没有花哨的散热片或标签贴纸，但提供了黑 / 蓝两种 PCB 配色选项，且性价比相当突出。

MP33 Pro 的持续读写可达 2100 MB/s 和 1700 MB/s，随机性能最高可达 220k 和 200k IOPS 。

容量方面，消费者可选 512GB / 1TB / 2TB 版本，售价分别为 59.99 / 103.99 / 239.99 美元（约 413 / 715 / 1651 RMB）。

2TB 的大容量版本，可满足具有大量素材的创作者和游戏玩家的使用需求。且 MP33 Pro 内置的智能管理算法可确保兼顾操作效率、访问速率、以及存储的稳定性。

从官方给出的参数来看，512GB / 1TB / 2TB 版本的标称写入耐久度分别为 400 / 600 / 100 TBW 以上，并且十铨提供了五年有限质保。遗憾的是，该公司尚未宣布确切的上市日期。
