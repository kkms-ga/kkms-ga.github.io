[图]微软重组内部团队 重心之一是提升Windows 10体验

August 07, 2020

今年年初，微软 Surface 掌舵者 Panos Panay 开始负责新成立的 Windows + Devices 集团。这意味着 Panay 能够同时负责 Surface、Windows 10，以及包括 Windows 10X 在内的其他 Windows 产品。

虽然在微软停止对 Windows 7 的支持之后，Windows 10 比以往任何时候都更受欢迎，但最近的错误更新已经严重损害了微软的声誉。根据报道称，微软正在进一步重组团队。

在本次团队调整中，微软计划将 Windows 工程团队从 Azure 中移出来，然后并入到 Windows + Devices 集团下，并直接由 Panos Panay 负责。

此举是微软在过去几个月中意识到 Windows 对其业务的重要性之后调整的。根据报道，微软的 Windows + Devices 集团将会在 Panos Panay 的领导下，创建平台和产品并提供神奇的体验。
