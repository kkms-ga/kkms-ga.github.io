多功能键盘+扩展坞：Kolude KD-K1 Keyhub简单上手

July 08, 2020

对于喜欢保持桌面整洁的用户来说，一款自带多功能扩展坞的高品质键盘，或许是一个不错的选择。好消息是，WCCFTech 刚刚分享了 Kolude KD-K1 Keyhub 键盘集线器的简单上手评测。作为一个众包项目，其不到一天的时间，就在 Kickstarter 上达成了既定的筹款目标，可见网友们对它的喜爱程度有多高。

（来自：WCCFTech，via Kickstarter）

外形方面，Kolude KD-K1 Keyhub 采用了时尚的铝制外壳，辅以剪刀式键轴，侧边还有完整的 USB 集线器，集成 TF / SD 读卡器、且支持 HDMI（4K / 30Hz）和 Power Delivery 。

对于 Mac 和 iPad 用户来说，KD-K1 可以完美地融入其中。在接入显示器等外设之后，可在保持桌面整洁的同时、实现更佳工作效率。

打开盒子之后，可在 KD-K1 Keyhub 键盘本体下方找到使用手册，以及随附的双头 USB-C 线缆，可轻松接入 MacBook 或新款 iPad 平板电脑。想要在 USB Type-A 设备上使用，还需另寻一条。

USB Hub 方面，该键盘集成了 USB 2.0 和 USB 3.0（三个），以及支持 PD 供电的 USB-C 接口（最高 100W 输出功率 / 两台 4K 显示器）。

银色金属 + 圆形黑色键帽的设计，以及钻石切割的边缘，与 MacBook 和 iPad 很是搭调。为了保持使用时的稳定，键盘底部还配有一些防滑脚垫。

如果一切顺利，Kolude KD-K1 Keyhub 或很快向早期支持者进行交付。感兴趣的朋友，不妨拿出 99 / 109 / 119 美元的零花钱去支持一下。
