[视频]Microsoft Lists向全球Microsoft 365用户开放：帮用户追踪工作进度

July 22, 2020

在 Build 2020 开发者大会上，微软正式宣布了一款名为 Microsoft Lists 的应用程序，帮助用户更好的追踪工作进度。今天，微软启动了该应用的推广计划，表示将于 7 月底上线全新的主页面以及 8 个现成的模板。

Microsoft Teams 中的 Lists 应用将在 8 月底前上线测试，预估会在今年 10 月正式发布。Microsoft Lists 移动端应用程序将会在今年年底上线，通过 Microsoft 365 或者 Office 365 计划推出。

区别于Microsoft To-Do，全新的Lists作为一款独立应用出现在Microsoft 365套件中，可在所有的订阅服务中共享和跟踪信息。微软表示通过Lists，用户将能够在Microsoft Teams、SharePoint和Outlook中跟踪工作进度、联系人、问题跟踪和状态报告等信息。

通过Lists，用户能够使用丰富的模板和彩色标记，并通过简单的“if this then that”来配置规范流程和警报。Lists还将与Power Apps（表单）和Power Automate（工作流）集成。微软表示

Microsoft Lists是在20多年SharePoint列表创新基础上创建的，现在将这些功能带到了整个Microsoft 365的更多用户中。对于那些熟悉SharePoint Lists的用户以及任何设备上的新用户来说，Microsoft Lists将简化用户体验。
