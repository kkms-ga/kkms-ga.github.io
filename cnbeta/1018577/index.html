索泰发布Zbox Q系列迷你工作站 i7-10750H处理器与Quadro RTX 3000加持

August 21, 2020

索泰（Zotac）近日推出了 Zbox Q 系列迷你工作站，型号为 QCM7T3000 。与此前推出的消费级产品相比，其最大的特色是采用了英特尔十代 Core i7-10750H 处理器 + 英伟达 Quadro RTX 3000 专业显卡。尽管体型很是小巧，但 Zbox Q 的性能依然相当强悍。从复杂的工业设计、特效制作，到复杂的科学可视化和大数据建模等应用，该机都可以从容应对。

（来自：Zotac）

Zbox QCM7T3000 迷你工作站配备了 6 核 / 12 线程的 Core i7-10750H 处理器，热设计功耗（TDP）为 45W 。

至于 NVIDIA Quadro RTX 3000 专业显卡，其拥有 6GB GDDR6 显存、以及专用的 Tensor 内核。除了 VR，它还支持实施光线追踪（RTX）和深度学习超级采样（DLSS）。

此外 Nvidia Quadro 平台已久经考验和备受创作者信任，完美兼容大多数专业应用程序。

与传统塔式工作站相比，Zbox QCM7T3000 的体积只有 2.65 升（长宽高 8.27×7.99×2.45 英寸 / 225×203×62.2 毫米），占地面积也可减少近半。

借助优化的风道设计和强大的散热硬件，Zbox QCM7T3000 可最大限度地导入冷风和排出热量，以确保长时间运行的性能。

值得一提的是，该机支持同时接驳四台显示器（双 DP + 双 HDMI），辅以包括 USB 3.1 Gen2 10Gbps 接口在内的完整连接 / 扩展选项。
