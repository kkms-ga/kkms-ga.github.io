Sabrent发布首款8TB NVMe SSD：读速3400MB/s

May 21, 2020

Sabrent 刚刚推出了 Rocket Q 系列 NVMe PCIe SSD 新品：其采用了 M.2 2280 的标准外形和 PCIe 3.0 ×4 接口，最高可选 8GB 容量（另有 500GB / 1TB / 2TB / 4TB 的选项）。搭配群联 E12S 主控和 3D QLC 闪存，最高可实现 3400 MB/s 顺序读取、以及 3000 MB/s 的持续写入。

（来自：Sabrent，via Amazon）

虽然 Sabrent 未披露 4K 随机读写性能，但在大容量 QLC 闪存的加持下，这款 M.2 2280 PCIe NVMe SSD 的参数应该还是相当可观的。

以下是 Rocket Q 系列 SSD 的主要特点：

● 采用 M.2 PCIe Gen 3 ×4 接口；

● 符合 PCIe 3.1 / NVMe 1.3 规范；

● 支持 APST / ASPM / L1.2 高级电源管理特性；

● 支持 SMART 和 TRIM 指令；

● 支持 ONFi 2.3、ONFi 3.0、ONFi 3.2、以及 ONFi 4.0 接口规范；

● 附赠 Acronis True Image 软件，轻松迁移数据。

目前亚马逊已上架 512GB / 1 TB / 2TB / 4TB 版本，售价分别为 69.99 / 129.98 / 239.99 / 749.99 美元（约 497 / 923 / 1703 / 5323 RMB），8TB 版本暂不得而知。
