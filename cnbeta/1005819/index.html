锐龙4700G台式APU达成冰冷全核5.0GHz与DDR4-5400内存超频纪录

July 21, 2020

在 AMD 今日官宣之前，网络上就已经曝光了锐龙 7 4700G 台式 APU 处理器的许多细节。现在，已拥有该芯片有段时间的 ITCooker，再次将 4700G 的非液氮超频纪录提升到了全核 5GHz 。作为一枚 8 核心 / 16 线程的锐龙 4000 系列“Renoir”芯片，其采用了不同于锐龙 3000 系列“Matisse”的单芯片设计，集成了 7nm Zen 2 CPU 与 7nm Vega GPU，因而具有更高的效率、更低的延时、以及明显改善的带宽。

从 CPU-Z 屏幕截图来看，锐龙 7 4700G 基于 7nm Zen 2 架构，具有 8 核心 / 16 线程、4MB L2 + 8MB L3（合计 12MB）缓存。

需要指出的是，锐龙 3000 系列“Matisse”台式处理器基于小芯片设计，但锐龙 4000 系列“Renoir”已将 CPU 和 GPU 部分整合到了一块。

该处理器的基础频率为 3.6GHz、动态可达 4.45GHz，与早期爆料相符。热设计功耗 65W，且沿用 AM4 插槽。

图形部分采用了增强的 7nm Vega GPU，具有 8 个 CU（512 核）、频率 2100MHz，算得上是目前最快的核显。

在技嘉 B550 AORUS Master 主板的加持下，ITCooker 将锐龙 7 4700G 台式 APU 推向了全核 4.96GHz 。

此时电压为 1.519V，但散热仍采用 360mm 一体式水冷（用塑料桶里的两袋冰块来辅助冷却），详情可移步至 Hwbot 查看。

此前 ITCooker 已多次借助冰水达成过非常规的超频纪录，而在某些高端一体式水冷的配置下，4.5 ~ 4.7GHz 的成绩也并非难事。相信随着 4700G 芯片的官方解禁，后续还会有更加惊人的液氮冷却超频纪录出现。

除了 CPU 超频，Benchlife 还分享了在技嘉 B550 AORUS Master 主板上达成 DDR4-5400 内存频率的 CPU-Z 截图。

截图显示其使用了英睿达的 Ballistix 游戏内存，可在 B550 平台上加载 XMP 内存配置文件。但搭配的似乎是一枚工程样品（ES）处理器，因为它的基准频率设置仅为 3.0GHz 。
