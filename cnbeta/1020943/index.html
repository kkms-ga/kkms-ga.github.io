Windows 10 19042/20201最新官方ISO镜像开始提供

August 27, 2020

按计划，下半年还有一次Windows 10正式版更新，即20H2，大版本号将从v2004（20H1）的19041迭代为19042。今晨，微软首次上线20H2版本的官方ISO镜像，供Beta会员纯净安装使用。同时，rs_prerelease分支的最新ISO镜像也一并上线，版本号为Build 20201，需要Dev通道的会员才能下载使用。

下载步骤：

注册 Windows Insider Program (点击这个链接).

确定您的PC满足系统需求.

然后从这里下载ISO镜像.

无论是此次的20H2（Build 19042.487）还是Build 20201，主要都着眼于BUG修复、稳定性改善等，20201日志中公布了大概5条，包括商店中带有反作弊程序的游戏运行失败、状态栏输入法指示图标错乱、部分HDR显示器开启HDR后黑屏等。

当然，20H2本身也预装了新Edge浏览器、调整了开始菜单的UI、对Alt+Tab进行了功能性增强等。

Build 19042.487修复的问题就超级多了，简单数数大概72条……这样来看的话，即便是不下载镜像，也值得在系统中检查更新。

另外，目前官方渠道可供下载的ISO镜像还有Release Preview分支的Build 19041和29H2正式版Build 18363。
