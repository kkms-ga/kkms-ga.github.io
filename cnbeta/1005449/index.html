Windows 10即将到来的新功能一览： 优化开始菜单 强化Alt+Tab

July 20, 2020

相信不少 cnBeta 的读者已经升级到 Windows 10 May 2020（20H1/Version 2004）功能更新了，那么引入的这些新功能解决你的痛点了吗？你还希望引入哪些新功能？外媒 Windows Lastest 近日盘点了近期 Windows 10 预览版中正在测试的新功能，让我们一起来看看。

Storage Recommendations

Windows 10 系统的存储设置页面中将允许用户扫描本地磁盘，从而释放临时文件夹、应用程序以及其他占用磁盘的数据。从下一个Windows 10功能更新开始，“存储设置 “将提供清理建议，帮助您释放空间，并删除可能不需要的未使用的应用程序或文件。

清理工具将所有的建议都列在一个页面上，从而方便用户腾出空间。从上面的截图中可以看到，清理建议会被分成几个类别，包括未使用的应用/文件和大文件。

Apps & Features

微软正在改进 Apps & Features 页面，日前在微软应用商城中发现了一款名为 AppManager 的应用程序。

这似乎是当前设置页面中 Apps & Features 功能的升级。看起来，Windows 10的应用程序和功能页面最终将允许用户查看后台运行的进程，你也可以卸载应用程序。

开始菜单

明年 Windows 10 也将迎来全新的开始菜单设计。微软最近发布了一个新的预览构建，其中展示了带有主题识别磁贴的开始菜单。虽然重新设计的开始菜单最初看起来并没有什么大的变化，但更多的工作正在酝酿中，因为微软正在寻求让操作系统看起来更干净、更统一。

动态磁贴已经更新为“theme-aware”，将能够跟随系统进行颜色背景的调整，从而获得统一的外观。此外，用户的动态磁贴的颜色将会匹配开始菜单，无论用户使用的是浅色模式还是深色模式。

Alt+Tab

微软已经确认，用户能够在应用程序、Edge 浏览器标签页之间进行切换。在 Windows 10 Build 20161 中，微软已经更新了 Alt+Tab 的默认行为，切换界面中包括了 Microsoft Edge 的标签页面（Chromium版本）。

虽然，将 Alt+Tab 快捷方式和 Edge 浏览器标签页组合能够提高工作效率，但是当你打开太多的标签页和应用窗口的时候，Alt+Tab 视图就会变得有点拥挤。

幸运的是，微软允许用户自定义交互体验，用户可以在设置中选择以下几个选项

● 打开窗口和 Edge 浏览器中的所有标签页

● 打开窗口和 Edge 浏览器中的 5 个最近标签页

● 打开窗口和 Edge 浏览器中的 3 个最近标签页

● 只打开窗口

音量优化

在最新的Windows 10 Version 2004预览版更新中，微软已经邀请Windows Insider成员测试音量窗口的新功能。

根据Albacore率先爆料，这项新功能提供了精细化的多媒体控制，允许用户使用全局媒体控件时对某个音乐进行的控制，而且目前正在测试多个运行界面，其中还包括锁屏界面下的多媒体控件。目前尚不清楚这项功能是否会进入Version 2004的正式版本中。

谣传中的新功能

音量布局

在Windows 10最新预览版Build 19577更新中，已经出现了改进音量控件的新代码，表明微软仍在评估并进行后续开发，并没有完全放弃。在新版本中当音乐播放中或者当用户点击音量按钮的时候，Spotify、Groove Music或者浏览器的音量控件就会显示在音量窗口的左上角。这个Windows 10音量布局允许用户通过多媒体键盘和鼠标来播放、暂停或者停止音乐。

从截图来看，微软计划将所有控件移动到新的音量布局中，用户能够点击系统托盘上的音量按钮进行访问。当前，任务栏弹出仅允许用户控制耳机或扬声器的音量，但是将来的Windows 10更新也将允许您访问媒体控件。新的弹出窗口还会显示音乐的专辑封面以及正在播放音乐的应用名称。

在外媒的测试中，只有在单击系统任务栏中的图标时，更新的弹出按钮才会启动。它目前不支持键盘多媒体键。今年晚些时候，这项新的音量控件可能会提供给Windows Insider项目成员进行测试，但是尚不清楚何时会整合到功能更新中。

现代化磁盘管理

在Windows 10 Build 19577及以上版本中，外媒发现在代码中提及了“Modern_Disk_Management_Link”，暗示微软正在开发现代化的磁盘管理工具。未来我们有望在设置应用中触发新的磁盘管理工具。
