Python 3.9.0a6 已可用于测试

April 30, 2020

Python 3.9.0a6 发布了。Python 3.9 仍在开发中。3.9.0a6 版本是六个计划的 alpha 版本中的最后一个。Alpha 版本旨在测试新功能和修复错误，新功能可能会一直添加到 Beta 阶段（2020-05-18）为止，如果有必要，也会对其进行修改或删除，直到发布候选阶段（2020-08-10）。要注意，这是预览版本，不建议在生产环境中使用。

3.9 系列的主要新特性包括：

PEP 584，Union Operators in dict

PEP 593，灵活的功能和变量注释

PEP 602，Python 采用稳定的年度发布节奏

BPO 38379，垃圾收集不会阻止复活的对象

BPO 38692， 添加了 os.pidfd_open，它允许进程管理

BPO 39926，Unicode 支持已更新至版本 13.0.0

BPO 1635741，在同一进程中多次初始化 Python 时，它不再泄漏内存

使用 PEP 590 vectorcall 加速了许多 Python 内置函数（范围、元组、集合、frozenset、列表）

许多标准库模块（audioop，ast，grp，_hashlib，pwd，_posixsubprocess，random，select，struct，termios，zlib）现在都使用 PEP 384 定义的稳定 ABI

Python 3.9 的下一个预发行版本将是第一个 beta 版本  3.9.0b1，计划于 2020-05-18 发行。
