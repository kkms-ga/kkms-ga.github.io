微软向Windows 10 2009测试者推送KB4566782累积更新

August 12, 2020

WCCFTech 报道称，微软刚刚向 Beta 测试通道发布了 KB4566782 累积更新。作为即将推出的 2020 年 11 月更新（Version 2009）的一部分，其将 Windows 10 20H2 的内部编译版本号提升到了 19042.443 。软件巨头在 Windows Blog 上写道：首先，本次更新修复了通用 Windows 应用（UWP）平台上的一个问题，以避免程序在不具有企业身份验证功能时允许单点登录身份验证。

（来自：Windows Blog）

随着 CVE-2020-1509 安全漏洞的修复，UWP 应用程序可能会开始提示用户输入登录凭据。

其次，KB4566782 累积更新修复了与微软脚本引擎、IE 浏览器、Windows 图形组件 / 内核、输入 / 合成、Windows Media、Windows Shell、Windows 钱包服务、经典版 Edge 浏览器、Windows 云基础架构、Windows 身份验证、AI 平台、Windows Fundamentals 基础、存储和文件系统、更新堆栈、文件服务器和集群、混合存储服务、应用程序平台和框架、Microsoft JET 数据库引擎、以及 Windows SQL 组件等有关的问题。

更多内容，还请移步至官方博客网站查看。
