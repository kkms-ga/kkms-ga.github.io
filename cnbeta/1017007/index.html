AMD Radeon 20.8.2驱动发布 优化支持微软飞行模拟器等游戏

August 18, 2020

在 NVIDIA 发布 452.06 新版驱动之后，AMD 今天也发布了 Radeon Software Adrenalin 2020 Edition 20.8.2 套件。本次更新重点对《微软飞行模拟器》、《漫威·复仇者联盟》公测版本、《致命躯壳》以及《全面战争传奇：特洛伊》进行了支持和优化。

下载链接：AMD

AMD 官方只公布了《全面战争传奇：特洛伊》这款策略游戏的详细优化信息，相比较本月早些时候发布的 20.8.1 版本，RX 5700 XT 在安装新驱动之后性能优化 12%。而且本次更新对所有 AMD 产品的 FPS 都有所提升。

新版本中修复了以下问题

● 报告称在一些带 Radeon™ 图形的 AMD Ryzen™ 3000 系列移动处理器上，退出系统睡眠时系统间歇性卡顿的情况。

● 在打开 Radeon Overlay 时按下按键，或在玩 Hyper Scape™ 时退出 Radeon Overlay，可能会导致玩家的操作冻结或无法识别用户的输入。

● 在 Windows 7 系统配置中打开的时候，Upgrade Advisor 可能会显示“Unable to get requirements”错误信息
