[视频]iFixit拆解Pixel 4a：维修难度要比Pixel 3a高

August 31, 2020

往往非高端机型都不够坚固，但会比较容易拆解和维修。Pixel 4a 却是一个例外。近日，知名维修团队 iFixit 分享了对该机的拆解视频，表示该机非常难以维修，虽然没有给出得分，但应该要比 Pixel 3a 要低。

拆解 Pixel 4a 都是从机面屏幕入手，跟 Pixel 3a 一样都是在屏幕加热， 再在边缘慢慢将屏幕跟机身分离。屏幕和设备靠接线连接，拆开螺丝便可将接线分开，屏幕就从装置分离。

不过和 Pixel 3a 的相似之处就到这里了。iFixit 指出，和 Pixel 3a 不同的是， Pixel 4a 的主板实际和中框连在一起，而且拆解之前还需要清除不少细小的线缆，并要小心翼翼地拆掉金属屏蔽层。

在机背和中框分离之后，就可以看到电池，松开接线再以酒精将电池的黏合胶溶解就可以抽出电池。其他部件也可以跟着逐一拆解。如视频中所见，若是 Pixel 4a 要替换屏幕的话，应该不会太困难。但若要替换电池就要花点功夫，移除屏幕再移去机背才可。
