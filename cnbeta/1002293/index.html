韩媒称三星Galaxy Note 20实际上可能会比预期的更便宜

July 12, 2020

三星正准备推出新的秋季旗舰Galaxy Note 20，离发布时刻越近，关于该设备的信息就越多。最新的报道显示，尽管Galaxy Note 20增加了新功能和最新一代硬件，但实际上可能比前代产品更便宜。韩国门户网站Naver引用业内人士带来的消息称，Galaxy Note 20的韩国起售价可能在999美元左右，不过，这很可能是该设备的标准版，Galaxy Note 20 Ultra将从1040美元起售。

换句话说，这意味着新一代机型可能会比前代机型便宜50美元左右，这对于每一个计划购买手机的人来说，无疑是一个好消息。

但另一方面，没有人知道的是，三星是否希望这种降价发生在所有国别市场，而不仅仅是在韩国。在美国，Galaxy Note 10的售价为949美元起，如果其他地方也采用这种方式，那么新手机可能会以900美元左右的价格上架。

对于三星最新和最强大的智能手机来说，近年来不断提升的价格到了今年反而下降令人不习惯，但另一方面，考虑到Galaxy S20的市场表现，这一策略却也是相当合理的。

Galaxy S20于2月推出，3月起上市，由于其销售主要依靠零售商，因此成为全球健康危机的受害者。Galaxy S20的表现远低于预期，现在相信三星也担心Note 20也会受到类似的影响。

降价可能有助于使Note20成为更有吸引力的购买产品，尽管这是否足以让该公司应对健康危机的影响还有待观察。我们将在8月5日的揭幕后了解更多信息。
