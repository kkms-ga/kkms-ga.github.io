《精灵宝可梦Go》将很快停止支持三星Galaxy S4等旧款智能机

June 13, 2020

《精灵宝可梦 Go》开发商 Niantic 的员工今日透露，他们将很快停止对某些旧设备的支持。除了将被放弃的三星 Galaxy S4 / S5 / Note 3 / J3、索尼 Xperia Z2 / Z3、初代 Moto G 等智能机，还有更多设备的功能也将受到限制。主要原因是《精灵宝可梦 Go》要全面转向 64 位版本，而那些在 2015 年之前发布的大量 Android 设备都是 32 位版本。

大限过后，这些旧设备的玩家将无缘继续体验《精灵宝可梦 Go》。如需检查自己的 Android 设备到底是 32 位、还是 64 位版本，可以下载 Device Info 等工具软件进行查看。

安装之后，可在 App 中查看 CPU 类型，确定其显示的是 32 位、还是 64 位的设备。当然，过去五年内购置的设备，大部分都已经是 64 位的平台。

Device Info 传送门（来自：Google Play）

Niantic 给出了下列 32 位 Android 设备的示例：

三星：Galaxy S4 / S5 / Note 3 / J2 / J3；

索尼：Xperia Z2 / Z3；

摩托罗拉：初代 Moto G / Nexus 6；

LG：Fortune / Tribute；

一加：OnePlus 1；

宏达电：HTC One（M8）；

中兴：Overture 3 。

最后，在玩家购置新设备前，Niantic 还建议大家保存账户登录信息（提供商、邮件地址和密码），以便在新设备上顺利地重新登录账户。
