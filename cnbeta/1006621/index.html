三星Galaxy Note 20渲染图和详细配置曝光：设计大改 配置旗舰

July 23, 2020

7月23日早间消息，三星Galaxy新品发布会将在北京时间8月5日举行，预计会发布Note 20系列、折叠屏幕手机Fold 2等产品。目前外媒Winfuture曝光了Note 20 系列的详细配置还有几张渲染图。先来看看标准款的三星Galaxy Note 20，预计采用6.7英寸FHD+ 60 Hz平面屏幕，分辨率为2400x1080，拥有金属边框 + 塑料背板，搭载Exynos 990 并辅以 8GB+256GB存储方案且不支持可扩展存储。

三星Note 20 Ultra渲染图

”超大杯“的三星Note 20 Ultra是这次曝光的重点，三星Note20 Ultra正面设计与Note10系列相似，均为居中打孔设计，但是屏幕曲率似乎有所增加，而且从渲染图效果来看，机身似乎也较上代更加修长。三星Note 20 Ultra配备了一块6.9英寸OLED显示屏，手感应该不错，屏幕像素为3200×1440，刷新率高达120Hz，将首发康宁大猩猩第7代玻璃。

详细配置

配置方面，三星Note 20 Ultra将搭载三星Exynos 990处理器，但是国行版本可能会改用高通骁龙865 Plus，辅以12GB运存和256GB/512GB内存，支持通过microSD卡扩充。

1.08亿像素主摄像头

相机部分，根据曝光的信息来看，三星Note 20 Ultra采用了重新设计的1.08亿像素主摄像头，内置f/1.8大光圈，辅以两颗1200万像素摄像头，其中一颗是潜望式摄像头，另一颗是超广角镜头，支持50倍变焦，前置1000万像素自拍镜头。

S Pen手写笔

S Pen手写笔一向是Galaxy Note系列的卖点，这次三星Note 20的手写别也有了很大改进，根据曝光的信息来看，延迟仅有9毫秒，操作体验更接近传统手写笔效果。

另外有传三星与微软Xbox合作，透过Xbox Game Pass可以让Galaxy Note 20 Ultra畅玩多达90多款游戏。

其他部分，标准版的三星Galaxy Note 20采用双SIM卡配置，拥有4300mAh 电池，而三星Note 20 Ultra配备了一块4500mAh电池，两款产品都支持快速充电，30分钟内可将手机电量充到50%，支持无线充电、NFC、WiFi 6、双GPS和IP68级防尘防水等。

三星Galaxy新品发布会将在北京时间8月5日22点举行，新浪数码也将全程报道。
