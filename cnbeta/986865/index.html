Red Hat Enterprise Linux 7.8 更新

June 04, 2020

红帽对两个月前发布的 Red Hat Enterprise Linux 7.8 进行了更新，7.8 是 RHEL 7 进入维护阶段的首个版本，此次更新同样是 7.8 的首次更新（也有可能是最后一次更新）。红帽表示在升级之前，用户应该开始认真思考他们升级至 RHEL 8 的策略。

6·18活动已全面开启 大促活动入口汇总：

阿里云6·18上云年中大促 点击领取最高12000元红包

2020天猫6·18超级红包在此领取 6月1日追加40亿元消费券

京东6·18十七周年庆大促主会场入口 - 最高可领618元红包

此次更新主要是针对容器工具进行升级。值得注意的是，从该版本起，用户将不会再接收到新版本的容器工具。需要使用最新版本 Podman, Buildah 和 Skopeo 的用户，应该迁移至 RHEL 8 中，因为这里的容器工具模块每季度更新一次。

此次对 RHEL 7.8 的更新则是为其中的容器工具提供最后一次更新，主要包括以下三方面：

Rootless Containers 现已正式 GA，在 RHEL Server 和 RHEL Workstation 中也完全受支持 (release notes)

对容器工具的主要更新：Podman 1.6.4, Buildah 1.11.6 和 Skopeo 0.1.41 (release notes)

升级容器镜像 (release notes)

详细内容查看发布公告。

下载地址：https://access.redhat.com/downloads/

Red Hat Enterprise Linux（RHEL）是红帽公司开发的 Linux 发行，它专攻商业市场。Red Hat Enterprise Linux 面向 x86、x86_64、Itanium、PowerPC、IBM System z 架构发布服务器版本，面向 x86 和 x86_64 处理器发布桌面版本。红帽公司的所有正式支持和培训，以及红帽认证计划，均围绕 Red Hat Enterprise Linux 开展。
