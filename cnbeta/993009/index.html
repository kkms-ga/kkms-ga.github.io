新扩展程序允许Chrome直接跳转并突出显示网页上的文本段落

June 19, 2020

谷歌为 Chrome 浏览器用户推出了一款名叫“Link to Text Fragment”的新扩展程序。顾名思义，它可以无视网页格式，将任意文本片段生成为一个特定的链接。安装该扩展之后，你只需选中要突出显示的文本段落，然后右键点击“将链接与选定文本”关联起来。

（来自：Chrome Web Store）

更棒的是，任何使用兼容浏览器的人们，都可以借此来分享和打开。比如点击 Google Chrome 或新版 Microsoft Edge 中的链接，即可直接跳转到凸显的文本部分。

据悉，谷歌近期向 Chromium 添加了一项名为“Text Fragments”的新功能。通过在 ＃之后，将额外的链接信息附加到 URL 上来实现新的功能。

谷歌官方偶尔也会利用这项技术，将链接指向网页搜索结果中的特定部分。然而需要链接到较长的文本部分或复杂网页的化（比如多次重复的相同单词），长网址就难以手动创建了。

好消息是，心扩展程序不仅简化了链接的创建过程，还兼容所有基于 Chromium 内核的新版浏览器（80+）。

对于希望在网页上轻松查找特定信息的人们来说，Text Fragments 确实是一项不错的功能。

遗憾的是，Mozilla Firefox 和 Apple  Safari 尚不支持这项新特性。点击链接后指挥跳转到网页顶部，而不会突出显示相关文本段落。

Link to Text Fragment extension demo（via）

此外，在非官方浏览器中，Text Fragment 可能还不够完美。比如在 Microsoft Edge 中测试时，有时链接会跳转到网页上正确的部分，但又无法突出显示相关文本。

以及在创建链接时，尽管已经选择了整个段落，偶尔还是会受到一条文本可能超长的报错信息。感兴趣的朋友，不妨去 Chrome 网上应用店下载体验一下。
