Windows 10 2004已重新支持蓝牙A2DP音频串流接收功能

May 19, 2020

外媒报道称，Windows 10 Version 2004（2020 年 5 月更新）重新引入了对蓝牙 A2DP 接收器的支持，以便用户通过蓝牙连接，将音乐从移动设备串流到 PC 扬声器。其实早在 Windows 7 时代，微软桌面操作系统就已经支持这项特性，但 Windows 10 平台却长期缺乏对该功能的支持。好消息是，随着 5 月更新的到来，软件巨头终于又把它捡了起来。

访问:

微软中国官方商城 - Windows

外媒 Windows Latest 在支持文档中发现，微软已证实可在 Windows 10 Version 2004 中配置开启蓝牙扬声器功能，同时仍可在移动设备端收听到音频。

据悉，AudioPlaybackConnection 被用于启用 / 禁止创建远程设备连接，不过用户也可以将连接到 PC 的有线耳机当做远程音源。

微软补充道：Windows 10 操作系统现支持通过底层蓝牙组件来处理远程音源，并在计算机音频节点上播放（比如扬声器或有线耳机）。

需要注意的是，在默认情况下，蓝牙 A2DP 接收器功能功能是保持禁用状态的，需要由负责远程音频流的应用程序进行管理。

按照计划，Windows 10 Version 2004（2020 年 5 月）更新有望在 5 月 26 ~ 28 日之间正式向公众推送，其中包含了许多面向消费者、企业、以及开发者的体验改进。

以 Windows Linux 子系统（简称 WSL）为例，新版 WSL2 提供了单独的 Linux 内核。

这套‘真内核’将成为 Windows 10 操作系统的一部分，并在精简版的 Hyper-V 虚拟机管理器程序上运行，且可通过 Windows Update 实施安全修复。

新的虚拟化方案类似于 Windows 沙箱，允许 Windows 和 Linux 共享相同的资源，以避免性能问题。

此外，Windows 10 Version 2004 带来了对 WDDM 2.7 显示驱动的支持，有助于提高帧率、减少某些游戏的延迟、改进了视频播放体验、以及修复了多屏显示等问题。
