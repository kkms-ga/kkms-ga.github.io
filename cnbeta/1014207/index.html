谷歌更新Chrome Apps淘汰时间表 2022年6月彻底结束支持

August 11, 2020

谷歌刚刚在 Chromium Blog 上发布了更新后的时间表，以便大家了解预知未来几年的支持计划。该公司在 1 月发布了最初的公告，当时指出 Chrome Web Store 将终止对 Chrome Apps 的支持。需要指出的是，这项政策仅适用于 Chrome 网上应用店中的 Chrome Apps，而不会影响 Chrome 扩展程序。

（来自：Chromium Blog）

Google 解释称，之所以宣布这一消息，是因为现代 Web 已经迎来重大发展，并且能够为用户提供一流的体验。该公司将继续投资 Web 平台上丰富的新功能，并致力于推动 Web 向前发展。

根据近日公布的时间表，Chrome Web Store 已于 2020 年 3 月份开始停止接收 Chrome Apps 的提交，但现有应用可继续服役到 2022 年 6 月。

不过为了照顾企业客户，Google 已经适当调整了这一政策，以便企业管理员继续向 Chrome Web Store 提交私有和不公开的新应用。

修订版的时间表补充道：到 2021 年 6 月，该公司将终止对 Windows、Mac 和 Linux 平台上的 Chrome Apps 的通用支持。

不过在 12 个月的扩展期内，Chrome OS 将继续提供支持，而无需对策略或设置进行任何调整。

最终到 2022 年 6 月的时候，Chrome Web Store 将彻底停止接收任何形式的 Chrome Apps 。

与此同时，Google 也将终止对所有平台上的 NaCl、PNaCl、以及 PPAPI 的支持。
