三星为售价999美元的Galaxy Note 20配备了塑料后盖

August 06, 2020

2020 是相当魔幻的一年，因此当我们得知三星仅为 Galaxy Note 20 Ultra 旗舰机型配备了玻璃后盖的时候，似乎也没有那么惊讶。与 Note 20 Ultra 相比，Galaxy Note 20 机型在硬件上有许多折衷。但若消费者知道售价 999 美元的它只配拥有廉价的塑料后盖，那 Note 20 的销售或遭到更艰难的考验。

Note 20 Ultra 提供了三种配色（图自：Samsung）

当几周前听闻 Note 20 的规格要比 Note 20 Ultra 差得多时，我们本以为 Note 20 系列的入手门槛会更加吸引人。只要价格到位，聚碳酸酯后盖并不是什么难以启齿的事情。

在周三的 Unpacked 2020 发布会上，三星宣布了 Note 20 的详细规格、并提及了对塑料的热爱。然而“聚碳酸酯增强材料”的说法，最终并未出现在新闻稿中。

Galaxy Note 20 也提供了三种配色

三星产品设计师 Dong Hub Kim 曾在 2014 年 4 月推销 Galaxy S5 时表示，其主要目标是“更人性化的友好设计的可用性”，当时的解释或许可套用到最新的 Note 20 身上：

我们想要带来一种更令人愉悦的感觉，比如更好的握持手感。塑料的质地更加温暖，如果使用金属的话，设计可能更显笨重和冰冷。

聚碳酸酯材质在视觉和体积表达上也是最佳的方案，我们相信用户会喜欢这种既温暖又友好的感觉，且其象征着更出色的设计理念。

YouTube 视频截图（via Samsung）

当年三星在平板电脑上也使用了塑料材质，但随着技术的发展和时间的推移，这种倒退也可能产生一些意想不到的后果。

虽然 OnePlus Nord 的塑料框架很容易被扭曲，但考虑到它的售价只有 450 美元左右，目标消费群体还可以勉强接受。

然而三星 Note 20 的起步价格已经接近 1000 美元，即便将同样的事情套到苹果身上，我们也实在无法想象消费者会将 1000 美元的塑料 iPhone 喷成什么样。
