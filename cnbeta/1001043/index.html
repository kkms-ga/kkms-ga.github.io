GitHub在接到DMCA通知后删除了Chimera13 iOS越狱工具项目

July 09, 2020

为了满足功能上的更多需求，一些用户宁愿借助某些工具将自己的 iOS 设备给越狱。通常情况下，越狱工具是可以免遭《数字千年版权法案》（DMCA）的打击的。然而本周，代码托管平台 GitHub 却移除了 Chimera13 的存储库。需要指出的是，这项有争议的撤除通知并非来自设备制造商苹果，而是同样开发越狱解决方案的竞争对手 Unc0ver 发出的。

尽管苹果在努力保护自家的封闭生态，但还是无法全面阻挡外部人士的越狱方案吗，只能在越狱工具推出后立即实施相应的漏洞修补。

但 Unc0ver 对阵 Chimera13 一事，还是引发了越狱社区的高度关注。其指责一位名叫“Coolstar”的第三方开发者盗用了自家团队的越狱代码，并将之作为基于 Swift 的 Chimera 13 越狱方案的一部分。

Unc0ver 在 DMCA 版权移除通知中写道 —— 被投诉人获取了 Unc0ver 私有信息库泄露的副本，但该副本此前仅可在 Unc0ver 团队的特定成员内使用。

Unc0ver 对此展开了调查，结果发现其首次出现在社交平台 Reddit 的 r / Jailbreak Discord Server 版块。后来他们宣布获得了相关源码，并在 Twitter 上公开叫嚣。

最终 GitHub 在接到 Unc0ver 的 DMCA 通知之后，按要求移除了 Chimera13 的存储库，以及数十个相关分支。现在访问的话，只能看到官方存储库和各个分叉都不可用的提示。

据悉，Chimera13 开发者 Coolstar 曾是前加州大学圣巴拉拉分校计算机科学系的一名学生。对于此事，其辩称 Unc0ver 的删除请求是毫无根据的。

目前他已经向 GitHub 方面发送了反 DMCA 通知，要求该公司恢复相关存储库。他写道：Chimera13 是我用 Swift 原始代码编写而成，主要依赖的是公共技术。

该存储库中唯一的第三方代码，就是 Jake James 的 time_waste 攻击，但这也是在遵从开放源码许可证（GPL）的情况下进行的。

在 Twitter 上，Coolstar 再次强调 Unc0ver 是出于恶意目的、并且滥用了 DMCA 删除通知。

由此看来，横亘在两个 iOS 越狱阵营之间的长期争执，似乎无法在短时间内轻易解开。
