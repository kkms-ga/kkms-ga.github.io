[图]Android端Chrome 85开始标记“fast page” 改善用户体验

August 18, 2020

谷歌本周一宣布，在被认为高质量的页面右键菜单中新增“fast page”标签。谷歌表示，这些标签是基于其 Core Web Vitals 指标的综合评价，这些指标跟踪用户体验，包括页面加载时间、响应速度以及加载时内容的稳定性等。

如果你在访问一个页面之前长按链接，如果符合谷歌的标准，那么就会得到“fast page”的标签。拥有“fast page”标签的页面在谷歌搜索中的排名也会更高。谷歌表示，其他 Core Web Vitals 指标也会纳入到搜索排名算法中。
