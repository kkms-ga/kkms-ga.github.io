三星Galaxy Note 20系列价格汇总

July 31, 2020

根据此前三星官方发布的预告，三星将于8月5日举行发布会，并推出Galaxy Note 20旗舰系列，Buds真无线蓝牙耳机、Z Flip 5G折叠屏手机等新品。随着发布会的临近，关于新机Galaxy Note 20的爆料信息也越来越多。

近日海外爆料人Ishan Agarwal就透露了他提前拿到的三星这些新品在欧洲的官方零售价。

具体为：Note 20 4G：999欧元（约合人民币8224元），Note 20 5G：1099欧元（约合人民币9048元）。

Note 20 Ultra 5G有两个版本，其中256GB版本的售价为1349欧元（约合人民币1.11万元），512GB版本的售价1449欧元（约合1.19万元，512GB）。

而蓝牙耳机Buds Live售价为189欧元（约合人民币1553元）。

配置方面，Note 20采用6.7英寸屏幕，分辨率和刷新率分别为1080P和60Hz，配备64MP+12MP+12MP三摄（支持30倍变焦）的后置三摄。

Note 20 Ultra则采用6.9英寸显示屏，具备2K分辨率和120Hz的刷新率，后置108MP+12MP+12MP三摄（支持50倍变焦）。
