马斯克透露柏林超级工厂将生产深红新配色的Model 3

July 22, 2020

在周一的一系列推文中，马斯克在与网友互动的同时，透露了柏林超级工厂（Gigafactory Berlin）将具有先进的喷漆车间，并且为 Model 3 用户提供全新的“深红”（Deep Crimson）配色选项。截至目前，特斯拉仅为消费者提供了少数的颜色选项。除了基本免费的珍珠白配色，纯黑（Solid Black）、午夜金属银 / 深蓝（Midnight Silver / Deep Blue Metallic）、多层次红漆（Red Multi-Coat）都需要加钱选购。

资料图（来自：Tesla）

回顾初代 Roadster 车型，特斯拉提供了 13 种不同的颜色选项，比如经典的跑车绿、亮黄、橙色、以及电蓝。

Model S 的可选范围也比较大，比如金属绿、金属棕等配色。初次选购的客户，亦有限量的标志红（Signature Red）选项。

尽管马斯克没有详细描述这种“深红”将具有怎样的质感，但他此前确实提到过柏林超级工厂的喷气车间将能够产出“令人惊叹的更多层次色彩，可随着曲率而发生变化”。

然而特斯拉在喷气工艺上的经验积累并不足，尤其是近年来，随着该公司加大对 Model 3 和 Model Y 跨界车型的投入，一些买家已发表过有关漆面品控不一致的抱怨。

庆幸的是，特斯拉已承诺改善车漆质量，并且已经取得了不错的进展。比如前通用汽车高管 Bob Lutz，就曾于去年发表过对 Model 3 制造质量的看法：

当我在 Ann Arbor 停车场检查金属红配色的 Model 3 车型时，不由地想到了有关装配工艺粗糙的早期评价。

然而当我站到车旁时，还是被几乎没有明显瑕疵的高精度车漆给震撼到了。从引擎盖到挡泥板、以及车门到框架的各种缝隙，看起来都相当均衡一致。
