[图]本月Windows 10累积更新再出BUG：安装时跳出错误代码

August 17, 2020

在本月的补丁星期二活动日中，微软为所有支持的 Windows 10 系统带来了新一轮的累积更新。和以往一样，本次发布的 KB4566782 和 KB4565351 两个累积更新又出现了新的 BUG。

根据用户反馈，Windows 10 Version 2004/1909/1903 功能更新安装这些累积更新之后，就会出现错误代码。其中包括 0x800f0988、0x800f081f 和 0x800f08a、0x80071160，其中 0x800f081f 是报告最多的错误代码。

其中一条错误信息写道：“安装更新时出现了一些问题，但我们稍后会再次尝试。如果你一只出现这条错误信息，请在网络上进行搜索或者联系支持信息。这可能会有所帮助。（0x800f081f）”。

此外，Feedback Hub、Reddit 和 Twitter 上也有用户反馈道：“我的桌面电脑目前运行 Windows 10 Version 2004 功能更新版本 Build 19041.423，尝试安装 KB4566782 更新来升至 Build 19041.450。不过它并不能完成安装，接收到 0x80071160 错误”。
