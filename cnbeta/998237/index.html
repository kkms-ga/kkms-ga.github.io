微软正在调整Windows 10窗口切换组合键的体验

July 02, 2020

7 月 1 日，微软向开发者通道 Insider 测试者们推送了 Windows 10 Build 20161 编译版本，改进了许多功能体验和实施了大量的 bug 修复。比如为“开始”菜单引入主题感知功能，以解决应用程序图标的纯色背景。此外 Windows 10 Build 20161 还为瓷贴提供了 Fluent 设计风格的半透明背景，以更好地迎合用户的自定义配色方案。

在微软将重心逐渐偏向 2021H1 的同时，Microsoft Edge 浏览器的多个选项卡也出现在了可由 Alt + Tab 组合键唤出的窗口切换界面中。

微软表示，调整后的 Alt + Tab 体验将帮助用户随时返回其在 Microsoft Edge 中的 Web 会话。

不过作为一项可选功能，用户依然可在“设置 -> 系统 -> 多任务处理”中切换至经典的 Alt + Tab 窗口切换体验。

微软还正在更新 Windows 10 中的通知体验，以使之同 Windows 10X 平台保持一致。

软件巨头表示，用户现可点击通知气泡右上角的“×”按钮将其关闭。或者点击齿轮图标，以跳转到全局通知设置。

最后，微软已开始在新安装的 Windows 10 操作系统上测试任务栏的个性化选项。目前该公司正在积极听取 Insider 测试者的反馈，以找到默认情况下最佳的任务栏图标固定体验。

遗憾的是，上述改进无法在今年晚些时候的 Windows 10 20H2 更新中得到体验。
