Reddit推出基于以太坊的代币 加密货币与《堡垒之夜》子版块正在公测

May 14, 2020

The Block 报道称，知名线上论坛 Reddit 正在对两款基于以太坊的加密货币令牌展开公测，贡献内容的用户将获得相应的奖励。作为新发起的“社区积分”（Community Points）计划的一部分，名为 $ MOONS 和 $ BRICKS 的两款代币将分别发放给 reddit 上的 / r / Cryptocurrency 和 / r / FortniteBR 坛友。

Reddit 发言人表示表示：“我们将通过持续的实验来探索能够吸引社区用户的功能，借助社区积分项目，我们希望这两个版块的网友能够积极提交反馈”。

此外 Reddit 推出了一个指导用户打开“Vault”的网页，这是一个用户管理 Reddit 发行的代币的新平台。根据移动 App 上的提出，用户可在多个 Reddit 子版块上查看和消费其“社区积分”。

实际上，这两款代币均基于以太坊区块链上发行的 ERC-20 令牌。通过创建保管库，用户实际上也是开设了一个用于存储代币的以太坊钱包。

据 Reddit 所述，目前该项目仍处于公测阶段，预计会持续到 2020 年夏季。为打消大家的顾虑，Reddit 还强调智能合约与移动 App 都经过了具有专业知识的独立安全公司 Trail of Bits 的审核。

此外 Reddit 鼓励安全研究人员帮助其查找漏洞，感兴趣的开发者可移步至 https://www.reddit.com/wiki/whitehat 查看详情。

Reddit 网友能够使用代币进行打赏和转移支付，论坛上还可以显示账户余额，以衡量其可信度。目前 / r / Cryptocurrency 和 / r / FortniteBR 子版块分别拥有 100 万和 130 万坛友。
