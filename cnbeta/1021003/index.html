Twelve South推出面向新款iPad Pro的BookBook系列书皮保护套

August 27, 2020

Twelve South 推出的 BookBook 系列保护套，能够为新款 iPad Pro 和 Magic Keyboard 用户带来更佳的防护。其中 BookBook Cover 系列有 11 / 12.9 英寸的版本，能够作为 iPad Pro 的集成支架使用，但不兼容与 Magic Keyboard 一同使用，售价为 69.99 / 89.99 美元。另有面向 11 / 12.9 英寸 iPad Pro 的 BookBook Case 系列，售价为 79.99 / 99.99 美元。

细节方面，BookBook Cover 和 BookBook Case 均采用了类似精装书壳的真皮材质。带有两个拉链头，以方便接入充电数据线。加固的书脊和四角可在意外跌落时提供一定的防护。

由于采用了类似皮革装订书的普通外观，BookBook 系列保护套或有助于防止用户的 iPad Pro 被小偷给盯上。

其支持第三代和第四代的 iPad Pro 机型，并且带有摄像头开孔，以及用于放置线缆和细长键盘的位置。

售价方面，适用于 11 / 12.9 英寸 iPad Pro 的 BookBook Cover 为 69.99 / 89.99 美元（约 482 / 620 RMB），BookBook Case 系列则是 79.99 / 99.99 美元（约 551 / 688 RMB）。
