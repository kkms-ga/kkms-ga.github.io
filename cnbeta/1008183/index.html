AMD锐龙4000 Renoir APU图形处理器能力强于PS4与Xbox One

July 27, 2020

近日曝光的 AMD 锐龙 Renoir APU 的基准测试数据，其 CPU 和 GPU 处理能力给我们留下了深刻的印象。在索尼 PS5 和微软 Xbox Series X 这两款次世代主机于今年晚些时候上市之前，可知新品已经大幅超越了 2013 年发布的 PlayStation 4 和 Xbox One 的整体机能。

据悉，2013 年发布的索尼 PS4 和微软 Xbox One 同样采用了 AMD 定制的 CPU + GPU 芯片架构，但锐龙 4000 Renoir APU 的表现更是让人刮目相看。

原始性能方面，PS4 和 Xbox One 都配备了主频 1.6GHz 的 AMD Jaguar 处理器（更高的 Xbox 机型版本为 1.75GHz）。CPU 本身的性能约 0.1 T-Flops，但 GPU 部分的配置截然不同。

在 PS4 上，索尼选用了基于 AMD GCN 架构的 GPU，具有 1152 个内核、理论计算性能为 1.84 T-Flops 。

Xbox One 则是768 核的 AMD GCN GPU，理论性能 1.3 T-Flops 。Xbox One S 的性能和效率都略有提升，高频 GPU 可输出 1.4 T-Flops 的算力。

时隔七年，我们见到了 AMD 锐龙 4000 Renoir APU，其原始性能较 PS4 和 Xbox One 有着天翻地覆的差别。

芯片制程方面，等于是台积电 7nm 吊打 28nm 工艺节点，辅以 Zen 2 架构的 CPU 和增强的 Vega GPU 。

即便 CPU 核心数量相同，甚至 GPU 的核数等地，但 Renoir APU 仍提供了卓越的性能。

根据 Reddit 上 AMD 讨论板下的 AIDA64 GPGPU 基准测试数据，锐龙 7 PRO 4750G 的 CPU 部分算力为 1.1 T-Flops，GPU 部分更是 2.15 T-Flops 。

作为对比，即将将 Jaguar CPU 的图形算力加进来，PS4 充其量也就能够输出 1.95 T-Flops 的算力。

Renoir APU 的 GPU 核数只有前者的一半不到，但先进的 7nm 制程架构让该 SoC 的总算力达到了 3.25 T-Flops 左右（其中 CPU 部分为 1.1 T-Flops）。

目前 Xbox One S 和 PS4 Slim 的售价仍在 400 美元左右，而最高端的零售版 Renoir APU 的预计售价为 299 美元。当然，这里并没有将主板、内存、电源、存储等系统组件的费用计算进去。

如果组装一套总计 500 美元的整机，想必也可提供相当不错的游戏性能。即使是 209 美元的 AMD 锐龙 PRO 4650G，也可提供 2.5 T-Flops 的不俗算力。

要求不高的话，售价 149 美元的锐龙 3 PRO 4350G 也是个不错的选择。算力 1.9 T-Flops，较 Xbox One S 更快，与 PS4 并驾齐驱。

值得一提的是，锐龙 7 PRO 4750G 的 7nm Vega GPU 拥有巨大的超频潜力，最高可达 2400MHz，意味着这枚 65W 的芯片拥有令人印象深刻的 2.5 T-Flops算力。

Tech Epiphany 的测试表明，在 1080p 的分辨率下，《刺客信条：奥德赛》、《侠盗猎车手 V》、《死亡搁浅》等游戏表现都相当亮眼。
