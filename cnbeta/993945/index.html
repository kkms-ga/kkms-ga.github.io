[图]扩展推荐：Tabs Aside可实现经典Edge的“搁置标签页”功能

June 22, 2020

在迁移到 Chromium 之后，微软一直在努力将经典版 Edge 的优秀功能移植到新版 Edge 浏览器中，不过稍感遗憾的是其中并不包括“Set aside tabs”功能。不过现在，你可以通过安装这款“Tabs Aside”扩展程序，在 Edge 和 Chrome 上实现该功能。

经典版 Edge 浏览器允许用户通过点击左侧的图标将所有打开的标签页先放置在一边，你可以在之后随时恢复并检查它们，这可以清除标签页的杂乱，也不会丢失打开的标签页。

操作步骤如下

1.Chrome 或者 Edge 浏览器安装该扩展程序

2.点击 Tabs Aside 按钮

3.点击“Set current tabs aside”（搁置当前标签页）

4.当前所有打开的标签页都会消失，就像你已经关闭了它们一样，点击扩展图标，然后点击 “恢复标签页的按钮，你想再次看到的标签。

通过点击 “还原标签 “旁边的菜单，你能够在不删除的情况下重新打开标签，并且可以对一旁的标签进行书签或收藏。目前，共享标签页功能在Edge中无法使用，Chrome也不支持该功能。
