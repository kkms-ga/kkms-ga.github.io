[图]Android端Chrome即将允许生物认证来自动填写支付信息

July 07, 2020

现在当你尝试在谷歌 Play 商城购买应用程序或者游戏的时候，允许通过生物识别认证程序来确认交易。谷歌正努力将类似的功能引入到 Android 端的 Chrome 浏览器中。

在 Chrome 浏览器中新增了“Allow using platform authentication to retrieve server cards”（允许使用平台认证检索服务器卡）的新 Flag，允许用户使用生物识别认证或者 Android 设备的屏幕锁来验证身份。

而目前，当用户自动填写支付细节的时候，他们被要求输入支付卡的 CVC 信息在服务器上检索数据。虽然该 Flag 已经在 Chrome 的 Stable、Beta、Dev 和 Canary 频道上线，但是启用之后无法正常工作。

启用该标志后，每当用户下一次尝试自动填写信用卡或借记卡信息时，都会收到通知，他们可以使用锁屏密码来确认身份。尽管如此，这个 Flag 证实了谷歌确实在努力将支付细节自动填写的生物识别认证带到 Chrome 浏览器中。
