2018-2020手机保值率排行榜：iPhone依然远高于Android阵营

July 03, 2020

由于技术发展和产品迭代实在太快，大多数电子产品都会随着时间的推移而越来越便宜。比如曾经动辄上万的大屏电视，如今已经只需几千块轻松带回家。不过在智能手机领域，苹果的 iPhone 虽然售价较高，但其保值率也一直被津津乐道。即便竞争对手在同期推出的旗舰新品的售价与之相当，但保值率却明显落后于 iPhone（比如三星的 Galaxy 系列）。

（来自：SellCell，via：PCMag）

手机零售商 SellCell 近日发布的数据显示，尽管苹果 iPhone 的售价较高，但其保值率却较一种 Android 旗舰竞品更加出色。

以 iPhone XR 为例，从 2018 ~ 2020 年，其价值仅损失了 53.2% 。iPhone XS / XS Max 的贬值率则是 56% 和 56.8% 。

作为对比，LG V40 ThinQ / 三星 Galaxy Note 9 在同时期内的贬值率分别为 79.10% 和 79.20%，三星 Galaxy S9 系列 / 索尼 Xperia X22 系列更是贬值超过 80% 。

Android 阵营中相对保值的 Google Pixel 3 / 3 XL，贬值率也达到了 72.40% 和 75.6% 。
