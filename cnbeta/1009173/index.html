光盘老矣：请记得对重要的资料进行二次备份

July 29, 2020

曾几何时，光盘是许多人保存和体验大容量媒体内容的首选介质。但是随着闪存和云存储技术的发展，光驱已开始逐渐淡出普通消费者的视野。除了音乐、游戏和影视资源，你或许会还在整理橱柜或打扫阁楼时，翻出几张充满回忆的碟片。然而当你想要重新读取的时候，可能会遭遇盘片因年久而损坏的局面。正因如此，二次备份的重要性也是不言而喻的。

作为一种基于特殊染料的激光存储解决方案，一张普通 CD 的容量在 6-7 百兆之间，而单面 DVD 则是 4GB 多点。至于更高容量的多层红蓝光碟片，或许也将在云时代销声匿迹。

在合适的环境中，光盘或许可以维持较长的保存年限（至少 15~25 年）。然而根据加拿大政府的一项研究，普通盘片的“最佳赏味期限”，或许只有 5~10 年的时间。

除了塑料外层的正常使用磨损，紫外线和脏污也会对光盘造成很大的伤害。随着时间的推移，粘合层迟早会氧化脱落。

为了避免重要数据毁于一旦，定期翻录和异地多重介质备份，就成为了长期保管光盘资料的一个良好习惯。

软件方面，Windows / macOS / Linux 平台都有比较受欢迎的刻录工具，比如 Nero Burning ROM、UltraISO 等。当然如果资料结构比较简单的话，系统自带的磁盘工具也可以轻松应付。

在光盘之外，建议有条件的人们还可以通过磁记录介质（机械硬盘 / 磁带机）和云端硬盘保存一份，毕竟闪存介质也是相对比较脆弱的。
