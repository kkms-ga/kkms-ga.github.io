[图]微软庆祝Windows 95发布25周年：对桌面体验影响深远

August 25, 2020

微软发布 Windows 95 系统至今已走过 25 个年头了。这款系统对桌面体验的影响是非常深远的，开始菜单和任务栏概念沿用至今，并且全球有数十亿用户使用。在过去 25 年里，Windows 系统不断更迭进化，已经成为很多人日常工作生活中不可或缺的重要组成。

在庆祝 25 周年之际，微软分享了一段对两位曾参与 Windows 95 开发的工程师采访，目前他们仍在微软工作。相关的细节如下：

The Start of Something Different, Part 1:

本月的两集视频中，记者 Jason 采访了 Windows 的首席软件工程师 Raymond Chen。他回顾了 Windows 95 系统的历史以及早期的应用程序兼容性和测试。你还可以通过 Raymond 的博客 “The Old New Thing “阅读更多Windows历史。

The Start of Something Different, Part 2:

第二集将会在本月晚些时候发布，Jason 采访了另一位软件开发者 Chris Guzak。他们探讨了Windows 95标志性体验的历史，包括开始菜单、任务栏以及软件可用性测试的早期。
