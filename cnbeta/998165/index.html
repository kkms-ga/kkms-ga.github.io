[图]Chrome OS推进Steam新线索：虚拟机系统要从Debian换成Ubuntu?

July 02, 2020

今年早些时候，有报道称谷歌正努力将 Steam 引入到 Chrome OS 中。而现在，外媒 9to5Google 发现了其中的一些细节，包括 Chrome OS 如何运行 Steam 以及 ChromeBook 如何获得支持。

谷歌在一年前就已经启动了“Crostini”项目，允许 Chrome OS 运行 Linux 应用程序。在后台，Crostini 在虚拟机上运行完整的 Linux 发行版本，有点类似于桌面上运行 Android 模拟器。

在过去几周内，该外媒一直在追踪代号为“Borealis”的 Chromium 开源代码中的新项目。根据一些相关的代码变化，Borealis 似乎也与 Chrome OS 的虚拟机有关。

经过一番挖掘，通过 Borealis 拷贝发现它其实就是另一个完整的 Linux 发行版本。与基于 Debian 的 Crostini 不同，Borealis 将于另一个 Linux 发行版 Ubuntu。就像现有的Linux应用支持一样，我们相信Borealis将与Chrome OS整合在一起，而不是成为一个完整的桌面体验。

随后外媒发现，Borealis 和普通安装 Ubuntu 之间有一个关键的区别，就是 Borealis 包含了一个预装的 Steam 副本。这与我们在 CES 2020 上了解到的情况一致，当时谷歌 Chrome OS 产品管理总监 Kan Liu 分享说，即将推出的 Steam 游戏支持将基于 Linux。

当然也引发了一系列问题，其中最明显的疑问就是：“既然 Debian 一直运行良好，那么为何要换成 Ubuntu 呢？”

Steam 的开发商 Valve 一直非常积极地推进 Linux 移植工作 。他们最大的贡献是开发了一个 “兼容层”–Proton，帮助 Steam 在 Linux 上运行原本为 Windows 打造的游戏。虽然 Proton 能在所有主流 Linux 发行版本上运行，但 Steam 的安装说明中规定，Ubuntu 是他们推荐给游戏玩家的 Linux 发行版本。

目前，我们还不确定谷歌是否计划让Chromebook用户从现有的 “Crostini “Linux安装版切换到 “Borealis”，或者两者是否打算共存。不过后者似乎不太可能，因为这会无谓地占用额外的存储空间。
