Windows 10失去市场份额 而macOS出现明显上升

May 03, 2020

微软的Windows 10操作系统在2020年4月罕见地出现了倒退，它失去了一直以来稳步增长的市场占有率，而macOS显然受益于这一下滑，市场占有率有明显增长。Windows 10已经是全球第一大桌面操作系统，目前每2台PC中就有1台在运行该系统。而且，虽然Windows 10的市场份额有望继续增长，尤其是在微软退役Windows 7之后。

微软中国官方商城 - Windows

更多的设备升级到新版本之后，Windows 10的市场份额还将增长，但上个月的情况却并没有遵循之前的趋势。

NetMarketShare的数据显示，Windows 10在4月份的市占率实际上从57.34%下降到56.08%，而macOS从3.41%提高到4.15%。

Windows 7方面下降到25.59%，这显得理所当然，因为它不再接收更新和安全补丁。与此同时，Windows 10仍然是大多数人在Windows生态系统中的首选。

安全专家警告说，针对Windows 7设备的漏洞的数量正在增加，这也是由于它与较新的Windows共享代码的结果，这些漏洞使设备面临潜在的攻击。

Windows 10是微软推荐给还在使用Windows 7的所有人的选择，但目前来看，很显然并不是每个人都愿意去做升级动作。毫无疑问，Windows 7已经成为了新的Windows XP，就像2014年微软退役2001年的操作系统一样，要知道，Windows XP在退役6年后的市场份额仍有1.32%以上。
