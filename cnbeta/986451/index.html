微软与BBC在英国开测Beeb语音助手

June 03, 2020

英国广播公司（BBC）与微软合作推出了先前宣布的 Beeb 语音助手的早期版本，感兴趣的朋友可注册成为 Windows Insider 测试者并下载体验 Beta 版本。需要指出的是，目前这项服务仅在英国地区开启有限测试。BBC 在去年宣布了 Beeb 语音助手项目，但它并非要成为亚马逊 Echo 硬件的简单替代，而是在自家产品（包括网站和 iPlayer TV 流媒体服务）中构建语音服务。

虽然只有英国地区的 Windows Insider 测试者才有机会在 Windows 10 PC / 平板电脑上访问，但 Beeb Beta 版本现已在英国开放下载，且任何人都可以轻松注册成为 Windows Insider 测试者。

起初，Beeb 仅支持通过语音控制来播放特定的 BBC 节目，比如“OK Beeb，播放 Radio 1”。不过用户也可以要求刷新内容和获取当地的天气信息，以及让 Beeb 讲个笑话。

个性化方面，首个公测版本仅支持男声，但 BCC 希望后续为用户提供更加丰富的音源选项。
