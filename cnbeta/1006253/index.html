Galaxy Note 20 Ultra完整规格：首配大猩猩7玻璃 S Pen延迟仅9ms

July 22, 2020

德国科技媒体 WinFuture.de 在最新报道中，分享了 Galaxy Note 20 Ultra 的详细规格，并表示在欧洲市场的售价将会超过 1000 欧元（约 8050.7 元）。

根据报道中提供的细节，Galaxy Note 20 Ultra 所搭载的 S Pen 手写笔将会采用改进后的“Real Feel”技术，最大的特点就是将延迟降低到 9 毫秒，提供类似于纸张的书写体验。

Galaxy Note 20 Ultra 还深入整合了诸多微软服务，包括对 Xbox Game Pass 和 Project xCloud 项目的优化。通过该服务，该旗舰能够在云端运行超过 90 款游戏，是其成为“便携式游戏机”。

在摄像头方面，装备了改进版的 1.08 亿像素的主摄，并大大提高了对焦功能。该机不仅提供双像素自动对焦，也集成了一个基于激光的自动对焦系统，以及5倍光学变焦。前置 1000 万像素摄像头也有双像素自动对焦功能。

机身正面装备了 6.9 英寸屏幕，确认支持 120Hz 的刷新率，并且能够根据你的工作、游戏进行动态调节。而且该机是首款搭载康宁大猩猩 7 玻璃的手机。

电池方面，该机装备了 4500 mAh 容量，能够在 30 分钟内将电量充至 50%。该机还配备了 wireless Dex、双卡双待、5G 和 WiFi 6。
