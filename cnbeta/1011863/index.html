AMD发布Radeon 20.8.1驱动更新 改进《地平线：零之曙光》等游戏支持

August 05, 2020

AMD 刚刚发布了 Radeon 20.8.1 显示驱动更新，主要改进了对《地平线：零之曙光》、奇幻生存类角色扮演游戏《Grounded》、以及育碧《棱彩维度》（Hyper Scape）等游戏的支持。此外在采用 RX 5700XT 独显的测试平台上，与上月的 20.7.2 驱动相比，20.8.1 还将《Grounded》的运行效率提升了 9% 。

（来自：AMD）

AMD Radeon 20.8.1 显示驱动下载地址：

Windows 10 64-bit | Windows 7 64-bit

以下是 Radeon 20.8.1 显示驱动中已修复的一些 bug：

● 修复在启用 Radeon FreeSync 的情况下，切换到另一显示器或应用程序时，可能间歇性地导致桌面刷新率锁定和画面卡顿的现象。

● 修复在 Windows 7 操作系统中，Radeon RX 5000 系列显卡无法启用串流录制功能的问题。

● 修复加入 AMD 用户体验计划后，Radeon 软件从睡眠状态唤醒时可能导致应用程序崩溃或挂起的问题。

● 修复在启用 HDR 的游戏中更改垂直同步设置时，《毁灭战士：永恒》可能出现暗屏的问题。

● 修复在未接通电源的 HDMI 显示器上执行热插拔操作，然后再通电，可能导致间歇性系统崩溃或死机的问题。

● 修复启用 Instant Replay 时，可能罕见地导致使用 Radeon RX 5000 系列显卡的系统死机 / 遭遇 TDR 故障的问题。

● 修复在打开‘兼容性检查器’时可能显示‘无法获取’报错信息的问题，用户或需要在 Radeon 软件中进行出厂重置。

此外 Radeon 20.8.1 显示驱动还有一些已知问题：

● 比如切换到其它显示器或应用程序时可能导致显示器闪烁片刻。

● Radeon RX 5000 系列显卡可能在软件或浏览器中播放某些 VP 视频内容时出现亮度闪烁。

● 在某些游戏和系统配置上启用增强同步后可能导致黑屏（可临时禁用该选项）。

● 性能指示器可能误报 Radeon RX 5700 系列显卡的闲置时钟频率（不影响性能和功耗）。

● 启用 HDR 之后，Windows 桌面可能出现闪烁，且游戏中切换任务可能导致褪色或过饱和。

● 在 Radeon RX 5700 系列显卡上通过 HDMI 连接音视频接收器时，音频输出可能不稳定。

● 在某些 AMD Ryzen 7 3000 / 4000 系列 APU 系统上扩展显示时，YouTube 播放器可能被 Microsoft Edge 或 Google Chrome 浏览器冻结。

● 修改 HDMI 缩放滑块，可能导致 FPS 被锁定为 30 。
