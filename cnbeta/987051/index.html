[图]Firefox 新功能：可将网站帐号和密码导出到 CSV 文件中

June 04, 2020

Mozilla Firefox 在近日的版本更新中引入了一项备受期待的功能：将 Firefox 登陆帐号和密码导出到 CSV 文件中。目前该功能还处于开发阶段，仅面向 Firefox Nightly 通道用户开放。相同功能已经在谷歌 Chrome 上线。

Firefox Nightly 最新版本中，你可以将浏览器的登录帐号和密码导出到 CSV 文件中。整个步骤也非常的简单：

1.打开 Firefox 浏览器

2.访问 about:logins 页面打开 Firefox Lockwise，或者点击菜单进入“登陆和密码”

3.点击“…”按钮，选择“导出登录”（Export logins）

在点击该按钮之后，你会看到一个警告页面，会强调你的密码会以明文方式进行存储，这样所有人都可以读取你的密码。在导出过程中可能需要用户输入 Windows 密码，点击确认之后可以导出为 CSV 文件。

输出的 CSV 文件包括以下字段：

● Webpage URL（网页地址）

● Username（用户名称）

● Password（用户密码）

● HttpRealm

● FormactionOrigin

● Guid

● Timecreated（创建时间）

● TimelastUsed（上次使用时间）

● TimePasswordChanged（最近一次密码更改时间）
