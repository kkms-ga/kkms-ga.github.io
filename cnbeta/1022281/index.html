三星Galaxy Book Flex 5G广告片曝光：再讽苹果

August 30, 2020

据外媒报道，三星在广告中公开嘲笑苹果的日子已经一去不复返了，即便如此，这家韩国巨头仍热衷于将其跟苹果进行比较。除了“old = bad（旧=差）”和“new = good（新=好）”的说明之外三星并没有更进一步给出实质性的解说，而这是Galaxy Book Flex 5G最新发布的视频的要点。据悉，该视频在官方发布之前就被泄露到了网上。

在这段的简短预告片中可以看到三星对Apple II和Commodore 64等产品的回顾，随后，三星得出结论，Galaxy Book Flex 5G是一种非常优秀的计算解决方案。

这支广告可以说是被一个由俗气的口号变成了行动呼吁，其邀请消费“把旧的（设备）抛到身后”。根据之前的报道，Galaxy Book Flex 5G将是三星现有笔记本电脑的一个迭代的后续产品。它是一款二合一的可转换笔记本电脑、支持S Pen和第10代英特尔CPU。最高12GB的内存、512GB SSD存储、AKG扬声器以及一系列现代连接选项–包括一个Thunderbolt 3端口和Wi-Fi - 6。

如果广告中渲染的笔记本的比例是正确的，那么Galaxy Book Flex 5G将拥有13.3英寸的QLED显示屏。而最初的系列还有一个15.6英寸的机型，至于它有没有也出现在全新的Galaxy Book Flex 5G中则还有待商榷。
