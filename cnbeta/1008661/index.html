[图]Android端Firefox大跃进：用户将默认切换至“Fenix”新版

July 28, 2020

适用于 Android 平台的旧版 Firefox 今天迎来最后一次更新。在下一轮发布周期（下月）中，Mozilla 将默认切换用户至新版浏览器中。新版 Firefox 的内部代号为“Fenix”，不过在正式推向用户之后依然称之为“Firefox”。该公司的新版 Firefox 已经打磨一年多时间，功能已日趋完善。

虽然目前桌面端 Firefox 已经到了 79 版本，但在 Android 平台上一直停留在 Firefox 68 版本上。该版本于2019年7月9日推出，与 Chrome 和其他 Android 系统上的浏览器相比，Firefox 68 已经开始显得非常陈旧。

新版 Firefox 浏览器使用了 GeckoView ，提供了更快的浏览速度、更优秀的浏览体验，相比较 Firefox 68 在外观和体验上进行了彻底的改造。
