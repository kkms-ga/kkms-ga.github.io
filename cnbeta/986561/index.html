Firefox 78测试版迎来了针对Linux和macOS平台的较大改进

June 03, 2020

尽管仍处于 Beta 测试阶段，但我们已经知晓定于本月晚些时候到来的 Firefox 78 将引入针对 Linux 和 macOS 用户的重大改进，以及 Windows 平台上备受期待的新功能。首先是面向 Windows 用户的一项福利，即 Firefox 78 将内置 PDF 阅读器。默认情况下，此类任务是由 Windows 10 上的 Microsoft Edge 浏览器代劳的（当然 Google Chrome 也支持）。

其次是面向 Linux 平台的最低系统要求改进，从 Firefox 78 开始，Linux 用户需要预备以下支持 —— 包括 GNU libc 2.17、libstdc ++ 4.8.1、GTK + 3.14 或更高版本。

然后是面向 macOS 平台的改进，Firefox 78 将放弃对较早的操作系统版本的支持，建议希望继续接受更新和安全补丁的 macOS 用户切换到 ESR 长期支持分支。

Firefox 78.0b2 下载地址：

https://ftp.mozilla.org/pub/firefox/releases/78.0b2/

Mozilla 表示：“Firefox 将是支持 macOS 10.9 / 10.10 / 10.11 的最后一个主要版本，明年将通过 Firefox ESR 78.x 来支援上述早期版本的 macOS 系统平台”。

最后，Mozilla 有望于 6 月 30 日向各平台用户推送 Firefox 78 的最新稳定版本，感兴趣的朋友可以通过 Beta 测试通道抢先体验所有功能。
