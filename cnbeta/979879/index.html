Microsoft Edge将免费保护用户免受假新闻的侵害

May 16, 2020

2019 年的时候，微软与 NewsGuard 达成了一项免费合作，以帮助 Edge Mobile 用户分辨网络上的虚假信息。现在，双方的合作已经进一步拓展到了桌面平台的 Microsoft Edge 浏览器。据悉，NewsGuard 拥有一支由 4000 名经验丰富、训练有素的记者组成的团队，致力于打击互联网上的虚假新闻和错误信息。

访问:

微软中国官方商城 - Windows

通常情况下，这项服务需要收取 2.95 美元的月份，但微软希望向桌面和移动平台的 Edge 用户提供免费福利。

根据协议，微软将赞助 NewsGuard 的新闻素养计划，旨在通过 700 多座图书馆，向在英、美、法、德、意的 700 万用户提供 NewsGuard 服务。

感兴趣的朋友，可下载 NewsGuard 扩展程序（Microsoft Store 传送门）。安装之后，你将在搜索引擎和社交媒体的链接上看到一个新图标。

今年晚些时候，双方还计划将该项目扩展到更多的市场区域，比如澳大利亚和加拿大，且微软旗下的必应搜索引擎也可访问 NewsGuard 的实时提要数据。
