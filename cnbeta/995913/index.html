Mophie等配件厂商推出手机专用的UV消毒解决方案

June 26, 2020

为了应对狡猾的新冠病毒，许多人在出门时变得特别小心，口罩和酒精消毒液已经成为了许多人的日常必备。但是对于需要频繁解除的智能手机来说，其实还有一种更加省心的方案，比如 Mophie 和 InvisibleShield 新推出的紫外（UV）消毒套件。

尽管酒精可以有效清理物体表面的细菌和病毒，但想要不破坏智能手机或其它小物件（比如钥匙和信用卡）材料的话，还是 UV 方案更加稳妥一些。

Mophie 和 InvisibleShield 推出的紫外消毒盒的外形大小和原理都差不多，主要借助 USB 线缆供电，然后通过内置的多个 UV-C 灯泡来杀死物体表面的常见微生物。

mophie UV sanitizer with wireless charging（via）

两款 UV 消毒装置最大都支持 6.9 英寸的平板手机，以及钥匙或信用卡等无孔物品。InvisibleShield UV Sanitizer 的售价为 59.99 美元（约 425 RMB）。

Mophie UV Santizer 的售价为 79.95 美元（约 566 RMB），特点是能够在消毒的同时为手机进行无线充电。
