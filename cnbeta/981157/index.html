Windows 10 PowerToys运行启动器迎来0.18版本更新 引入键盘管理器功能

May 20, 2020

微软近日发布了适用于 Windows 10 操作系统、类似于苹果 macOS 上的 Spotlight 的新版 PowerToys 运行启动器。其旨在取代当前通过“WinKey + R”组合键唤出的“运行”功能，并引入更多现代化的特性，比如在系统中快速检索程序和文件、朝招正在运行的进程、以及调用计算器等实用插件。

早期测试的版本将支持基础的搜索任务，此前通常由 Windows“开始”菜单中内置的搜索功能来代劳。

感兴趣的朋友，现在就可以下载并体验新版 PowerToys 运行启动器（GitHub 传送门），且未来还会引入对更多自定义插件的支持。

显然，微软有意将之打造成类似于 macOS 上的 Alfred、且较苹果的 Spotlight 搜索共更强大的综合性工具。

当前的“运行”功能可用于调出 CMD 命令提示符、regedit 注册表编辑器、powershell 实例、甚至某些系统功能（如 control 控制面板）的快捷方式。

通过与开源社区展开深入的合作，新版 PowerToys 运行启动器不仅支持当前 Windows 中执行的所有相同命令，未来还将变得更加强大。

比如软件巨头一直在与 Wox 和 WindowsWalker 的开发者合作，以将这些项目整合到 PowerToys 运行启动器中。尽管早期版本还存在一些 bug，但它已经受到了许多用户的欢迎。

举个例子，通过插件选项，用户可以添加自定义 Web 搜索或摘要等功能。不过微软很可能留有一点私心，比如更加无孔不入地推广自家的必应搜索引擎的内容、以及打开 Edge 浏览器。

此外，微软今日还推出了键盘管理器功能，Windows 10 用户可借此来轻松重映射相关键位。只要软件在后台运行，软映射配置表就会默认生效，甚至允许交换 WinKey 按键。

据悉，键盘管理器是 PowerToys 0.18 最新版的一部分，可通过 GitHub 项目页下载体验。
