Nokia 6.3/7.3/9.3 PureView有望在三季度末四季度初发布

June 11, 2020

91Mobiles 报道称，期待已久的 Nokia 6.3、Nokia 7.3 和 Nokia 9.3 PureView 智能机，有望在今年 3 季度末、或 4 季度初发布。HMD Global 原计划在今年早些时候发布多款新机，可惜受新冠病毒引发的 COVID-19 疫病大流行而一再推迟。不过现在看来，厂家已经在原型测试等方面取得了不错的进展。

NokiaPowerUser 援引知情人士的话称，尽管手机开发已步入正轨，但受 COVID-19 传播趋势的影响，HMD Global 或被迫将发布日期延后至 2020 年 4 季度初。

规格方面，据说 Nokia 9.3 PureView 配备了 6.29 英寸的 Quad HD+ P-OLED 屏，支持屏下指纹识别和 120Hz 高刷新率。

处理器为高通骁龙 865，集成 Adreno 650 GPU，辅以 8GB RAM + 128 / 256GB ROM，且预装 Android 10 移动操作系统。

尽管该机采用了后置五摄的设计，但可能无法与 Light 继续合作。主摄为支持光学防抖的 64MP 传感器，另有 1.08 亿像素的广角、长焦、人像、以及微距镜头。

机身正面升级了 48MP 的前置单摄，有望带来自拍效果的改进。内置 4500 mAh 电池，以及支持 Qi 无线快充。

至于 Nokia 7.3，据说 HMD Global 目前正在测试两款原型，其中一款采用了高通骁龙 765 5G SoC 。

与前代机型相比，其采用了后置四摄，具有改进的图像质量和更好的夜拍效果。主摄可能是 48MP 或 64MP 的规格，前摄则由可能是 24MP 或 32MP 的规格。

最后 Nokia 6.3 有望采用高通骁龙 730 平台，辅以蔡司加持的后置四摄，包括 24MP 主摄 + 12MP 超广角 + 2MP 深度 + 微距传感器的组合。

指纹传感器可能与电源键二合一，3GB RAM + 64GB ROM 版本或售 249 欧（约 283 美元 / 1999 RMB）。
