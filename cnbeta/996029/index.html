谷歌正在为Chrome OS开发剪贴板管理器

June 27, 2020

Windows 10以及Android版Gboard比较有用的功能之一，就是可以临时存储和管理你最近复制到剪贴板的东西。现在，谷歌正在寻求将类似的剪贴板管理器带到Chrome OS中，并支持文本、图像等。

在Windows 10上，你可以使用Windows + V键组合来调出你最近复制东西的列表，这对于你不小心复制到剪贴板上的东西时特别方便。安卓系统和其他许多操作系统一样，只能通过安装应用来提供这一功能，比如Gboard，而Gboard仅在今年才获得了保存最近复制图像的支持。根据新发布的代码修改，chrome://flags中附带了一个标志，谷歌正在为Chrome OS准备一个内置剪贴板管理器。从标志的描述来看，它似乎受到了Windows 10自带剪贴板管理器的启发，使用了类似的按键组合。用户按“搜索+V”会显示一个菜单，允许你粘贴之前复制的东西。

“搜索+V”这个键组合本身并不太令人惊讶，因为Chrome OS在许多其他有用的方式中使用搜索键，例如搜索+左键用于Home键，搜索+A用于谷歌助手。在这种情况下，剪贴板管理器选择“搜索+V”很可能是为了类似于大多数电脑上粘贴的Ctrl + V组合键。在目前的早期开发状态下，Chrome OS的剪贴板管理器将存储你最近复制的五个东西，计划支持文本、格式化文本、图像、书签格式的网络链接，以及一种叫做 “网络智能粘贴 “的东西。Chrome OS的剪贴板管理器被赋予了一个开发代号 “tote”。从这同一段代码中，我们还相信，“tote “除了可以通过搜索+V轻松打开外，还将在Chrome OS的快速设置托盘中拥有一个专用按钮。

总而言之，如果这个新的剪贴板管理器真的推出，它将为帮助Chrome OS与Windows等操作系统并驾齐驱又迈出了一步。
