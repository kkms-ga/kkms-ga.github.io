[视频]Windows 10 Build 20161上手体验

July 04, 2020

本周早些时候，微软向开发者通道（此前被称作 Fast Ring）的 Insider 测试者们推送了 Windows 10 Build 20161 编译版本。从去年 12 月起，微软一直在积极在系统体验上作出一系列较小的改动，比如针对“开始”菜单的 UI 设计、以及通知等体验调整。在今日分享的一段视频中，外媒 Neowin 就着重介绍了 Windows 10 Version 2004（Build 19041）之后的版本中所包含的各项新特性。

需要指出的是，由于微软现正在使用开发者通道（Dev Channel）来测试新功能、且尚未将它们附加到 Windows 10 的特定功能更新中，因此这里无法保证这些改动都将很快向公众推送。

Hands-on with Windows 10 build 20161（via）

在做好相关准备之后，微软会先向 Beta 通道引入 Windows 10 功能更新，接着在完成最终测试后向普通用户推广。
