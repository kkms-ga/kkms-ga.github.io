万元机皇最好戴壳:三星Galaxy Note 20 Ultra首碎

August 10, 2020

8月10日消息，三星Galaxy Note 20系列国行版尚未发布，这款旗舰迎来首碎。据悉，三星Galaxy Note 20 Ultra正反两面采用的是康宁大猩猩Victus玻璃（全球首发），康宁称Victus玻璃能从高达2米的高度跌落至坚硬粗糙的表面保持完好，抗刮擦性能比康宁第六代大猩猩玻璃高两倍，与竞品铝硅玻璃相比更是高出四倍之多。

尽管如此，Galaxy Note 20 Ultra使用过程中仍然要小心，戴上保护壳不失为一种有效的防碎手段。

当前Galaxy Note 20 Ultra已经在三星商城开启预订，目前参与三星先行者计划的订购名额已经满员，8月28日起陆续发货。

价格方面，Galaxy Note 20 Ultra 12GB+256GB尝鲜价位9599元，12GB+512GB尝鲜价为10399元，提供曜岩黑、迷雾金、初露白三种配色（顶配版仅曜岩黑一种配色）。

需要注意的是，尝鲜价并非手机最终零售价格，最终售价将在三星Galaxy Note 20 Ultra国行发布会上正式揭晓，我们拭目以待。
