[图]Edge 84起默认启用“使通知请求静音”功能

July 24, 2020

在最新的 Edge 84 版本更新中，微软推出了一项新功能，能保护用户免受来自网站的恼人通知请求。在启用“使通知请求静音” 功能之后，所有通过 Notifications 或者 Push API 都会在地址栏上出现“通知受阻”的铃铛图标，恼人的通知就不会弹出。

如果用户想接受通知请求，可以点击铃铛图标，接受请求。目前所有 Microsoft Edge 用户都默认启用了这项功能，如果用户依然喜欢传统的通知风格，那么可以在“edge://settings/content/notifications”页面中关闭“使通知请求静音”的选项。
