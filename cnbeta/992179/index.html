苹果新专利将能让自动驾驶系统向iPhone发出汽车下一动作提醒

June 17, 2020

据外媒报道，苹果汽车(Apple Car)自动驾驶系统或能通过在车内iPhone和iPad屏幕发出通知来预先向车内的驾驶者和乘客发出其将要进行的操作的提醒。苹果汽车项目开发的自动驾驶系统可能会让汽车上的每个人包括驾驶者在路上能更容易度过时间。

根据自动驾驶的进展情况，车辆的指定驾驶员可能没有太多的动力去积极观察道路以确保一切顺利进行。关于人们开车时发短信和使用智能手机的报道以及越来越多的批评促使苹果推出了“开车时请勿打扰”的功能–尽管该功能的作用有限。

在自动驾驶汽车中，司机们可能更希望用他们的移动设备来打发时间而不是盯着道路或仪表盘。由于自动驾驶汽车可能需要在任何时候将控制权交给驾驶员，所以这种情况是不安全的。

当地时间周二，苹果在美国专利与商标办公室(USPTO)获批的专利“Device, method, and graphical user interface for presenting vehicular notifications”或将能解决这个问题。通过将警报推送到iPhone或iPad的屏幕上，苹果希望让驾驶员有充分的机会在出现故障时承担起控制责任或至少能知道汽车将要做什么。苹果认为操作手持设备的驱动程序很难将注意力从设备上转移开，因此在设备上显示这些警告信息是不可避免的。

如果汽车不得不停车、遇到交通灯或出于安全或其他原因需要移交控制权，而警报遭到忽略那么可以通过多种方式显示出来。一个通用的状态指示器可以提供基本的交通更新，而在屏幕中间的部分显示或动画中更多的通用通知将更能吸引司机的注意力。

这些指示器还可以提醒驾驶员在最后一次交互之后的预定时间内跟车辆进行交互，同时还有触觉反馈或音频提示通知。在某些情况下，显示器本身可能会变得模糊，视频或音频内容会暂停甚至屏幕的其余部分也会变得不会有响应。

反之，如果系统判定不再需要驾驶员保持注意力，那么它就会撤销通知、使无响应的屏幕区域再次响应并重新播放媒体。

通知的数量跟驾驶者可以用他们的移动设备做什么可能会根据需要的注意力水平而改变，而这反过来又取决于自动驾驶级别。

据悉，这项专利的发明者为Andre Boule、Stephen Chick、Thomas R. Powell和Emily Schubert。

虽然苹果每周都会提交许多专利申请，并且专利申请的存在表明了苹果研发团队感兴趣的领域，但这并不能保证它就一定会出现在未来的产品或服务中。
