Chrome将迎来GPO和Tab Throttling两项性能改进

August 26, 2020

谷歌刚刚通过 Chromium 博客宣布了 Chrome 即将迎来的两项性能改进，得益于配置文件引导的优化（PGO）和标签页节流（Tab Throttling），谷歌浏览器的速度将迎来显著的提升。今日推出的最新版本，已经自带 PGO 功能。至于 Tab Throttling，用户还得再等等。

（来自：Chromium Blog）

据悉，微软早在基于 MS VC++ 的 Windows 平台上（Chrome 53 中）引入了“配置文件引导的优化”（Profile Guided Optimization）。

随着 Chrome 85 的到来，GPO 功能已可在基于 Clang 的 Mac 和 Windows 新平台上使用。遗憾的是，该公司尚未提到 Linux 版本是否有这一特殊加成。

测试表明，PGO 可让页面加载速度提升 10% 。当 CPU 忙于大量标签页的处理时，速度改善会更大明显。除了微软 Windows，苹果 Mac 用户也将获益于此。

至于 Tab Throttling，顾名思义，它能够让长期未访问的后台标签页省下更多资源，减少内存占用、提升电池续航、以及改善浏览器的整体速度体验。

目前尚不清楚谷歌将于何时向所有人推送 Tab Throttling 功能，感兴趣的朋友可下载 Chrome Beta 版本抢先体验。
