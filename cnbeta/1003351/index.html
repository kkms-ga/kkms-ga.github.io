[视频]OpenCV众筹两款4K/60fps空间AI摄像头 售价199/299美元

July 15, 2020

从 CommuteGuardian 分支出来后，OpenCV 人工智能套件（OAK）项目今天推出了两款 4K/60fps 的空间人工智能摄像头，可应用于多个现实场景中。公司将这些产品描述为小型计算机，通过内置的人工智能芯片消除使用云端带来的安全和延迟问题。

OpenCV 表示，OAK-1 和 OAK-D 两款摄像头都能满足“类人感知”的广泛需求，可以实时确定物体在周围空间中的位置，以及它们的轨迹。这些模型内置 Myriad X 芯片，用于运行完全本地系统的人工智能。

使用云服务进行人工智能处理存在多个潜在问题，其中最严重的是不可靠和安全威胁。Myriad X 芯片可提供本地人工智能，包括为检测和分类对象、估计人类姿势和其他类似的事情而定制的深度学习模型。而且该芯片的尺寸非常小巧，接近于树莓派的尺寸。

OAK-1 和 OAK-D 模块都有多个共同点，包括 USB3（兼容USB2）。不过OAK-1 只有一条线可以插入，而 OAK-D 除了数据线外，还包括一条 5v 的电源线，可以连接到墙壁插座中。所有的AI处理都是在OAK硬件上进行的，这意味着它不会影响更广泛的主机系统的性能。

这项技术的潜在用途是什么？OpenCV表示，它们可以用于将AI功能添加到各种系统中，包括用于机器人和无人机，食品加工商确定农产品是否成熟销售，用于监控工作场所是否存在安全违规行为，以及其他任何涉及人工智能的东西。

该硬件包括了一个 1200 万像素的自动对焦摄像头，具有宽阔的水平视场和4056×3040的分辨率。原始视频输出的速度高达 4K/60fps，硬件 4K H.265 视频编码。该产品已经在 Kickstarter 上发起了众筹，OAK-1 和 OAK-D 的零售价预计分别为 199 美元和 299 美元。
