System76发布Pop!_OS 20.04 LTS大版本更新

May 02, 2020

System76 刚刚发布了 Pop!_OS 20.04 LTS（长期支持版本），作为该公司“迄今为止最大的 OS 更新”，其引入了令人印象深刻的功能阵容。比如改进了键盘导航，让用户可以不像以前那样频繁地触碰鼠标。另有一批新增的快捷键，可方便用户切换设置、以及在应用程序执行执行常规的操作。

Pop!_OS 20.04 LTS 下载地址（System76 官网传送门）

开发团队补充道：“通过替换默认的快捷方式，你还可以使用类似 Vim 组合键的方式来浏览桌面，而不必离开首页”。

此外，新版 Pop!_OS 引入了一项被称作“自动平铺”的新功能，可在启动应用程序时帮用户整理窗口。

System76 Pop!_OS 20.04 - Keyboard Shortcuts（via）

在将组织工作交给操作系统来负责后，Pop!_OS 让你有更多时间专注于工作，而不是浪费在窗口的摆弄上。

快捷键让自动平铺变得轻而易举，你可以按自己的喜好来移动、调节和切换窗口，而无需触碰鼠标。如果实在不想使用，亦可在设置中禁用该功能。

Pop!_OS 20.04 LTS 现还支持 Flatpak，在系统菜单中启用混合显卡（支持 Intel / Nvidia / AMD GPU），以及脱机升级等显著的改进（联机后台下载 / 然后在断网情况下安装 OS 更新）。

此外新版固件管理器可执行自动更新，且用户无需购买 System76 计算机，亦可受益于上述新功能。
