[图]售价174.99美元 微软认证合作商竟放出Windows 11预购页面

June 14, 2020

在 Windows as a Service 发展策略下，微软此前曾宣布 Windows 10 将会成为最后一个 Windows 系统。自然此前很多设计师也带来了各种后续版本的概念设计，但当微软认证的合作伙伴在其网站上放出 Windows 11 的预购页面[目前该页面已下架]，零售价格为 174.99 美元，你会如何看？

6·18活动已全面开启 大促活动入口汇总：

阿里云6·18上云年中大促 点击领取最高12000元红包

天翼云“年中上云节”已上线 云产品全场0.6折起 8888元礼包全场可用

2020天猫6·18超级红包在此领取 6月1日追加40亿元消费券

京东6·18十七周年庆大促主会场入口 - 最高可领618元红包

或许你会认为这是一个骗局，毕竟微软官方从未有关于 Windows 11 存在的相关消息。不过在 SoftwareKeep 上却提供了这样一个订购新操作系统的机会。

SoftwareKeep 是（至少在官方宣传中是）“您值得信赖的微软认证合作伙伴”，提供“最全面、最现代化的操作系统”，甚至提供了 Windows 11 的相关描述和产品亮点。

点击进入预购页面（同样是下面的截图），显示的不仅仅是一张包装盒照片，还提供了更多关于操作系统的细节信息。

概览

Windows 11 是传说中的 Windows 的最新一代系统。该更新通过新的设计、新的功能和更好的优化，将您的系统导航提升到一个新的水平。今天，通过向我们购买Windows 11，在您的计算机上获得最易理解、最流畅和最现代化的操作系统。

微软Windows 11家庭版引入了精简的界面、吸引人的外观和一大堆 “通用应用”；在Metro风格的应用上进一步改善。这些应用被设计成可以在多个微软产品系列中运行，具有密切相关的相同代码–包括平板电脑、PC、智能手机、Xbox One、嵌入式系统、Surface Hub 和 Mixed Reality。这只是这款操作系统能够以最快速、无障碍的方式为用户提供服务。

功能介绍

开始菜单：

快速访问常用的程序和文档。该菜单是完全可定制的。你可以调整它的大小，以管理交互式动态磁贴的布局。

Hybrid PC支持：

当设备转移到另一种操作模式时，Windows 11会自动检测，并独立优化显示，以在适当的模式下操作。

全新的图形主题：

旨在让 Windows 更加流畅和动画化的 Fluent Design 之后，微软推出了全新的清晰图形主题，即Windows Light（见下图截图）。

Device Guard:

Device Guard 提供了对新的未知版本的恶意软件和不断增加的复杂威胁的保护。

Cortana 回归

这款语音助手从操作系统的开始菜单和搜索引擎中剥离，现在直接集成到 Windows 任务栏中。“这将使每个体验都能从独立的创新中获益，以更好地服务于受众和他们的用例。“微软认为。

改进的电子邮件客户端

快速连接并与最重要的人和团体共享文件。通过帮助你控制电子邮件和日程的工具，简化你的生活。

Windows 11 安装要求

处理器

最低：1.0GHz 的 32 位处理器

推荐：2.0GHz 的 64 位处理器，至少双核

内存

最低：2GB

推荐：4GB

图形：

最低：兼容 DirectX 9的显卡

推荐：兼容 DirectX 11/12的显卡

硬盘

最低：20GB

推荐：100GB
