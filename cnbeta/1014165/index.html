Android版Gboard输入法迎来智能完成和语音听写等功能更新

August 11, 2020

上月针对 Android 版 Gboard APK 安装包的拆解分析表明，谷歌正努力为消息传递应用引入智能撰写功能。而在近日发布的新版 Gboard 软件中，搜索巨头再次补全了这款移动输入法的智能功能。@MishaalRahman 在推特上指出，新功能中还包括了在其它非消息传递 App 中提供预测性文本建议。

（来自：XDA-Developers）

据悉，该功能由机器学习（ML）提供技术支撑。用户只需在建议内容上向右滑动，即可快速注入文本。

近年来，谷歌一直努力将智能撰写功能推广到其它产品和服务，而 Gboard 就是利用该功能的最新工具。

此外 Gboard 现已引入了一个可快速撤销自动更正（Auto-correction）的选项，以便用户通过退格键来快速删除被自动更正的单词。

鉴于有时 Gboard 会矫枉过正，这项改进还是相当受欢迎的。不过其默认处于禁用的状态，有需要的用户可在设置中找到并启用该选项。

最后是谷歌在 Gboard 中测试的基于 Google Assistant 智能助理的全新语音功能，当按下 Gboard 上的语音输入按钮时，它将显示一个新的语音输入 UI 。

New Google Assistant Keyboard Dictation（via）

用户可在说完消息后直接发送，通过“关闭”指令来切换界面，或借助“删除”指令来清除已转换的文字内容。

需要指出的是，谷歌正在将这些功能作为服务器端更新的一部分而推出，因此并非所有人都可立即使用。
