Android版Project xCloud将于9月15日上线 XGP终极版会员抢先体验

August 04, 2020

微软今日宣布，Xbox Game Pass终极版会员将能通过安卓手机/平板上的云游戏服务——Project xCloud，体验超过100多款游戏，该过程不会额外收取任何费用，时间是从9月15日开始，支持22个市场（国家）。介绍视频：

目前支持的游戏包括《命运2》、《战争机器5》、《我的世界：地下城》、《Tell Me Why》、《如龙：极2》以及更多其他游戏。此外微软所有第一方游戏都会在首发当天通过Project xCloud支持。

你的好友列表、成就、手柄设置和游戏存档也会同步在你的手机上，这意味着你能在主机和手机之间无缝切换，不会失去任何的人性化私人设定。

微软还公布了最新的“为Xbox设计”的手机游戏配件来支持在手机上游玩Project xCLoud。

“为了提升手机游戏体验，我们正与雷蛇、PowerA、8BitDo和Nacon合作，打造专为云游戏设计的全新配件。这些产品包括可适应多种设备的不同类型的手机夹、可装进任何包的旅行手柄，以及可分离并附在手机侧面的专属xbox品牌手柄。当然玩家也可以使用Xbox One蓝牙无线手柄和PS4手柄。”

“你可以在第一个月花1美元加入Xbox Game Pass终极版，之后每个月花14.99美元。除了云游戏，Xbox Game Pass 终极版还提供了100多款高质量的主机和PC游戏，独家会员折扣，交易，Xbox Live金会员。如果你在你的手机或平板电脑上玩游戏，可以从三星Galaxy商店或ONE商店(这两家商店都有完整的、功能齐全的体验)或谷歌play商店下载Xbox Game Pass应用程序。”
