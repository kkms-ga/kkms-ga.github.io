威刚发布P20000QCD与P10000QCD移动电源新品 支持多种快充协议

August 20, 2020

经过长期的发展，电力服务已经变得相当普及。但在大自然面前，极端气候灾害仍是一个难以回避的话题。比如美国东北部近期遭遇的热带风暴塞亚斯（Isaias）就导致当地许多家庭断电数日，使之难以保存冷藏食品、甚至无法为智能手机充电。此时如有一款大容量的快充移动电源，显然就能够满足个人应急通讯的需求。

如果你尚未用拥有一款快充移动电源，那么不妨考虑下威刚（ADATA）近日推出的 P20000QCD 或 P10000QCD 新品。顾名思义，两者分别内置了 20000 / 10000 mAh 的电池。

接口方面，两款快充移动电源都提供了两个 USB Type-A 口，但 P20000QCD 还提供了两个 USB Type-C 口，而 P10000QCD 则只有一个 USB Type-C 口。

威刚表示，新款快充移动电源兼顾了大容量和紧凑外形，支持高通 QC 3.0 和 USB PD 3.0 快充协议。其中 P20000QCD 能够为手机完成大约 6 次充电，支持高达 5V @ 3A 的输出，以及同时为三款设备充电。

容量较小的 P10000QCD 则更具便携和通用性，支持广泛的快充协议（QC 3.0 / USB PD 3.0 / MTK PE 1.1 / 三星 AFC / 华为 SCP / OPPO VOOC），是不同品牌 / 制造商的移动设备的一个理想选择。

值得一提的是，P10000QCD 的 USB-C 端口支持双向充电，搭配另外两个 USB-A 口，能够同时为三款设备充电。

遗憾的是，威刚尚未公布 P20000QCD / P10000QCD 快充 USB 移动电源的确切售价和上市日期。不过等它们上架之后，消费者将能够在黑、白、蓝三色中进行选择。
