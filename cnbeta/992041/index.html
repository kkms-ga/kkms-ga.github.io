微软介绍Edge浏览器的PDF功能改进路线图

June 17, 2020

尽管 Microsoft Edge 浏览器已经提供了 PDF 阅读器的笔记和编辑支持，但这家软件巨头仍希望进一步改善人们的使用体验。比如填写 PDF 表格、通过用户选择的颜色和笔触宽度在 PDF 文档上签注，凸显文档中的某些部分、在黑暗模式下查看，打开受 MIP 信息保护和 IRM 权限管理的文档，以及键盘、屏幕阅读器、朗读（最新的 Canary 开发分支）、高对比度模式等可访问性更新。

此外作为从经典版 Edge 浏览器延续下来的一个项目，新版 Edge 开发团队已针对平滑滚动体验进行了长时间的改进。

团队还致力于提供 PDF 文档的目录功能支持，以及在编辑时提供一个突出显示模式的选项，以便选择特定的文本、并将注释添加到 PDF 文件中，且对数字签名的支持也在不断发展。

安全方面，微软信息防护（Microsoft Information Protection）功能对受保护的 PDF 文档带来了改进，用户可见到敏感度标签、以及查看来自其它组织的受保护文件。

最后微软承诺带来可访问性方面的改进，使用屏幕阅读器来填写表单、以及借助符号插入模式来导航。

当然，上述介绍都属于路线图方面的更新，具体部署还请留意后续正式的产品公告。
