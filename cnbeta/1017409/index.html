[图]微软飞行模拟器正式上架 免费赠送“飞行者俱乐部”制服

August 19, 2020

2019年6月，微软正式宣布热门游戏《飞行模拟器》特许经营权的回归。正如此前承诺的，在经历了一段私有 Alpha 测试阶段之后，这款游戏已经于今天正式发布。为庆祝新版本发布，微软将发布一套免费的“飞行者俱乐部”（Aviators Club） 制服，玩家可通过游戏内市场领取。从明天到9月30日之间，任意新玩家都可以免费领取。

《飞行模拟器》的上次更新是1982年11月，至今将近 38 年时间。虽然这本身就给游戏带来了一段传奇的历史，但这也让它成为微软目前运行时间最长的软件系列。它甚至比1985年首发的 Windows 还要早。

飞行模拟器有标准版、豪华版和高级豪华版三个版本，每种口味的飞机和手工制作的机场数量越来越多。标准版包含在PC的 Xbox Game Pass 中。
