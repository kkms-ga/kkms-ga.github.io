苹果为iOS和Xcode中的新冠病毒密切接触者API做好了准备

April 30, 2020

苹果今日发布了面向开发人员的 iOS 13.5 Beta 3 和 Xcode 11.5 Beta 1，允许其自愿选择是否加入新冠病毒密切接触者 API 。作为与谷歌合作推出的一款公共健康应用，其旨在通过 iPhone 上的蓝牙技术来记录 COVID-19 接触史。从 iOS 13.4.5 升级到 13.5 版本后，系统已囊括了运行所需的代码，开发者亦可通过 Xcode 11.5 的 SDK 来引入通知 API 。

苹果在线商店(中国)

说完苹果这边，与之合作的谷歌公司，亦在 Beta 版 Google Play 服务更新中引入了通知 API 和 SDK，允许 Android Developer Studio 开发人员选择是否自愿加入。

在新冠病毒大流行面前，公共卫生当局希望借助这些工具来追溯 COVID-19 患者的接触史。测试结束后，苹果计划在 5 月中旬正式推出该 API 。

早在 4 月 10 号的时候，苹果和谷歌两家公司便已经宣布了要开发基于蓝牙的 COVID-19 密切接触者追踪应用，并且承诺以隐私为导向。

今日发布的 Beta 版 API，也是为医疗机构提供相关数据的多方努力的一部分。在经过验证的卫生机构的引导下，该 API 允许智能机彼此间交换一个匿名标识符。

若期间有用户后被确诊患有 COVID-19，则会向其他有近距离接触史的用户发出通知，以便采取居家隔离和诊断检测等措施。
