[图]微软创建新GitHub issues repo：在Windows上开发问题都可提交

July 29, 2020

昨晚，微软在 GitHub 上发布了新的 Windows Developer (WinDev) Issues repo（代码仓库）。微软的计划是收集和编目 Windows 系统上存在的开发问题。这个 repo 不仅面向那些为 Windows 平台开发的开发者，而且还面向所有在 Windows 平台上编写代码的开发者。

微软已开始接受面向开发者的性能问题。这种问题包括：

● 开发工具（如编译器，链接器等）在 Windows 上的运行速度比预期的要慢

● Runtime 平台（例如node, .NET, Python）在 Windows 平台上的运行速度要比其他平台慢

● 您的应用程序遇到文件IO/网络/进程创建相关的Perf问题。

Windows 开发者团队高级项目经理 Rich Turner 写道：“我们创建这个 repo 是为了让使用 Windows 的开发者能够直接向还没有自己 repo 的 Windows 工程团队提交和讨论问题，这里你不仅可以提交与使用 Windows 本身有关的问题，而且也可以提交任何与在 Windows 上运行你的代码有关的问题。”
