Inkscape 1.0 发布 - 超强的跨平台矢量图形编辑软件

May 05, 2020

Inkscape 是一个矢量图形编辑软件，与 Illustrator、Freehand、CorelDraw、Xara X 等软件很相似，它使用 W3C 标准的 Scalable Vector Graphics (SVG) 文件格式，支持包括形状、路径、文本、标记、克隆、alpha 混合、变换、渐变、图案、组合等 SVG 特性。

它也支持创作共用的元数据、节点编辑、图层、复杂的路径运算、位图描摹、文本绕路径、流动文本、直接编辑 XML 等。它可以导入 JPEG、PNG、TIFF 等格式，并输出为 PNG 和多种矢量格式。

经过 3 年多的开发，Inkscape 1.0 终于发布了，现在支持 20+ 语言，其中包括中文。此版本主要亮点包括流畅的性能、HiDPI 支持、改进的实时路径效果和原生 macOS 应用。

更流畅的性能、支持 macOS

此最新版本适用于 Linux、Windows 和 macOS，这也是首次支持 macOS，在 macOS 上被标记为“预览”，这意味着将为下一个版本计划进行其它改进。总体而言，1.0 在 Linux 和 Windows 上提供了更平滑、更高的性能体验，在 macOS 上提供了更好的系统集成（没有 XQuartz!）。

工具箱与实时路径效果

重组后的工具箱，其逻辑顺序更加合理。有许多新的和改进的实时路径效果（LPE）功能，新的可搜索 LPE 选择对话框现在具有非常优美的界面、描述，甚至可以标记喜欢的 LPE。当编辑大量节点的对象、使用对象对话框以及进行分组/取消分组时，性能改进最为明显。

画布灵活性以及更多用于自由式绘图的功能

现在可以镜像和旋转画布，并使用 X 射线和拆分视图模式。铅笔工具的新 PowerPencil 模式提供了与压力有关的宽度，最终可以创建闭合路径。Inkscape 允许在新的统一“跟踪位图”对话框中矢量化线条图。新路径效果包括 Offset、PowerClip 和 PowerMask LPE。

技术图纸复制助手、角线和细线

使用技术图纸的用户会喜欢能够创建重复向导、将页面对齐网格、测量工具的路径长度指示器和倒置的 Y 轴，从而使 SVG 代码与 Inkscape 用户界面之间的坐标匹配。那么他们最喜欢的新 LPE 可能是“角”（Fillet/Chamfer），用于均匀倒圆/切割路径拐角；“椭圆”用于构建圆和椭圆；“测量线段”用于建筑平面图和其它实际对象测量。圆形工具的新功能意味着它可以创建闭合圆弧（圆角）。对于 SVG 和 CSS，Inkscape 1.0 可以利用 SVG2 矢量图案填充，并且可以渲染和导出细线。

PDF 导出，文本和文档修复

设计师将能够导出具有可单击链接和元数据的 PDF。他们可以在 Web 浏览器中使用的新调色板和网格渐变，以及对象的便捷画布上对齐。现在可以提供适用于不同屏幕尺寸的新模板、边距指南和彩色棋盘背景。Inkscape 1.0 甚至具有用于创建交互式模型的扩展，可以在 Web 浏览器中模拟用户与应用程序的交互，以向客户端演示和进行可用性测试。

可自定义的主题、图标、字体和 UI

从菜单和工具栏到页面大小和自定义字体目录，都可以进行自定义。可以从已安装的主题中进行选择，为 Inkscape 提供深色或明亮的界面，然后在图标集中选择，其中包括可自定义的单色图标和新设计的多色图标集。用于将当前文件另存为模板的新对话框意味着可以使用所需的模板。
