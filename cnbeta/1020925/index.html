[图]微软：用户安装第三方杀毒软件后将自动关闭Defender

August 27, 2020

根据微软近期更新的支持文档，用户将更难禁用 Windows 10 系统中的 Microsoft Defender，从而为用户提供更妥善的安全保护。默认情况下，它总是在后台运行，以检测、隔离和删除恶意软件感染、勒索软件尝试和其他威胁。

此前微软允许用户通过修改注册表的“DisableAntiSpyware”键值来关闭 Microsoft Defender，但是现在除非用户运行的是 Windows Server 系统，否则单单修改这个注册表项已经无法关闭 Defender。

部分用户认为，当打算安装新的杀毒软件的时候需要禁用 Microsoft Defender。这也是部分科技媒体所倡导的。不过微软表示这种理论是错误的，Defender 不会与另一个杀毒程序发生冲突。

该公司现在已经更新了文档，说明如果用户安装了杀毒软件，Microsoft  Defender 将自动关闭。如果你卸载了杀毒软件解决方案，Microsoft Defender 将再次开启。

微软在文档中指出：“这一变化不会影响第三方杀毒软件与Windows安全应用的连接。这些仍将按预期工作”。
