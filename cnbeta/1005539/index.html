Chrome for Android即将迎来“稍后下载”功能选项

July 20, 2020

Techdows 报道称，谷歌正努力改善 Chrome for Android 的下载体验，比如为用户提供“稍后下载”的选项。在这项功能可用时，用户将无需依赖下载管理器来安排 Chrome 浏览器在特定日期 / 时间进行文件下载，而是在开始下载时弹出一个对话框，可选“现在”、“Wi-Fi 网络环境下”、或“指定一个日期和时间”。

此前访问 Chrome for Android 的菜单时，浏览器只能在下载管理界面显示已完成和正在下载的项目，且未提供按日期、大小和名称进行排序的选项。

好消息是，用不了多久，我们就无需借助第三方文件管理器来轻松实现更多功能。启用标记（flag）之后，Chrome 86 Canary 已允许用户自行安排下载时间。

感兴趣的朋友，可在 Chrome for Android 地址栏中跳转至 chrome://flags，搜索“稍后下载”（download later），在下拉列表中选择“启用”，然后重新启动浏览器。
