[图]用户强烈要求下 Edge地址栏重新显示“http://”和“www.”

July 17, 2020

Edge Canary 频道最新版本中做了一项很小但非常重要的调整：在地址栏上隐藏了“http://”或者“www.”，目的是让 URL 地址更容易阅读和理解。不过微软强调，这并非是最终的决定，不过希望在正式进入稳定版之前希望能收集更多用户的反馈。

目前很多用户都对这项调整持反对意见，因此在最新的 Edge Canary 86.0.579.0 版本中重新显示了这些文字。微软现在引入了 Omnibox UI Hide Steady-State URL Subdomains Beyond Registrable Domain 这个功能 Flag，在默认情况下被禁用。

虽然微软表示通过启用该 Flag，能够在地址栏上隐藏“http://”，不过经过测试发现并非完全生效。例如访问cnBeta、新浪、网易门户网站依然会显示 URL 全地址，但是访问微博就会隐藏掉。
