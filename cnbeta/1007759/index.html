Ubuntu Web：基于Firefox的Web操作系统 旨在替代Chrome OS

July 26, 2020

开发人员 Rudra Saraswat 计划创建一个新的基于 Web 的操作系统。该操作系统名为 Ubuntu Web，旨在构建结合了 Ubuntu Linux 和 Mozilla Firefox 浏览器功能的 Chrome OS 替代操作系统。与 Chrome OS 不同的是，它基于 Firefox，用户可能可以在任何计算机上运行。

Coming soon! Stay tuned!

— Ubuntu Web (@ubunweb) July 22, 2020

Saraswat 在创建基于 Ubuntu 的操作系统方面有一定的经验。他是 Ubuntu Unity Remix 和 Ubuntu Education 的创建者和主要开发者，并参与了针对 Raspberry Pi 计算机优化的 Krob Linux 项目。

不过截至目前，除公告外，仍没有有关 Ubuntu Web 的其他信息被披露，社区仍在讨论默认情况下是否选择轻量级图形界面。项目成员之一的 Allan Carvalho 称，Ubuntu Web 是一个以 Firefox 浏览器为基础的与 Chrome OS 进行竞争的项目。

对此，FOSSBytes 则认为，Ubuntu Web 尚不足以成为基于 Linux 的 Chrome OS 的新竞争对手。因为此前就有出现过几种基于云的操作系统，例如 Apricity OS 或 Cub Linux，他们都旨在成为 Chrome OS 的替代品。

然而结果是，这两款软件都已停产；且在当下的市场上，也还没有 Chrome OS 的直接竞争对手。因此，Ubuntu web 能否真正实现它的目标值得期待。
