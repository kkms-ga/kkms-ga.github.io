微软发布Windows 10 KB4556799五月累积更新

May 13, 2020

作为“星期二补丁”的一部分，微软向 Windows 10 1903 / 1909 用户推送了 KB4556799 累积更新。由于两个系统版本的代码基本相同，因此本次累积更新的变更日志也是一模一样。此外 1909 中的新功能也包含在了 1903 最近的每月质量更新中，但直到用户选择启用，这些隐藏特性才会被激活。

访问:

微软中国官方商城 - Windows

本次累积更新的重点放在系统漏洞的安全修补上，安装 Windows 10 KB4556799 更新后，1903 / 1909 的内部版本号将分别升至 18362.836 / 18363.836 。

包括 Windows Linux 子系统、Microsoft Edge 浏览器、Windows Media、Windows 升级堆栈等在内的一些系统组件也得到了更新。

Office 生产力套件的安全性得到了增强，键鼠 / 手写笔等输入设备的使用体验也得到了改进，此外还有用于存储 / 管理文件和验证用户名 / 密码的更新。

以下是本次累积更新的一些摘要：

● 更新提升使用 IE 和 Edge 时的安全性；

● 更新提升使用键鼠手写笔等输入设备时的安全性；

● 更新用户名和密码的验证；

● 更新提升使用 Microsoft Xbox 时的安全性；

● 更新提升 Windows 执行基本操作时的安全性；

● 更新文件的存储和管理功能；

● 更新提升使用 Office 产品时的安全性。

好消息是，本次更新没有带来一些已知问题。众所周知，此前 Windows 10 用户最怕的就是在安装累积更新时遭遇各种意外，轻则安装失败、重则导致某些功能损坏。

据悉，Windows 10 KB4556799 累积更新可通过标准的分发渠道获得，比如等待 Windows Update 的自动更新推送、

手动安装、或通过企业部署的 WSUS 升级服务来实现。
