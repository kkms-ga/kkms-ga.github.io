泄露表明AMD锐龙C7移动平台或带来令人惊叹的智能机使用体验

June 02, 2020

凭借锐龙（Ryzen）在 PC 处理器市场的优秀表现，AMD 再次受到了游戏玩家和发烧友们的喜爱。不过最新爆料称，该公司很可能乘胜追击，在移动设备市场掀起新的风浪。若真如此，那 AMD 将直面与高通、三星、联发科、华为海思等厂商的竞争。当然，该领域还有类似瑞芯微（Rockchip）等小企业在深耕细分市场。

Twitter 网友 HansDeVriesNL 透露，AMD 可能正在酝酿一款打着“Ryzen C7”头衔的智能机 SoC 。如果爆料靠谱，它很可能像锐龙的台式机和工作站处理器产品一样，为移动市场带来新的震撼。

这枚 SoC 具有两个最新的 3GHz @ ARM Cortex X1 内核（Gaugin Pro）、两个 2.6GHz 的 Cortex A78 核心、以及四个低功耗的 2GHz @ Cortex A55 核心。

更有趣的是，Ryzen C7 SoC 似乎集成了 AMD 与三星在 2019 年合作开发的、基于 RNDA 架构的神秘移动 GPU 解决方案。

事实证明，该 GPU 的速度明显快于三星 Galaxy S20 等旗舰智能机上所使用的 Adreno 650 图形硬件方案。

制程方面，传闻称该 SoC 将采用台积电 5nm 工艺，并集成联发科的 5G UltraSave 双 SIM 卡基带。

辅以 LPDDR5 RAM 和 2K 144Hz 支持，Ryzen C7 有望引领移动计算行业的新发展。
