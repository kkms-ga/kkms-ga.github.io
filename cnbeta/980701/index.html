Adobe预告Premiere Pro 14.2更新将大幅提升视频编码速度

May 19, 2020

Adobe 即将于本周发布 Premiere Pro 的 14.2 更新，主要改进了针对英伟达和 AMD GPU 的硬件加速特性，有望大幅提升视频编码的速度。以英伟达 Quadro 和 GeForce 系列 GPU 的硬件编码器为例，其能够较纯 CPU 方案五倍的速率导出高分辨率视频。此外这项改进也适用于 Adobe 旗下的 Media Encoder、After Effects 和 Audition 等应用程序。

长期以来，Premiere Pro 一直支持利用英伟达 GPU 的 CUDA 核心来加速视效和缩短导出时间。

不过 14.2 版本更新优化了该应用程序，以便在英伟达 GPU 上使用单独的硬件编码器，有助于缩短 H.264 或 H.265 / HEVC 内容的导出时间。

Adobe 高级工程经理 Manish Kulkarni 表示，这些改进是双方持续多年向创作者提供高品质工具的合作成果。

借助 Windows 平台上对英伟达 GPU 新特性的支持，Premiere Pro 可以变得更加强大，并使视频创作者保持生产力和敏捷性。

本次更新还引入了对 After Effects 中 Apple ProRes RAW 格式的支持，意味着创作者可借助英伟达 GPU 的 CUDA 核心，在 Windows 平台上导入 ProRes RAW 文件、而无需对其进行转码。

Adobe 和英伟达都希望通过这项重大的改进，为用户带来全新的体验。以 RTX 2060 显卡为例，英伟达声称新方案可将基础 4K 转码所需的时间降低至 4 分 54 秒。

作为对比，使用英特尔酷睿 i9-9750H CPU 的纯软件转码方案需要耗时 11 分 43 秒。最后，Adobe 还引入了对 AMD GPU 的支持，并于近期开展了相关 Beta 测试。
