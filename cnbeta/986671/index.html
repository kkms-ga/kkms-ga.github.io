蓝牙SIG认证披露Galaxy Note 20阵容 将包括Galaxy Note20 Ultra

June 03, 2020

之前有消息称，三星即将推出的旗舰机Galaxy Note 20系列将包括Galaxy Note 20和Note 20+，而Galaxy S20阵容中还包含了第三款机型–Galaxy S20 Ultra。现在，我们刚刚获悉，三星Galaxy Note20也将包括第三款机型，而且它同样将被命名为Galaxy Note 20 Ultra。

近日，Galaxy Note20 Ultra在蓝牙SIG认证网站上浮出水面，从而证实了Note 20系列将包含第三款机型的事实。我们很少在蓝牙SIG认证网站上看到智能手机的零售名称，所以这次显然是个例外。

除了确认了Galaxy Note20 Ultra的存在之外，上市后的认证也为我们提供了一些关于Note20 Ultra的关键信息。根据认证信息，Note20 Ultra的型号将为SM-N986U，将具备蓝牙5.0连接功能。

虽然我们还不知道Note20 Ultra的关键规格，但可以肯定的说，Galaxy Note20 Ultra将是Note20阵容中最贵的机型。
