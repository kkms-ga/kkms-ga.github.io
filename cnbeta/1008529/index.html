百事为27万名员工装备Microsoft 365与Teams生产力套件

July 28, 2020

微软今日宣布，其已同百事（Pepsi）公司签订了为期五年的合作协议。其中包括让 Microsoft Azure 云服务为该公司提供基础架构支持，在此之上完成 ERP 部署和数据资产的现代化整合。此外该公司还将为其在全球的 27 万名员工装备 Microsoft 365 和 Microsoft Teams 生产力套件。

（来自：Microsoft）

这项伙伴关系有望为百事集团的运营目标和积极创新计划提供助力，同时扩大微软与全球领先的快消品提供商的合作深度。

百事首席信息官 Seth Cohen 表示：作为全球快消品行业的领导者，该公司从未像现在这般需要兑现及时交付的承诺。

通过与微软的合作，百事将致力于提交交付能力，以满足对基础商品不断增长的需求。同时推动新的创新，使员工和运营队伍变得更加强大，以及更适应未来的任务要求。

今年 4 月，微软已经与可口可乐签署了类似的协议。作为合作的一部分，该公司将积极使用 Microsoft Azure、Dynamics 365、Power Platform 和 Microsoft 365（包括 Microsoft Teams）等生产力工具。

基于这些新服务，可口可乐公司将能够全面洞察其业务范围、获得新的见解、以及提供更好的员工和客户体验。
