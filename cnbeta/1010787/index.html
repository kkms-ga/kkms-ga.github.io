Xbox Game Pass更换新LOGO 去除Xbox字样

August 03, 2020

上个月中旬，微软取消了Xbox Live年费续费通道，业界对此举猜测微软准备在下个世代取消联机收费，并将整合Live和XGP。昨天，Xbox Game Pass官方社交媒体发布新消息，更换了全新的LOGO，也许大家的猜测正在逐步变成现实。

新LOGO

昨天，Xbox Game Pass官方推特账户发布消息，更换了账户的头像。本次更新删除了原本LOGO中的Xbox字样，左侧的Xbox图形相比过去的LOGO更加扁平化，这些设计方面的改动显然是为了迎合最新的潮流。不过删除Xbox文字应该是微软希望Game Pass能够给外界“不局限于Xbox”的印象，因为用户已经可以通过非Xbox设备，甚至云游戏Project xCloud等平台利用Xbox Game Pass服务享受相同游戏。

更改前的LOGO

微软将在今年正式发售下一代游戏主机，同时包括Xbox Live和Xbox Game Pass等服务有望得到更新。
