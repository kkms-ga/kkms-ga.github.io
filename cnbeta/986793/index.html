[图]Edge Canary 85新版发布：支持 PWA 应用随 Windows 开机启动

June 04, 2020

在最新发布的 Chrome Canary 通道版本中，允许 PWA 应用程序成为 Windows 系统中的开机启动程序。而同样基于 Chromium 的Edge浏览器，在最新发布的 Canary 通道 85.0.535.0版本更新中同样引入了这项功能。

现在微软也为 Edge Canary用户带来了这项改进，能够让 PWA 应用随 Windows 开机启用。这项功能最初被外媒 TechDows 发现，隐藏在一个 Flag 后面。如果你正在使用Microsoft Edge Canary，那么你可以按照下面的步骤来启用该功能。

1.启动Edge Canary（85.0.535.0或更高版本），在地址栏中键入 “edge://flags”。

2.在搜索栏中键入 “Desktop PWA run on OS login”。

3.单击下拉菜单并选择 “已启用”。

4.重启浏览器

在重启 Edge 浏览器之后，用户能够访问支持 PWA 的网站（例如 twitter.com ）并在地址栏上点击安装。在 “将此站点安装为应用 “对话框中，勾选 “登录电脑时启动应用程序 “选项，然后点击安装。这样该 PWA 应用下次就会随Windows开机而启动。

目前尚不清楚用户如何从开机启动项中删除该应用程序，因为谷歌并没有提供取消勾选"登录电脑时启动应用程序 “选项。不过用户可以打开启动文件夹，手动删除PWA应用程序。要做到这一点，按Windows+R打开运行，输入”%APPDATA%MicrosoftWindowsStart MenuProgramsStartup “打开启动文件夹并删除快捷方式。
