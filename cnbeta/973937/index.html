Windows 10商店惊现Xbox游戏串流应用

May 01, 2020

今年早些时候，微软向所有市场开放了 Xbox 主机串流功能。作为 Project xCloud 的一部分，其允许用户在 Android 或 PC 端体验游戏，或者通过云端进行串流。去年 11 月的时候，软件巨头就已经透露 Project xCloud 将于 2020 年登陆 Windows 10 平台。现在，WalkingCat 又在 Twitter 上分享了他的新发现。

访问:

微软中国官方商城 - Xbox

（Test App 传送门：Microsoft Store）

近日，微软商店里出现了一款名叫“Xbox Game Streaming”的应用程序。由描述可知，它似乎主要面向开发人员。

目前尚不清楚面向普通用户的版本将于何时到来，且 Project xCloud 预览仍仅限于在 Android 和 iOS 设备上使用。

虽然知之甚少，但从 Aggiornamenti Lumia 分享的屏幕截图来看，这款测试应用似乎仅限于那些为主机提供串流功能的游戏开发人员。

对于普通用户来说，目前依然只能通过移动平台来体验 Project xCloud 。不过到项目落成的时候，微软还将同时为基于 ARM 硬件平台的 Windows 10 设备提供支持。
