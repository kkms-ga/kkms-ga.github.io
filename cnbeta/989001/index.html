抛弃曲面屏 三星Galaxy Note 20将回归直屏设计

June 10, 2020

6月9日，据数码大V@i冰宇宙 爆料称，三星Galaxy Note 20将抛弃曲面屏，改为直屏设计。Galaxy Note 5之后的所有Note系列均搭载了曲面屏，S系列在S6之后也是如此，不过近年来三星对于曲面屏的热度直线下降，不仅曲面屏的曲度越发缓和（实际上更类似于2.75D屏幕），甚至在Galaxy S10e和S10 Lite身上取消了曲面屏设计。

至于原因，大概率可能是曲面屏会引起严重的误触问题，这在其他国产曲面屏/瀑布屏旗舰上也屡见不鲜，传统的直屏就没有这一问题。

当然更高端的Galaxy Note20+、Galaxy Note 20 Ultra机型应该依然会搭载曲面屏设计。

预计三星将于今年8月5日正式发布Note 20系列，此外，本次发布会还会推出Galaxy Fold 2，加上Galaxy Note 20、Galaxy Note 20 Ultra，这场发布会会有三款旗舰登场，我们拭目以待。
