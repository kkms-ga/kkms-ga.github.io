人们显然不打算放弃Windows 7 四分之一的PC依然在顽强运行它

August 07, 2020

Windows 7在2020年1月达到了延长支持期限，显然微软希望它的用户数量在过去八个月中急剧下降，但是他们错了。这款古老的操作系统的市场份额几个月来变化细微，它仍占所有运行中 PC设备的近四分之一。

Windows 7于2009年首次发布，最后错误修复和技术支持于2020年1月14日结束，但很少有用户放弃它。根据NetMarketShare的最新数据，Windows 7运行在23.34％的设备上。这比一月份的25.56％市场份额只低了2％。

不过，Windows 7的持久用户群并不等同于Windows 10的坏消息。微软终于在三月份实现了其运行最新操作系统的十亿台设备目标，比最初的预测晚了大约20个月。自发布以来，Windows 10一直在缓慢地增加其市场份额，目前为59.37％。

Windows 8.1的下降趋势仍在继续，但我们仍在2.87％的设备上发现它。 Windows XP拥有0.82％的市场份额，而且由于某种原因，令人难以置信的是，极少数人仍在使用Windows Vista。

总体而言，Windows操作系统现已在所有设备中的运行比例占87.03％。 Mac OS以8.95％位居第二，其次是Linux的3.57％。

为什么有这么多人顽强锁定在Windows 7上？习惯成自然只是其中一个原因，而Windows 10令人讨厌的系统更新后频繁发生故障的问题也很要命，2020年5月的更新毫无意外地又引起了很多问题，其中包括一个非常令人讨厌的Internet连接错误，尽管微软想尽方法试图解决这些问题，但依然没有太多改善。
