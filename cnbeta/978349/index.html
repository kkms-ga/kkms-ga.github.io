亚马逊宣布即日起正式开放Graviton2 AWS实例

May 13, 2020

在去年 12 月发布正式公告，并开展了持续数月的早期预览之后，亚马逊终于在今日宣布了 Graviton2 实例已正式在 AWS 上线。作为基于自家 ARM 新平台的 m6g 实例，外媒 AnandTech 曾在 3 月份的一篇专题文章中作过深入的研究和解析。亚马逊希望通过 Neoverse-N1 内核架构，为客户实例带来单线程和多线程等方面的可竞争性能。

更重要的是，与基于英特尔或 ARM 平台方案的竞争实例相比，m6g 可让 AWS 具有更高的实例运行成本效益。

即日起，亚马逊已在全球主要区域开放全新的 m6g 实例，包括美国东部（弗吉尼亚背部 + 俄亥俄州）、美国西部（俄勒冈州）、欧洲（爱尔兰 + 法兰克福）、以及亚太地区（东京）。
