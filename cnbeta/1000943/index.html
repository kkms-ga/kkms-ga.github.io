OWC开售Mac版ThunderBay Flex 8存储扩展塔

July 09, 2020

知名 Mac 配件制造商 OWC 今日宣布，其专为媒体创作专业人士推出的 ThunderBay Flex 8 塔式存储服务器已经正式上市。其提供了八个驱动器托架，提供了用于连接外围设备的多个端口和一个 PCIe 功能扩展插槽，且用户只需通过一条雷电 3 线缆来轻松连接到 Mac 计算机。

（来自：OWC）

从外形上来看，ThunderBay Flex 8 很像是黑色版本的 Power Mac G5 和初代 Mac Pro 。

其提供了一个 PCIe x16 连接器（x4 通道），可用于附加音视频采集扩展卡、或者额外的 SSD 存储。

八个 3.5 / 2.5 英寸 SATA / SAS 硬盘位可以轻松访问，并且支持 U.2 / M.2 NVMe 驱动器。

在 ThunderBay Flex 8 的正面底部附近，有一串用于连接外设的端口，包括一对 10Gbps 的 USB Type-A、一个 10Gbps 的 USB Type-C、以及 CFexpress 和 SD 4.0 读卡器。

塔式机箱的背部还有一对雷电 3 接口（一个上行 / 一个 15W 下行），以及一个 DisplayPort 1.4 显示输出端口。

OWC ThunderBayBay Flex 8 存储扩展塔现已上市，准系统的售价为 1299 美元（约 9098 RMB）。如出厂选配各种存储组合，还可轻松堆出高达 8000 美元（约 5.6 万 RMB）的配置选项。
