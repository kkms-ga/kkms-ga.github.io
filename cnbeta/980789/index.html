微软展示了Fluent UI设计的最新改进

May 19, 2020

微软正在缓慢地为 Windows 10 系统和应用生态引入 Fluent 设计语言，甚至影响到了 Xbox、IoT 和 Web 端。对于用户来说，最明显的变化就是图标和应用程序的 UI 。显然，微软希望借助半透明和渐变的设计，为用户带来模仿真实世界的视觉效果。

访问:

微软中国官方商城 - Windows

据悉，Fluent Design 基于光源、深度、运动、材质、比例这五种基础元素。

微软计划在 2020 年统一跨平台组件，以便 JavaScript 开发者能够更轻松地交付 Fluent 设计体验。

除了微软自家的 Windows 10 和 Xbox，Fluent 设计元素还将遍布 Web、iOS、macOS 和 Android 平台。

近日，该公司披露了 Fluent UI 的最新改进，比如一个定制的导航栏。设计团队称，该导航栏可以更加贴合 iOS 和 Android 系统。

那些使用了新主题架构的应用程序，亦可访问可扩展的设计系统，以便开发者轻松地刷新所有平台上的视觉设计。

目前微软仍在积极开发 Fluent Design，且计划每六个月实施一次更改。不过普通用户不见得会见到 UI 的大改，因为它更聚焦于开发者层面。

预计本月晚些时候，Windows 10 2020 年 5 月更新包中会带来一批基于 Fluent 设计元素的新图标集。

此外微软计划在 5 月 19 ~ 20 日举办的 Build 2020 开发者大会上讨论有关 Fluent Design、Win32 应用程序、以及 UWP 应用的更多细节。
