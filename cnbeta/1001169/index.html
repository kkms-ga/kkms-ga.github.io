macOS版Fujifilm X Webcam工具上线：让X系列无反相机当网络摄像头

July 09, 2020

富士（Fujifilm）今天发布了适用于 macOS 系统的软件-Fujifilm X Webcam，能够让你使用 X 系列无反相机作为 Mac 的网络摄像头使用。该软件于今年五月率先登陆 Windows 平台，当时富士承诺 macOS 版会在今年七月中旬推出。

Fujifilm X Webcam 适用于 X-T200、X-A7、X-H1、X-Pro2、X-Pro3、X-T2、X-T3 和 X-T4 X 系列 APS-C 无反相机，以及所有三款 GFX 中画幅相机。对于 Mac 用户来说需要注意的是，富士只表示它可以在 Chrome 或新版 Edge 浏览器上使用 Zoom、Teams、Skype 和 Meet，因此可能无法在 Safari 浏览器中使用。

该软件现在可以从富士的网站上免费下载。
