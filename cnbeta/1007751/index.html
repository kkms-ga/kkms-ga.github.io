英特尔10代酷睿KA系列处理器曝光 定价介于K与KF系列之间

July 26, 2020

WCCFTech 报道称，欧洲某些线上零售商店已于近日曝光了“KA”系列的英特尔 10 代酷睿 i9 产品线，可知其定价介于 K 与 KF 系列之间。在此之前，我们只听说过 K 与 KF 后缀的处理器 SKU，而 KA 版本似乎为全新引入的版本。根据论坛上某些网友的讨论，KA 系列处理器的睿频参数可能略低一些，但 L3 缓存保持不变。

资料图（来自：Intel 官网）

从欧洲零售商线上挂出的价格来看，KA 系列 10 代酷睿 i9 处理器的价格比 KF 高、但低于 K 系列。

首先是 Core i9-10900KA，其部件号为 BX8070110900KA，基础频率 3.7GHz，售价 525 欧元（约 4291 RMB）。

其次是 Core i9-10850KA，其部件号为 BX8070110850KA，基础频率 3.6GHz，售价 485 欧元（约 3964 RMB）。

接着是 Core i9-10700KA，其部件号为 BX8070110700KA，基础频率 3.8GHz，售价 408 欧元（约 3335 RMB）。

最后是 Core i9-10600KA，其部件号为 X8070110600KA，基础频率 4.1GHz，售价为 278 欧元（约 2272 RMB）。

Core i9-10900KA 的售价较 Core i9-10900K 低 2.1％，而 Core i9-10850KA 的售价与 Core i9-10850K 相同。

此外 Core i9-10700KA 的售价较 Core i7-10700K 低 1.2％，且 Core i9-10600KA 的价格比 Core i5-10600K 低 2.8％ 。

总体而言，KA 系列 10 代酷睿 i9 处理器的售价较 F 系列低 2% 左右。传闻称这些新型号的内部样品或于 8 月 9 日到来，至于消费者是否买账，仍有待时间去检验。
