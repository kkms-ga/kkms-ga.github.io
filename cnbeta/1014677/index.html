密码管理器应用1Password终于登陆各大Linux发行版平台

August 12, 2020

近年来的网络账户安全，正面临着密码易被泄露和“一锅端”的严峻考验。为了提升安全性，双因素身份验证（2FA）已经变得相当普及。此外专家建议大家尽量为不同的服务设置不同的复杂密码、并做到定期更换，然而这也为记性不好的用户带来了较大的麻烦。为妥善管理账号密码，Windows 和 Mac 平台上已有不少知名的解决方案，但 Linux 平台却有些尴尬。

好消息是，作为目前市面上最好用的密码管理器之一，1Password 刚刚推出了面向 Linux 平台的开发预览，并且支持 Ubuntu、Debian、Fedora 等知名发行版，感兴趣的朋友可以下载体验一番。

下载地址：https://snapcraft.io/1password

不过 Agilbits 的 Dave Teare 指出：目前 Linux 上的 1Password 仍处于早期开发阶段（而不是长期支持版本），许多功能尚未最终完成，预计未来数月内会迎来较多的更新和改动。

比如当前该应用程序只提供只读状态，尚未开放项目的编辑和创建。初始发行版本仅用于测试和验证的目的，暂不适合在关键业务环境中运行。

认识 1Password for Linux：https://support.1password.com/cs/getting-started-linux/

Teare 补充道，新应用程序旨在满足 Linux 用户对安全和性能的需求，因此他们在后端完全用 Rust 来编写。

作为一种以安全为主打的系统编程语言，其已在 Linux 社区内引发广泛的关注。尤其是运用了令人难以置信的环形加密库，以增强端到端加密的数据安全。
