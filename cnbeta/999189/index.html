[图]Windows Update故障导致部分设备循环安装旧版驱动程序

July 05, 2020

除了功能更新和累积更新之外，Windows Update 还负责安装显示器、蓝牙、音频以及其他外设的驱动程序。尽管大部分用户都能获得流畅的 Windows 10 升级体验，但仍有少部分用户由于各种兼容性问题导致无法正常使用 Windows Update。

Image Courtesy: PCWorld / IDG

在 Windows 10 May 2020（20H1/Version 2004）功能更新发布之后，华硕等部分 OEM 厂商开始推送新的图形驱动程序，以便于更好的支持该功能更新。不过遗憾的是，部分用户显示图形驱动程序升级过程中出现卡死停止情况，并提示模糊的错误信息。

根据微软 Answer 社区的几个帖子内容，造成上述问题的原因是 Windows Update 一直在推送旧版的驱动程序，即使驱动程序已经成功安装，安装进度也会卡在安装循环中。

此外还提供了一个比较模糊的提示信息：“当前 PC 上安装的驱动程序可能比我们尝试安装的驱动程序更好。我们将会继续尝试安装。”

那么具体是什么原因导致了 Windows Update 不断推送旧版驱动程序呢？通常情况下有两个因素

● Windows Update 缓存

● 阻止 Windows Update 检查当前已安装驱动版本的BUG。
