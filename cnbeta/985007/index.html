蓝屏乱入：玩家们可在《动物森友会》中体验各种微软元素

May 30, 2020

拥有任天堂 Switch 掌机和《动物森友会》游戏的微软粉丝们，现在可以在网络上获取各种微软风格的“装饰周边”。如果你想纪念一下上周刚结束的 Build 2020 开发者大会，不妨查看 Jen Gentleman 和 Reddit 社区成员参与设计和整理的微软主题衬衫系列，涵盖 Build 2020、Microsoft Edge 浏览器、XP 桌面壁纸、甚至 Windows 10 蓝屏死机等各种元素。

如上图所示，Build 2020 系列服装周边包括了 VS 徽标、Linux 文化衫、Channel 机械手等。感兴趣的朋友可以输入创建者代码 MA-2729-6723-4412，在 Able Sister 商店中获取。

看起来丑得有些辣眼睛的 Windows 95 毛衣，让我们回想起了微软在 2018 年 12 月向记者朋友们赠送的哪一款，Jen 的第二创作也是这个款式，在白色背景上放置了 Win95 的老图标

在 Able Sister 商店中输入设计 ID 时，可使用 MO-W5W3-L4SP-CCH6 代码进行获取。

Jen Gentleman 为 Microsoft Edge 衬衫选用了微软在 2019 年 11 月公布的那版徽标，可使用设计 ID MO-06MC-4SGJ-VHBK 获取。

霸王龙（T-Rex）文化衫与“忍者猫”一样，都是深受 Insider 测试者们喜爱的非正式吉祥物，玩家可使用 MO-GT2B-GB6X-1WHF 代码获取。

Office 大眼夹可能会暴露你的年龄，请悄悄地使用 MO-2HLW-0SNK-5R6B 代码获取。

Windows XP 壁纸更不用多说了，记得给你的电脑桌面设置一下（通过 MO-WM9X-BJFQ-5W5R 代码获取）。

最后是喜闻乐见的“传统艺能”时间 —— 由 Reddit 网友（u / Minifig81）创作的 Windows 10 蓝屏死机界面（可使用 MO-F0RT-YHDV-M5RD 代码获取）。
