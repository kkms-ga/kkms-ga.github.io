[图]PowerToys 0.19发布：修复超100个bug 改善搜索和FancyZones体验

July 01, 2020

微软刚刚发布了适用于 Windows 10 的 Windows PowerToys 0.19 版本更新，重点改善了该工具的质量和稳定性。新版本中修复了超过 100 个 BUG，改善了搜索体验和 FancyZones 的线程模型。

访问：https://github.com/microsoft/PowerToys/releases/

以下是PowerToys 0.19版本的改进、修复和已知问题的完整列表。

PT Run 的已知问题依然存在

● 新安装的应用程序无法找到 #3553

微软已经找到了一个修复方案，不过没有在 0.19 版本中上线

● CPU/内存问题，目前仍在调查 #3208

微软已经修复了一个问题，目前还在进行其他方面的改进。

● PT Run 的 WinKey 重映射可能会混乱 #4578

PowerToys Run 改进

● 现在使用低级别 Hooks 从而能够重映射 Alt+Space 为基于 WinKey 的快捷键 #2325

● PWAs 正确地重新识别 #3199

● 开发模式的应用找不到#3330

● 由于 romaing 个人资料数据可能出现重复结果 #3180

● Steam游戏找不到 #3425

● 隐藏文件不在结果中 #3449

● 主题改进 #3485

● 缺少图标 #3541

● Supress Alt-F4 #3796

● 网络路径不工作 #4096

● 默认搜索环境路径 #4200

● 默认搜索桌面 #4056

● 无法找到 Terminal #3974

● 找不到ClickOnce应用 #4025。

● 开始输入文字可见，然后冷启动时消失 #2974。

● 右键菜单高亮显示问题  #2559, #3483

● 鼠标悬停时显示上下文菜单按钮 #2420

● 放置带监视器的shutters #3535, #3018

● 空白的结果框，当没有任何搜索查询 #3186

● 结果框尺寸问题 #2456

PowerRename

● 检测何时使用模式匹配#1002

FancyZones

● Outlook 新邮件窗口在最近使用的区域不显示 #1192。

● Teams 通知并不弹出 #1931

● 改进多线程设计。#961

● 多个虚拟桌面修复。#2725, #2684, [#2678]#2678), #2633

键盘管理器

● 翻转到日文版IME #3397时，键的重新映射保持有效。

● Ctrl/Alt重映射时Tab修正 #3294

● 修正重映射到alt-tab/win+space风格的快捷键#3331。

● 修正了对扩展键（Home, End, Ins, Del, Pg, Arrow键）的重新映射#3478。

● 游戏有错误的键位映射#3647

● 修正numlock重映射#3277

● 稳定性修复

设置

● 设置v2为空白 #3384

● “检查更新 “实际上是检查更新，而不是进入GH页面 #2535。

● 文档指向重定向URL了，所以我们可以在不更新应用的情况下更新#3088。

● 多项用户体验改进

● 从登录不被尊重开始 #3875

● 现在使用低级钩子，因此可以将Alt+Space重映射为基于WinKey的快捷键#2325。

● 在#3992的热键中添加无效输入的警告。

安装器

● 可以通过控制面板卸载#2012

● 0.19.0之后，资源管理器无法重启 #2730

● 多项构建系统改进以减少冲突 #3809, #3812, #3867, #3948

根据微软的说法，在 0.19 和 0.20 版本中，安装程序体验将会得到进一步改善。此外，这将是你最后一次在升级过程中看到 Windows 资源管理器闪动，在 PowerToys 0.20 版本开始，.NET Core的安装体验应该会更加流畅。
