[图]Android端Opera 59发布：新增支付请求API

July 01, 2020

适用于 Android 平台的 Opera 59 今天正式发布。新版本带来了包括支付请求API，站点设置改进在内的诸多细节改进，为用户提供了更精致的浏览体验。此次升级中最显著的新增功能是支付请求API。该公司表示，随着越来越多的购物移动到网上，对 W3C 认可的支付请求API等标准的需求将会增加。

这个特殊的 API 将允许网站访问客户数据，如地址、联系信息和支付细节，这将为客户节省时间和精力。Opera 承认，这个 API 还没有被广泛使用，但为被其他浏览器接纳奠定了基础。

新版本中，Opera 还调整专用应用程序的处理方式，使用户更容易决定是在浏览器中保持网站打开，还是切换到外部应用；这一功能将对YouTube和Reddit等网站有所帮助。为了访问这一设置，只需按下全能栏中的绿色挂锁，在那里你还可以找到控制权限等设置。
