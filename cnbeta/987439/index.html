亚马逊AWS已普遍提供基于AMD二代霄龙平台的EC2 C5a实例

June 05, 2020

经过多年的努力，AMD 终于凭借锐龙（Ryzen）和霄龙（Epyc）处理器，在消费级与服务器市场受到了更多客户的青睐。最新消息是，从今天开始，亚马逊 AWS 将普遍提供基于 AMD 二代霄龙平台的 EC2 C5a 实例。作为 EC2 C5 系列的优化衍生版本，此类实例主要面向批处理、分布式分析、以及数据转换等工作负载。在降低 10% 成本的同时，还可带来较以往同类实例更高的性能。

6·18活动已全面开启 大促活动入口汇总：

阿里云6·18上云年中大促 点击领取最高12000元红包

2020天猫6·18超级红包在此领取 6月1日追加40亿元消费券

京东6·18十七周年庆大促主会场入口 - 最高可领618元红包

据悉，C5a 实例细分为八档规格，起步为拥有 2 个虚拟处理器核心（vCPU）和 4GB RAM 的 c5a.large，最高可选拥有 96 个 vCPU 和 192GB RAM 的 c5a.24xlarge 。

目前 C5a 实例已在美国东部的北弗吉尼亚 / 俄亥俄州、美国西部的俄勒冈、欧洲的爱尔兰 / 法兰克福、以及亚太的悉尼 / 新加坡等区域节点上线。

C5a 实例支持许多 AWS 资源，包括 AWS Batch、Amazon EMR、Elastic / Elastic 容器服务，且支持基于 Nitro 的跨平台管理。

最后，亚马逊补充道，C5a 随附的本地 NVMe 实例存储和 C5an.metal / C5adn.metal 变种也将很快与大家见面。
