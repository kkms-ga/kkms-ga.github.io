Windows 10系统优化工具Ninjutsu OS因遭微软DMCA打击而下架

June 12, 2020

尽管微软在 Windows 10 发布时承诺尊重用户的隐私、并给予更大的隐私设置控制权，但事实是许多用户希望更进一步。好消息是，目前市面上有许多免费的第三方工具可以对系统软件进行各种更改，比如移除恼人的捆绑软件，以提升用户的隐私安全感。然而 TorrentFreak 近日报道称，Windows 优化工具 Ninjutsu OS 已经遭到了微软的 DMCA 投诉打击，并在 GitHub 平台下架。

软件巨头在投诉中称：Ninjutsu OS 工具的以下功能存在侵权行为，因绕过相关技术限制而违反了微软的软件许可条款。

● 通过强大的调整功能和优化来定制 Windows 10；

● 通过调整和定制 Windows 10 以保护隐私；

● 禁用 Windows 内置的许多恼人功能；

● 删除不需要的 Windows 组件；

● 删除 / 禁用大量 Windows 程序和服务。

Ninjutsu OS 宣称可添加数百种安全工具并移除“不必要”的 Windows 功能，对系统的定制和修改能力甚至超越了许多同类工具软件的常见范畴，这或许也是微软“枪打出头鸟”的主要原因。

商业软件联盟（BSA）代表微软发起了本次 DMCA 投诉，并要求 GitHub 平台下架存在侵权行为的 Ninjutsu OS 工具软件的托管项目。

然而开发者 Hasan 拒绝了相关指控，称其工作并不涉及任何侵犯微软版权的行为。
