机械革命高配轻薄本S2 Air与程序员本Code 01开启预售

May 28, 2020

机械革命日前推出两款高性价比AMD锐龙4000标压本，分别是主打轻薄的S2 Air和面向程序员的Code 01，定于5月31日和29日开启预售。S2 Air采用铝镁合金金身，仅重1.13kg。基本配置方面，14英寸72% NTSC屏幕，AMD锐龙4600H或4800H标压满血处理器（54瓦TDP释放）。16GB DDR4-3200双通道内存，全系标配PCIe x4 SSD，双风扇双热管，支持USB-C双向充电，配90瓦电源适配器。

访问:

机械革命自营旗舰店

价格方面，R5 4600H/16G/512G定价3999元，R7 4800H/16G/512GB定价4699元。

Code 01升级为15.6英寸100% sRGB屏幕，同样铝镁合金机身，1.47Kg重，标配AX200 Wi-Fi6网卡，91瓦时电池，R5-4600H/8G/512GB定价4299元，R7-4800H/166G/512GB定价4999元。
