Windows 10可能改为一年一个功能更新 以更快推进Windows 10X

July 27, 2020

援引内部知情人士透露，为了更快推进 Windows 10X 系统的开发速度，微软正在考虑将 Windows 10 系列的更新频率从现有的“一年两个功能更新”缩减为“一年一个功能更新”。在经历了十月更新（Version 1809）的巨大尴尬之后，微软已经放慢了更新节奏，通过机器学习等方式进一步提高系统的稳定性和可靠性。

在做出调整之后，每年上半年的功能更新更注重新功能的引入，而下半年的功能更新更注重 BUG 的修复，后台的优化。这种“主-次”的节奏对于用户来说，事实上就是一年一次的功能更新，下半年的功能更新可以看作是超大号的累积更新。

据悉，微软计划将 Windows 10 的频率调整为每年一次功能更新，这将使工程师们腾出时间来研发Windows 10X，这是一款面向低成本硬件、网络优先的轻量级操作系统。

作为新时间表的一部分，微软将在2021年春季推出Windows 10X，2021年秋季更新成为Windows 10的主要功能更新。放弃主要-次要升级节奏，一年只发布一个版本，这将是一个很大的变化，而且也可以让Windows 10的bug减少。
