[图]System76升级Oryx Pro：十代酷睿+RTX 20系列 起售价1623美元

June 26, 2020

今天，知名 Linux 厂商 System76 对旗下的 Oryx Pro 笔记本电脑进行了升级，在预装系统中用户可以选择 Ubuntu 20.04 或者 Pop!_OS 20.04。这款性能强悍的笔记本拥有前沿的英特尔第十代酷睿 i7-10875H 处理器，提供 8 个核心和 16 个线程。

此外在 GPU 方面，用户最高可以选择 NVIDIA RTX 20 系列的显卡。得益于 Pop!_OS 20.04 系统中内置的智能图形切换功能，它可以与英特尔显卡协同工作。

除了令人印象深刻的 CPU 和 GPU 之外，你还可以获得大量的端口：包括 1 个 Mini DisplayPort，1 个 HDMI 和 1 个 Thunderbolt 3 端口。此外，它还有 3 个 USB-A 端口，专用耳机和麦克风端口，以及 1 个千兆以太网接口。它甚至可以容纳 2 块 M.2 PCIe SSD 和 2 根内存–所有这些都是用户可以更换的，方便升级。此外键盘部分还有数字键。

最重要的是，焕然一新的 Oryx Pro 笔记本搭载了 “System76 开放固件”（System76 Open Firmware）。公司对其的描述如下

System76 Open Firmware 是由 coreboot 固件、EDK bootloader 结合 System76 Firmware Apps 构建的开源固件。它的设计是为了轻量级的代码，以获得更好的速度和安全性。这意味着用户可以获得闪电般的启动时间，增强的安全性，以及通过操作系统访问的固件更新。

此外，开源固件提供了一个内部代码的外观，所以用户可以跟踪他们的数据发生了什么。我们还添加了 System76 嵌入式控制器固件，这是 GPLv3 授权的代码，允许您访问和控制重要功能，如键盘、风扇和电池。

Oryx Pro 笔记本提供 15.6 英寸和 17.3 英寸两个版本，起售价格为 1623 美元。
