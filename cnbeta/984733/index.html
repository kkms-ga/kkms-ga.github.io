本田最强B级车讴歌TLX发布：海外21万元起售

May 29, 2020

讴歌TLX是本田旗下高端品牌的一款中型轿车，由于性能更强的TLX Type S的存在，该车也被称为本田最强的B级车。日前，全新换代讴歌TLX也已发布，该车是首款按照品牌Precision Crafted Performance精神推出的新车，起售价30000美元（约合人民币21万元），将于今秋在美上市销售。

随着TXL的到来，断代近十年的TLX Type S也重回了人们视线。消息称，Type S车型将于明年一季度开售。

据官方介绍，全新换代讴歌TLX将会是该品牌，35年来最快、操控最敏捷、功能最齐全的轿车。

具体来看，新车采用讴歌家族化设计元素，前脸搭载钻石型进气格栅，扁平化点阵式LED大灯，造型更加犀利。前唇配备熏黑运动化包围套件，整车的运动风格更加鲜明。

同时，讴歌TLX Type S车型还拥有专属标识、20英寸轮毂、Brebmo高性能刹车和双边四出排气。

内饰部分，TLX沿用讴歌的传统设计风格，配备本田标志性的按键式换挡机构。中控部分采用分级式造型，呈现出丰富的空间层次感。仪表盘由传统指针和液晶显示屏组合而成，整体风格较为时尚。

车身尺寸上，新车长宽高分别为4942/1910/1432mm，轴距为2870mm，相比于目前在售的标准轴距车型，车长增加73mm，轴距增加93mm。

动力方面，讴歌TLX搭载2.0T发动机，最大马力272Ps，峰值扭矩380N·m。TLX Type S为3.0T V6发动机，最大马力预计可超过400Ps，两车均配备10AT自动变速器。普通版TLX可选配讴歌SH-AWD四驱系统，而Type S车型则标配四驱系统。
