[图]微软：Project Cortex项目将于2020年夏季正式上线

May 20, 2020

去年11月，微软启动了Project Cortex项目，借助AI构建知识网络，对组织内的数据进行梳理，并自动按照项目或者客户等相关性转换为成共享话题。在今天的Build 2020开发者大会上，微软表示该项目将于2020年夏季正式上线。

访问:

微软中国官方商城 - 首页

在Build 2020大会上，微软表示已经将该项目介绍给超过75位私人预览客户，包括Arla Foods、Mott MacDonald和Siemens Healthineers。

此外微软还发布了新的Microsoft Graph API和Graph扩展，以及Project Cortex和Microsoft Graph之间的交互连接器。此外，开发者还将能够利用Azure语言理解和AI Builder等外部资源来创建更丰富的集成。

Project Cortex 用 AI 生成的知识网络能够将有关某一主题或内容（例如演示、视频和对话）的概念解释、关联信息、相关专家等等都集成到一张类似维基百科的知识话题卡片中，分享给组织内的成员，这将大大增加整个公司或组织在相关员工中共享知识和经验的效率。

Project Cortex主要功能有：

● Project Cortex将使用AI来理解组织的数据，并且自动组织到诸如项目和客户等共享主题中。

● Project Cortex使用主题卡和主题页面为组织中的人员提供相关知识。

● Project Cortex还将通过智能内容摄取方面的创新将您的内容转变为交互式知识库，以分析文档并创建复杂的内容模型。机器教学，使主题专家可以教系统如何理解半结构化内容；知识检索，使人们可以轻松访问经常被锁定在文档，对话，会议和视频中的宝贵知识。
