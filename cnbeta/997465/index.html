Mozilla Firefox 79.0 Beta 1 发布

June 30, 2020

Mozilla Firefox 79.0 Beta 1 今日发布，新版加入了一个便利性方面的功能，可以将浏览器中保存的网站账户和密码输出到一个CSV文件保存。macOS 10.9、10.10 和 10.11 版本不再受 Firefox 支持。建议这些macOS版本的用户切换到Firefox ESR 78.x，以便继续接收安全更新。

更新列表：

New

Saved logins and passwords can now be exported to a CSV file.

macOS versions 10.9, 10.10 and 10.11 are no longer supported by Firefox. Users of these macOS versions are recommended to switch to Firefox ESR 78.x in order to continue receiving security updates.

下载地址:

http://ftp.mozilla.org/pub/firefox/releases/79.0b1/
