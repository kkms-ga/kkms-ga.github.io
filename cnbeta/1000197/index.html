[技巧]Windows 10如何卸载基于Chromium的新版Edge浏览器

July 07, 2020

拥抱 Chromium 开源平台，让 Edge 浏览器获得了更多用户的青睐。虽然新 Edge 浏览器的性能和可靠性广受赞誉，但通过 Windows Update 安装新版 Edge 就会自动移除经典版 Edge，更遗憾的是你无法直接从 Windows 10 中卸载新版 Edge。

如果你想要继续使用经典版 Edge，那么这里有个小技巧能够让你卸载基于 Chromium 的新版 Edge 浏览器，然后你再安装经典版 Edge 就可以了。

卸载步骤

1.输入左下角的 Windows 图标

2.输入“Command Prompt”，看到该应用之后右键点击“以管理员身份运行”

3.在命令提示符窗口逐一复制以下命令并执行

C:/Program Files (x86)/Microsoft/Edge/Application/83.0.478.58/Installer

setup.exe –uninstall –system-level –verbose-logging –force-uninstall
