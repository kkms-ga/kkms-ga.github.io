《使命召唤：现代战争》扩大不雅ID封锁 排除种族歧视

June 05, 2020

开发商 Infinity Ward 近期宣布《使命召唤：现代战争》、《战区》以及《使命召唤 Mobile》的新赛季无限期延期，藉此声援美国正在进行的抗议运动。除此之外，InfinityWard 祭出更进一步的策略，决定力除《使命召唤》游戏中种族歧视的问题。

6·18活动已全面开启 大促活动入口汇总：

阿里云6·18上云年中大促 点击领取最高12000元红包

2020天猫6·18超级红包在此领取 6月1日追加40亿元消费券

京东6·18十七周年庆大促主会场入口 - 最高可领618元红包

《使命召唤：现代战争》于去年发行后，游戏中不乏出现使用不雅 ID 的玩家，例如 N 开头的禁忌词汇。

Infinity Ward 表示：「我们的游戏中绝不容许种族歧视的问题存在，自从游戏发售后便致力于修正问题，但我们必须做得更好。我们每天都会封锁数千个违反规则的 ID，但我们知道可以做得更多。」

官方表示他们将扩大对不雅 ID 的封锁规模，透过增加资源、改善系统来增加不雅 ID 的侦测与封锁，同时在更名上新增更严苛的限制。举报系统上也会进行改善，让玩家更简单地举报玩家。最后，重复违规的容忍度降低，提高永久封锁的执行量。

Infinity Ward 似乎也了解这项改变不应该是因为目前美国的抗议运动而发起，因此他们在文末向玩家社群道歉，表示他们能够做得更好。
