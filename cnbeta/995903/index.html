Canonical因在Ubuntu MOTD中植入广告而遭到抨击

June 26, 2020

想要从 Windows 平台逃离的用户，有很高的概率被推荐使用 Ubuntu 。作为当前最受欢迎的 Linux 发行版之一，Canonical 为其提供了开箱即用的直接体验。然而近日，该公司却因为在 Ubuntu MOTD 中植入广告而遭到了社区的猛烈批评，一些人甚至考虑投奔其它 Linux 发行版。

在一则讨伐 Canonical 的 Reddit 帖子中，已经收到了上千网友的评论。屏幕截图显示，Ubuntu 中混入了一条附带链接的文字广告：

如果你一直在等待面向 macOS 的完美 Kubernetes 开发解决方案，现在是时候体验 Microk8s 了！

尽管许多人会无视在 MOTD 中出现的文本广告，但对于 Ubuntu 这样的热门 Linux 发行版来说，Canonical 公司显然背负着更大的期待和压力。

实际上，Ubuntu 中的 MOTD 广告已经持续有段时间了。从社区 Bug 提交和追踪平台上的一篇文章来看，最早可追溯到 2017 年 6 月的 Ubuntu 17.04 身上。

不过正如 Reedit 网友 mralanorth 在帖子中解释的那样，用户可以通过手动禁用 motd-news.service 和motd-news.timer 这两个系统组件，然后执行以下命令（各版本不尽相同）来实现“眼不见心不烦”的效果。

/etc/update-motd.d/99-esm # Ubuntu 14.04

/etc/update-motd.d/10-help-text # Ubuntu 14.04+

/etc/update-motd.d/50-motd-news # Ubuntu 16.04+

/etc/update-motd.d/80-esm # Ubuntu 16.04+

/etc/update-motd.d/80-livepatch # Ubuntu 18.04+

截至发稿时，Canonical 仍未对 MOTD 中包含的广告链接发表评论。至于后续是否会在新版中默认禁用，仍有待时间去检验。
