微软连发三条视频 介绍Surface Duo的特色功能

August 15, 2020

微软为 Surface Duo 双屏折叠智能机配备了两块超薄的高分辨率触摸屏，能够帮助用户更轻松直观地完成复杂的任务。今天，软件巨头还在 YouTube 频道上连发了三条新视频，以深入介绍 Surface Duo 的重点功能。目前 Microsoft Store 官网商店已经开放了 Surface Duo 的预订，128 / 256GB 版本的售价为 1399 / 1499 美元（约 9723 / 10418 RMB）。

Surface Duo 开启后的大小为 145.2×186.9×4.8 mm，闭合时为 145.2×93.3×9.9 mm，重 250 g 。

两块 Dual PixelSense Fusion AMOLED 屏支持 100% sRGB 和 100% DCI-P3 色域，辅以康宁大猩猩玻璃防护。

开启后的大小为 8.1 英寸（2700×1800 / 3:3 / 401 PPI）单块屏幕为 5.6 英寸（1800×1350 / 4:3 / 401 PPI）。

1 How To Do One Better - Two sites, side by side（via）

处理器为高通骁龙 855 SoC，可选 6GB RAM + 128 / 256GB UFS 3.0 存储组合，两块电池容量 3577 mAh（典型值），自带 18W 快充电源适配器。

拍照方面，Surface 提供了 11MP @ f/2.0 镜头（对角线 FoV 为 84.0°），支持相位检测自动对焦和 4K @ 60fps 视频录制（电子防抖），并且优化了前后 AI 拍摄体验。

2 How To Do One Better - View across both screens（via）

连接方面，该机支持 802.11ac 双频 Wi-Fi、蓝牙 5.0、4x4 MIMO LTE（下行 1.2Gbps / 上行 150Mbps）、以及 GPS / Galileo / GNLONASS / 北斗 /  QZSS 定位。

安全方面，Surface Duo 配备了指纹生物识别传感器，并且预装了 Microsoft Authenticator 安全验证器。手写笔方面，该机支持所有在售的 Surface Slim Pen / Surface Pen / Surface Hub 2 Pen 。

3 How To Do One Better - Compose an email（via）

软件方面，Surface Duo 预装了基于 Android 10 定制的 UI 。设备启动的时候，还可分别在两块屏幕上显示“田牌”和“机器人”的 Logo 。
