[图]又是累积更新惹的祸？导致Outlook无法正常运行

June 23, 2020

近日有不少微软 Outlook 用户反馈称，在运行 version 2005 build 12827.20268及更高版本之后出现：“你的某个数据文件出现错误，Outlook 需要关闭。Outlook 可能会修复你的文件，点击 OK 运行收件箱维修工具”。

如果如果你点击确定启动收件箱修复工具，你最终会得到同样的错误。所以你就会被陷入到这样一个循环中，无法正常使用你的电子邮件客户端。

那么是什么原因导致了这个问题呢？微软并不确定，在这份技术支持文档中解释道

在 Insider Beta 版本 13004.10000 中提供了一个初始客户端修复。 Outlook 团队正在与 Windows 团队一起调查这个问题。

我们还不确定这个问题是来自 Outlook 还是 Windows。 当我们有更多关于修复细节的信息时，我们会在这里及时更新。

CW 的 Woody Leonhard 说，罪魁祸首是最新的累积更新，并补充说，PST 文件是好的，没有任何损坏。微软还在上面链接的页面上发布了一个临时解决方法，该公司建议用户在启动应用程序时，将 Outlook 配置为忽略 PST 文件的检查。这一切都可以通过注册表编辑来实现，不过对于部分用户来说还是等待微软修复吧。
