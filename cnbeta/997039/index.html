[图]Windows Insider告别Ring拥抱Channel：Fast自动升级为Dev频道

June 30, 2020

两周前，微软宣布 Windows Insider 项目启用全新的分发方式，原有的 Fast、Slow 和 Release Preview Ring 升级为和 Edge 相似的 Dev、Beta 和 Release Preview 频道（Channel）。而现在Windows Insider 已经陆续获得这项改进。

Windows Insider 官方推特账号发布推文

Hey，伙计们– Channel的时代已经到来。预计在接下来的几个小时内，您的#WindowsInsider设置中会出现这些变化。请记住，我们已经为你做了映射，所以你不需要做直接的改变。查看博客了解详情 ^AL https://t.co/auc8wvrxRy

Insider 成员将很快会发现 Fast Ring 会变成 Dev Channel，Slow Ring 会变成 Beta Channel，而 Release Preview Ring 会变成 Release Preview Channel。如果 Insider 成员想要更改渠道并尝试新的体验，可以随时通过 “设置">"更新与安全">"Windows内部人员计划 “来调整这些设置。
