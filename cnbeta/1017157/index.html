英睿达P5 1TB NVMe SSD上机实测 原厂闪存+新主控表现亮眼

August 18, 2020

继入门级的 P1 和 P2 系列之后，美光旗下消费级存储品牌英睿达（Crucial）又于今年 4 月发布了更高定位的 P5 系列 M.2 NVMe SSD 产品线。近日，英睿达 P5 SSD 的 500GB 和 1TB 版本终于在国内开售，我们也有幸拿到了 1TB 版本来体验。其采用了标准的 M.2 2280 板型，支持 PCIe 3.0 x4 通道和 NVMe 1.3 协议，具有高达 3400 MB/s 和 3000 MB/s 的持续读写性能（250 入门型号的写入为 1400MB/s，最高可选 2TB 版本）。

英睿达 P5 沿用了自 P1 以来的家族式包装，左上角是蓝底白字的英文商标，右下角标明了所属系列，同时通过条码贴纸来区分版本容量。

包装盒背面是多种语言的介绍、五年有限质保的承诺、并且可透过小窗可看到 PCB 板上的二维码。

取出后可看到多种语言的安装说明，以及留在塑料保护壳内的英睿达 P5 M.2 SSD 本体。

由官网资料和外媒早期报道（via The SSD Review）可知，英睿达 P5 SSD 选用了美光原厂高品质 96 层 3D TLC 闪存（两颗 NW969）、以及自研的八通道 NVMe 主控（型号为 DM01B2）。

此外据 PC World 报道，在每 TB 闪存搭配 1GB DRAM 的基础上（P5 1TB 搭配的是 D9CMZ），英睿达 P5 SSD 还支持动态分配 SLC 缓存，1TB 对应的标称写入量为 600 TBW 。

此外与一众采用公版方案的 PCIe 3.0 x4 / NVMe 1.3 M.2 SSD 相比，英睿达 P5 的 PCB 上还有一颗编号为 905400X101 / U00BYR 的芯片（或可提供一些特色功能）。

本次测试选用的是 AMD 锐龙 3700X 处理器 + 微星 B450M Mortar Max 主板的台式机平台，辅以 8GB * 2 双通道 DDR4-2666 内存（OC-3600）。单面的 PCB 设计，也能够很轻松地放入笔记本或 ITX 机箱。

离线测试用的系统为 Windows 10 2004 五月更新（编译版本号 19041.264），安装在了下方半速（PCIe 2.0 x4）的 M.2 插槽上，以便将上方的全速（PCIe 3.0 x4）M.2 插槽留给英睿达 P5 1TB SSD 。

安装 AMD 芯片组驱动后，NVMe 和 SATA 控制器驱动都使用了系统默认自带版本，并且在电源选项中开启了锐龙高性能模式。

Windows 10 2004 系统自带磁盘管理器认出英睿达 P5（型号为 CT1000P5SSD8）的初始化容量为 931.51GB，这里选择指定为一整个 GPT 分区。

测试期间，室内空调设置 27 ℃ 。为优化风道，小编挑选了一张单槽设计的亮机卡，且机箱前后还各有一枚 TT 的 12CM 风扇。

经过多次测试，英睿达 P5 1TB 版本均可在 Crystal Disk Mark 7.0.0 中轻松跑出 3500 MB/s 和 3300 MB/s 以上的顺序读写成绩，4K 随机性能也不俗。

AS SSD Benchmark 测出的顺序读写也有 3055 MB/s 和 2860 MB/s，总分在 5300+ 。

ATTO 磁盘基准测试，64MB 文件传输的读写性能可达 3.28 GB/s 和 2.89GB /s 。

实际文件复制，以 4.89GB 的 Windows 10 2004 五月更新 ISO 镜像为例，从 PCIe 2.0 x4 副盘拷贝到 PCIe 3.0 x4 主盘的速度也可以稳定在 1.4GB/s 左右，可在转瞬间完成操作。对于需要经常调动创作素材的生产力人士来说，这一点还是相当吸引人的。

即便将文件打散，英睿达 P5 SSD 的小文件混合写入性能依然出众。

空盘状态下，HDTune Pro 的全盘读写曲线也是相当平稳。温控方面，英睿达 P5 1TB M.2 SSD 大多在五六十度之间。

此外 CPUID HWMonitor 记录下的最低闲置温度为 48℃，最高工作温度为 68℃，室内空调环境下的表现还是挺让人满意的。

考虑到英睿达 P5 SSD 的主要信息都在 PCB 背部的贴纸上，不怕影响保修的“装甲爱好者们”也可以将正面贴纸揭掉，换成 DIY 散热片或主板附带的 M.2 SSD 散热马甲。

目前英睿达 P5 系列 PCIe NVMe M.2 SSD 的 500GB / 1TB 版本已在京东上架开售，首发价 699 / 1299 RMB（亚马逊 87 / 180 美元），感兴趣的朋友可趁着 599 / 1199 RMB 的秒杀活动期间入手。

至于入门级的 250GB 和更高的 2TB 版本，亚马逊上的参考报价为 63 / 400 美元，约合 438 / 2780 RMB 。
