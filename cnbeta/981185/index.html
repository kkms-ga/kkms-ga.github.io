微软宣布几周内正式推送新版Edge浏览器 带来一系列新功能

May 20, 2020

在今日的 Build 2020 线上开发者大会上，微软介绍了许多与 Edge 浏览器有关的内容。不过我们最关注的，莫过于这家软件巨头终于要在几周内正式向公众推送 Edge 浏览器了。其实早在今年 1 月份，Edge 开发团队就已经宣布了“全面的可用性”，并承诺“缓慢推出”，没想到一等就是好几个月。

微软中国官方商城 - Windows

用户终于有机会体验 Edge 稳定版本

今年 2 月，微软开始向 Release Preview 通道的 Windows Insider 测试者们推送 Edge 浏览器。然而除了公测用户，其他人尚未接收到此更新。

尽管博客文章文章中未明确提及，但 Edge 正式版似乎会作为 5 月更新的一部分发布（开发者已抢先体验，并将于本月晚些时候到来）。

届时用户有望通过 Windows Update 接收到新版 Edge 浏览器的推送，以及一系列的新功能。比如收藏夹与 Pinterest 深度集成，可将网络图片、文本、链接分组保存。

用户能够在收藏夹的底部看到 Pinterest 推荐，且支持将收藏集导出到这项服务，Edge Insider 测试者可于下月抢先体验。

此外侧边栏提供了新的搜索选项，不仅可以减少新建选项卡（标签页）的烦恼，还能够直接在侧边栏中显示搜索结果，该功能同样定于未来几周内向 Edge Insider 测试者推送。

必应搜索体验的改进，总与 Edge 浏览器相辅相成，这次也不例外。比如当你使用工作账户登录的时候，就会在图像、购物、新闻等标签之外，再多出一个新的“工作”页面。

其想法是便于通过 Windows 搜索功能查找组织机构的内部区域，可检索相关文件、人员、内部站点里的资源。与 Power BI 的集成，将于 6 月底到来。

最后，微软发布了 WebView2 的预览。作为 Project Reunion 项目的一部分，其旨在统一 Win32 和 UWP 应用。

开发者可将基于 Chromium 的 WebView2 集成到他们的应用中，而不是初始的 EdgeHTML WebView 。
