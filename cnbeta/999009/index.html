为与Chromium保持节奏一致 微软宣布暂缓两周的Edge浏览器更新

July 04, 2020

为了与 Google Chromium 的节奏保持一致，总部位于雷德蒙德的软件巨头微软宣布，其将暂缓 Microsoft Edge 浏览器的某些更新。鉴于 Windows 10 上预装的默认浏览器早已迁移至 Chromium 内核，我们对此并不感到意外，毕竟这么做可以让两者实现更无缝的同步。微软表示，该公司将于接下来的两周时间内暂停 Edge Dev、Beta、以及稳定版的推送。

与此同时，每日更新的 Canary 分支将照常推进。作为当前发布计划的一部分，Dev 版本的更新频率为每周一次，而 Beta 通道则是每六周一次。从 7 月 13 日所在的那周开始，Microsoft Edge 将恢复既定的发布节奏。

在迁移至 Chromium 引擎的同时，微软也在努力提升 Edge 浏览器的更新频率，使之不再受到操作系统更新周期的限制，可通过 Windows Update 更快地获得维护更新。

同时该公司在努力将新版 Edge 浏览器带到 Windows 10 之外的平台上，比如 macOS 和 Linux，可惜后者的预估到达时间（ETA）一直杳无音讯。
