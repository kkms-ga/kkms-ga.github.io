一切为了玩家 云游戏将于9月加入XGPU

July 16, 2020

微软Xbox大佬菲尔·斯宾塞今日发表了一封关于关于次世代的公开信，除了介绍Xbox关于次世代的承诺以外，其中重点提到了Xbox Game Pass Ultimate将于今年9月免费加入Project xCloud云游戏支持，此外微软确认目标在Xbox Series X发售时，让玩家在新主机上玩到Kinect以外的所有Xbox游戏， 现有的控制器（例如精英手柄等）也都支持XSX。 以下为具体内容：

官方原文：

游戏是快乐、灵感以及社交的源泉。它们的力量让我们汇聚在一起，互相理解，让我们更加团结。在我们紧锣密鼓地为新世代做准备时，我们也在为让游戏体验更具包容性、更加沉浸、让更多人互联、更具社交性这些同样重要的方面付出努力

Xbox团队一直在努力倾听广大玩家、开发者、内容创作者和直播播主对于游戏未来的期许。基于大家的反馈，我们正在打造能够让大家通过Xbox主机、PC以及移动平台都可以感受到沉浸感、交互性和深厚友谊的游戏体验。

在新世代即将到来之时，我们认为让大家能够清晰地了解Xbox的未来至关重要。

我们的愿景围绕着一个中心来逐步实现：您。

今天，我想与大家分享我们对大家的承诺：

·Xbox的大门永远为您敞开: Xbox的游戏体验围绕您而打造 —— 我们的玩家来自世界各地、各行各业。我们希望您所在的Xbox群体是安全、具有包容性，且能够让您畅享游戏乐趣的。正如我们在社区行为准则中所提出：种族主义、以及所有形式的歧视和仇恨在Xbox没有立足之地。如果您认为社区中其他玩家违反了相关的准则，我们的安全团队将会根据您的报告进行严格的调查，并为您提供24/7/365的不间断支持。同时，我们也在不断加速推进新的技术，来减少仇恨言论的出现，为您提供打造安全游戏环境的工具。

除了工具之外，我们承诺为您带来更具多样性的Xbox体验。我们正在助力来自不同文化及社会背景的内容创作者来创作新的游戏体验，让各类元素在游戏中都能够有真实、准确的体现，同时也在努力让所有人都能够畅享游戏的乐趣。此外，有超过30万的Xbox大使通过他们对游戏的热爱帮助Xbox群体成为最能够让玩家体验游戏乐趣的大家庭，在此我也诚挚地邀请所有玩家和我们一起实现这一愿景。我们还有许多的工作要去完成并且在每一个玩家被欢迎、倾听和珍视之前，不会止步。

·Xbox Series X让您的游戏更加绚丽更加好玩。Xbox Series X专为更高标准且前所未有的逼真度、游戏体验、性能和精准度而打造。游戏在Xbox Series X上将更加绚丽更加好玩 —— 无论它们来自15个Xbox Game Studios旗下工作室 （如 《光环：无限》）还是来自我们业界知名的开发商及发行商伙伴。Xbox Series X承载超过12 TFLOPs的GPU处理能力、诸如硬件加速的Direct X光线追踪和多变速率渲染等新科技、以及4倍于Xbox One X的处理能力。Xbox Series X让开发者能够通过更加丰富、生动的游戏世界、更加逼真的AI及动画，以及对于高至120 fps的帧率支持为您打造具有革新性的游戏体验。

Xbox Series X还将让您减少等待时间，将更多的时间投入到真正的游戏中去。它通过40倍于上一代主机的I/O吞吐量，几乎完全消除了游戏的加载时间。借助于我们的新世代定制SSD以及Xbox快速架构，游戏中几乎方方面面都得以改进。游戏的世界更加广阔、更具生机，同时也将在一瞬间加载完成。游戏中的快速旅行也可以做到真正的快速。Xbox快速架构还将为快速唤醒等全新平台技术提供支持。玩家通过快速唤醒可以瞬间继续先前的游戏进度，并可以在多个游戏间无缝切换而不必再观看冗长的加载画面。现在，Xbox Series X已经分发给我们15个Xbox Game Studios旗下工作室以及数以千计的第三方开发伙伴，助力他们为您提供新世代的游戏体验。

·Xbox Game Pass让您在首发当日即可畅玩游戏大作。所有来自Xbox Game Studios  的游戏作品将会在其全球首发当日同步加入Xbox Game Pass,无论是通过您的Xbox Game Pass  订阅服务畅玩，还是对游戏进行单独购买，选择权都掌握在您的手中。Xbox Game Studios旗下系列作品在发售首日加入Xbox Game Pass的包括： 《光环》,《极限竞速》,《帝国时代》,《战争机器》,《我的世界》,《地狱之刃》,《天外世界》,《脑航员》,《微软飞行模拟》,《腐烂国度》,《废土》,《我的世界：地下城》 以及 《盗贼之海》—— 更多的游戏系列内容也正在开发之中。所以，当《光环：无限》上市之时，您和您的好友可以决定是通过Xbox Game Pass畅玩还是单独购买游戏。

·您将不会被迫进入新世代。我们希望每一个Xbox玩家都可以畅玩来自Xbox Game Studios的全新游戏作品。这也就是为什么诸如《光环：无限》等Xbox Game Studios的系列作品在Xbox Series X和Xbox One主机上均可畅玩。即便是Xbox Series X上市之后，您仍可以选择您喜爱的平台来畅玩来自Xbox的独占游戏作品。

·借助向下兼容，您的游戏将和您一起步入新世代。您将可以通过Xbox Series X畅玩横跨4个世代的游戏作品。这也使得Xbox Series X拥有了游戏主机有史以来最庞大的游戏发布阵容。您将可以在数千款游戏中进行选择和畅玩。我们的向下兼容工程师通过数年的努力、创新的当世代及新世代技术，让您正在不断壮大的游戏库变得更加精彩，而这一切也不需要您付出额外的成本，也无需开发人员进行额外的工作。我们的目标是在新主机上市当天，所有不需要借助Kinect运行的Xbox One游戏在Xbox Series X上均可畅玩。同时，借助Xbox Series X的强劲能力，大多数您喜爱的游戏都将更快加载，更加绚丽，更加好玩。

·您的Xbox One游戏配件也将和您一起步入新世代。Xbox精英手柄和Xbox无障碍控制器都将兼容Xbox Series X,所以您不必购买新手柄。与他人不同，我们坚信您为游戏所作出的投入也应该和您一起步入新世代。

·一次购买，无需额外付费。借助我们全新的智能分发技术，您将不必为同一款游戏在不同平台进行多次付费。无论您在哪一个Xbox平台游戏，您都将在该平台获得游戏的最佳游戏体验。如果您拥有支持智能分发技术的游戏如《命运2》，《战争机器5》和《光环：无限》，您将自动获得该游戏在您所选择平台上的最佳版本。 目前，《刺客信条：英灵殿》、《赛博朋克2077》，《漫威复仇者联盟》以及更多即将上市的精彩游戏大作也都将支持智能分发，我们也会在未来公布更加详细的支持阵容。

Xbox Play Anywhere数字版游戏也将让您可以通过一次购买即可在Xbox主机和Windows 10 PC进行畅玩。

·健康及平衡的游戏生活方式由您来掌控。如果您是家长或监护人，全新的iOS及Android平台Xbox Family Settings app (预览版)将为您提供简单便捷的方式来创建未成年人账户，自定义家庭游戏设置来确保未成年家庭成员以健康积极的方式进行游戏。

·您的Xbox Game Pass Ultimate订阅服务将为您带来更多内容。今天，我们隆重宣布：从九月起开始，我们将在已支持Project xCloud的市场将该服务加入Xbox Game Pass Ultimate，已经订阅Xbox Game Pass Ultimate的用户将无需支付额外费用。借助Xbox Game Pass Ultimate中的云游戏服务，您将可以在您的手机或平板设备上畅玩Xbox Game Pass游戏库中超过百款游戏作品。同时，您的Xbox Live订阅功能也可以让您和全球各地超过1亿Xbox Live用户共同游戏。在《光环：无限》上市时，您和您的好友将可以在任何地点、通过您所选择的设备扮演士官长进行《光环》宇宙中的冒险。

在Xbox Game Pass Ultimate中加入云游戏功能意味着您的游戏将不再局限于客厅。正如您喜爱的电影、音乐和直播内容一样，在云游戏加入Xbox Game Pass Ultimate之后，您可以通过任意设备继续您的游戏进度。

游戏的未来从未如此令人振奋。这将是一个由您决定如何进行探索的未来。您的游戏征程将和您一同步入新世代。如果您在新世代到来之际仍选择继续留在目前所持有的主机上一段时间，您也依旧可以畅玩来自Xbox Game Studios的全新独占游戏作品。

这是一个您可以和好友在不同设备上共同领略沉浸式生动世界的游戏未来；这是一个通过游戏连接世界各个角落的游戏未来；这是一个您可以通过游戏获取更多价值的未来，而这个未来，也欢迎每一个人。

我们诚挚地邀请您和我们共同观赏北京时间7月24日0点开始的Xbox Games Showcase并和我们共同了解来自Xbox Game Studios第一方游戏作品的最新消息。

感谢您，和我们共同打造游戏的未来。

菲尔·斯宾塞
