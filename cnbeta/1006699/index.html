[图]部分AMD设备依然无法升级Build 20175 微软：正积极修复

July 23, 2020

对于那些运行 Windows 10 Build 20161 以及此前版本，尤其是使用 AMD 处理器的用户来说，今天发布的 Build 20175 能够让他们摆脱“7月31日过期”的困扰。不过遗憾的是，不少受影响用户在社交媒体上反馈，依然无法升级到新版本。

随后 Windows Insider 团队的 Brandon LeBlanc 表示：“我们正在积极解决这个问题：配备 AMD 处理器的 PC 无法获得今天应该获得的新版本。”
