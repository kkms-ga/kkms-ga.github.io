Windows 10已允许用户获取可选的驱动和系统更新

August 19, 2020

微软表示，从 Windows 10 Build 19041.450 开始，其已恢复 Windows 7 时代的“可选更新”选项，以帮助用户获取可选的驱动程序和非安全性的功能更新。在此之前，有需要的用户需要通过繁琐的操作，才能在设备管理器中获取驱动更新。但是现在，你可在“设置 -> 更新与安全 -> Windows Update -> 查看可选更新”页面下轻松获取可选更新。

改进后的这项功能，可在一个区域中显示所有这些可选更新，无论是质量更新、非安全更新、或是可用的驱动程序更新。

需要注意的是，除非你手动点击“暂停更新”、或使用注册表来屏蔽更新，否则 Windows Update 仍会在未经用户许可的情况下自动安装更新。

这项变动将于今年 8 月被引入 Windows 10 2004，等不及的朋友可通过早期渠道开展体验。

不过只有在“有可选更新可用”、或微软已批准并在 Windows Update 目录中发布了 OEM 驱动情况下，可选的驱动 / 非安全更新才会显示在 Windows 10 的“设置”页面上。

另一种更直接的方法，就是通过“运行”命令来搭载“可选更新”页面。方法是通过 WinKey + R 组合键唤出小窗口，然后执行“ms-settings:windowsupdate-optionalupdates”命令。

之后你将被允许访问隐藏的可选更新页面，如果没有新的候选驱动可用，则页面会显示一片空白。
