Parallels助力Chrome Enterprise支持Windows应用

June 17, 2020

为了给 Chrome Enterprise 企业与云客户带来功能齐全的 Windows 应用程序，谷歌已经同 Parallels 达成了一项新的虚拟化合作。得益于 Parallels 的软件解决方案，企业、组织和个人用户可以轻松地在任何设备或操作系统上访问和使用其所需的应用程序和文件。

Parallels Desktop 6·18 促销 使用优惠码可享7折

（来自：Google Cloud Blog）

举个例子，谷歌客户将能够把包括 Microsoft Office 在内的功能齐全的 Windows 应用程序，无缝地添加到 Chromebook Enterprise 设备上。

基于此，客户能够随时随地地实现高效协作、消除额外的硬件成本、并将总体拥有成本（TCO）降至最低。

目前 Parallels 正与 Chrome OS 团队紧密合作，以确保每家公司都能够在这项云端功能创新中获益，更多细节将在未来几个月内陆续公布。
