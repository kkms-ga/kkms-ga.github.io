[图]贝尔金推出紧凑型Thunderbolt 3 Dock Core 售价170美元

June 30, 2020

贝尔金（Belkin）今天推出了 Thunderbolt 3 Dock Core 扩展坞，“双电源”选项支持从笔记本电脑直接给外设供电，因此可以不用独立的电源线。贝尔金公司表示，Dock Core的目标是针对那些寻求快速传输速率和单线解决方案来连接笔记本电脑和外设的用户。

Thunderbolt 3 Dock Core 专为 Mac 和 Windows 设备设计，提供了一个紧凑的外形，非常适合台式机使用。该扩展坞支持 40Gb/s 的传输速率，60W 的电源上行充电，并支持一台 8K 显示器或两台 60Hz 的 4K 显示器。

在端口方面，提供一个USB-C PD电源，一个 DisplayPort 1.4 端口，一个 HDMI 2.0 端口，一个 1Gb 以太网端口，音频输入/输出端口，一个 USB-A 3.2 Gen 2（10Gb/s端口）和一个 USB-A 2.0 端口。

当连接到笔记本电脑时，Dock Core能够为连接的外围设备供电，而不需要单独的电源线。贝尔金的 Thunderbolt 3 Dock Core 将于2020年7月上市，零售价格为 170 美元。
