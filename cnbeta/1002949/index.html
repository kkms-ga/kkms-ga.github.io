[图]微软骚操作：通过Outlook在Android长按菜单中嵌入Bing搜索

July 14, 2020

向 Android 平台输出 Microsoft Launcher 等优秀应用的同时，微软也采取激进的方式来推广自己的应用程序。去年，微软就在 Android 的分享列表中插入了 OneDrive 广告，而援引外媒近日报道，微软现在通过 Android 端的 Outlook 应用，选中任意文本之后长按菜单中会有 Bing 搜索的选项。

援引外媒 Android Police 报道，Pixel 社区论坛上部分已在设备上安装 Outlook 的用户反馈，在长按菜单中新增了“Bing Search”的新选项。但令人困扰的是，用户不仅没有安装 Bing 应用程序，而且该 Bing 搜索选项是系统级别的，选中任意文字长按都会出现这个选项。点开该选项，会通过 Outlook 的内置浏览器访问 Bing 搜索。

外媒 XDA-Developers 随后对 Outlook 应用进行了分析，确认这种行为来自于该应用程序。不过，它似乎还需要安装 Edge 浏览器才能触发。

Outlook在其Manifest文件中有一个 intent-filter，可以使其在高亮显示文本时出现在长按上下文菜单中。但是，除非启用了“Edge Integration”功能标志，否则这个 intent-filter 是被禁用的。

用户在安装 Outlook 和 Edge 之后，并将 Edge 作为默认浏览器之后，就能将 Bing 选项显示出来。在卸载 Outlook 之后，Bing 搜索方式就消失了。随后外媒再次尝试安装，发现一直无法让它再次显示。

无论如何，这是一个相当公然滥用上下文菜单的行为，也是一种不恰当的宣传 Bing 的方式。
