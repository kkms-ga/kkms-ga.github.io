美国微芯新发PCIe 4.0 SSD主控：低功耗飙上6GB/s

August 04, 2020

PCIe 4.0 SSD越来越丰富，从消费级到企业级都在普及，而说起PCIe 4.0 SSD主控，作为普通人大家接触最多的就是群联电子PS5016-E16，但是在企业市场上，这种产品多着呢。早在两年前，美国微芯半导体(Microhips)就发布了企业级的PCIe 4.0 SSD主控方案“Flashtec NVMe 3106”，最近终于投入量产，现在又推出了它的降级版本“Flashtec NVMe 3108”，规格更低，但是功耗也更低。

3106、3108都采用此类产品最通用的台积电16nm工艺制造，架构也完全相同，只是规格有明显差别。

3106内部采用Tensilica CPU架构，支持PCIe 4.0 x8、16个闪存通道、1CE，NAND闪存支持1200MT/s，最大容量超过200TB，DDR4内存支持3200MHz频率、40(32+8)/72(64+8)-bit位宽，随机读取性能超过2M IOPS。

3108改成了ARM CPU架构，而且核心数更少，支持PCIe 4.0 x4、8个闪存通道、16CE，NAND闪存规格不变，最大容量仅为64TB，DDR4规格则降级为2400MHz频率、16/32/40(32+8)-bit位宽，持续读取性能超过6GB/s，随机读取性能则超过1M IOPS。

精简规格后，3108的功耗大大降低，芯片封装面积也大大缩小，可以放入标准的M.2 SSD。

另外，二者都支持NVMe 1.4、安全启动、数据加密、加密验证固件、虚拟化、SLC/MLC/eMLC/TLC/QLC闪存类型，3106还额外支持压缩加速器硬件、扩展端口(可以两个合并支持PCIe 4.0 x16)。

微芯3108目前处于试样阶段，明年上半年投入量产，面向入门级企业市场。
