安全研究员指责红米Note 8收集隐私数据 小米回应称被误解

May 02, 2020

根据市场研究机构 Canalys 数据分析，红米 Note 8 是 2019 年最畅销的 Android 智能机之一。然而近日，一位安全研究人员却指责小米公司的手机和网络浏览器在收集用户的隐私数据，比如打开了哪些 App 和文件等细节。在周五的博客文章中，小米回应称，这些统计数据是为了优化程序响应和性能，并不涉及可识别的身份信息。

周四的时候，《福布斯》援引多名安全研究人员的话称，该公司正在收集网络历史记录和电话数据，比如可悲用于识别特别设备的 Android 版本唯一编号。

安全研究员 Gabi Cirlig 向《福布斯》表示，结合收集的数据和标识号，即可识别特定的用户。

且红米 Note 8 上默认的浏览器会记录用户访问过的任何站点，包括搜索引擎查询和新闻提要功能里查看的每条内容。

即使在隐身模式下浏览，这种追踪也似乎不会停止。此外还有已打开的文件夹和屏幕滑动信息，这些数据都被发送到了该公司租赁的阿里云服务器。

尽管 Google Chrome 和 Mozilla Firefox 之列的第三方浏览器也在收集有关网址访问等在内的汇总用户信息，但这些制造商也公布了如何保护数据的详情。

谷歌表示，Chrome 会收集与用户标识符无关的使用情况的‘匿名化随机数据’。

Mozilla 亦在 2017 年启用了一款程序来收集 Firefox 用户的使用情况数据，并通过隐私差分的方式进行保护，以增加特定的个人数据的定位难度。

不过小米辩称，《福布斯》误解了该公司的数据隐私原则和政策，并否认有任何不当的行为。因为只有用户在设置中启用了该功能，才会同步 Web 浏览器的历史记录。

小米将用户的隐私和安全视为重中之重，我们将严格遵守全球范围内的用户隐私保护法律和法规。

当然，无论是苹果 iOS、还是 Google Android 移动操作系统，隐私问题都是绕不开的一环。

通常情况下，研究人员不得不深入研究这些设备，以查看第三方 App 是如何收集、并向广告商发送有关用户位置和使用情况等数据的。
