Chrome遭遇无法在Windows 10上正常播放HDR视频的bug

July 14, 2020

Reddit 网友抱怨称，他们在使用 Chrome（v81+）和系统自带的 Edge 浏览器的时候，遇到了浏览器无法正常播放 HDR 内容的问题。有网友写道：在对“视频播放”设置项中的滑块进行调节之后，YouTube 上的 HDR 内容的亮度级别就出现了不准确的问题。

（来自：MSPU）

在理想情况下，调节“SDR 内容外观”的滑块只会影响 SDR 内容、而不会波及 HDR 视频内容。

然而 Windows 10 的问题在与，若在“HD 颜色设置”中将该滑块调到最大，也将自动增加 YouTube 上 HDR 内容的亮度级别。

某位 Reddit 网友描述其使用的是 Windows 10 1909 系统和最新版的英伟达 RTX 2080 Ti 独显驱动，但却在播放 YouTube HDR 视频时被亮瞎了眼。

在典型设置下，游戏、媒体播放器和观看 4K HDR 蓝光资源都一切正常。但修改了系统颜色设置中的 SDR 外观滑块，就会波及 Chrome 中播放 YouTube HDR 内容的亮度。

（图 via Techdows）

许多网友在评论中反馈遇到了同样的问题，有人将滑块调到 100% 以增加桌面亮度，结果后来在播放 YouTube HDR 视频时被亮瞎了眼。

另一位 Reddit 网友指出，即便将滑块降低到 50，同一视频的亮度也会伴随着降低，这种情况是本不该发生的。

庆幸的是，Chromium 团队已知晓该问题，并做好了推出修复程序的准备，预计很快就会向 Canary 频道推送。

（来自：Reddit）

想要复现这个问题，可先准备好一台支持 HDR 的显示器，然后再最新版的 Chrome 浏览器中启用 HDR 。

方法是在桌面上点击鼠标右键，选择“Windows 桌面 -> 显示设置 -> 播放 HDR 游戏和应用”。

接着在这个显示设置窗口中点击“Windows HD Color 设置”，选中“播放 HDR 游戏和应用”与“流式传输 HDR 视频”，向下滚动并将“SDR 内容外观”滑块拉到最高。

完成上述操作后，可在 Chrome 或 Edge 浏览器中点开某个 HDR 视频（比如 LTT 介绍 Pro Display XDR 这款 HDR 显示器的这篇）。

在被亮瞎之后，还请及时返回 Windows 10 显示设置，并将滑块调回最低。尽管桌面上其余部分会因太暗而看起来有些难受，但至少 Chrome 上播放 HDR 视频时不会再亮瞎眼。
