Ubuntu 20.10 将为 ZFS 提供更多改进

May 07, 2020

从 Ubuntu 20.04 LTS 发布到现在已有两周，Canonical 和 Ubuntu 社区也已将注意力转移到 Ubuntu 20.10 Groovy Gorilla 上。开发团队最新的每周会议总结显示，在 ZFS 方面似乎将会进行一些更令人兴奋的变更。

早在 Ubuntu 19.10 中，开发团队就已将 ZFS (OpenZFS/ZFSOnLinux) 的实验性根文件系统添加到其桌面 Ubiquity 安装程序中，以作为实验性的选项。到了 Ubuntu 20.04 LTS，使用 ZFS 作为根文件系统仍是一个实验性的选项，但他们对安装程序的 UI 做了一些改进，升级了 ZoL 快照，并围绕 Zsys 守护进程提供了更多的改进，其中包括在 APT 事务上自动生成 ZFS 快照。这与 GRUB 引导加载器的支持相配合，如果出现严重的软件包故障，可以快速回滚到系统的早期版本。

ZFS 将是 Ubuntu 20.10 的开发工作重点，Canonical 员工 Jean Baptiste Lallement 提到，他已开始分析 ZFS 加密技术并准备参加内部各种关于“ZFS和企业级桌面”的会议。

虽然还需要再过几个月才能看到团队为 Ubuntu 对 ZFS 改进的结果。每周会议中显示的加密功能意味着可能会出现加密的 Ubuntu ZFS 安装，类似于目前的 LVM 全盘加密。相信除了这个变化，还会有更多有趣的内容。
