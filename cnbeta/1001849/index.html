Mozilla Firefox 78.0.2 修正版发布

July 11, 2020

Mozilla Firefox 78.0.2 修正版现已发布，正式版之后的0.x后续版本主要带来一些 bug 修复。Firefox 78 虽然改进幅度并不是很大，但完善了用户体验，例如卸载器新增了刷新按钮，能够让 Firefox 恢复到初始配置，与此同时还内置了 PDF 阅读器，并放弃了对某些旧版本操作系统的支持。

具体更新列表如下：

Fixed

修复了阅读器模式下的可访问性回归（错误1650922）

使地址栏对用户配置文件中的数据损坏更具弹性（错误1649981）

修复了打开某些外部应用程序的回归（错误1650162）

Developer
