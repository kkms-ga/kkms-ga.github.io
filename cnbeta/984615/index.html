[图]Chrome Canary更新：允许PWA应用随Windows开机启动

May 29, 2020

谷歌一直致力于改善Chrome浏览器上的PWA应用体验。上月在Play Store为Chrome OS用户将部分Android应用替换为PWA应用之后，今天谷歌为Chrome Canary用户带来了新改进，能让PWA成为Windows的开机启动程序。

现在谷歌为Chrome Canary用户带来了一项改进，能够让PWA应用随Windows开机启用。这项功能最初被外媒TechDows发现，隐藏在一个Flag后面。如果你正在使用Google Chrome Canary，那么你可以按照下面的步骤来启用该功能。

1.启动Google Chrome Canary，在地址栏中键入 “chrome://flags”。

2.在搜索栏中键入 “Desktop PWA run on OS login”。

3.单击下拉菜单并选择 “已启用”。

在重启Chrome浏览器之后，用户能够访问支持PWA的网站（例如cnbeta.com ）并在地址栏上点击安装。在 “安装应用程序 “对话框中，勾选 “登录电脑时启动应用程序 “选项，然后点击安装。这样该PWA应用下次就会随Windows开机而启动。

目前尚不清楚用户如何从开机启动项中删除该应用程序，因为谷歌并没有提供取消勾选"登录电脑时启动应用程序 “选项。不过用户可以打开启动文件夹，手动删除PWA应用程序。要做到这一点，按Windows+R打开运行，输入”%APPDATA%MicrosoftWindowsStart MenuProgramsStartup “打开启动文件夹并删除快捷方式。
