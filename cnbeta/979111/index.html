无缘屏下摄像头 三星Galaxy Note 20系列曝光：挖孔屏/120Hz高刷

May 14, 2020

5月14日消息，Android Authority曝光了三星Galaxy Note 20系列的核心参数。报道指出，Galaxy Note 20系列包含Galaxy Note 20和Galaxy Note 20+两款，其中Galaxy Note 20的屏幕尺寸为6.42英寸，Galaxy Note 20+的屏幕尺寸为6.87英寸。

SAMSUNG - 三星旗舰店

二者都将搭载高通新一代3D超声波屏下指纹，报道称新一代指纹技术比上一代大17倍，解锁速度更快，而且可以录入两个指纹，安全性相比上一代大幅提升。

Galaxy Note 20渲染图（真实性有待证实）

核心配置上，Galaxy Note 20系列搭载高通骁龙865旗舰平台，最高配备16GB内存，电池容量为4300mAh（Galaxy Note 20+的电池容量未知）。

最后Android Authority表示，三星Galaxy Note 20系列均采用挖孔屏方案，前置摄像头挖孔位置与Galaxy Note 10系列一致（居中），无缘屏下摄像头。

之前有爆料称屏下摄像头技术将在今年下半年量产商用，这将是行业和用户都期待的真全面屏的终极解决方案。

不过屏下摄像头初期成像表现不及挖孔屏，这是因为目前的技术能力还没有找到兼顾显示效果和相机效果的有效方案。

小米集团中国区总裁、Redmi品牌总经理卢伟冰之前曾科普过，目前的屏幕PPI大约在400左右，而PPI在这个水平，相机中光的透过率是很低的，这样就严重影响了拍照效果。如果把PPI降低，提高透过率，就会出现显示区域的PPI和整个屏幕的PPI差距很大，从而出现显示区域有色块等现象。

目前来看，三星为了保证前摄成像选择了更成熟的挖孔屏方案，屏下摄像头最快会在明年Galaxy S系列上使用。

按照惯例，Galaxy Note 20系列将在8月份发布，9月份上市开卖。
