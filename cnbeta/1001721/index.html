AMD发布Radeon 20.7.1驱动更新 引入新的错误报告工具

July 10, 2020

时隔一个月，AMD 于今日发布了 Radeon Software Adrenalin 2020 Edition 20.7.1 版驱动套件。本次更新主要围绕 bug 修复，但也引入了一些新的功能。比如上月发布的 FPS RTS 混合游戏《瓦解》（Disintegration），就迎来了 AMD 显卡的官方支持。至于玩家可期待哪些变化或性能提升，目前暂不得而知。

值得一提的是，在 Radeon Software Adrenalin 2020 Edition 20.7.1 驱动套件中，还包括了一款新的错误报告工具。不喜欢打包安装的朋友，也可以单独去下载。

据悉，AMD Bug Report Tool 可在必要时自动采集系统详情，并允许用户填写表格，以直接向 AMD 驱动程序团队提交反馈，而无需通过老旧且麻烦的 Web 门户。

该工具可用于报告 GCN 和 RDNA 架构的显卡错误，并且同时支持 Windows 7 和 Windows 10 平台。

需要指出的是，尽管此前 AMD 已允许在 Beta 版驱动中启用某些 Navi GPU 的硬件加速调度，但似乎并不适用于此版本。

以下是 Radeon Software Adrenalin 2020 Edition 20.7.1 中修复的某些 bug（完整列表请移步至官网查看）：

● 修复在 Radeon RX 5000 系列 GPU 系统上启用 Instant Replay 时，某些游戏可能卡顿或冻结的问题；

● 修复在游戏运行时开启 Metrics Overlay 时，Radeon RX Vega / VII 显卡可能出现性能下降的问题；

● 修复在 Radeon Software 软件的各个选项卡之间进行切换时，有时显示错误消息、而非流媒体预览的问题；

● 修复在 Radeon 性能调节选项卡中应用更改之后，自定义风扇和频率可能被重置为默认值的问题；

● 修复在某些系统启动后，自定义配置文件可能无法正确加载或应用的问题；

● 修复《反恐精英：全球攻势》中启用显示缩放时，显示分辨率可能无法拉伸到全屏的问题。

Radeon Software Adrenalin 2020 Edition 20.7.1 可选驱动程序现已放出，感兴趣的朋友可从官方发行页面上下载安装包、或者在旧版 Radeon Software 中手动检查更新。

下载地址：Windows 10 64-bit | Windows 7 64-bit

最后，20.7.1 版可选驱动仍有一些已知问题，比如旧版中保存的性能调试文件无法与 20.7.1 或更高版本兼容，完整列表还请移步至官网发行页面查看。
