报道称ThinkPad X1 Nano为13英寸机型 或于11月上市

June 17, 2020

多次曝光后，我们对 ThinkPad X1 Nano 这款机型终于有了更多的了解。近日在接受 ThehnoBok 采访时，联想南非区经理披露了有关这款尚未发布的新机的一些细节，以及可能的发布日期。其实早在 2 月下旬，就有人在 GitHub 上发现了它的驱动程序页面。

（图 via NotebookCheck）

上个月的时候，我们在泄露的 ThinkPad 2020 路线图中再次发现了它的身影。可惜除了下半年上市这件事，联想并未在路线图中透露太多。

近日，联想南非区域经理 Thibault Dousson 在接受 YouTuber 采访时透露，ThinkPad X1 Nano 将配备 13 英寸的屏幕、专注于轻量级体验、并于 11 月前后投放市场。

尽管并非 ThinkPad X1 家族的首款 13 英寸机型，但 ThinkPad X1 Nano 却是自 2011 年以来的首款非平板 13 英寸 ThinkPad X1 笔记本。

毕竟粉丝们最为熟悉的 ThinkPad X1 Carbon，早已经用上了 14 英寸的较大屏幕。甚至连 ThinkPad X1 Yoga 这款变形本也在坚持使用 14 英寸的屏幕，毕竟这就是商务本市场中最受欢迎的尺寸。
