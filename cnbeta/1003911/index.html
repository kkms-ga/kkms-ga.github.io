微星被曝试图贿赂YouTube科技博主以消除游戏本的负面评价

July 16, 2020

微星近日陷入了一场与油管科技博主有关的负面公关事件，起因是 @TechTeamGB 在一段标题为《MSI Bravo 15 Review》的评测视频中对这款游戏本表达了失望之情。然而从一连串的推文爆料来看，微星最先想到的不是感谢其提出批评意见、并在将来作出改进，而是试图通过贿赂的方式来公关。

视频截图（来自：TechTeamGB / YouTube）

其实在正常的商业运作模式下，产品有那么一些无关紧要的负面评价，并不全是一件坏事。如果一边倒地在吹捧，反而会引发人们的怀疑。

许多网友因为自己对技术不那么精通，才需要参考可信赖的独立评测来作出明智的购买决定。遗憾的是，某些制造商（或 PR 人员）无法接受哪怕是一丁点的负面评价。

事件起因是 TechTeamGB 在 YouTube 上发布了有关微星 Bravo 15 游戏本的一段独立评测视频，其中指出了该机了一些缺陷，比如屏幕和散热可以进一步加强，触控板的表现也让人难以接受。

MSI Bravo 15 Review - Disappointing - TechteamGB（via）

在评测内容发布之前，TechTeamGB 试图与微星联系以证实上述问题是否为自己的个案，没想到该公司 PR 人员宁可想着通过贿赂的方式来堵住负面评价，而不是展开更加真诚的沟通。

评测视频发布之后，微星 PR 人员仍要求 TechTeamGB 移除视频标题中的“令人失望”一词，但遭到了硬气的拒绝。

而后 PR 方面甚至想要通过 AMD 这层关系（这款笔记本采用了锐龙 4000 系列处理器）来迫使 TechTeamGB 就范，庆幸的是，AMD 选择了站在博主这边。

最后，尽管 AMD 和微星方面都没有回应这个问题，但 LinusTechTips 和 Steve’s Hardware Unboxed 等知名科技评测媒体纷纷向 TechTeamGB 伸出了援手。

对于一家推出过大量热销游戏本的企业来说，微星明显可以对自家设备的产品力抱有更大的信心。但目前尚不清楚此番公关翻车事件是否得到了公司的默许，或仅是个别 PR 人员在无上级授权情况下的过激之举。
