直观对比：macOS Big Sur较Catalina迎来了多大程度上的UI设计改进？

July 08, 2020

为帮助大家更加直观地了解 macOS 11 Big Sur 较 macOS 10.15（Catalina）迎来了 UI 设计上的哪些变化，Andrew Denty 特地制作了一段对比演示视频。作为 Mac 操作系统的下一个主要版本，苹果已经在上月的 WWDC 2020 开发者大会上介绍了它的预览。至于新版 UI 将对 Mac 应用程序的开发带来哪些影响，仍需进一步的深入学习。

左侧为 macOS Catalina，右侧为 macOS Big Sur 。

Andrew Denty 在默认安装设置下进行了两版 macOS 操作系统的对比，并且有意识地不调整任何窗口的大小和设置。

并排比较下的访达、系统偏好设置、菜单栏、通知中心、Safari 浏览器、日历、联系人、提醒、便签、照片、Apple Music、播客和其它许多原生应用的图标都进行了风格重绘。

总体而言，Andrew Denty 的感受是：macOS Big Sur 中的 UI 差异并没有最初想象的那么剧烈，而是包含了“一系列的增量变化”，使之变得与 iOS 和 iPadOS 更加协调一致。

即便如此，他仍觉得苹果会在 macOS Big Sur 测试结束前投入大量的工作精力，以进一步完善新的 macOS UI 。比如让应用程序的启动体验更加一致，以及让状态栏和路径栏等各种视觉元素更好地融入整体。

macOS Big Sur - Completely New Look（via）

目前苹果已经向开发者开放了 macOS Big Sur 的预览访问，并计划在本月开放 Beta 公测，最终在秋季正式发布。
