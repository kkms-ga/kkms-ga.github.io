iFixit称Galaxy Note 20配备的不同散热系统不成问题

August 27, 2020

三星旗舰智能机的部分消费者，经常为同系列机型选用的高通骁龙和三星 Exynos 芯片组之间的差异而感到困恼，比如今年早些时候推出的美版 / 国际版 Galaxy S20 。随着 iFixit 拆解报告的发布，下半年刚推出的 Galaxy Note 20 系列，再次引发了网友们对于其散热方案的争议。因为他们发现，三星为某些装置配备了热管、另一些则采用了石墨导热垫。

（图自：iFixit）

为了化解纷争，iFixit 不得不强调两种散热系统与内置的处理器型号无关。此前许多人猜测，Exynos 版 Galaxy S20 系列机型在散热上有短板。

不过现在，iFixit 和 JerryRigEverything 的 Zack Nelson 都证实，三星在骁龙和 Exynos 版本上都使用了热管 / 石墨散热方案。

拆解专家补充道，目前尚无证据表明使用石墨散热的装置导致手机异常发热，因为大家无法透过外壳或包装来分辨设备到底采用了 哪种散热方案。

事实上，如果用户能够感觉到烫手，则意味着冷却系统实际上正在发挥作用，将热量从敏感组件转移到机身外部。

iFixit 工程师 Greg kramer 解释称，基于石墨或热管的散热方案，其实没有太大的差异。只不过后者的成本相对更高，且难以在机身内安装。

鉴于三星总有“两手准备”，这样的拆解结果可以说是在情理之中、意料之外。只是希望在下一次舆论发酵的时候，该公司的 PR 部门能够更快更好地安抚消费者的情绪。
