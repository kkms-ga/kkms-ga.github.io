错字乌龙：苹果意外邀请测试用户体验macOS Bug Sur

June 23, 2020

苹果在 WWDC 2020 主题演讲期间，介绍了新版 macOS 操作系统的正式名称，它就是出乎很多人意料的“Big Sur”。根据该公司为 macOS X 引入的命名习惯，Big Sur 与 Sierra、Catalina 和 Mojave 一样，都属于加州的一处风景名胜。

截图（来自：Stephen Shankland / Cnet）

在大家努力适应 Big Sur 这个代表内华达山脉的新名词时，苹果工作人员似乎不小心地将新系统称作“Bug Sur”。

有测试者体验者在其接收到的邮件中发现，官方错误地在邮件标题中打上了“Bug Sur”的字样，但正文随附的图片又明确印着 Big Sur 。

苹果尚未对本次邮件错字事件给出回应

作为苹果生态大一统的新产物，苹果不仅让 macOS Big Sur 兼容即将推出的 ARM Mac 设备，还在界面和功能设计上努力贴近 iPadOS 。

自 2006 年转向使用 Intel 处理器以来，macOS Big Sur 意味着该公司的又一次重大转型，系统版本号也终于从 10.15 一跃提升到了 macOS 11 。
