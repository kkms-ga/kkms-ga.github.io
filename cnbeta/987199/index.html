[图]Xbox Game Bar 开始测试插件商城：首批共有 7 款

June 05, 2020

两个月前通过支持第三方插件，微软进一步提高了 Windows 10 的 Xbox Game Bar 通用范围。在本周发布的 Xbox Game Bar 更新中，全新的插件商城（Widget Store）已进入测试阶段。当你打开游戏的“叠加画面”（overlay）时候，你就会看到宣布全新插件商城的弹出通知。

6·18活动已全面开启 大促活动入口汇总：

阿里云6·18上云年中大促 点击领取最高12000元红包

2020天猫6·18超级红包在此领取 6月1日追加40亿元消费券

京东6·18十七周年庆大促主会场入口 - 最高可领618元红包

访问:

微软中国官方商城 - Xbox

想要访问全新的插件商城，你需要点击游戏栏左侧的 Widget 菜单。目前该商城只提供了 7 个插件，包括 RAZER Cortex 性能优化应用、热门游戏直播应用 XSplit（可替代近期从 Xbox Game Bar 中移除的 Mixer 直播功能）等。

在后台方面，Xbox Game Bar 的第三方插件是使用 Microsoft Store 进行安装的，因此在你按击 Win + G 的时候会在叠加画面中显示出来。不过，诸如 XSplit Game Bar HUD 等少量插件需要以桌面应用的形式安装使用。
