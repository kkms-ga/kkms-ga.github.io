[视频]Pie Chart Pirate回顾了浏览器市场在过去25年里的份额变化

August 31, 2020

现如今，浏览器（Web Browser）已经在 PC、平板、智能机上无处不在。尽管其最初是在 PC 上启动的，但世界正在加速向移动领域转型。在移动体验变得越来越重要的同时，浏览器和站点也必须充分考虑到跨设备漫游时的平台一致性。为便于大家了解全球浏览器市场在过去 25 年时间里发生的变化（1995 ~ 2020），Pie Chart Pirate 特地在 YouTube 上分享了一段动画视频。

视频截图（来自：Pie Chart Pirate / Youtube）

近日，有网友在 Reddit 上分享了一张展示浏览器市场在过去 25 年里的变化的动画图表，可见曾经的霸主（IE）和行业领军者（Mozilla Firefox）的份额有了翻天覆地的变化。

根据微软的计划，其将于 2021 年停止对 IE 浏览器的支持，以全面转向基于 Chromium 内核定制的新版 Microsoft Edge 浏览器。不过正如老当益壮的操作系统那样，两者将继续共存很长一段时间。

browser world changed in the last 25 years（via）

通过拥抱 Chromium，新版 Microsfot Edge 正在以 Windows 10 为主的平台上吸引越来越多的用户，看起来很有希望重现 IE 的辉煌。

不过至少现在，Google Chrome 仍然是浏览器市场的绝对主导。第三方统计数据表明，其在桌面平台上占据大约 70% 的市场份额。
