[图]美版特斯拉Model 3即将内置Qi无线充电面板和USB-C端口

June 14, 2020

援引外媒 Elektrek 报道，位于美国加州弗里蒙特的特斯拉工厂已经开始为 Model 3 车型内置无线手机充电器和 USB-C 端口了。这则消息最初由 Drive Tesla Canada 媒体报道，这些手机功能已经装备到 Model Y 车型中，并于上月开始在国产 Model 3 中进行装备。

在今年三月向美国消费者交付的 Model Y 车型中，已经包含了标准的 Qi 无线手机充电器，不过在 Model 3 上只是作为提车后加装配件方式提供。特斯拉上个月降低了其在北美的Model 3、Model S和Model X的价格，以及在中国销售的Model S和Model X汽车的价格。其在中国生产的Model 3汽车并没有降价。据报道，降价是为了帮助提升销量。
