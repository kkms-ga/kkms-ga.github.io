谷歌计划在Android 11中默认禁用过度扭曲的美颜滤镜

July 17, 2020

随着智能手机和移动互联网的兴起，相机的美颜滤镜也迎来的飞速的发展。起初这些滤镜旨在调节色彩平衡以实现独特的创意，或通过多图拼接来确保合照中的每个人都处于睁眼的状态。然而 AI 美颜技术的突飞猛进，同样在社交网络上遭到了猛烈的批评。由“照骗”爱好者们掀起的这股潮流，被指责传递了不健康的价值观（比如身体扭曲畸形、肤色惨白）。甚至让涉世未深的儿童和成年人去盲目追求“病态美”，结果导致饮食的失调。

XDA-Developers 总编辑 Mishaal Rahman 在一份泄漏的 Android 11 兼容性定义文档中发现，谷歌似乎已经制定了向这一有害的潮流功能说不的计划。

不过这项政策并非一刀切地彻底禁用 OEM 厂商默认提供的所有美颜功能，而是希望相机应用默认禁用那些可能造成“扭曲”的滤镜。

原文如下：

[C-0-12] 对于任何 android.hardware.camera2.CameraDevice 照相设备或 Android 硬件，必须确保相机 API 不会改变被拍摄者的面部外观，包括但不限于变更面部的几何形状、面容肤色、以及皮肤的平滑度。

鉴于大多数消费者不大可能对相机的默认设置进行任何修改，且连苹果也开始在其照相应用中添加面容修饰功能，OEM 厂商和最终用户可能都会对此感到不满。

比如在升级 Android 11 之后，手机用户可能感觉设备的成像质量“有所下降”，但这其实更接近于真实的效果。
