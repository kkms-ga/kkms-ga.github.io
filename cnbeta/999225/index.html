[多图]乔思伯推出V8 ITX机箱 独特抽拉式结构 售价599元

July 05, 2020

以 CPU 散热和一体式水冷散热而享誉业内的乔思伯（JONSBO），近日推出了全新的 ITX 机箱–V8，目前已经在乔思伯官网、天猫商城店、京东上架，零售价格为 599 元。

乔思伯 V8 机箱是 Mini-ITX 主板的完美选择，提供了简约、时尚的外观设计。而且该机箱采用了独特的组件安装方式，主板托盘可以从机箱中拉出。

兼容性方面，本机箱的特点是支持 ITX 主板和 SFX 或 SFX-L 电源。支持的 SFX-L 电源长度最大为125mm，这意味着只能安装更紧凑的电源。这款机箱的特点是支持 195mm 高的 CPU 散热器和长度高达 330mm 的显卡。

该机箱非常紧凑，其长、宽、高分别为249.5毫米、290毫米，高度仅为260毫米。这种较小的尺寸很容易让 PC 放置在显示器旁边，同时几乎不占用您的桌面空间。

机箱前部设有一个 120 毫米的机箱风扇，以推动机箱内的空气流通。这款机箱最多可以安装 4 个不同的风扇，顶部可以安装 2 个 120mm 的风扇，背部可以安装一个 120mm 或 140mm 的风扇。顶部的特色是最多可以安装两个 120mm 风扇，背部的特色是支持一个 120mm 或 140mm 机箱风扇。

机箱正面采用了铝制拉丝设计，虽然前置IO可能有限，但较小的IO让机箱的设计几乎不受影响。前置IO只有一个USB 3.1、一个USB Type-C接口和一个音频接口。
