[图]谷歌优化Chrome Omnibox自动完成功能 使其更加智能

August 25, 2020

为了让 Chrome 用户更方便地导航到自己想要的网址，谷歌正计划对浏览器 Omnibox 中的自动完成功能进行优化，使其变得更加智能。谷歌将全新的自动完成体验称之为“Rich autocompletion”（富自动完成）。

目前，谷歌 Chrome 浏览器用户只需要输入 URL 或者某个标题的开头，就能自动完成 URL 的剩余部分。换句话说，你只有在记住标题或者 URL 的某个部分的情况下自动完成才能生效，而它本身并不能给你提供建议。

好消息是，现在自动完成将不再局限于用户输入 URL 的某个关键词。如果用户省略了URL的开头，或者包括标题中的部分内容，那么现在就会自动完成建议，这要归功于新的 “富自动完成"功能。谷歌还对 Omnibox 进行了另一项改动。Chrome 浏览器 Omnibox 现在将显示两行来同时显示建议的标题和URL。

目前所有频道版本均可以使用全新的自动完成功能。要想在 Chrome Omnibox 中体验新的自动完成体验，你需要打开一个名为 “Omnibox Autocomplete Titles"的标志。

但如果你希望对该功能进行更多精细化控制，你可以使用 Chrome Canary版本，因为它提供了额外的实验 Flag，包括 “Omnibox Rich Autocompletion”、“Omnibox Rich Autocompletion Min characters”、“Omnibox Rich Autocompletion Show Additional Text”。
