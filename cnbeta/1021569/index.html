[图]Edge 85发布：优化集锦功能 增强PDF处理

August 28, 2020

继数日前 Chrome 85 发布之后，基于 Chromium 的 Edge 85（Build 85.0.564.41）版本也于今天正式发布。Edge 85 引入了诸多新功能，除了面向消费者之外，还面向开发者优化了开发工具和其他安全改进。

更新日志：Edge 85

Edge 85 改进之一是能够在Azure Active Directory (AAD)配置文件之间同步浏览器数据，如扩展名、密码、收藏夹和收藏。该功能针对的是正在为返校季做准备的教育客户。不过遗憾的是，浏览器缺席的功能是同步历史记录和标签的能力，相信会在未来的版本更迭中引入。

Edge 85 还带来了新的 PDF 高亮工具，可以让用户在PDF上进行高亮和书写，这是从经典版 Edge 迁移过来的新功能。集锦功能也可以直接发送 OneNote、Excel 和 Word。

在安全改进方面，该版本包含了最新Chromium项目的安全修复。最后，DevTools的改进包括对Surface Duo仿真的支持和与VS Code相匹配的键盘快捷键，这两点已经在Edge 84的说明中列出。还有一些实验性的功能，比如CSS网格调试功能、在面板之间移动工具的能力等等。
