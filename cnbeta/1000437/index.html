[图]Android端Edge现也可访问实验Flag功能了

July 08, 2020

和桌面端一样，适用于 Android 平台的 Edge 浏览器同样是基于 Chromium 浏览器。桌面端和移动端新版 Edge 共享诸多特性和功能，不过 Android 端此前缺少一项至关重要的特性–对实验 Flag 的支持。现在，这个不足已经被弥补了。

如果想要在桌面端 Edge 浏览器启用实验性质的某些特性，用户只需要在 URL 地址栏上输入 edge://flags 就能访问。不过，此前在 Android 智能手机上是无法访问这个页面的。

在最新的发布的 Android 端 Edge Beta v45.06.24.5042 版本更新中，终于添加了对实验 Flag 功能的支持。这意味着，你现在可以通过在URL栏中输入 edge://flags 来开启 Android 版 Edge 的实验性功能，就像你在 Edge 桌面浏览器上的操作方式一样。
