Windows Hello已可用于谷歌Chrome浏览器中的支付认证

May 22, 2020

早在 2018 年的时候，微软就已经为 Edge 浏览器引入了 Web 身份验证规范，从而为基于 Windows Hello 的无密码登陆方案铺平了道路。当年晚些时候，微软跟进了使用 FIDO2 密钥登录自家帐户的能力。现在，谷歌也做好了在 Chrome 浏览器中支持 Windows Hello 支付认证的准备。

即便此前该公司已支持 Web Authentication API，但新功能允许用户通过 Windows Hello 来便捷支付。

多年来，人们已经习惯了手动输入信用卡背面的 CVC 码来完成支付。但在启用了 Windows Hello 认证支付功能之后，用户便可轻松地在 Chrome 浏览器中实现刷脸或指纹来验证支付。

如果你需要经常网购，那不妨直接转到 Google Chrome 的设置页面，然后在“付款方式”中查找“Windows Hello”选项、选择开启或关闭，之后便可以在支付的时候点击使用。
