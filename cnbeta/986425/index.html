[图]硬件加速 GPU 调度选项即将登陆 Windows 10 平台

June 03, 2020

Windows 10 May 2020（20H1/Version 2004）在游戏方面引入了诸多改进，其中就包括了对 WDDM 2.7 的支持，不过遗憾的是其中并没有包含启用硬件加速 GPU 调度的选项。硬件加速 GPU 调度是 WDDM 2.7 的主要功能，它允许集成/独立显卡直接管理自己的显存，从而让你获得更好的性能，降低延迟，提高平均或最低帧率，改善视频播放效果。

在英特尔和NVIDIA发布的最新驱动中，都包含了对 WDDM 2.7 的支持，但问题是在 Windows 设置中并没有硬件加速 GPU 调度的选项。

NVIDIA 表示在 Windows 10 May 2020 中已经获得支持，但是最新的 Game Ready 446.14 WHQL 驱动并不基于 WDDM 2.7，这也是为何没有硬件加速 GPU 调度的原因。

而英特尔方面并就没有就硬件加速 GPU 调度提供任何沟通，AMD 和 NVIDIA 已经确认会在未来的版本更新中提供支持。
