[图]仍无法升级Windows 10 Version 2004？试试禁用网络吧

July 31, 2020

在5月27日发布之后，已经有很多用户升级到 Windows 10 May 2020（20H1/Version 2004）功能更新了。但部分设备尽管已经解除了升级限制，但是至今仍无法获得升级，其中就包括微软自家的 Surface Pro X。

Surface Pro X 和其他用户反馈称，当他们尝试安装 Windows 10 Version 2004 功能更新的时候，会跳出：“当前设备无法升级到 Windows 10”的错误提醒。不过由于微软没有提供更详细的说明，因此尚不清楚具体是什么原因阻止了升级。

幸运的是部分用户发现，在 Update Assistant 下载 Windows 10 Version 2004 功能更新之后禁用网络连接，就能绕过神秘的升级阻止。

在外媒 Windows Lastest 的测试中还发现，阻止升级 Windows 10 Version 2004 的原因还包括 Synaptics 触摸板驱动程序、OEM 应用程序或使用 OEM 定制的显示器驱动程序等等。

如果你使用更新助手工具，并收到一条错误信息，说明 “此电脑无法升级到Windows 10”，那么在初始下载后，你就可以通过禁用互联网连接来半强制安装。

不过，有用户报告说，在应用 Version 2004 更新后，他们的无线键盘、鼠标，甚至Realtek音频适配器都出现了问题。一位用户指出：“我的一台戴尔笔记本电脑也有同样的问题。自从升级到2004年版本后，无论是内置键盘还是触摸板鼠标都无法工作”。一些用户能够通过重新安装驱动程序来解决这个问题，而其他用户则回滚到 Windows 10 Version 1909。
