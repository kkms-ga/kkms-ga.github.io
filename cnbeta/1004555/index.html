[图]不只是移动端 微软Project xCloud目标是登陆任意屏幕设备

July 17, 2020

微软的云游戏服务 Project xCloud 已经确认将会在今年 9 月正式发布，将会包括在 Xbox Game Pass Ultimate 订阅服务中。近日在接受外媒 The Verge 采访时候，Xbox 掌舵者 Phil Spencer 表示 Project xCloud 的商业模式和平台规模将会进一步拓展。

虽然 Project xCloud 主要在移动设备上首发，不过最终目标是“未来任何人可以通过任意屏幕访问流媒体游戏”，提供可用性和流畅的游戏体验。Spencer 表示

随着时间的推移，我们将继续拓展流媒体游戏规模，能够让你畅玩订阅部分之外的游戏。该服务将会整合到 Ultimate 订阅中，用户不需要支付额外的费用。无论对于我们，还是玩了很多游戏的玩家来说，这都是一个非常好的开端。

我们的目标是将 Project xCloud 引入到任意屏幕设备上。现阶段该服务仅限于移动设备。不过我们正在积极讨论，解决一些问题。我们会通过8月和9月的发布会更具体地谈论哪些移动设备。

我希望游戏的试用就像今天的音乐和视频一样简单，我今天可以在 Spotify 里给你发一首曲子，你就可以立即听到了。随着时间的推移，在任何地方一个游戏，都能进入游戏。

另外值得一提的是，Project xCloud 并不是微软云游戏服务的最终名称，在 9 月正式发布的时候应该会公布新的名称。
