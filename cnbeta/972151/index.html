黑暗模式很酷炫 但Windows 10的浅色主题要养眼得多

April 27, 2020

作为一款不断发展中的操作系统，Windows 10 在视觉效果上的变化却有些缓慢。比如许多人期待已久的黑暗模式，就是千呼万唤始出来。与此同时，用户也希望微软能够对 Windows 10 的浅色主题带来更彻底的更新，其中一些想法已经接近了苹果的 macOS 。

访问:

微软中国官方商城 - Windows

随着黑暗模式从移动设备火到 PC 端，几乎每款热门应用的开发商都已经提供了对黑暗模式的支持。

为了让系统中的各个元素能够实现更高的一致性，微软也对 Windows 10 的全局黑暗模式，在视觉样式上引入了更多的一致性。

对于经常熬夜、或者习惯了在暗处工作的人们来说，黑暗模式在酷炫之余，也提供了一些切实的帮助。

不过在 2019 年 5 月更新（Version 1903）中，微软也为 Windows 10 引入了许多人喜欢的全新浅色系主题。

现在，随着 20H1 更新（Version 2004）即将向正式版用户推送，Windows 10 的浅色系主题也有望看起来更加养眼。

上述屏幕截图展示了基于全新浅色系主题的快速设置选项，可知其中一些是默认启用的。

结合透明和其它视觉效果（比如强调色和新图标），Windows 10 Version 2004 的浅色系主题，总算是为广大用户带来了期盼已久的 Modern UI 改进。

虽然黑暗模式很适合在夜间关了灯使用，但在旁边有一盏台灯的情况下，浅色主题显然也更加养眼。

微软最近为 Windows 10 发布的新图标，也是浅色系主题更新的一个重要组成部分。

至于“开始”菜单，动态瓷贴（Live Tiles）已经被桌面用户嫌弃很久，甚至许多时候都是毫无意义的存在。无论微软再怎么改进，其特性还是更适合平板电脑或展示等应用。

最后，我很欣喜地看到 Windows 10X 开发分支的最新进展，其似乎生动地证明了微软正在朝着正确的方向去发展。
