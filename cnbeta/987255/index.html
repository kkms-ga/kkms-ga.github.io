亚马逊宣布与通讯公司Slack合作 后者将采用AWS视频会议技术

June 05, 2020

北京时间6月5日早间消息，据外媒报道，亚马逊与通讯软件公司Slack正在深化合作。在周四宣布的新的合作协议基础上，所有亚马逊员工可以使用Slack的职场通讯工具。同时，Slack也深度挖掘亚马逊网络服务（AWS，Amazon Web Services）技术，更新视频会议功能，充分利用被称为Chime的AWS服务。

随着新冠疫情期间很多公司和学校寻求软件召开远程会议，近几个月来，与Slack相比，微软公司的团队通讯服务已大受欢迎。而亚马逊与Slack的合作联盟可以让两家公司与微软一较高下。

虽然很多公司员工居家办公，但Slack公司在疫情期间的业务增长保持稳定，并没有增速。周四早些时候Slack透露了公司第一季度财报，公司盘后股价下跌了15%。

亚马逊与Slack的往来由来已久。自亚马逊2014年推出亚马逊网络服务（AWS）以来，Slack公司就在其平台运行，而且在疫情期间一直依靠AWS满足其他需求。

Slack表示，双方新协议独立于Slack的云服务合同，但是拒绝透露具体条款。周四，一份文件透露，Slack将为AWS付费至少4.25亿美元，合作期为5年，截至2025年4月，较之前承诺的合作期至2023年、金额至少2.5亿美元有所增加。

亚马逊的部分团队已经开始使用Slack，但是合作协议将包含所有员工。亚马逊是美国第二大私营部门雇主，第一季度全职员工和兼职员工超84万人。
