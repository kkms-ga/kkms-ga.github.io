虚幻引擎推出借助Face ID捕获3D动画的《Live Link Face》应用

July 10, 2020

Unreal Engine 刚刚为 iPhone 用户推出了一款名叫《Live Link Face》的应用，特点是能够充分利用苹果智能手机上的 Face ID 和运动协处理器来捕获 3D 动画。如此一来，配备 Face ID 功能的 iPhone 就可以实现高预算的电影风格效果，记录实时的 3D 角色动画或面部追踪数据以备后续使用。

据悉，Unreal Engine 在周四上午发布了全新的《Live Link Face》应用，并夸耀了 Face ID、TrueDepth 摄像头和 ARKit 在 3D 动画捕捉中的实用性，比如捕获现场动作和为数字花生制作实时动画。

借助 iPhone 的运动协处理器，《Live Link Face》还可根据使用方式进行智能调整。比如在对准头部的时候，它就只会将面部动作记录到数据中。若放在桌子上，又可用于追踪头部和颈部动作。

据悉，《Live Link Face》是为专业的协作环境而构建，允许 App 与采用 Unreal Engine 的作品编辑器联动。通过蓝牙对接舞台时钟，还可确保时间码与录音完全同步。

目前这款《Live Link Face》应用已可通 App Store 中免费获取，而且不包含应用内购买的选项，但还需需要与 Unreal Engine 的专业生产力软件配套使用，才能充分利用期间采集的 3D 动画数据。

Unreal Engine 以推动跨平台游戏的开发而被人们所熟知，开发者可借此为 iOS、macOS 和游戏主机平台进行创作，甚至被迪士尼用于《曼达洛人》等数字电影的媒体创作。

Verizon Media 动画技术总监 Addy Ghani 亦对《Live Link Face》赞赏有加，称之可借助 iPhone ARKit 实现高质量的面部动画捕捉，让手机轻松变身为一款简化的生产工具，很适合专业工作室或小作坊使用。
