但迁移到 Windows 10 并不是轻松简单的事情。例如对于公司而言，升级至 Windows 10 系统还会涉及到其他成本，例如硬件升级、应用兼容性等等。而对于普通消费者来说，升级过程虽然不用考虑这么多，但依然存在一些其他问题。

但无论是企业还是普通消费者，现在升级 Windows 10 都需要购买许可证。尽管微软已经实现了装机量超 10 亿的目标，但让更多人使用 Windows 10 系统的门槛之一就是价格。

目前在美国，Windows 10的许可售价为

Windows 10家庭版–139美元

Windows 10专业版– 199.99美元

Windows 10 工作站专业版 – 309美元

在 Feedback Hub 的一则帖子中，有用户推荐微软降低 Windows 10 的售价，这样才能让更多的人使用该系统。在帖子中写道

无论是购买整机还是自己装机，人们已经在电脑上花费了太多的钱了。大部分消费者会选择仅下载 Windows 而不去激活它。你们已经从中赚取了足够多的钱，那么现在依然采取如此高昂的售价还有什么意义？

你通过这种方式赚取的钱财难道不够金钱浴吗？如果你不希望免费，那么请将Windows 10家庭版的定价设置为10美元，专业版定价设置为35美元，最终反而你可能会赚取更多的钱，因为实际上很多像我这样的人会购买它。希望微软将操作系统的价格设定在每个人都能承担的范围内。

Windows 7/8.1 免费升级 Windows 10 的一年有效期虽然在2016已经结束，但仍然有一种简便的方法可以免费获得正版许可证。

该技术可以在用户当前使用正版Windows 7或Windows 8.1 PC的前提下使用。如果是这种情况，则可以通过下载Microsoft工具并在设备上运行它来升级到Windows 10。在继续执行本指南之前，请确保备份所有现有文件。

首先从Microsoft的“下载Windows 10”网页下载Windows 10 Media Creation Tool。这将为您提供执行升级所需的一切。下载后运行程序。

片刻之后，您会看到“您想做什么？” 页面。选择“立即升级此PC”，然后按“下一步”继续升级。Windows 10现在将安装在您的PC上-这将需要重新启动几次，并且可能需要几个小时才能完成。

最终，您的设备将显示Windows 10桌面。您可以通过单击开始>设置>更新和安全性>激活来确认Windows 10已激活。
