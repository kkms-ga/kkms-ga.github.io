《微软飞行模拟》发布和下载时间表公布 各地区不尽相同

August 17, 2020

上个月，微软宣布新版《飞行模拟》终于将从8月18日开始在PC上为用户提供服务。这款模拟游戏最新版本在去年E3期间公布，并在2020年7月13日开始预购。现在，微软已经提供了各个地区的发布时间，为用户提供了何时能够开始下载和玩这款游戏的信息。

那些从微软商店预购了游戏的用户将可以按照既定发布时间表开始下载，这意味着可用性将根据地区和时区的不同而有所不同。游戏将于午夜时分在全球25个时区上线，新西兰等国家的用户已经开始有资格下载，中国用户从18日零点开始有资格下载。

那些选择从Steam购买游戏的玩家将能够在UTC凌晨4点（格林尼治标准时间4点，美国东部时间12点，中部时间11点（8月17日），太平洋时间9点（8月17日）获得完整下载。这意味着，部分地区将能在午夜之前下载《微软飞行模拟》。

虽然公告中没有明确提到Xbox Game Pass Ultimate或Xbox Game Pass for PC用户的时间表，但可以肯定的是，商店购买下载游戏的时间表也适用于订阅者，因为订阅中包含标准版。此外，微软没有公布这款游戏在游戏机上推出的时间表。

该游戏提供了三个版本。标准版（59.99美元）、豪华版（89.99美元）和高级豪华版（119.99美元）。版本之间的差异包括更多的机场和飞机。微软的最新产品利用了DirectX 12的技术，提供了实时天气和动态场景。
