三星Galaxy Note 20摄像头阵列模块曝光 透露拍照关键细节

May 21, 2020

今天，三星Galaxy Note 20摄像头阵列模块相关信息泄露，透露了该手机拍照能力的一些关键细节。根据这次泄露，Galaxy Note 20 其背部配备一个看起来像是潜望镜式长焦镜头。泄露的摄像头阵列似乎还包括一对至少两个其他摄像头，LED闪光灯，一个（可能是几个）麦克风孔，以及一个传感器，可能会提供更快更准确的对焦。

这里的传感器很可能是激光雷达或ToF（飞行时间）传感器，这将使设备能够更快地对焦，并轻松使用高级增强现实功能。据悉，Galaxy Note 20将抛弃Galaxy S20系列中现有的100倍变焦功能，并将为该系列带来成功的108MP HM1相机传感器。

据悉，Galaxy Note 20很可能会有两种尺寸，其中至少有一种会带来一块6.9英寸的显示屏，分辨率为3200×1440像素。另一款Galaxy Note 20很可能会有一块6.42英寸的显示屏，如果传闻被证明是准确的，那么该系列将在今年9月以Galaxy Note 20和Galaxy Note 20+的形式出现。

 基础版机型被推测将配备12GB RAM和128GB内部存储。这款设备将搭载当时最新的Android系统，很可能是Android 10和三星One UI 2.x用户界面。上周有消息称，Galaxy Note 20将搭载高通公司的新指纹传感技术。那将是高通的第二代3D Sonic Max指纹识别技术，采用大尺寸的超声波技术，可以同时进行一到两个手指读取。

三星Galaxy Note 20很可能会与三星Galaxy Fold 2同时在2020年8月的活动中正式发布。
