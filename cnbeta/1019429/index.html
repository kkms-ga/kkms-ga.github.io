[图]继Android平台后，Chrome桌面端也获密码检查功能

August 24, 2020

针对 Android 和桌面端 Chrome 的密码检查功能，谷歌在近日的版本更迭中对其进行了大量改进。继此前 Android 端 Chrome 允许检查已泄漏的密码，并让你查看、编辑和复制这些密码之后，Chromium 团队向桌面端 Chrome 也添加了这项功能。

Chrome 浏览器此前已经获得密码泄漏保护功能，当保存在 Chrome 浏览器的密码泄漏之后就会向用户发出警报。除此之外，你还可以手动访问 chrome://settings/passwords 页面，利用检查密码选项来了解你的密码是否被泄露。

Chromium团队今天在 chrome://settings/passwords/check 页面的标志后面增加了密码强弱检查功能。想要测试该功能，可以按照以下步骤

1.启动最新的 Chrome Canary 版本

2.访问 chrome://flags 页面

3.搜索 “Passwords weakness check”

4.从下拉菜单中选择 “已启用”，然后重新启动浏览器。

之后，前往chrome://settings/passwords页面，点击 “检查密码”。然后，Chrome浏览器不仅会检查是否有密码泄露，还会检查你保存的密码是否被泄漏，如果有发现，可能会发出警告。
