AMD正式发布线程撕裂者PRO：一颗64核心秒杀两颗28核心

July 15, 2020

AMD刚刚正式发布了全新的锐龙线程撕裂者PRO(Ryzen ThreadRipper PRO)系列处理器，专门面向专业工作站，拥有无与伦比的超多核心/线程优势，以及不俗的频率，非常适合艺术家、建筑师、工程师、数据科学家等专业人士，由联想首发。

线程撕裂者PRO系列继承了桌面版线程撕裂者的基本属性，比如7nm工艺制造，Zen 2微架构，最多64核心128线程、288MB缓存，PCIe 4.0总线通道从64条开放到完整的128条，内存支持从四通道DDR4-3200 256GB UDIMM开放到完整的八通道DDR4-3200 2TB UDIMM/RDIMM/LPRDIMM/3DS DIMM，并支持AMD PRO技术，热设计功耗则维持在280W。

这是世界上第一颗64核心工作站处理器，也是第一颗主频突破4GHz的12核心工作站处理器，还是首个支持PCIe 4.0的工作站处理器，无论单线程还是多线程性能都拥有突出的表现。

线程撕裂者PRO系列包括四款不同型号：

－ 3995WX：

64核心128线程，主频2.7-4.2GHz，二级缓存32MB，三级缓存256MB。相比3990X低了200/100MHz。

－ 3975WX：

32核心64线程，主频3.5-4.2GHz，二级缓存16MB，三级缓存128MB。相比3970X低了200/300MHz。

－ 3955WX：

16核心32线程，主频3.9-4.3GHz，二级缓存8MB，三级缓存64MB。

－ 3945WX：

12核心24线程，主频4.0-4.3GHz，二级缓存6MB，三级缓存64MB。

专业技术方面，Intel有大名鼎鼎的vPRO博锐，AMD PRO也不甘示弱，包含四个方面：

－ AMD PRO安全性：内置多层安全功能，帮助保护敏感数据。

－ AMD PRO可管理性：简化与当前基础设施兼容的部署、映像和管理。

－ AMD PRO业务可靠性：18个月的软件稳定性支持，24个月的产品持续供货支持。

－ AMD Memory Guard：内存全面加密，有助于防止对敏感数据的物理攻击。

AMD强调，相比于竞品，线程撕裂者PRO系列只有单独一个平台、四款型号，整个平台的特性保持一致，Intel至强则有至强可扩展系列、至强W-2200系列、至强W-3200系列多个复杂的产品线，各种型号超过85款，而且不同型号的特性差别非常大。

AMD也精心为线程撕裂者PRO系列挑选了各自的对手，比如3995WX对标的是Intel最顶级的至强铂金8280，2.7-4.0GHz/205W，而且敢于一颗对两颗，因为后者只有28核心56线程，双路也不过56核心112线程。

3975WX对标至强W-3275，28核心56线程，2.5-4.4GHz，205W。

3955WX对标至强W-2295，18核心36线程，3.0-4.6GHz，165W。

3945WX对标至强W-3235，同为12核心24线程，3.3-4.4GHz，180W。

除了核心数量上的碾压，线程撕裂者PRO系列在扩展性上也更胜一筹，比如至强W系列最多仅支持64条PCIe 3.0，而内存最多仅支持六通道DDR4-2933 1TB，也不支持3DS RDIMM。

至于性能，AMD给出的数据自然是各种超越和秒杀，比如一颗3995WX对两颗8280都能在至少25个项目中取胜，综合领先幅度达27％，图形性能领先最多37％，对一颗至强W-3275更是综合领先136％。

同样是12核心24线程，3945WX对W-3235单线程性能可领先12％，多线程性能则可领先28％。

联想ThinkStation P620工作站全球首发AMD锐龙线程撕裂者PRO系列，今秋上市。

它采用单路设计，最高可选64核心的PRO 3995WX，还有八条内存(最大容量1TB)、四条PCIe 4.0 x16和两条PCIe 4.0 x8扩展插槽(可选最多四块Quadro RTX4000或两块Quadro RTX8000)、四块硬盘(最大容量20TB)、两个USB-C 10Gbps和六个USB-A 10Gbps接口。
