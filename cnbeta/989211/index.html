[图]IBM Cloud出现全球性宕机 目前已经修复

June 10, 2020

IBM Cloud 今天遭遇了重大宕机故障，平台上托管的多项服务也因此中断，其中就包括知名科技新闻聚合网站 Techmeme。本次宕机事件从太平洋时间6月9日下午2点30分左右开始，并快速蔓延至全球。

6·18活动已全面开启 大促活动入口汇总：

阿里云6·18上云年中大促 点击领取最高12000元红包

天翼云高性能云主机1折起 低至74元/年

2020天猫6·18超级红包在此领取 6月1日追加40亿元消费券

京东6·18十七周年庆大促主会场入口 - 最高可领618元红包

目前 IBM 自家的状态页面已经无法加载，并返回了一个内部服务器错误，目前尚不清楚本次宕机事件的受影响范围，也不清楚是什么原因所导致的。

在宕机事件发生之后，IBM Cloud 官方推特并没有更新推文，不过外媒发现了一个托管在第三方服务器上的IBM Aspera 的状态页面，这似乎证实了这很可能是一个全球性的网络问题。

更新：IBM Cloud 官方推特在1小时前发布推文，表示本次宕机事件已经得到全面修复。
