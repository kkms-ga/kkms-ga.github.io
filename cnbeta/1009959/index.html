Memblaze发布PBlaze5 52X系列低功耗企业级SSD新品

July 31, 2020

Memblaze 刚刚推出了 PBlaze5 520 系列低功耗企业级 SSD 新品，在致力于提供出色性能的同时，还可保持较低的功耗水平。主打存储效能的有 PBlaze C520 / D520 / C526 / D526 几个产品线，并且每种都有 2.5 英寸 U.2 或 HHHL 的外形选项、以及两档的容量。

（来自：Memblaze）

与去年更彪悍的 C900 系列不同，52X 系列仍在使用 PCIe 3.0 x4 接口，顺序读取的峰值性能为 3.3 GB/s 。

其中 PBlaze C520 型号可选 1.92 TB / 3.84 TB 两个版本，而 PBlaze C526 型号可选 1.6 TB /  3.2TB 两个版本。

所有型号的随机读取性能均为 520k IOPS，但 PBlaze C520 型号的随机写入性能为 68k / 70k IOPS（对应两种容量）、PBlaze D526 / C526 则是 135k IOPS 。

此外与去年 900 系列采用的 32 / 64 层企业级 3D TLC NAND 相比，今年的 PBlaze 52X 系列已经全系升级到了更新更节能的 96 层闪存。

最终，PBlaze 52X 系列企业级 SSD 的闲置功耗从 7W 降到了 4W，工作状态下从 25W 降到了 9W，峰值状态下也只有 12W 。

遗憾的是，作为面向企业客户的产品，Memblaze 未公开 PBlaze 52X 系列 SSD 的确切价格，因其取决于客户订单所涉及的确切配置和数量。
