[图]Windows 10 May 2020已无法使用Fresh Start 微软或有意删除该功能

June 14, 2020

自2017年的 Windows 10 创作者更新开始，微软就在 Windows Security 中引入了 Fresh Start 功能，允许用户在保持数据完整性的情况下重新安装 Windows 10 系统。重要的是这项功能并不会尝试保留用户现有的应用程序，为用户提供“Fresh Start ”，就像是首次开机一样。不过在 Windows 10 May 2020 功能更新中，导致这项功能无法正常使用，移除了“Get Started”按钮。
