Pixel 4a基准测试成绩曝光 骁龙730表现亮眼

May 14, 2020

尽管 Pixel 4 / 4XL 旗舰机的表现让人感到有些失望，但中端入门的 Pixel 3a / 3a XL 还是给谷歌赚回了一些面子。今年的 Pixel 4a 系列，显然也会延续这一趋势。近日曝光的基准测试信息表明，Pixel 4a 是一款采用高通骁龙 730 平台的产品。虽然比不上骁龙 765 和 768 5G SoC，但还是较 Pixel 3a 的骁龙 670 有了很大的进步。

由 TecnoLike Plus 的 Julio Lusson 向 XDA 分享的跑分数据可知，即便骁龙 730 的定位低于骁龙 800 系列，但设备的实际体验未比 Pixel 4 上的骁龙 855 差多少，某些情况下甚至超越了 Pixel 3 搭载的骁龙 845 芯片组。

当然，这并不是说 Pixel 4a 的性能可与高端旗舰产品匹敌，而是暗示该机的最终定价可能相当平易近人。

相比 Pixel 4 的高昂售价、以及 Project Soli 传感器等问题，Pixel 4a 显然有望吸引更多的用户去选购。
