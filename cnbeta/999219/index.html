[图]Eurocom推MXM-to-PCIe x16适配器 移动GPU秒变桌面PCI-e显卡

July 05, 2020

Eurocom Riser 卡本质上一个反向的 eGPU 扩展坞，它允许用户将笔记本上的 MXM3 显卡直接安装到桌面端的 PCIe 插槽上。该产品是面向专业人士和工作站环境的专业 SFF 主机，而非家庭办公。

就像通过 USB-A 转 USB-C 适配器，允许 USB-A 设备连接到 USB-C 端口上一样，MXM 转 PCIe 适配器能够让 MXM 图形显卡连接到大多数桌面 PC 主板的 PCIe 插槽上。

Eurocom将这些适配器营销为 “MXM3转PCIe Riser Cards”，能够支持移动 MXM3 Radeon RX 580 或 GeForce RTX 2080 显卡，最高 TDP 为 150W。在散热方面提供了一个小型风扇和散热器，因为 MXM 卡原有的笔记本散热方案显然无法沿用。

虽然概念不错，但这并非是面向普通用户的，而是面向专业用户的，产品售价在 500 美元到 1300 美元不等。由于入门价格较高，而且首先MXM显卡的数量有限，受众面会相当小。比如大多数新式游戏笔记本都会采用焊接的 GeForce GPU 而不是可拆卸的 MXM GPU 来减轻重量和厚度。不过，至少对于一小部分能够利用 MXM-to-PCIe 适配器的用户来说，这个选择还是存在的。
