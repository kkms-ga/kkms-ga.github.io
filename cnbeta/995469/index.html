ARM Mac将无法通过Boot Camp支持Windows

June 25, 2020

苹果将于今年晚些时候开始将 Mac 切换到自研的 ARM 处理器平台。但是对于不习惯原厂 macOS 操作系统的用户来说，这里有个坏消息，因为你将无法通过 Boot Camp 模式去安装运行微软 Windows 操作系统。微软发言人在致 TheVerge 的一份声明中称，该公司只会向 OEM 厂商提供 ARM 版 Windows 10 操作系统的授权。

近年来，苹果与微软之间的合作算是相当密切的，比如为今年晚些时候的 ARM Mac 提供 Office 生产力套件的支持。

可惜在 WWDC 2020 期间，双方都对基于 Boot Camp 的 Windows 安装部署避而不谈。

在被问及“将来是否会修改策略，以允许在 ARM Mac 上通过 Boot Camp 进行部署”时，该公司表示目前没有其它信息可向大家分享。

如此看来，想要在换用 Appel Silicon 的 Mac 上运行 Windows 10 操作系统，只能等待微软进一步开放 Windows 10 for ARM 的广泛授权了。

当然，用户并不是完全无法在 ARM Mac 上体验 Windows，毕竟苹果官方已经演示过虚拟机方案（在 Parallels Desktop 中运行 Linux），因此基于 VMWare 和 Parallels 等虚拟化软件的解决方案还是很有希望的。
