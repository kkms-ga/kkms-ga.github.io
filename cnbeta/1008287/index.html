三星向印度投放Galaxy M01 Core入门新机 起售价5499卢比

July 27, 2020

三星刚刚向印度市场投放了一款 Android Go 入门新机，它就是采用了塑料机身、四核 SoC、8MP 后置单摄、内置 3000 mAh 的 Galaxy M01 Core 。软件方面，该机采用了定制的 One UI，并且提供了集成的黑暗模式支持。售价方面，1GB RAM + 16GB ROM 版本为 5499 卢比（约 514 RMB），2GB RAM + 32GB ROM 版本则是 6499 卢比（约 608 RMB）。

三星为 Galaxy M01 Core 配备了 5.3 英寸 HD+“智能”显示屏，主打动态节能特性。处理器为联发科 6739 四核芯片组，辅以最高 2GB RAM + 64GB ROM 存储空间。

拍照方面，该机采用了前置 5MP、后置 8MP + LED 闪光灯的组合。内置电池容量为 3000 mAh，预装基于 Android Go 定制的 One UI 界面 + 一系列 Android Go 定制应用。

最后，Galaxy M01 Core 将于 7 月 29 日在三星官网、零售门店和主力电商平台上销售。

可选黑、蓝、红配色，低配 / 高配机型的售价分别为 5499 / 6499 卢比（约 514 / 608 RMB）。
