小米全新系统MIUI 12发布：挑战iOS、22款机型首发升级

April 27, 2020

4月27日，小米正式推出了MIUI 12，这个被雷军称为“惊艳”的全新OS，从底层技术开始自研重构，号称要挑战最强的iOS，从动画到引擎再到图标等元素焕然一新，目前已有22款机型开始内测升级。

访问:

小米有品

在MIUI 12上，小米的目标是挑战最强动画，也就是iOS，为此小米从底层技术开始自研重构，完成了一套焕然一新、极其先进的技术架构。小米将这套包含动画引擎、渲染引擎、插画引擎的技术架构，命名为“米柚光锥动效架构”。

MIUI 12米柚光锥架构中，小米开发的全新渲染引擎、物理引擎、分层动态图标、动态窗体、动态天气等设技术和设计，在全新架构下，核心场景动画焕然一新，首次跟iOS不相上下。

UI方面，MIUI 12带来了超级壁纸功能，打通息屏、锁屏、桌面壁纸，小米重建了真实的火星、地球等模型，将它们做成了动态壁纸，开启深色模式后，星球也会进入夜色，视觉体验更完美。

安全隐私方面，针对目前日益严重的信息泄露，MIUI过去10年做了大量领先行业的工作，获得多家权威隐私认证。这一次MIUI 12又引入了照明弹、拦截网等功能，前者可以将应用行为忠实记录下来，向用户报道一切切敏感行为。

此外，当应用使用相机、录音、定位权限时，系统会给出明显提示，点击提示即可管控。

拦截网功能则进一步规范了APP权限，完全禁止后台使用相机、后台唤醒。应用获取权限时，可以选择仅在使用中允许或本次允许。

在安全方面，MIUI 12是全球首家通过德国莱茵TÜV《安卓系统增强隐私保护测试》的手机操作系统，全面守护用户的数据安全。

在多任务方面，MIUI 12支持了全局自由窗口，符合手势操作直觉，适应多种使用场景。来消息时轻轻下滑即可展开进入小窗模式，上滑即可退出。

对于只需要查看状态的应用，拖至屏幕角落即可悬挂。MIUI全局自由窗口，可能是全球最优的手机多任务解决方案。

AI方面，MIUI 12的AI电话助理正式升级为AI通话，用户可以自定义开场语、自由切换自动应答或手动回复，还可以定向自动接听，再也不用在意骚扰电话，同时还能AI拨打电话。

在健康方面，MIUI还带来了全新的功能，托自研灵弦算法，AI智能感知运动状态。手机揣兜里，即可自动记录一整天的步行、跑步、骑行和爬楼梯。更重要的是，一整天耗电量小于1%！

据悉，MIUI 12已经开启内测申请。目前可申请MIUI 12内测的机型有Redmi K30、Redmi K20、小米CC9 Pro、Redmi Note 7 Pro、小米9 SE、小米CC9、小米CC9美图定制版、Redmi Note 8 Pro、Redmi Note 7、小米8、小米8屏幕指纹版、小米MIX 3、小米8透明探索版、小米MIX 2S、小米Note 3、小米Max 3、小米8青春版、小米MIX 2、Redmi Note 5、小米6X、小米8 SE、小米6等，总计22款机型。

MIUI 12申请内测之后，今晚18点就陆续OTA升级。

最后，小米还推出了一个MIUI的菜单，将过去10年的经典主题高清复刻，米粉可以重新使用过去MIUI中备受好评的主题了。
