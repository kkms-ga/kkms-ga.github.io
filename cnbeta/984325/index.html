AMD发布Radeon 20.5.1驱动更新 支持Windows 10 2020五月更新

May 28, 2020

等待了一个月多之后，AMD 终于在今日发布了 Radeon Software Adrenalin 2020 软件更新。版本号为 20.5.1，主要提供了对 Windows 10 2020 五月更新（即 Version 2004）的支持，以及其它改进和修复。然而据 Reddit 网友所述，虽然新版驱动引入了对 WDDM 2.7 的支持，但却失去了基于硬件的调度功能。

访问购买页面:

AMD旗舰店

与英伟达那边一样，AMD 未在发行说明中提及针对 Windows 10 2004 的详细改进。与此同时，20.5.1 版 Radeon Software 升级了对 AMD Link Xinput Emulation 的支持，有望改善当前和未来游戏的兼容性。

下载地址：

● Radeon Software Adrenalin 2020 Edition 20.5.1（Windows 10 x64）

AMD 介绍了 Radeon Software 20.5.1 中已经修复的一些问题（完整列表请移步至官网查看）：

● 修复软件可能在混合交火的系统配置下崩溃或死机的问题。

● 修复在某些显示器上启动游戏可能导致间歇性黑屏，需要重启才能解决的问题。

● 修复 AMD Link 可能随机或间歇性断开连接的问题。

● 修复执行任务切换时，《命运 2》主程序可能卡死或黑屏的问题。

● 修复加入比赛时，《守望先锋》可能间歇性崩溃或出现黑屏的问题。

● 修复软件可能误报 Radeon RX 5700 系列 GPU 内存频率的问题。

● 修复 Radeon RX 5500 系列 GPU 可能在混合交火平台上安装驱动时可能出现蓝屏的问题。

此外 Radeon Software 20.5.1 仍有一些已知问题（以下简单摘录其中几条）：

● 运行游戏室打开性能指示器，Radeon VII / RX Vega 系列显卡可能出现性能下降。

● 在 Radeon RX 5000 系列 GPU 系统上配置 Instant Replay 后，部分游戏可能报错。

● 某些游戏和系统配置上启用增强同步，可能导致黑屏（解决方案就是暂时禁用）。
