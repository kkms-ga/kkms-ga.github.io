Android版Gboard输入法正测试面向IM应用的自动撰写功能

July 01, 2020

2018 年的时候，谷歌为 Web 版 Gmail 邮件服务推出了“智能撰写”（Smart Compose）功能，并于 2019 年扩展到了 Android 平台。现在，这家搜索巨头正在展开新的测试，以便通过 Android 版 Gboard 舒服法，将“智能撰写”功能覆盖到其它消息传递应用程序中。

借助“智能撰写”（Smart Compse）功能，Gboard 会提供更加智能的输入建议，直至完成整个句子。作为对比，普通输入法的“智能联想”功能，通常只会给出下一个单次的建议。

首次启动的时候，用户可查看有关“智能撰写”的快速解释。如果接受 Gboard 提供的建议，用户可直接在所选的文本上向右滑动。

据悉，目前谷歌正在面向部分运行 v9.5.12.317844448 版 Gboard 输入法的用户开展“智能撰写”功能的 Beta 测试。至于何时向公众全面推送，目前暂不得而知。
