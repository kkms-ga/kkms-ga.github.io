[图]Windows Virtual Desktop新功能上线：优化微软Teams体验

July 28, 2020

在今年四月宣布重大更新之后，今天微软向所有用户开放了 Windows Virtual Desktop 的新功能。其中包括 Microsoft Teams 用户可以通过全新 A/V redirect 在 Windows Virtual Desktop 上享受更好的体验、IT 管理员可以通过 Azure Portal 轻松管理虚拟桌面。

以下是本次更新的主要内容

优化管理经验–微软将管理经验深度整合到 Azure Portal 中

你能够在 Azure Portal 中设置主机池、管理应用程序或桌面，并分配用户等一系列操作。微软还通过与Azure自动化和Azure Logic Apps的集成，改善了自动扩展体验。

以前，你只能将远程应用和桌面发布给单个用户。现在，您可以将资源发布到Azure Active Directory (Azure AD)组，这使得扩展更加容易。

您可以使用 Azure RBAC 为您的 Windows 虚拟桌面资源提供更精细的访问控制。有四种内置的管理角色，您可以开始使用，必要时可以创建自定义角色。

对 Microsoft Teams的支持

微软将改善Windows虚拟桌面上的Teams体验。微软将使用 “A/V重定向 “进行视频通话。这将在共享视频时在您的用户之间创建一条直接路径，显著改善视频和音频体验。

一旦您在Windows的桌面客户端中启用A/V重定向，音频和视频将在本地处理 Microsoft Teams 呼叫和会议。在没有优化呼叫和会议的情况下，您仍然可以在Windows虚拟桌面上与其他客户端使用Microsoft Teams。
