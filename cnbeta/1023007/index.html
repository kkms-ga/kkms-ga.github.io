AMD锐龙5000桌面APU具有8个Zen 3核心：主频3.6GHz 增强7nm Vega核显

September 01, 2020

OpenBenchmarking 基准测试数据库表明，AMD 新一代 Ryzen 5000“Cezanne”APU 具有 8 个 Zen 3 CPU 内核、以及增强的 7nm Vega 核显，意味着其在两项性能上都有巨大提升。网友 Coelacanth’s Dream 在 Twitter 上分享了他的最新发现，指出这颗处理器似乎属于桌面平台，并在一款内部测试用的主板上进行了 Xonotic 跑分。

移动平台上的 Ryzen 4000 Renoir APU

这枚 Ryzen 5000“Cezanne”桌面 APU 具有 3.6GHz 的主频（动态加速频率未知），8 核心 / 16 线程（Zen 3 架构），且 7nm GPU（核显）部分也较 Ryzen 3000 系列 APU 有所增强。

这枚 Cezanne APU 的编码为 100-000000263-30_Y

参考早期 Vermeer 样品的 4.9GHz 频率、且 Ryzen 7 4750G 也有 4.4GHz 的加速频率，预计 2021 年推出的 Ryzen 5000“Cezanne”APU 也将有 4.5GHz 保底的频率。

GPU 方面，Ryzen 5000“Cezanne”与 Renoir APU 有很多相似之处（Vega 核显），但新一代产品将采用升级后的 7nm+ 制程、且具有更高的时钟频率（本文曝光的样品的 GPU 为 1800MHz）。

有趣的是，这枚 Ryzen 5000“Cezanne”APU 出现在 AMD 内部的 Artic-CNZ 测试平台上。

通过 Xonotic 套件和 Ubuntu OS 上的一些基准测试工具，可知其较 Ryzen 7 4750G 分数略低。一个猜测是，目前新一代芯片的核显等驱动尚未完善。
