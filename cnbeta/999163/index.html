三星Galaxy Note 20和Note 20 Ultra可能成为史上最贵的Note机型

July 04, 2020

我们距离三星秋季旗舰机Galaxy Note 20和Note 20 Ultra的发布仅有数周时间。预计这两款手机将在Note 10系列的基础上进行改进，采用类似的设计，但有望迎来更强大的摄像头和更细的边框。尽管在核心技术上没有突破新的领域，不过据意大利泄密者Ricciolo透露，这款手机有望在价格上“向上突破”。

可靠的三星泄密者Ice Universe似乎用两个预测价格印证了这个建议。他预测三星Galaxy Note 20的价格将为999美元，比Note 10上涨50美元，而Note 20 Ultra的价格将为1299美元，高于Note 10+的1199美元上市价格。

规格方面，Note20 Ultra搭载骁龙865+，采用120Hz LTPO显示屏，屏幕分辨率为2560×1440(QHD)。Note20 Ultra的另一个有趣的功能是，用户将可以同时开启QHD+120Hz，这是S20用户无法做到的。此外，三星还将为这两款智能手机推出新的S Pen，预计也会有一些新功能。

我们还听说三星将继续使用ISOCELL Bright HM1 1.08亿像素的主传感器，搭配激光自动对焦。搭配全新的广角摄像头传感器：1200万像素ISOCELL Fast 2L3 1/2.55传感器，像素尺寸为1.4um，Dual Pixel PDAF，支持智能WDR，可实现最高1440fps(裁剪后720p)的慢动作录制，并且可以输出RAW10和RAW8(通过DPCM/PCM压缩)格式。长焦方面，最大支持50X混合变焦，潜望镜头中的4800万像素传感器替换为1300万像素 Slim 3M5 1/3传感器，并支持WDR（宽动态范围），这可以改善低光性能，通常在远距离放大时尤为重要。

据报道，三星在Galaxy Note 20 Ultra上运用新的激光自动对焦元件取代了飞行时间传感器，据说已经解决了所有的自动对焦问题。

这款手机将在2020年8月5日的三星Unpacked活动上发布。据悉，三星还计划在2021年发布一款廉价版三星Galaxy Note 20 Lite，以解决价格高高在上，用户望而却步的问题。
