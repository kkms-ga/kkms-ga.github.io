富士发布Fujifilm X WebCam应用程序 无反相机秒变网络摄像头

May 27, 2020

继佳能之后，富士也为部分 X 和 GFX 无反相机用户推出了一款应用程序，以便通过有线连接，将相机作为高素质的 PC 网络摄像头来使用。据悉，这款程序名叫《Fujifilm X Webcam》，可通过 USB 连接笔记本或台式 PC 。与市售普通网络摄像头相机，相机的传感器和镜头显然能够将画质推升至新的台阶，甚至可以套用富士经典的胶片模拟模式。

受新冠病毒引发的 COVID-19 疫病大流行的影响，许多人被迫在家学习和远程办公，网络摄像头也因货源紧缺而普遍涨价。

早些时候，佳能已经为旗下多款数码相机发布了类似的网络摄像头应用程序。现在，富士也紧随其后，推出了《Fujifilm X WebCam》。

Fuji Guys - FUJIFILM X Webcam Tutorial（via）

下载地址：

https://dl.fujifilm-x.com/support/software/x-webcam-win100-rrg5d27f/XWebcamIns100.zip

据悉，Fujifilm X WebCam 可配合 X-H1、X-Pro2、X-Pro3、X-T2、X-T3、X-T4 和全系 GFX 机型免费使用，但目前仅限支持 Windows 操作系统。
