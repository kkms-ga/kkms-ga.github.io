微软宣布与OpenAI合建一台人工智能超级计算机

May 21, 2020

在线上举办的 Build 2020 线上开发者大会活动期间，微软宣布了将与 OpenAI 合建一台超级计算机的消息。显然，微软希望通过对 AI 模型引入持续的改进，让自家 Azure 云平台在未来的人工智能领域赢得先机。此外，软件巨头还在会上介绍了 Windows 10 UWP 应用开发、面向高级用户的诸多实用工具、以及 Windows Linux 子系统等项目的新进展。

微软与 OpenAI 合作构建了全球排名前五的 AI 超算之一，其本质上仍基于 Azure 云端，方便研究人员测试 OpenAI 的人工智能模型。

无论是编写力压顶级高手的《刀塔 2》游戏 AI，还是能够改善机器人技术和决策能力的使用模型，都可以在这台 AI 超算上测试实现。

尽管未深入介绍这台新型 AI 超算的技术细节，但这家软件巨头还是披露了 28.5 万个 CPU 内核、以及一万个 GPU 组件，且分组后的服务器可通过 400 Gbps 的带宽互联互通。

之所以为 AI 研究投入如此惊人的算力，主要是因为模型大小每隔几个月就要翻番。以自然语言处理（NLP）的图灵（Turing）模型为例，其参数已从去年的 10 亿、增长到了现在的 170 亿。

2019 年的时候，微软同 OpenAI 达成了多年的伙伴关系，并向这家由伊隆·马斯克成立的 AI 初创企业注资 10 亿美元，以使其能够借助 Azure 来训练 AI 。

更重要的是，通过这项合作，微软能够向外界证明该公司在 AI 研究领域的优势地位，从而吸引更多企业选择 Azure 云平台。
