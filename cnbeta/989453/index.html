升级加量不加价 小米推出Type-C充电器65W疾速闪充版

June 11, 2020

2020年3月份小米一口气推出四款65W PD充电器：AD65G、AD653、AD651、MDY-11-EB，其中AD651充电器比较另类，因为只有他不支持小米私有协议，其他三款产品均支持。销售了一段时间后AD651下架，最近该产品又重新上架，型号改成AD651P，缺失的小米私有协议也给“补”上了。

小米AD651P新增的私有协议是什么？部分小米手机采用了基于PPS协议开发出10V5A、10V4A、10V3A等等不同功率快充方案，对应旗舰机型、主流机型。此前旧版本AD651只支持PD3.0，不包含私有协议无法支持小米大功率快充，哪怕是小米10Pro这种旗舰机型充电也只有18W左右的功率。而这次加入了10V5A私有协议后，小米10Pro也能获得50W MAX疾速快充了。

日常使用中，AD651P与常见的PD充电器没什么区别，包含5V3A / 9V3A / 12V3A / 15V3A / 20V3.25A这些常用的PDO，可以提供最高65W的PD快速充电，适合为笔记本、平板电脑服务。小米官网透露的信息中介绍，1小时50分即可充满MacBook Pro 13，2小时25分钟充满小米笔记本Pro 15.6。

小米AD651P三围尺寸50mm x 50mm x 28mm，相比起上一代小米65W PD充电器，体积缩减了27%。采用了可折叠插头设计，外出时可将插脚收纳进壳体内，避免与行李物品互相刮蹭。

小米AD651P可以看做是此前旧版的固件升级款，新增了10V5A私有协议的支持，可以更好地为小米设备提供大功率快充。售价方面与之前一样保持99元，加量不加价。
