苹果发声明解释云游戏流媒体服务在App Store无法上架问题 但等于没说

August 08, 2020

苹果公司已经提供了一份声明来解释其立场：为什么微软的xCloud和谷歌的Stadia云游戏流媒体服务不能在iPhone和iPad的App Store中上架。值得注意的是，在声明中苹果并没有正面说明用户和其它公司公开抱怨的原因，而是坚持老一套，大谈App Store的好处和功能。

早些时候，微软公开宣布不会将其xCloud流媒体服务带到iOS上，并提前停止测试，因为苹果的App Store政策不允许该服务在目前的状态下工作。

由于苹果的App Store政策，Facebook在iOS上应用中推出的游戏频道里也没有游戏。

以下是苹果提供给Business Insider的声明全文：

创建App Store的目的是为客户提供一个安全可信的地方，让客户发现和下载应用，同时也为所有开发者提供一个巨大的商机。所有应用在上架之前，都会根据同样的准则进行审查，目的是保护客户，并为开发者提供一个公平、公正的竞争环境。

我们的客户享受着来自数百万开发者的优秀应用和游戏，游戏服务只要遵循适用于所有开发者的同一套准则，包括单独提交游戏进行审核，以及出现在排行榜和搜索中，就完全可以在App Store上推出。除了App Store之外，开发者还可以选择通过Safari和其他浏览器在App Store上通过网络接触所有iPhone和iPad用户。

Business Insider将这一说法解读为苹果不愿意发布此类应用，因为它无法对这些服务提供的每一款游戏进行审查。这完全没有道理，因为Spotify和Netflix等流媒体应用中也存在很多不需要苹果审查的内容。并且Spotify和Netflix的内容以及许多其他服务的流媒体内容也不会出现在Apple TV或iTunes应用的搜索结果，或其附带的排行榜中。

针对苹果的声明，微软发表了一份新的声明，呼吁苹果重新考虑这一武断做法：

我们对iOS版Project xCloud预览应用的测试期已经结束。遗憾的是，我们没有办法通过苹果应用商店将我们通过Xbox Game Pass Ultimate实现云游戏的愿景带给iOS上的玩家。苹果作为唯一一个拒绝消费者使用Xbox Game Pass等云游戏和游戏订阅服务的通用平台，独树一帜。而且它一贯对游戏应用区别对待，对非游戏应用适用更宽松的规则，即使它们包含互动内容。Xbox Game Pass目录中的所有游戏都由独立的行业评级机构（如ESRB和地区性同等机构）进行内容评级。我们致力于寻找一条将云游戏与Xbox Game Pass Ultimate带到iOS平台的道路。我们相信，客户应该是游戏体验的核心，玩家告诉我们，他们希望在任何地方游玩、连接和分享。我们对此深表同意。

苹果的首要目标似乎是保持对App Store以及允许在iPhone和iPad上已经存在的应用和游戏的控制。该公司显然害怕通过允许游戏以其无法控制的方式运行而失去控制权，这可能会开创一个先例，让第三方开发者依靠xCloud和Stadia等平台将他们的游戏带到iPhone和iPad上，而不是使用App Store。

苹果是否会从便利客户角度出发而尽快改变政策，目前还不得而知。Facebook Gaming在等待苹果让步数月后，刚刚推出了一个令人非常尴尬的没有游戏的游戏频道。而对于想要在智能手机上运行xCloud和Stadia的游戏玩家来说，最好的选择是使用Android。
