Microsoft Edge Dev更新 可安装Chrome网上应用店里的主题

August 06, 2020

本周，谷歌向 Edge Dev 通道推送了 86.0.594.2 版本更新，添加了可从 Chrome 网上应用店获取主题的功能。此外 Edge Dev 迎来了可靠性修复和其它改进，并且为那些喜欢对浏览器进行各种调整的用户引入了一个新的标记（flags）。启用之后，用户将能够通过 Shift + Backspace 组合键来执行网页的前进或后退操作。

该标记默认处于禁用状态，感兴趣的用户可以手动启用。此外设置中添加了在地址栏搜索时关闭历史记录和收藏夹建议的选项，以及如下改进：

● 为 VR / AR 网站添加了相应的权限；

● 在‘密码设置’和‘密码监视器设置’页面之间加入了跳转连接；

● 在策略管理器中添加了‘允许 Edge 浏览器在关闭时保存 Cookies’的选项。需要注意的是，该策略旨在与“关闭浏览器时清除浏览数据”的策略配合使用，管理模板将在后续迎来更新。

可靠性修复方面，Edge Dev 86.0.594.2 修复了在某些 Windows Insider 测试版中切换选项卡时，浏览器可能崩溃等问题：

● 修复在 IE 模式选项卡中访问某些站点时，浏览器可能崩溃的问题。

● 修复选项卡可能在加载时偶然崩溃的问题。

● 修复网页翻译完成后，地址栏有时仍保持‘翻译’状态的问题。

● 修复条目有时无法正确添加到收藏的问题。

● 修复 Ctrl + F 查找时，偶尔会在页面上突出显示的问题。

● 以及改进安装渐进式 Web 应用（PWAs）时的 UI 。

当然，Edge Dev 86.0.594.2 仍有一些已知问题，比如 Mac 用户可能无法顺利登录浏览器或同步数据。目前微软已经查明了原因，并计划在下一版本中修复。

其它已知问题还涉及广告屏蔽扩展程序、卡巴斯基互联网安全套装、收藏夹重复、窗口变黑、以及滚动时存在抖动。
