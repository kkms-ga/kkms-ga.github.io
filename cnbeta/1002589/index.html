必应弹广告强推Edge浏览器 微软称正在研究相关问题

July 13, 2020

尽管微软和谷歌都在利用自家的搜索服务页面强行推广 Edge 或 Chrome 浏览器，但用户对这种骚扰方式也积怨已久。近日，外媒报道称必应搜索又开始在结果页面底栏强推新版 Micrsoft Edge 浏览器的广告。问题在于，即便你已经在使用 Edge 浏览器，这个推广栏也不会消失。

（截图 via Techdows）

在 Reddit 和 Edge Insider 测试者论坛上，都已经有不少用户提交了这个问题报告。

某位网友写道，‘获取新版 Edge 浏览器’的弹出窗口不仅令人生厌，还对用户的正常工作流程造成了干扰。

更难受的是，即便已将“诊断和反馈设置”改为“完全”，必应仍无法正确检测到用户已安装并正在运行 Chromium Edge 。

对于这种令人难以置信的行为，Edge Insider 测试者无奈地向微软发出了警告，因为该公司的营销人员正在搬起石头砸自己的脚。

近日一位 Reddit 网友在 r/MicrosoftEdge 讨论板上请求官方别在用户搜索 Firefox 或 Chrome 浏览器的时候强推广告。

庆幸的是，微软项目与社区经理 Fawkes 回应称已知悉此事，并表示营销团队正在检查该问题。

感谢您提交的关注，虽然这并不是我们的本意，但还是对您受到的体验影响深感抱歉。

我们会认真对待此类报告，尤其是来自一位长期的 Insider 测试人员的报告。

通过与市场营销团队紧密合作，我们会将这些信息转达给他们。

与此同时，大家仍可通过浏览器来提交反馈和诊断数据，这将对我们很有帮助。

然而外媒 Techdows 指出，尽管这个问题已不再出现在 Edge 浏览器的必应搜索结果页面，但仍会在 Firefox 和 Chrome 浏览器中使用必应搜索服务时出现。
