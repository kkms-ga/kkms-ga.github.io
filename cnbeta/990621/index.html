WSL 2.0公告意外泄露了Windows 10 21H1预览

June 13, 2020

微软已经开始向快速更新（Fast Ring）通道的 Insider 测试者们推送 mn_release 分支的预览版本。“mn”前缀的全称为 Manganese，是定于今年晚些时候推出的 20H2 服务更新（Service Pack）的内部代号。在这之后，开发团队将把重心转移到明年春季的 21H1 更新。根据近日一篇博客文章的意外爆料，微软将从 6 月份开始，将 Iron 版本交付给 Fast Ring 的 Insider 测试者。

据悉，微软在 Windows Linux 子系统（简称 WSL）的 2.0 公告中意外泄露了 Windows 10 21H1 预览版本的信息，不过原文很快就经过了重新编辑。

即便如此，还是有眼尖的人们在第一方应用程序更新包中发现了对 Windows 10 21H1（Build 20141.1000）的引用，表明 21H1 的首个预览版本或在原计划的几周内到来。

尽管 Microsoft Insider 的时间表可能会有所变化，但从目前来看，Windows 10 21H1 预览更新似乎仍在路上。

正如 Windows Insider 团队在今年早些时候展示的那样，预计明年我们将看到“开始”菜单的重大改变。

新的“开始”菜单将进一步融入 Fluent 设计元素，以便在减少视觉干扰的同时，进一步提升整体的质感。

微软还计划为 Windows 10 引入 Windows 10X 中的一些特性，比如允许测试人员在 Windows 10 中使用新的 Win32 容易，类似于在“沙箱”环境中运行 Win32 应用程序。

此外 Windows Update 的更新机制也会做出一番调整，让升级流程变得更快，以减少用户在更新安装时的漫长等待。
