Windows 10 2004改进了账户头像在其它服务上的刷新速度

April 27, 2020

Windows 10 2020 五月更新（Version 2004）发布在即，预计首轮推送会在未来几周内到来。与 2019 年 11 月更新（Version 1909）相比，其不仅仅是一个 Service Pack 升级包，而是引入了许多新功能和修改，甚至在某些起眼的地方。其中之一涉及 Windows 10 中的账户图片，从 Windows 10 20H1 开始，用户头像可更快地在各项服务中得到刷新。

访问:

微软中国官方商城 - Windows

此前，用户只会在当前 Windows 10 系统中见到账户头像的即时更改。然而在其它地方（比如关联服务），旧头像要过段时间才能获得刷新。、

随着 Windows 10 Version 2004 的正式发布，这方面的体验将得到极大的改善。微软解释称：

新版中，更改 Windows 账户头像的速度将比以往任何时候都要快，新头像可在系统、应用程序和网站服务商获得更快速的刷新显示。

相比以往，用户可能在不同的应用程序和网站中见到 3~4 张不同的头像照片。现在只需在 Windows 10 设置里转到个人信息页面，便可立即更改生效。

需要指出的是，并非所有网站和关联服务都已经做好了对接 Windows 10 Version 2004 相关变动的准备。如果在升级后的一段时间内遇到问题，还请再耐心等待一段时间。

感兴趣的朋友，可参考如下方式来修改 Windows 10 账户头像 —— 转到 Windows 10 设置 -> 账户 -> 你的信息 -> 创建图片。
