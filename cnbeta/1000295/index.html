华为携手统信UOS国产系统：EulerOS开源迈出一大步

July 07, 2020

日前，openEuler技术委员会听过讨论决定，openEuler开源社区正式成立DDE SIG特别兴趣小组，进一步完善和丰富openEuler的桌面环境，而第一个维护者就是出品了国产操作系统UOS的统信软件。DDE SIG成立后，openEuler社区参与者和用户可获得由专业团队维护的最新桌面环境DDE版本，并通过openEuler社区获得技术支持。

统信软件表示，作为操作系统领军企业，统信软件拥抱开源，贡献开源，未来将和众多开发者一起，全力支持构建openEuler开源社区。

DDE是统信软件自研的一款全功能桌面环境，为用户提供美观易用、智能协同的界面交互，广受国内外用户的喜爱，已经被移植到Archlinux、OpenSUSE、Ubuntu、LinuxMint、Fedora、Manjaro等主流Linux发行版本中，搭载DDE的统信软件社区版操作系统deepin，也长期位居全球开源操作系统排行榜上前十名。

目前，统信软件已经将最新版DDE集成至统信UOS Server Euler之中，它基于openEuler 20.03 LTS(Linux 4.19内核)社区发行版，通过整合自研DDE桌面环境和图形界面的应用程序、第三方软件以及相关开源软件等内容，形成针对常规、云、虚拟化、容器、大数据等应用场景的通用操作系统方案。

openEuler是华为服务器操作系统EulerOS开源之后的产物，是一个免费的Linux发行版平台，通过开放的社区形式，与全球的开发者共同构建一个开放、多元和架构包容的软件生态体系，同时也是一个创新的平台，鼓励任何人在该平台上提出新想法、开拓新思路、实践新方案。

openEuler通过社区合作，打造创新平台，构建支持多处理器架构、统一和开放的操作系统，推动软硬件应用生态繁荣发展。

目前已经有众多操作系统厂商参与到openEuler社区，并积极贡献自己的力量，包括普华基础软件、麒麟软件、中科院软件所、拓林思等，都发布了基于openEuler的商业发行版，产业链上下游也都在纷纷参与openEuler。
