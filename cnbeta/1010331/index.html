基于暴露通知API开发 加拿大部署COVID Alert应用追踪新冠疫情

August 01, 2020

“暴露通知”（Exposure Notification API）是由谷歌和苹果公司联合开发的 API，能够帮助政府和相关机构更快、更准确地追踪新冠病毒的传播。该 API 在今年五月上线以来，包括加拿大、巴西在内的部分国家已经基于它开发了相关的应用程序。

加拿大开发的应用程序名为 COVID Alert，是首批基于该 API 开发的应用程序。加拿大地区的所有 iPhone 和 Android 用户都可以使用。

当你第一次启动该应用时，你会看到一系列的屏幕内容，解释该应用如何工作以及收集哪些数据。基本上，该应用会在附近同样使用该应用的设备之间传递蓝牙令牌，如果你靠近某人的时间足够长，就会有一个日志。

当某人的 COVID-19 测试呈阳性时，他们会从医生那里得到一个一次性的密钥（仅限加拿大地区）。一旦输入诊断密钥，几周内你接触过的所有人都会被提醒你的诊断结果。

该系统不会在应用内存储任何凭证，以保证用户的安全。唯一需要的信息就是那个一次性代码，它不会将你的身份透露给被提醒的阳性诊断的用户。

只有当民众大规模采用该系统时，联系追踪系统才会有效。如果你住在加拿大，并且有iOS或Android设备，请下载应用参与。
