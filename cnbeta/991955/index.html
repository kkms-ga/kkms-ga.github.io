[图]Windows 10 Version 1909/1903/1809获累积更新 修复打印机问题

June 17, 2020

在安装本月的补丁星期二活动日发布的累积更新之后，很多用户和IT管理员都反馈网络打印机出现故障。目前已经确认 Windows 10 Version 2004/1909/1903/1803，甚至是 Version 1507 都存在该问题。今天，微软面向 Version 1909/1903/1809 以及此前版本发布了新的累积更新，以修复管理员、普通用户的打印机问题。

微软表示，适用于 Windows 10 Version 1909/1903 的累积更新 KB4567512，以及适用于 Version 1809 的 KB4567513 将解决打印机相关的问题。不过这些更新仅通过 Microsoft’s Update Catalog 提供下载，用户需要手动安装进行修复。

更新目录工具提供三个选项。32、64位或ARM64版本的累积更新。选择合适的版本，点击 “下载”。下载完软件包后，双击.MSI文件，等待更新安装，需要时重启系统。

如果你当前运行的是 Windows 10 Version 1903，请下载以下链接

目前微软还没有发布适用于 Windows 10 Version 2004/1607/1507 的修复累积更新。如果你在使用过程中出现了这个问题，那么可以卸载六月累积更新，等到微软修复之后再升级。

此前，不少管理员确认受影响的打印机包括理光、惠普、兄弟、佳能、东芝和松下。我们初步发现，Windows 10 KB4560960和其他更新似乎会导致与基于网络的打印机上的PCL 5和PCL 6 v2驱动程序发生冲突。
