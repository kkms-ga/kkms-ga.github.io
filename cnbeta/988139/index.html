[图]微软商城概念设计：突显应用集锦 整合不同主题的应用程序

June 08, 2020

尽管自 Windows 10 系统推出以来，微软就大力完善 Microsoft Store 的应用生态，但始终收效甚微。无论是开发者还是消费者，都不会将微软商城作为发布和获取应用的首选之地，依然将更多精力放在 Win32 应用中。

尽管在可预见的未来一段时间内，Win32 应用依然是 Windows 10 系统的主要打开方式，但微软商城通过用户界面的优化调整、应用的丰富，或许能培养出一些忠实用户。

除了应用这个最大短板之外，微软商城的用户界面也有很大的改进空间。近日知名设计师 Zee-Al-Eid Ahmad Rana 发布了一个微软商城概念版，进一步突显集锦（Collections）在微软商城中的重要性。

集锦是用户在微软商城上寻找应用的一种有效途径，通过主页面的集锦能够根据场景的不同提供个性化推荐，帮助用户发现适合自己的应用程序。例如“自定义 Windows 桌面”、“适合资深用户的应用”等等。
