Windows Defender将CCleaner标记为“可能不需要的应用”

July 30, 2020

CCleaner 是一款深受用户欢迎的电脑优化软件，不过近日在 Windows Defender 平台更新之后，该应用程序被认定为是可能不需要的应用程序（PUA）。CCleaner 基本上可以让你检查所有主要浏览器中的 cookie、缓存文件和下载文件来优化系统，并附带了一个功能完善的注册表清理器。

不过令人惊讶的是，Windows Defender Antivirus 已经开始将 CCleaner 实用程序标记为潜在不需要的应用程序。

可能不需要的应用程序（PUA）保护是一个允许 Windows Defender Antivirus 检测和阻止不需要的应用程序的功能。

根据用户的报告，Windows Defender 将 CCleaner 标记为潜在不受欢迎的程序，警报级别较低。但 CCleaner 并没有被检测到对系统的威胁，用户仍然可以手动允许该应用在系统上运行。

微软在支持文档中指出，PUA 并没有被归类为病毒、恶意软件或其他类型的威胁，但这类应用可能会 “对终端性能或使用产生不利影响”，并提出安装其他符合PUA条件的应用。Windows Defender还将 “声誉不佳 “的应用标记为潜在的不受欢迎的程序，原因是 “某些类型的不良行为”。
