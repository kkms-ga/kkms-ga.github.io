华为平板MatePad T10/T10s规格曝光：麒麟710A芯片

July 11, 2020

日前新的曝光揭示了即将推出的两款华为平板电脑的规格，曝光的两款设备分别是 MatePad T10 和 MatePad T10s，它们都搭载了麒麟 710A SoC。Roland Quandt 在 Twitter 上表示，华为 MatePad T10 搭载 9.7 英寸屏，分辨率为 1280 x 800 像素，像素密度为 156ppi。

搭载八核麒麟 710A 芯片，该平板电脑预装 EMUI 10.1，有两个内存版本，即 2GB RAM + 16GB 存储和 2GB RAM + 32GB 存储，并支持高达 512GB 的 microSD 卡扩展。

MatePad T10 由 5100mAh 电池组供电，并支持常规的传感器。拍照方面，背部采用 500 万像素的主镜头，前置为 200 万像素的传感器。

另一方面，华为 MatePad T10s 则具有更大的 10.1 英寸屏，分辨率为 1920 x 1200，像素密度为 220ppi。在摄像头，电池，操作系统和连接性方面，则具有与 MatePad T10 相同的功能。除了屏幕尺寸之外，主要区别还在于存储版本，T10s 具有 2GB RAM + 32GB 存储和 3GB RAM + 64GB 存储版本。

需要指出的是，目前两款产品的价格和发布细节尚不清楚。
