金邦发布Orion系列DDR4内存模组 最高可选2*32GB DDR-4000套装

June 12, 2020

金邦（Geil）刚刚发布了 Orion 系列 DDR4 内存模组，并且提供了标准和 AMD 平台专用条两个版本。消费者可选 8GB 到 64GB 容量的套装，比如两条匹配过的 32GB 内存模组，频率从 DDR4-2666 到 DDR4-4000 不等。外观上，Orion 系列 DDR4 内存模组可选赛车红（Racing Red）或钛金灰（Titanium Grey）两种马甲配色。

对于不喜欢光污染的用户来说，缺少 RGB LED 点缀的金邦 Orion 系列 DDR4 内存模组的外观还是相当简洁的。

即便如此，与普条相比，Orion DDR4 内存模组的马甲高度还是有点高，需考虑避免与散热器等其它配件产生冲突。

目前官方尚未公布 Orion DDR4 内存模组的物理尺寸，选购 CPU 散热器或风扇时请考虑是否留有足够的空间。

产品 SKU 上，金邦提供了单条 8GB / 16GB / 32GB，以及双通道 16GB（28GB）/ 32GB（216GB）/ 64GB（2*32GB）的选项。

Orion DDR4 内存模组的频率有 DDR4-2666 / 3000 / 3200 / 3600 / 4000，工作电压 1.20 / 1.35V，高频条的时序也相当诱人。遗憾的是，我们尚不知晓这些产品的确切售价和上市日期。
