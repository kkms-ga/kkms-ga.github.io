[图]越狱工具Odyssey Beta 2发布：适用于iOS 13-13.5

July 29, 2020

虽然苹果已经不再签署 iOS 13.5/13.5.1 了，但 CoolStar 团队日前发布了 Odyssey 越狱工具的第二个公测版本。此前有消息称，Unc0ver 团队向 CoolStar 团队提交了 DMCA 通知，要求下架后者的 GitHub 代码仓库。目前最终的判决还没有公布，不过对于普通用户来说又多了一个新的越狱工具。

Unc0ver 团队表示，CoolStar 团队盗用了他们的代码。虽然已经提交了 DMCA，但目前仍未最终的判决。正如前文所述，Odyssey 越狱工具的首个公测版本发布了。虽然没有太多值得兴奋的地方，但很高兴看到用户现在有了其他选择

本次发布的越狱工具适用于 iOS 13~13.5.x 版本，兼容所有运行苹果 A8~A13 处理器的设备，该越狱工具支持包括 iPad Pro 在内的多款设备。

向公众发布越狱的目的是提供比 Unc0ver 和 Checkra1n 越狱工具更优秀的用户体验。需要注意的是，如果你打算使用 Odyssey 越狱工具来越狱你的iPad，你应该首先要在 Odyssey 中使用恢复 rootFS 功能，然后再转移到它。

你应该注意的另一个主要方面是，Odyssey 越狱工具默认带有 Sileo 包管理器. Sileo 最近收到了一个重大更新，因此能够比此前版本更快。和 Unc0ver 或 Checkra1n 另一个差别就是，Odyssey 越狱工具是完全基于 Swift 语言编译的，因此运行速度更快，用户体验更好。
