Exynos 990版Galaxy Note 20 Ultra跑分曝光 性能没有很大提升

July 31, 2020

最近Galaxy Note 20 Ultra和Galaxy Note 20都很火爆，在正式公布不到一周的时间里，所有的目光都集中在这几款设备上。 首先，我们知道Galaxy Note 20将搭载60Hz平板显示屏，售价999美元。然而，更糟糕的是，该设备的全球版本将配备Exynos 990而不是Exynos 992，而像美国这样的一些市场的版本将采用高通Snapdragon 865 Plus。

虽然我们曾报道过Galaxy Note 20系列的Exynos 990会如何进行更多的优化，但最近的报告表明并非如此，因为分数并不值得一提。根据Galaxy Note 20 Ultra的Geekbench 5跑分，这款手机的单核成绩为928，多核成绩为2721。为了方便对比，采用Exynos 990的Galaxy S20 Plus单核成绩为938，多核成绩为2726。因此，对于一款距离上市仅一周的全新设备来说，性能和老产品差不多是一样的，令人失望。

Galaxy Note 20 Ultra售价1400美元，相对于价格，这些测试结果是令人失望的。而骁龙865 Plus版本表现则会好很多很多。遗憾的是，该机型将仅限于美国、中国和其他几个市场，至于全球版本的Galaxy Note 20 Ultra将坚持采用Exynos 990。

这意味着目前Galaxy Note 20或Note 20 Ultra最大的卖点就是S-Pen，因为S20系列和Note 20系列的差异比我们想象的要小很多。
