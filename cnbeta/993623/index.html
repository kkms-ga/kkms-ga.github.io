卸载Chrome换回Edge 这也许才是最好用的浏览器

June 20, 2020

话说去年九月微软开始测试 Chromium 内核的全新 Edge 之后，我就开始关注这个来自大厂开发的可能代替我一直在用的 Chrome 的浏览器。事实证明微软这一次做对了，用上全新内核的 Edge 浏览器不但在内存占用、流畅度方面都比原来强悍 N 倍，还能安装 Chrome 商店的插件，甚至于还支持 macOS 系统，所以理想情况下它是能和 Chrome 一战甚至代替 Chrome 的。

但在我用了一个多月以后出现了问题，主要就是当时的 Edge 浏览器不能跨电脑同步插件，我安装了大概十几个插件（这还不包括油猴脚本），如果换一台电脑再让我手动安装一次实在是很麻烦，这一点当时 Chrome 依然领先，再加上在手机端配合也不够完美，所以后来我就弃坑了，想要等到功能完善再换回来。

刚好就在最近，微软终于把 Edge 浏览器上的这些槽点都解决了，而且宣布未来将作为 Win10 默认浏览器启动，所以我又换回了 Edge 浏览器，打算作为未来长期工具来用了。作为一个从早期就开始体验的用户，今天我就来分享一下它有哪些好用的功能以及一些非常方便的插件。

好用的功能之一：跟踪防护

打开 Edge 设置页面，选择隐私和服务，你会看到它包含了三种防护模式，包括基本、平衡、严格三种，这三种能够满足不同用户对于浏览网页时候的需求，个人认为非常方便而且实用，如果你对隐私非常非常在意，可以选择严格防护来保护自己。

好用的功能之二：集锦

如果你看到一个喜欢的网页，一般会选择把它收藏起来，但是你不一定会再想起来查看，集锦这个功能可以让你在看到有自己需要的资料时直接同步到 Word 或者 Excel 里，因为 Edge 浏览器和 Office 的深度绑定，所以这个功能是 Chrome 不可能拥有的，个人觉得它比书签更好用。

好用的功能之三：大声朗读

这个功能就在设置当中，快捷键 ctrl+shift+U 就能启动，会把当前页面朗读给你听，语音包括中文、英文还有很多其他语言，基本上涵盖了主流语种，虽然不能直接翻译英文页面，但是如果你很忙觉得看网页麻烦，或者是那些弱视患者，那它能有一定作用。

好用的功能之四：标签页静音

Edge 浏览器在打开网页播放音乐或者视频的时候，如果你这时候切换到别的页面，可以直接点击标签上的小喇叭来关掉声音，相比之下 Chrome 是把音乐播放都集合在最右上角，虽然谁好谁坏各有说法，但是我觉得 Edge 的这个快捷操作更符合逻辑直觉。

以上几点是我个人觉得 Edge 浏览器做的比 Chrome 好的一些细节，虽然说 Edge 相比 Chrome 还有一些不足，比如加载速度上有微弱劣势，再比如只有黑色白色两种外观没办法更换皮肤，但是无论是内存占用还是浏览时的流畅程度都不输给 Chrome，所以已经可以满足 Chrome 用户的大部分需求了，同步 Chrome 书签也非常方便，接下来我跟大家分享一下我在用的插件：

1，AdBlock：屏蔽广告，好用程度满分。

2，I’m a Gentleman（图片下载助手 可代替）：点击一下，你就能下载当前页面所有图片内容，特别适合写文章、做视频找素材用。

3，Infinity 新标签页：允许你添加首页标签，还有一些好用的插件，国内网站基本都有收藏，很方便。

4，Universal Video Downloader：我一般用它来下载微博中的视频。

5，哔哩哔哩助手：B 站页面黑暗模式、下载 B 站上一些鬼畜素材或者高清素材视频。

6，彩云小译 - 网页翻译插件：给当前页面添加中文翻译，可以让你中英文对照来看，而不是谷歌翻译直接完全替换。

7，捕捉网页截图 - FireShot：捕捉当前页面截图，你可以自由选取或者长截图。

8，Augmented Steam：Steam 比价工具，打折的时候看它是不是历史最低。

9，Google 翻译：支持多语种，准确度高。

10，Sexy Undo Close Tab：快速浏览历史打开页面

11，Dark Reader：实时生成当前网页的黑色主题，为每一个网站启用黑暗模式。

12，油猴：安装各种强大的脚本，它的强大超乎你的想象，具体可以去知乎看看。

以上就是我目前在用的浏览器插件，有的你可以直接登录微软插件商店下载，但有些是靠科学上网进入 Chrome 插件商店才可以的，所以大家如果下载不了不妨在微软插件商店看看有没有可替代的，如果有什么好用的大家也不妨在留言区分享。

对我而言新版 Edge 一大好处在于你可以稳定快速的去插件商店下载 Edge 插件，还能不断更新最新版本，对于网络不方便（你懂的）的用户来说是很方便的。

另外还要提一点是，如果你的电脑上是老版本的 Edge，那么去官方下载并安装新 Edge 之后会自动覆盖老版本，省去你要卸载旧版本的操作。
