[视频]LastPass发布Security Dashboard： 新增暗网监控功能

August 06, 2020

知名密码管理器 LastPass 今天宣布了全新的“Security Dashboard”，帮助用户更好管理密码和账号的安全性及完整性。该功能适用于该服务的所有用户，不过更多的暗网监控选项仅限于 Premium, Families 和 B usiness 用户。

Security Dashboard 为用户提供了账号安全的概况，根据待处理的行动项目数量给用户一个总分。该面板还建议了“风险密码”的数量，提示用户根据密码的强度来更改密码。评分还考虑到账户是否具备多因素认证功能。

暗网监控功能则是通过对照合作伙伴 Enzoic 的数据库，检查是否有泄露的电子邮件 ID 和凭证，进一步提高了链接账户的安全性。如果发现此类凭证或账户，该服务会通过电子邮件和产品内通知通知用户，提示他们更新密码。该公司还补充说，所有的监控都是 “在后台 “发生的，而且不涉及任何用户操作，直到该工具发出警报。
