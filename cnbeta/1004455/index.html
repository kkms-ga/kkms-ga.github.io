[技巧]如何在树莓派4开发板上使用Windows 10 IoT Core？

July 17, 2020

作为一款体积小巧的开发板，树莓派已经提供了预配置的系统环境。但对于追求更丰富的功能体验的用户来说，Windows 10 IoT Core 显然也是一个不错的选择。好消息是，曾于去年为树莓派 3 创建了 Windows on ARM 安装程序的开发者 José Manuel Nieto，现在又为树莓派 4 开发板的用户带来了 Windows 10 IoT Core 的轻松部署解决方案。

（图自：树莓派基金会）

据悉，树莓派 4 的入门价格为 35 美元（约 245 RMB），顶配版则是 55 美元（约 385 RMB）。

对于动手能力强的开发者来说，可通过 WoA Installer 在这款开发板上手动安装 Windows 10 IoT Core 操作系统。

受硬件性能的限制，此前在树莓派上运行 Windows 10 的体验可能会让不少人感到抓狂。

因为该项目最初仅支持 1GB RAM，从‘开始’菜单到 ARM64 版 Edge 浏览器的各种操作都很是卡顿，更别提恼人的‘蓝屏死机’问题。

庆幸的是，开发者 Amir Dahan 现已设法突破了相关限制，让 DIY 爱好者可在具有 4GB RAM 的树莓派开发板上尽情体验流畅的 Windows 10 ARM 。

（图 via MSPU）

以下是在树莓派 4 上安装 Windows 10 后的已知可用功能：

● 板载的 USB 端口；

● micro-HDMI 视频输出；

● 3GB RAM 运存；

● 蓝牙+以太网（需要 USB 接口的适配器）；

● YouTube、Microsoft Edge、Paint.net 和 Microsoft Store；

● 支持基于 x86 的应用程序，以及大多数 ARM64 应用程序。

以下是尚未解决的问题：

● 板载以太网卡、音频插孔和第二个 HDMI 端口不可用；

● 图形卡加速还没搞定；

● 板载的 Wi-Fi 和蓝牙功能暂不可用；

● 基于仿真的 X64 应用程序无法运行，禁用补丁后的 VLC 无法播放 1080p @ 60fps 视频，大多数 Steam 游戏也没指望。

如果能一路克服以上障碍，完成所有必须的硬件和驱动，便可在树莓派 4 上获得最佳的 Windows 10 性能体验，比如通过 USB 网卡来观看 YouTube 视频。

资料图（来自：Microsoft）

软硬件方面，最好通过 Minitool 或 Easeus 分区管理器，将系统安装在 Class 10 及以上的 32 / 64GB microSD 存储卡上，镜像文件可从 Discord 服务器上获取。

● 下载完成后，在 .rar 压缩包中提取 OS 文件。

● 找到选定的 Windows 10 编译版本所在的路径。

● 打开 Ether for Windows 软件，将系统文件写入到 microSD 存储卡中。

● 使用分区管理器软件，将 microSD 卡上的未分配空间扩展出来。

● 在树莓派 4 上插入 microSD 卡，连接键鼠和显示器。

● 运行补丁以解锁 3GB RAM 运存（可在同一个 Discord 服务器上获取补丁）。

需要指出的是，写入和安装 Windows 10 的整个过程，可能需要耗费一个小时的时间。想要加快这一流程的话，还是建议使用 Class 10 及以上标称速率的存储卡。
