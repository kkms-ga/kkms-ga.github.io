[图]无需启用实验Flag：新版Edge可直接安装Chrome主题了

August 02, 2020

想要在基于 Chromium 的 Microsoft Edge 浏览器上使用 Chrome Web Store 的主题，用户需要启用某个实验 Flag 才能下载和安装主题。不过在 Edge Canary 通道最新版本中，用户可以不再通过 Flag 就能安装 Chrome 主题了。

如果你正在使用Edge Canary，你可以访问Chrome Web Store，选择一个主题，然后点击 “添加到Chrome”。要删除或卸载主题，可以进入Edge浏览器设置>外观，点击 “自定义主题 “对面的 “删除 “按钮，默认主题就会恢复。
