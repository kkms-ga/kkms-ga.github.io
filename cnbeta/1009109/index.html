三星Galaxy Note 20搭载Exynos 990优化版：与骁龙865 Plus一样快

July 29, 2020

7月29日消息，三星将于8月5日发布Galaxy Note 20系列。和Galaxy S20系列一样，Galaxy Note 20系列不同市场分别使用Exynos版本和骁龙版本。其中Exynos版本使用的是Exynos 990，骁龙版本使用的是骁龙865 Plus。

有报道指出，三星Galaxy Note 20系列搭载的Exynos 990是官方优化改进版本，其运行速度与骁龙865 Plus版本一样非常快。

三星方面此前也曾提到，Exynos版本使用的是自家的芯片、闪存、电池等等，可以实现更好的协同优化，从而充分发挥其实力。

除了芯片不同之外，Exynos与骁龙版本其它规格预计保持一致。

具体来说，Galaxy Note 20采用了6.7英寸直屏，分辨率为2400×1080，刷新率60Hz，配备8GB内存+256GB存储，后置1200万像素主摄+1200万超广角+6400万长焦三摄，支持30倍数码变焦，前置1000万像素，电池容量为4300mAh。
