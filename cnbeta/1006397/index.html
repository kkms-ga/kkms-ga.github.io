[图]Edge路线图公布：跨设备同步浏览历史功能即将上线

July 22, 2020

尽管近日 Edge Dev 频道发布的新版本更新幅度并不是很大，但昨日微软透露了即将登陆 Edge 浏览器的诸多重磅功能。在 Edge Dev 博客上，微软表示已经根据用户反馈的情况，优先安排用户期待比较高的功能。

根据微软公布的列表，微软预估会先部署两项功能，分别是在PDF文件中可以通过目录来进行导航，以及允许在设备之间同步浏览历史。不过遗憾的是，微软并不计划添加鼠标手势的支持，而且“在关闭浏览器窗口时询问用户是否要关闭所有标签页”功能尚处于讨论中。
