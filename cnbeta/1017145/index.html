HTC Vive Sync企业级VR工具现已支持Android和Web浏览器

August 18, 2020

作为一款企业级工具，HTC Vive Sync 旨在帮助团队使用基于 VR 头戴式装置的相互交互。在近日的重大更新中，该公司称其引入了用户期待已久的诸多功能，尤其是 Web 浏览器的支持。升级之后，用户可在参加虚拟会议时直接访问 Web 。

（来自：HTC Vive）

HTC 表示，Vive Sync 是一款专为企业用户而设计的工具，能够让员工创建自己的 3D 化身，然后在虚拟环境中与同时和其它参与者进行交互。

以 VR 会议为例，用户可借助 VR 头戴式装置，开展协作、演示、头脑风暴等活动。与借助网络摄像头的普通方案相比，Vive Sync 可提供更具沉浸式的体验。

不过 VR 会议的短板，就是对场地和设备提出了较高的要求。为克服这个难点，HTC 还引入了对 Android 平台的支持。

这样当用户无法使用 VR 头显时，亦可通过智能手机或平板电脑来参加 Vive Sync 会议，目前新版应用已在 Play 商店上架。

此外，新版 Vive Sync 还具有内置的 Web 浏览器。虽然仍处于预览阶段，但用户已能够开展内容搜索、网站访问、以及在会议过程中可能需要执行的其它操作。

最后，本次更新带来了专为会议主持人打造的新版控件，允许其对每个参与者的权限进行管理，比如静音 / 取消静音、更改会议室等。
