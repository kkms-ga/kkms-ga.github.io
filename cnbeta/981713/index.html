Galaxy Note 20/Fold 2发布会将采取线上直播方式 具体时间未确定

May 21, 2020

援引韩媒The Korea Herald最新报道，即将于今年夏季召开的Galaxy Note 20/Galaxy Fold 2发布会将会采用线上直播的方式进行。报道称三星内部已在讨论Unpacked线上发布会了，但是目前具体的时间还没有确定。

根据往年的情况，三星会在8月上旬推出新旗舰。通常情况下，Galaxy Note系列新品的发布时间会比iPhone系列提前1个月时间。而最新的iPhone会在9月份发布。这样一来，三星有几周的时间来销售这款手机，而不用担心其主要竞争对手。

根据此前透露的信息，Galaxy Note 20 其背部配备一个看起来像是潜望镜式长焦镜头。泄露的摄像头阵列似乎还包括一对至少两个其他摄像头，LED闪光灯，一个（可能是几个）麦克风孔，以及一个传感器，可能会提供更快更准确的对焦。

据悉，Galaxy Note 20很可能会有两种尺寸，其中至少有一种会带来一块6.9英寸的显示屏，分辨率为3200×1440像素。另一款Galaxy Note 20很可能会有一块6.42英寸的显示屏，如果传闻被证明是准确的，那么该系列将在今年9月以Galaxy Note 20和Galaxy Note 20+的形式出现。

基础版机型被推测将配备12GB RAM和128GB内部存储。这款设备将搭载当时最新的Android系统，很可能是Android 10和三星One UI 2.x用户界面。上周有消息称，Galaxy Note 20将搭载高通公司的新指纹传感技术。那将是高通的第二代3D Sonic Max指纹识别技术，采用大尺寸的超声波技术，可以同时进行一到两个手指读取。
