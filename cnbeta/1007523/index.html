微软推送Windows 10 Build 19042.421测试版 引入新版开始菜单

July 25, 2020

微软刚刚通过 KB4568831，向 Beta 通道的 Insider 测试者（原 Slow Ring 慢速更新通道）推送了 Windows 10 20H2 Build 19042.421 更新。其主要引入了新版“开始”菜单，支持网站的 Alt+Tab 快捷键切换，以及任务栏等方面的改进。

（来自：Microsoft）

首先介绍“开始”菜单中对主题有感知的动态瓷贴：微软为其引入了更加简洁的新版设计，剔除了应用图标的纯色背景板，并换成了均匀、部分透明的背景。

在统一的 Fluent Design 设计风格下，无论是浅色或深色主题，包括 Office、Edge 浏览器、计算机、邮件、日历等在内的应用，都已变得更加养眼。

感兴趣的朋友可在“设置 -> 个性化 -> 颜色”中对“开始”菜单和瓷贴进行自定义配置。

对于经常需要进行多任务处理的人们来说，Windows 10 Build 19042.421 还改进了 Alt+Tab 切换组合键的功能。

包括应用程序和 Edge 浏览器（v83.0.475.0+ / Canary 或 Dev 分支）内的网站标签页，都可在前后台切换界面中显示，以便用户随时迅速切回某个地方。

如果不习惯新版 Alt+Tab 功能，亦可在“设置 -> 系统 -> 多任务”中配置仅显示最后 3~5 个选项卡、或完全关闭此功能。

微软一直努力改进 Web 浏览的效率，比如提供了快速访问已固定的网站标签页的方法。现在，当你点击任务栏上的网站快捷方式时，亦支持显示该站点在 Edge 窗口中打开的所有选项卡。

需要注意的是，作为一项早期预览的功能，其需要配合 v85.0.561.0（Canary 或 Dev 分支）或以上版本的 Edge 浏览器使用。

任务栏的个性化方面，新版提供了更加清爽、个性化的体验，同时能够减少内容的混乱。

借助云端驱动的灵活基础架构，系统能够感知用户对默认任务栏布局的接纳程度，以便后续做出相应的调整。

不过这项体验仅限于创建新账户、或首次登陆时的情况。

通知体验方面，Windows 10 Build 19042.421 也引入了一些改善。通过查看顶部的应用程序图标，可了解消息来自哪款软件。点击右上角的“X”按钮，可方便地将之忽略。

其次在默认情况下，Windows 10 将关闭“专注助手”（Focus Assist）。如果对此感到不习惯，可在“通知设置”中将它改为旧版的样式。

为了推进将“控制面板”中的功能转移到“设置”里，Windows 10 Build 19042.421 已将“控制面板 -> 系统”页面链接到“设置 -> 关于”。

此外用户可复制简要的设备和安全信息，想要查找“控制面板 -> 系统”中的高级控件的话，亦可从“设置 -> 关于”页面中找到。

最后，Windows 10 20H2 还改善了二合一设备的平板体验。此前在卸下键盘时，可能会弹出询问用户是否要切换到平板电脑模式的通知。

如果选择“是”，则会切换到默认的平板电脑模式。若选择不启用，Windows 10 2020 五月更新也引入了新的平板使用姿势（或维持 Windows 10 早期版本的桌面体验），并引入了触控方面的一些改进。

用户可在“设置 -> 系统 -> 平板电脑”中更改这项设置，一些 Surface 设备用户或已更早地知晓这项更改。至于更多变动，还请移步至微软官方博客查看。
