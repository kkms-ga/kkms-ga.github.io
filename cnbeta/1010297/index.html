[图]PowerToys 0.20版本发布：新增拾色器功能

August 01, 2020

对 Run、FancyZones 以及整体性能进行一系列优化之外，在今天发布的 0.20 版 PowerToys 工具还添加了拾色器（Color Picker）功能。目前新版本已经在 GitHub 上开放下载，用户也可在应用设置中检查更新来安装。

PowerToys 上次重大版本更新是 0.18 版本，为 Win+R 键盘快捷方式带来了 PowerToys Run 替代方案，并引入了 Keyboard Remapper。在随后发布的 0.19 版本更新中，主要是修复现有的 BUG，并没有引入太多新功能。

顾名思义，Color Picker 可以让用户从屏幕上挑选任何颜色的 RGB 和 HEX 值。这个灵巧的工具还可以让用户放大预览，以准确地挑选所需的颜色。可以通过 Windows+Shift+C 快捷键访问它。该功能是与 Martin Chrzan 共同开发的。

该公司指出，这个版本的重点是带来稳定性和质量修复，同时引入了颜色选取器。团队还解决了长期存在的设置 BUG，并改善了 OOBE 体验。如果你想下载 PowerToys 0.20，可以前往 GitHub 发布页面。
