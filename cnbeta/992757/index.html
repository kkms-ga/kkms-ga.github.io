亚马逊推出坚固便携的AWS Snowcone边缘计算设备

June 18, 2020

亚马逊云服务（AWS）今日宣布了坚固耐用的 Snowcone 安全边缘计算设备，其大小仅为 9×6×3 英寸（约 22.9×15.2×7.6 CM），重量也只有 4.5 磅（2 kg）。即便机身很是小巧，AWS Snowcone 还是内置了两颗处理器、4GB RAM + 8TB 存储控件、且支持有线 / 无线访问。除了内置电池，用户也可通过 USB-C 线缆来供电。

（来自：Amazon）

Snowcone 是 AWS 坚固型边缘计算设备家族中最轻巧的成员，用户可借其在隔离环境中收集和处理数据，然后将之传输到云端。

此外该设备具有防篡改功能，借助可信平台模块（TPM）来确保数据的安全和完整性，以及通过 AWS 密钥管理服务（KMS）进行加密。

值得一提的是，该设备还带有电子墨水运输标签，以便轻松追踪设备、并将之运动到正确的 AWS 设施。

以及支持 AWS IoT Greengrass，可运行 Amazon Elastic Compute Cloud（简称 EC2）实例。

AWS Snowcone - Edge Computing & Data Transfer Device（via）

感兴趣的客户可在各种环境中部署 AWS Snowcone，包括数据迁移、医疗保健、物联网、工业物联网、运输、内容分发、物流、战术边缘计算、以及自动驾驶汽车等用例场景。
