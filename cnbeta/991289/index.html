减配了气囊的国产特斯拉Model 3 居然比进口版还更安全？

June 15, 2020

6月10日，中保研公布了国产特斯拉Model 3的碰撞测试评价结果。结果显示，车内成员安全指数、车辆辅助安全指数两项获得优秀（G）评价；车外行人安全指数获得良好（A）评价；耐撞性与维修经济性指数获得一般（M）评价。

用一个不太恰当的词总结一下，国产版Model 3是一辆“谋财但不害命”的车，即安全性不错，但撞了以后修车费钱。

不过中保研测试过的所有车型中，“耐撞性与维修经济性”这一项似乎只有上汽斯柯达科迪亚克拿了G（优秀），拿A（良好）的都不多（吉利星越、长安福特福克斯等），得P（较差）的我倒是见过不少；原因除了维修经济性不好外，还有不少车是由于在低速碰撞中引爆了气囊。

根据测试规则，气囊引爆会让该项成绩直接变为P（就是说低速碰撞你就给我引爆气囊，纯属浪费钱），大家理解一下。

上图能明显看出Model 3的耐撞性表现不错，但维修经济性扣分严重

但是广大Model 3车友似乎对这个成绩非常不满意，他们认为Model 3应该拿“PGGG”，不然对不起自己“韭菜”的称号。

从中保研发布的碰撞测试图片及视频来看，Model 3在“车内乘员安全指数”大项中表现优秀。在最难应对的正面25%偏置碰撞测试中，虽然左前车轮脱落，但并未侵入乘员舱，气囊也正常弹开；A柱没有断裂，车门依然可以正常开关。

从详细结果来看，国产Model 3的“腿部和脚部”得分拖了后腿，这与该车没有膝部气囊有一定关系。

说到这个正面25%偏置碰撞测试，就必须得鞭尸一下帕萨特，俗话说没有对比就没有伤害，来看看帕萨特的测试结果：

特斯拉不光在电子电气架构上给大众打了个样儿，在碰撞安全上也引领潮流，这难道就是智能又安全的汽车吗？爱了爱了。

侧碰方面，Model 3一片飘绿，表现出色。受撞击一侧的侧气帘全部打开并承托住车内人员头部，B柱发生变形但并未出现断裂。

在头枕/座椅及车顶强度两项中，Model 3也都拿到了G（优秀）评价。

至于“行人保护”这一大项，Model 3拿了A（良好），由于这个项目的测试和评分方法都比较复杂，就不多解释了。不过要是想拿高分的话，吸能保险杠、软性材料前机舱盖都是可用的方案；当然像雅阁那样，多花点钱搞个可弹起的发动机盖效果更佳。

“车辆辅助安全指数”包含前碰预警和主动刹车，这两项配置属于自动辅助驾驶配置，是特斯拉的强项，自然是全部满分。

在整个碰撞测试过程中，Model 3并没有出现电解液泄露、起火等情况，电安全性也有所保障。不过中保研并没有专门针对电安全的测试，C-NCAP倒是有；鉴于C-NCAP风评不好，我倒是很希望中保研能搞一套专门针对电动车的碰撞测试规范。

之前Model 3在欧洲E-NCAP和美国IIHS的测试中都取得了不错的成绩，因此很多人都猜测国产版肯定也没问题；不过因为有帕萨特这个坑爹的前车之鉴，使得很多人开始担心起国产Model 3 来了。不过这次特斯拉算是证明了自己，虽然芯片“不小心”混装了一下下，但安全这块还是有保障的。

既然聊到了国外的碰撞测试，不如一起回顾下Model 3 在欧美的测试情况。E-NCAP（可理解为欧洲的C-NCAP）给了它五星最高评级；IIHS（可理解为美国的中保研）给到了最高级别的安全评级——2019 TOP SAFETY PICK+ 。

▲ 特斯拉Model 3 在E-NCAP的得分

▲ 特斯拉Model 3 在IIHS的得分

要拿到IIHS的 TOP SAFETY PICK+，就必须在驾驶员侧前方小角度碰撞、前方中等面积碰撞、侧面、车顶强度和头部约束测试以及乘客侧前方小角度碰撞、前大灯测试中都拿到G。此外，前方碰撞预警要拿到先进或优越的评分，特斯拉交上了一份近乎完美的答卷。

跟国内一样，在IIHS的测试中，Model 3 的前排腿部依旧是扣分项，而且能明显看出国产版减配了膝部气囊，希望未来特斯拉能针对这一点做出改进吧。

单看两次碰撞结果，国产版的Model 3 在正面25%偏置碰撞测试中表现的比美版要好（中保研和IIHS测试标准相似），国产版A柱弯折角度较小，驾驶员腿部的受入侵程度也较小。进口版车主欲哭无泪，原来我们才是真韭菜。

邦点评

总的来看，国产版特斯拉Model 3在各类碰撞测试中表现都可圈可点，安全性有保障，此前担心碰撞测试结果的车主朋友和吃瓜群众们可以放心了。目前国产版有标准续航升级版/长续航版/高性能版三种可选，售价分别为27.115/34.405/41.98万元，并且还申报了搭载磷酸铁锂电池的新车型，也就是说售价有可能进一步下降！

从各种角度来说，它都算得上一款有理由面、值得入手的电动车，如果你是持币待购的新能源潜在消费者，不妨考虑一下国产Model 3。
