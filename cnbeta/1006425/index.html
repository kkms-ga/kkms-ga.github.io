Sabrent开售Rocket Q4系列4TB NVMe PCIe 4.0 SSD新品

July 22, 2020

Sabrent 新发布的 Rocket Q4 系列 PCIe 4.0 NVMe 固态硬盘已在亚马逊上架。其采用了 M.2 2280 的标准外形，目前可选 2TB / 4TB 版本，官方指导价为 320 / 770 美元（约 2239 / 5389 RMB），但不带原厂散热装甲的价格要更实惠一些（750 美元 / 约 5250 RMB）。参数方面，该 SSD 可达成 4900 和 3500 MB/s 的顺序读写速度。

由于 PCIe 4.0 主控的性能和功耗都较 PCIe 3.0 提升不少，Sabrent 显然希望消费者使用原厂的热管散热器。

不差钱的朋友，还可选购 8TB 版本的 Rocket Q4 系列 PCIe 4.0 NVMe SSD，预计售价在 1600~2000 美元之间。

近段时间，Sabrent 的高规格 SSD 已经在市场上惊起了相当大的波澜，现在我们很高兴又迎来了市面上首款 4TB 容量的 PCIe 4.0 NVMe SSD 。

早些时候，Sabrent 还发布了 Rocket XTRM-Q 系列的外置式 8TB 固态硬盘，以及全球首款 8TB NVMe M.2 SSD（Rocket Q 系列 PCIe 3.0 NVMe SSD）。

不过现在，PCIe 4.0 版本的 Sabrent Rocket Q4 系列 NVMe M.2 2280 SSD 终于登陆亚马逊线上商店。

2TB / 4TB 版本的当前售价为 319.99 / 749.99 美元（约 2240 / 5250 RMB），预计 500GB / 1TB 版本或许也将很快到货。
