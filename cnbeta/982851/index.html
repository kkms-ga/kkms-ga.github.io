三星Galaxy Note 20+渲染图曝光：6.9寸挖孔屏 首次加入潜望镜头

May 25, 2020

5月25日消息，知名爆料人士Onleaks放出了三星Galaxy Note 20+的渲染图。如图所示，三星Galaxy Note 20+采用6.9英寸双曲面OLED显示屏，机身三围尺寸为165mm×77.2mm×7.6mm，底部的S Pen手写笔移至机身左侧位置。

访问购买页面:

SAMSUNG - 三星旗舰店

和Galaxy S20 Ultra一样，Galaxy Note 20+采用了潜望式长焦镜头（Galaxy Note系列首次使用潜望镜头），摄像头模组有所凸起，厚度达到了10.7mm，关于摄像头的规格暂时不得而知，之前有爆料称三星Galaxy Note 20+主摄为一亿像素。

核心配置上，Galaxy Note 20+搭载的是高通骁龙865旗舰平台，配备16GB内存。

该机将于8月份发布，考虑到疫情影响，三星有可能会在线上举行Galaxy Note 20系列发布会。

值得注意的是，本次发布会还将推出标准版Galaxy Note 20。

Onleaks爆料，Galaxy Note 20的屏幕曲率要比Galaxy Note 20+低，与直屏较为接近。

此外，Galaxy Note 20似乎采用的是60Hz刷新率显示屏，分辨率可能是FHD+，而Galaxy Note 20+的刷新率为120Hz，分辨率为QHD+。
