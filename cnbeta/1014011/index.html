Galaxy Note 20在韩国市场有三种额外的机身颜色

August 11, 2020

在Galaxy Note 20上，三星公司也在为其家乡保留一些专属色调。韩国买家显然得到了一些优惠待遇，让Galaxy Note 20多了三种颜色，你可能在其他地方买不到，至少现在是这样。在韩国国内市场销售的Galaxy Note 20会有三种额外的颜色，它们分别是神秘红、神秘蓝、神秘粉。

神秘红和其他红色手机一样，总是很抢眼，但是并非耀眼，而神秘粉色则看起来有点独特，蓝色虽然是个人最喜欢的颜色，可惜现在已经太常见了，无法真正打动人。凭借Galaxy Note 20和Galaxy Note 20 Ultra，三星试图从Galaxy S20销量的打击和COVID-19的影响中恢复过来。正如预测的那样，它收到的反馈相当复杂，主要是由于它的价格标签。三星和零售合作伙伴肯定会抛出一些优惠和赠品，让手机看起来更实惠，有更多的颜色选择也可以帮助左右一些对它有利的印象。
