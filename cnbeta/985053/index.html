宝马新产品规划曝光：全系推广48V轻混 升级7.0中控系统

May 30, 2020

近日，有海外媒体曝光了宝马最新的产品规划，其中多款车型将会搭载48V微混系统，BMW Operating System 7.0中控系统还将迎来升级。此外，针对欧洲市场，部分车型的动力参数也将调整。

具体来看，宝马计划在今年夏季推出多款搭载48V轻混技术的车型，包括3系、5系、6系和7系、X3、X4、X5、X6和X7。此外，所有配备48V微混系统的车型还额外会配备一块较小容量的电池。

这些新车在搭载48V微混系统后，除了进一步降低排放及油耗水平，电机还会为整车额外提供11马力（8千瓦）的动力输出。

此外，宝马还特别针对欧洲市场，推出最新一代的3.0升六缸柴油发动机，新发动机的动力参数有所提升，并且新车同样会搭载48V轻混系统。宝马还为插混车型更新了最新的eDrive Zone功能，该功能仅在欧洲地区使用，可以让车辆进入低排放低区时自动切换至EV模式。

配置方面，宝马Driving Assistant Professional系统应用在了3系、5系、6系、8系、X5、X6、X7、M5及M8中。

该系统增加了主动导航功能，驾驶员设定好目的地之后，系统可以按照规划线路自动变更车道。同时，该系统还更新了紧急车道辅助功能，而全速域ACC自适应续航也支持更多复杂路况。

值得一提的是，车辆的BMW Operating System 7.0（宝马操作系统7.0）已经可以支持Android Auto，该功能支持访问Google Maps（谷歌地图）及Google Assistant（谷歌助手），今年夏天该系统会陆续安装至宝马多款车型上。

具体车型方面，宝马3系也会更新至最新的BMW Operating System 7.0，同时还增加了新的M选装运动座椅以及新的Phytonic Blue金属车漆。而宝马7系和8系还增加了优化过后的集成式主动转向系统。
