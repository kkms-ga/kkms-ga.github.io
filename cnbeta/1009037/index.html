微软首次公布GitHub产品路线图

July 29, 2020

微软于今日首次公布了 GitHub 产品路线图，以便开发者预先了解 GitHub 团队正在打造的项目。这个新建的公共信息库允许任何人访问，除了做好在未来几个季度内迎来新功能的准备，开发者还可积极地向开发团队提交反馈。感兴趣的开发者查看各个项目版块的更新预告，点击链接后即可跳转到相关详情页面。

（来自：GitHub | Projects）

路线图上的每个项目都可以当做一个提问版块，并且带有如下内容标签：

（1）发布阶段：在路线图中描绘特定项目在未来将有哪几个预期的阶段。

（2）功能区域：用以指示特定项目所述的产品领域。

（3）特性介绍：描绘项目的特性或相关产品。

（4）产品 SKU：通过一个或多个产品标签，罗列出哪些预期功能即将到来。

（5）部署模型：如非特别说明，意味着默认将登陆云端（而不是服务器端），然后再通用发布。
