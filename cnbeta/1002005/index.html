固件信息显示Galaxy Note 20系列的全球版本将继续搭载Exynos 990

July 11, 2020

曾有人猜测，三星将推出的Galaxy Note 20系列会采用自研Exynos 992芯片，而不是S20系列中的老款Exynos 990。然而，目前情况看起来有所不同。我们已经知道，Galaxy Note 20设备将在美国和中国等一些市场配备骁龙865+。然而，根据最近的传闻显示，不少国际版本可能会继续使用Exynos 990而不是高通芯片或者是升级后的992芯片。

该传闻来自Max Weinbach，他在Twitter上分享了所有版本的Note 20固件中都没有Exynos 992处理器的证据，但固件中确实包含Exynos 990的信息。

如果这还不够，Max还分享了一份三星One UI 2.5将支持的芯片清单，One UI 2.5就是三星下个月推出的新设备上将运行的定制系统。

通过列表我们可以看出，列表中没有直接或间接提到Exynos 992。这进一步表明，我们可能不会看到Exynos 992成为Galaxy Note 20的国际版处理器。如果你还记得的话，曾有报道称Exynos 992将在8月份进入量产，也就是三星在Galaxy Unpacked活动上发布Galaxy Note 20系列的时候，难道三星决定使用老款芯片只是因为他们无法按时推出新款芯片？

三星决定在国际版本机型上坚持使用老款芯片，这肯定会引起那些对处理器不满而跳过Galaxy S20系列等待Note 20系列的人感到沮丧。
