谷歌发布全新AR技术 单摄像头即可实现AR景深感应

June 25, 2020

早在去年12月，Google就展示了如何使用单个摄像头为AR增强现实创建深感图，而在今天，ARcore深度API终于在Android上线，并且已有多个第三方应用程序已经开始使用。虽然其他厂商也在试图为自己的产品添加AR功能，但大多都是通过额外的硬件设备，诸如增加ToF模组。

使用双摄等方法实现深感图的感知和创建，而单摄象头因为信息不足，所以很难判断画面中的物体距离摄像头的距离到底如何。

而谷歌利用动态深度算法，仅靠单摄就实现了深感图的创建，并且拥有不错的精度。保证虚拟物体被正确的遮挡，不在空间中漂浮或者放置在物理层面上不可能放置的位置。

据悉，随着ARCore 1.18版（适用于AR的Google Play服务）的推出，Depth API将在“数亿个兼容的Android设备”上提供。Google将首先通过AR Animals in Search演示了该功能，之后将有新的合作伙伴陆续展示该技术。

不过目前，作为开发该技术时的合作方，Snapchat以及三星都已经开始上线利用Depth API的功能。Snapchat更新了多个滤镜，可以利用Depth API的优势，为单摄手机实现AR效果。

而三星，将在Galaxy Note 10+和S20 Ultra上将深度API用于其Quick Measure应用程序中，虽然已经拥有ToF传感器，但是结合新的算法可以进一步提升质量，减少扫描时间并加快测量速度，三星将在为了几个月内更新其应用程序。

同时，Google Creative还展示了一个新玩法，比如利用Depth API 玩多米诺骨牌。在谷歌提供的深度实验室应用程序中，海战时了Depth API的其他应用，比如逼真的物理、表面交互效果和环境遍历功能。
