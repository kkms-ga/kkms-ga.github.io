三星Galaxy Note 20机型360°渲染图曝光 采用平面显示屏

July 18, 2020

三星即将于 8 月 5 日的 Unpacked 发布会上正式揭晓 Galaxy Note 20 系列智能机，但 Evan Blass（@evLeaks）已经抢先在 Patreon 网站上分享了 Note 20 机型的 360° 渲染图。动图揭示了 Note 20 的一些有趣细节，比如“普通”的平面显示屏、以及熟悉的打孔屏前摄。上个月的爆料还指出，与 Note 20+ 和 Note 20 Ultra 相比，该机可能只支持 60Hz 的刷新率。

渲染图揭示 Note 20 采用了后置三摄，不免让我们想起了今年早些时候发布的 Galaxy S20 。

不过与 Galaxy S20 相比，Note 20 的后摄模组看起来要更“壮硕”一些，预计三星将引入至少一项的光学器件升级。

最后，虽然 @evLeaks 这次没有分享有关 Note 20 的更多规格，但早前爆料称该机将在某些市场采用高通骁龙 865+ 芯片组，但其余国际市场的 Note 20 机型或配备较老的 Exynos 990 芯片组。
