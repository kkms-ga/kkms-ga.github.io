Xbox Game Pass韩国真人广告片 Faker出演

August 24, 2020

韩国电讯公司——SK电讯今日发布了一个Xbox Games Pass真人广告宣传片，请来了《英雄联盟》职业玩家Faker和韩国足球明星孙兴慜联合出演，同时还宣传了韩国三星手机Galaxy Note 20。

除了直接的分发和通过社交网络外，Xbox正在和电讯供应商合作以增加Xbox Game Pass以及xCloud的普及，特别是在Xbox在历史上影响力不够充分的地区。

云游戏是5G技术的很好使用范例。此前韩国电信推出了自己的云游戏平台GameBox，支持100多款游戏，每月8.5美元。

Gamebox

LG U+目前支持Nvidia的GeForce Now。

安卓版Project xCloud将于9月15日上线，仅面向Xbox Game Pass终极版会员，支持22个国家，亚洲地区目前仅韩国在内。
