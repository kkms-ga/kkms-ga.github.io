[视频]谷歌Pixel 4a和苹果iPhone SE对比评测

August 06, 2020

本周，谷歌推出了售价为 349 美元的 Pixel 4a 手机，旨在与苹果的 iPhone SE 等其他设备竞争。那么相比较 iPhone SE，Pixel 4a 在哪些方面更有优势，在哪些方面有所不足？外媒 MacRumors 就带来了两款中端机型的详细评测。

首先，Pixel 4a 和 iPhone SE 均为“预算”手机，缺乏旗舰机型的某些功能，主要任务是吸引更多用户选购。Pixel 4a 采用 5.81 英寸的 OLED 面板，比 iPhone SE 的 4.7 英寸明显更大。该机正面有个打孔前置摄像头，因此屏占比更高，可视面积更大。

在显示方面，Pixel 4a 的 OLED 面板更好，但是两者的差距并不是特别大，具体可以看视频中两者的屏幕对比。

iPhone SE的底部边框有一个Touch ID Home键，Pixel 4a也使用了指纹传感器，但它位于设备的后部。在准确性和手机解锁时间方面，Touch ID击败了大多数其他指纹传感器，这里也不例外。谷歌的指纹传感器很好，但Touch ID更准确，更不容易出现故障。

为降低成本，Pixel 4a 使用塑料材质，而 iPhone SE 则是采用玻璃和铝合金中框材质。相比较而言，iPhone SE 给人的感觉更高端，但玻璃也有个缺点就是易碎。

iPhone SE 配备了苹果最新的A13芯片，速度非常快。这让iPhone SE在性能方面与苹果旗舰iPhone不相上下，这个因素让iPhone SE的399美元的价格对于那些更注重性能的用户来说是很划算的。

Pixel 4a拥有八核高通骁龙730处理器，但其性能并没有达到A13的水平。在日常使用中，你可能不会注意到明显的差异，但在一些任务中，iPhone SE会比 4a 更胜一筹。

在电池续航方面，Pixel 4a的电池容量为3140mAh，而iPhone SE的电池容量为1821mAh，MacRumors摄像师Dan在测试中觉得Pixel 4a的电池续航明显更好。

Pixel 4a还拥有6GB内存和128GB基础存储空间，而iPhone SE拥有3GB内存和64GB基础存储空间，所以你用更低的价格买到了 Pixel 更多的存储空间。

iPhone SE 提供了 Lightning 接口和无线充电功能，而Pixel 4a则没有无线充电功能，只有一个USB-C接口，这对于喜欢无线充电或USB-C接口接入的用户来说是一个考虑因素。
