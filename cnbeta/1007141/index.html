iOS 14为AirPods带来哪些改进？设备切换更自如

July 24, 2020

7月24日早间消息，AirPods可谓是最受 iPhone 用户欢迎的配件。在iOS 14 中，苹果又为它增添了不少新亮点。外媒Macrumors汇总了在iOS 14 下AirPods的一些新功能，新浪数码为您编译。苹果iOS 14拥有多项新功能，其中包括多项对AirPods和AirPods Pro与iPhone和iPad 的协作方式的改进，比如空间音频、自动设备切换，低电量通知等功能。

空间音频

空间音频

这个功能支持仅AirPods Pro，需要搭配iPhone 7或以上机型来使用。根据苹果的介绍，通过使用定向音频滤波器和对每只耳朵接收的频率的微调，空间音频能够将声音“放置”在空间中的任何位置，从而让用户在观看电视节目和电影时获得身临其境的音频体验。同时空间音频还能够利用‌AirPods Pro‌和‌iPhone‌中的陀螺仪和加速度计来跟踪头部的移动和设备位置。

音乐更有沉浸感

在AirPods Pro中实现空间音频需要固件更新，该固件尚未发布。

自动切换设备

在iOS 14当中，无需进行手动设置， AirPods能在不同设备之间无缝切换配对。比如当用户结束iPhone上的通话，拿起iPad准备听播客时，AirPods会自动切换过来。

自动切换设备

此功能支持 AirPods Pro、AirPods （第二代）、Powerbeats、Powerbeats Pro 和 Beats Solo Pro。

AirPods实现自动切换需要用户拥有iCloud帐户并且iPhone，iPad，iPod touch，Mac和 Apple Watch运行了最新的固件。

建议用户等测试完毕后，苹果发布正式版时再统一升级，避免影响日常使用。

低电量提示

低电量提示

如果AirPods快没电了，在iPhone中会出现提示，方便用户及时为AirPods充电。

优化充电

优化的充电

为了最大限度地延长AirPods的续航时间，苹果在iOS 14中新增了优化电池充电功能。优化电池充电功能可以让AirPods学习你的日常充电习惯，苹果在iPhone和Mac上使用过类似的电池优化功能，这个功能以延长设备的电池寿命。避免让锂离子电池持续处于最大充电状态，可以更好地长期保持电池健康。

耳机调节器

耳机适应功能是为听力不好的人提供的一项无障碍功能，它能够放大柔和的声音，调整频率，让音乐、电影、通话等听起来更加清晰。

声音自动调节

通过设置可以将音频调整为平衡音、人声的范围和亮度，或将柔和声音的音量调整为更大。

此外，还有一个自定义音频设置，可以通过轻声说话和不同的音乐进行测试，以确定你是否有特定的音频偏好，应通过耳机适应功能进行补偿。

也能与AirPods Pro上的透明模式配合使用

耳机适应功能也能与AirPods Pro上的透明模式配合使用，让安静的声音变得更响亮，并调整周围环境的声音以满足你的音频需求。

听力健康

听力健康

苹果iOS 14和watchOS 7中扩展了听力健康保护功能，当用户以过大音量听音乐时iPhone会向Apple Watch发送通知，一旦用户达到了世界卫生组织的建议值时，AirPods会将耳机的音量降低到合适的水平。“声音和触觉”中的一项新功能会与 “降低声音”一同启用以分析耳机音频并减少超过一定分贝级别的声音，用户也可以选择自行设置。

在控制中心添加“听力”功能

控制中心音量监控器

苹果在控制中心中添加了 “听力”功能，用户可以看到实时音量等级以确保声音音量安全。只需将向下滑动，然后点击耳朵图标（将其添加到 Control Center 后）即可查看。

AirPods Pro运动 API

苹果公司为AirPods Pro设计了一个 Motion API，开发人员可以充分利用它。Motion API 使开发人员可以访问方向，用户加速度和旋转速率，这对于健身应用程序和游戏非常有用。开发人员可以将 API 内置到AirPods Pro用户可以下载的应用程序中。
