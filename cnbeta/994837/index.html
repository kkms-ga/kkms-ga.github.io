宏碁发布ConceptD 2020个人电脑 为图形专业人员提供强大功能

June 24, 2020

宏碁已经为旗下ConceptD系列推出新产品，包括Windows PC和一系列外设产品，旨在诱导创意人员摆脱对Mac的痴迷。最新阵容包括一对笔记本, 一个2合1笔电和另一个传统的笔电，跨越了便携性和性能之间的界限。

作为2020年ConceptD阵容的旗舰产品，ConceptD 3 Ezel将于8月到货，售价1499.99美元起。宏碁为其搭配了独特的白色全金属机身，配备14或15.6英寸FHD触摸屏和琥珀色背光键盘，并通过360度铰链将两者连接起来。

无论哪种尺寸的屏幕，都通过了Pantone认证，100%sRGB色域覆盖，delta E<2色彩精度。除了触摸，它还可以与宏碁的Active Stylus笔配合使用。另外，对于更传统的使用，还有一个超大的玻璃保护触控板。内部有第十代英特尔酷睿i7-10750H或酷睿i5-10300H处理器可供选择，而图形是NVIDIA Quadro T1000在ConceptD 3 Ezel Pro，或GeForce GTX 1650Ti提供。存储是1TB PCIe SSD, 而端口包括一个SD卡阅读器和雷电3接口。14英寸型号重3.7磅, 而其15.6英寸的型号重4.29磅. 宏碁表示，电池寿命长达18小时。

同时，那些不需要可旋转显示屏的用户还有ConceptD 3可以选择，它将在10月份上市，售价999.99美元起。与它的旋转版本一样，有14或15.6英寸FHD触摸屏可供选择，并有Pantone验证。

内部有第十代英特尔酷睿H系列处理器，以及ConceptD 3 Pro中的NVIDIA Quadro T1000，或者GeForce GTX 1650Ti显卡。电池续航时间长达20小时。

台式机ConceptD 100本月在欧洲上市，起家799欧元，目标用户那些做图形密集型 2D 和入门级 3D 设计的人员，其中采用第九代英特尔酷睿处理器和 NVIDIA GeForce 或 Quadro 图形卡。尽管是SFF尺寸，但有高达256GB的SSD存储空间和高达32GB的DDR4内存。宏碁表示，ConceptD 100的噪音保持在40分贝以下，即使在负载状态下，这要归功于精心设计的冷却系统。

宏碁ConceptD显示器三巨头中的第一款ConceptD CM3在8月上市，售价709.99美元。它针对图形设计师和照片编辑人员，它拥有4K 3840 x 2150显示屏，刷新率为60 Hz，支持99%的Adobe RGB色彩空间。它还通过了Pantone验证，并支持Delta E<2和VESA DisplayHDR 400，另外还兼容AMD Radeon FreeSync。

对于那些预算较高的用户，ConceptD CP3在8月份登陆，售价619.99美元。它拥有一块WQHD 3840×2160显示屏，刷新率高达165Hz。还支持VESA DisplayHDR 400。支持高达98-%的DCI-P3色域。

最后，ConceptD CP5于8月到货，售价为749.99美元。它承诺Delta E<1色彩精度和Pantone验证，其WQHD 2560×1440分辨率的屏幕具有10.7亿色彩，并支持VESA DisplayHDR 600和170 Hz刷新率。还有99%的Adobe RGB覆盖率和自适应同步支持。
