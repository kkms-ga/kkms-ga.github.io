[图]Windows 10硬件加速GPU调度即将上线 AMD和Intel的已在路上

July 01, 2020

Windows 10 May 2020（20H1/Version 2004）功能更新中，微软引入了 Windows Display Driver Model (WDDM 2.7)改进，为 GPU 添加了硬件加速调度（HAGS）功能。

硬件加速 GPU 调度是 WDDM 2.7 的主要功能，它允许集成/独立显卡直接管理自己的显存，从而让你获得更好的性能，降低延迟，提高平均或最低帧率，改善视频播放效果。

在 Intel 和 NVIDIA 此前发布的驱动中，虽然包含了对 WDDM 2.7 的支持，但问题是在 Windows 设置中并没有硬件加速 GPU 调度的选项。

英特尔表示：“WDDM 2.7引入了一项名为硬件加速GPU调度的新功能。目前最新的英特尔DCH驱动程序并不支持这一功能，[但]英特尔希望在未来的驱动程序版本中加入对这一功能的支持。”

昨日，AMD 悄然放出了 20.5.1 Beta GHS 版驱动，其中 GHS 正是代表“Graphics Hardware Scheduling”(图形硬件调度)。AMD显卡终于开始真正支持Windows 10 v2004的硬件加速GPU调度功能，可在Windows系统设置的图形设置处开启。目前AMD显卡仅限最新的RX 5700/5600系列高端卡，即便是RX 5500系列还未加入，更别说更老的RX Vega、RX 500/400系列了。
