Facebook发布工具从单张照片实现高清人像3D数字化

June 23, 2020

深度神经网络推动了基于图像的三维人体形状估计技术的进步，但现有的技术还无法产生细节丰富的高清人体三维结构。主要限制是创建精确的 3D 模型需要更多的上下文背景和更高的图像分辨率，以及更多的内存。

南加州大学和 Facebook  的研究人员提出了 PIFuHD（PDF） 去解决这个问题，代码发布在 GitHub 上。从简单图像重组高清三维人像的演示动画。
