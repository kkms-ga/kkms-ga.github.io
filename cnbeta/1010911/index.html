Galaxy Note 20将搭载三星全新UWP技术：传输速度比NFC更快

August 03, 2020

8月3日消息，爆料人@SaudiAndroid得到独家消息称三星将推出一种全新的技术UWP，它可以以极高的速度进行数据传输，比NFC更快。该技术将首次被应用在Galaxy Note 20系列上，后续多家品牌将跟进这一技术。目前关于UWP的细节不得而知，从爆料信息来看，三星Galaxy Note 20系列将带来一项比NFC更好的方案，值得期待。

据悉，三星Galaxy Note 20系列将于8月5日正式发布，包含Galaxy Note 20、Galaxy Note 20 Ultra两款。

爆料指出，Galaxy Note 20、Galaxy Note 20 Ultra为不同市场提供了Exynos和骁龙两款平台，欧版搭载的是Exynos 990，国行、美版搭载的是骁龙865 Plus。

有消息称Galaxy Note 20系列搭载的是Exynos 990优化版，其速度和骁龙865 Plus版一样快。

核心配置上，Galaxy Note 20采用6.7英寸直屏，分辨率为FHD+，刷新率为60Hz，Galaxy Note 20 Ultra采用采用6.9英寸AMOLED柔性屏，分辨率为3200×1440，刷新率为120Hz。
