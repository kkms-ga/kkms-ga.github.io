[图]联想发布ThinkStation P620：首次使用AMD Threadripper PRO

July 15, 2020

今天，联想官方发布了首先使用 AMD Threadripper PRO 处理器的工作站– ThinkStation P620。该工作站介于单插槽的 P520 和双插槽的 P720 之间，提供最高 64 核的 AMD 单插槽解决方案，满足用户的个性化需求。

ThinkStation P620 是目前唯一一款你能买到的搭载64核128线程工作站。它的时钟频率最高可以达到 4.0 GHz，最高获得 64 个核心，并搭载 AMD 的 2019 Premium Chipset BXB-B。

该 CPU 支持最高 128 通道（lane）的 PCIe 4.0 带宽（P620 在实际使用过程中不会使用所有 128 通道），支持最高 1TB DDR4 3200MHz 的八通道内存。存储方面最高可以使用 4TB 的 M.2 PCIe SSD，最高 16TB 的 3.5 英寸 7200 转机械硬盘。M.2 支持 RAID 0/1，SATA 支持 0/1/5/10。联想在机箱中适配了 1000W 的电源。

在显卡方面，最多支持两块 NVIDIA Quadro RTX 8000 或四块 Quadro RTX 4000 显卡。这总共可以为 16 台显示器供电，最高可提供 96GB 帧缓冲。

联想在主板上加入了 10GB 以太网，以腾出一个 PCIe 插槽。这一点很重要，因为–每条 PCIe 通道的传输速度是旧标准的两倍–你可以充分利用这些 PCIe Gen 4.0 插槽。

和英特尔工作站相比，联想和 AMD 承诺：相比于基于两个 CPU 的英特尔双处理器工作站，P620 装备的 64 核 Threadripper PRO 3995X 显然可以提供 20％ 以上的性能。如果搭配高端 GPU，可用的图形性能应该也会更高。

虽然这款工作站主要面向媒体和娱乐用户，不过 Epic Games 以及一些不同垂直行业的用户也会对此感兴趣，这包括金融和保险、能源/石油/天然气行业、人工智能和软件开发等。

在拥有强悍硬件配置的同时，联想把这一切都装进了熟悉的机箱中，采用和 P520 相同的 33 升塔式机箱，采用了传统的风冷散热。AMD和联想共同为Threadripper定制了一个散热器，同时在机箱中采用了通道散热。这样就可以避免高端 Threadripper 在消费端对液冷的要求。

前面有两个USB 3.2 Gen 2 Type-A接口和两个USB 3.2 Gen 2 Type-C接口，以及一个麦克风/耳机组合插孔。背面有4个USB 3.2 Gen 2 Type-A、2个USB 2.0 Type-A、2个PS/2、1个万兆以太网、音频线输入、音频线输出和麦克风输入。里面有一块带蓝牙的英特尔PCIe WiFi卡。

此外，还提供了超薄光驱、15合1媒体读卡器和前置式存储外壳。总共最多可以容纳6个硬盘的空间，最多可以容纳4个内部存储托架。联想表示将会在今年 9 月底上市发售，起售价格为 4599 美元。
