[视频]Visual Studio Live Share新增聊天和语音功能

May 20, 2020

2017年，微软发布了Visual Studio Live Share。这是一项面向开发者的云服务，以提高他们在编码过程中的协作能力。通过Visual Studio Live Share，开发人员可以与队友或其他开发人员共享项目。即使使用不同的操作系统，也可以在他们的个性化编辑器或IDE中编辑和调试相同的代码。

访问:

微软中国官方商城 - Visual Studio

今天微软宣布Live Share将为Visual Studio Code和网页端Live Share整合聊天功能。在官方简短的介绍中写道

Live Share的聊天和语音能够让用户在Live Share会话中进行全面的沟通。在使用Visual Studio Code和Live Share web时，会话中的主持人和嘉宾都可以访问这些功能。

这些预览功能将在未来一周内提供给Visual Studio Code和Live Share web客户端。
