Galaxy Note 20 Ultra最新渲染图曝光 显示可能带有红外遥控功能

July 15, 2020

Galaxy Note 20 Ultra最近备受关注，这很简单。它将定义Note 20系列的最高端作品。因此，它所受到的关注度是显而易见的，至少可以说。在过去的几周里，有大量的相关消息泄露和曝光。

俄罗斯三星意外地在其网站上发布了神秘的青铜色Galaxy Note 20 Ultra的照片。起初，我们以为这些照片只是概念，但之后泄露的消息就开始源源不断。此时，关于Note 20 Ultra的一切都已经呈现在公众眼前。据报道，Galaxy S21 Ultra正在进行三种屏幕尺寸的测试，三三星准备超越7英寸显示屏障碍。

现在，Galaxy Note 20 Ultra再次泄露，展示了它的曲线和更平坦的顶部设计，以及一个IR 发射孔，泄露的消息来自于冰宇宙，他最近一直在发布很多与Note 20设备相关的泄露信息。但这次的泄露中又有了新的内容。渲染图印证了我们过去看到的一切。然而，这里的新因素是，就在你的SIM卡托盘旁边，我们看到的东西，看起来很像一个红外发射器，这至少可以说是奇怪的，因为我们上一次在三星手机上看到红外发射器是在2014年发布的Galaxy S5，所以看到三星带回这样的老技术是令人惊讶，它主要用于控制电视或空调，或者其他有遥控器的设备。

据悉，Galaxy Note 20系列将在8月5日的Galaxy Unpacked活动中正式上市。
