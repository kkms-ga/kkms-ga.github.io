扎克伯格：抵制Facebook的广告主很快就会回归

July 03, 2020

北京时间7月3日凌晨消息，500多名广告主以Facebook没有控制仇恨言论为由对其发起联合抵制，但该公司CEO马克·扎克伯格（Mark Zuckerberg）对此并不感到担心。扎克伯格在Facebook内部会议上表示：“我猜想，所有这些广告主很快就会回到我们的平台上来。”

据外媒获得的会议记录显示，扎克伯格在会上谈到了Facebook员工对各大品牌暂停在其平台上投放广告的担忧。

尽管遭到广告主的抵制，但扎克伯格的立场很坚定。他在内部会议上说道：“我们不会因为公司收入的一小部分受到威胁而改变自己的政策或方法。”

广告业务是Facebook最大的收入来源，扎克伯格在此前向美国国会作证时的名言“参议员，我们做广告”（Senator， we run ads）就证明了这一点。去年，Facebook营收高达近700亿美元。

虽然大公司暂停在Facebook平台上投放广告对该公司来说是个令人担忧的结果，但到目前为止，这种抵制活动还处于早期阶段。据外媒报道，虽然可口可乐、微软、星巴克和塔吉特等大公司都参与了抵制活动，但这些公司并非在Facebook平台上花钱最多的公司。报道指出，在Facebook平台上25个花钱最多的广告主中，只有3个公开确认将会加入抵制活动。这25个广告主为Facebook贡献了近20亿美元收入，在其营收中所占比例为3%。

迄今为止，许多广告主都加入了为期一个月的抵制活动，要求Facebook承担更多责任。有报告称，Facebook是仅次于谷歌的全球最大数字广告网络，占据着超过21%的市场份额。因此，对于许多公司来说，如果没有后备的社交媒体营销计划就贸然退出Facebook广告网络，这种做法可能是行不通的。

扎克伯格认为广告主将会重返Facebook平台，他的这个想法可能是对的。但是，Facebook必须确保采取足够的建设性措施，让广告主相信该公司正在认真对待仇恨言论。
