《对马岛之鬼》游戏地图究竟有多大

July 18, 2020

Sucker Punch工作室最新发售的日本武士题材开放世界动作游戏《对马岛之鬼》在PS4世代末期为玩家们献上了最后的独占大餐。而作为一款开放世界游戏，对比世界地图的尺寸是许多玩家十分关心的事。

以前测量游戏地图尺寸是一件很麻烦的事，不过《对马岛之鬼》的地图方便地以千米为单位指示了距离，所以估测起来很容易量化。

媒体twinfinite用地图工具分别测量了南岛和北岛的最长轴和最宽轴，估算出了大致的矩形面积：

北岛 = 7.5 x 2.5 = 18.75 km2

南岛 = 4.6 x 2.1 = 9.87 km2

所以合集地图面积约为28.62 km2

这个数字在众多开放世界里并不能说是特别大，不过考虑到开放世界内容密度的问题，尺寸也并不能说明一切。参考其他开放世界游戏地图面积：

《塞尔达传说：荒野之息》：74.85km2

《巫师3》53.87km2

《对马岛之鬼》28.62km2

《天国：拯救》15.80km2

《漫威蜘蛛侠》11.97km2

而在现实世界中，日本的对马岛面积则要大许多，足足有708平方千米。当然这个面积依然比不上电子游戏中最大的开放世界地图：《正当防卫4》的1000平方千米大地图。
