[图]微软：更多设备将自动升级至Windows 10 Version 2004

July 24, 2020

Windows 10 May 2020（20H1/Version 2004）功能更新已经于今年 5 月正式发布，但依然有部分设备没有进行升级。除了部分设备由于兼容性问题导致外，还有部分符合条件的设备用户不选择升级。上月在强制 Version 1809 升级之后，今天微软表示面向 Version 1903/1909 功能更新发送自动更新。

在 Windows 10 Version 2004 功能更新的官方支持文档中写道

截至2020年7月22日的现状

对于当前运行 Windows 10 Version 1903/1909 ，也就是那些可以在 Windows Update 中手动“检查更新”的用户，现在开放 Version 2004 了。我们现在开始进入新的推广阶段。利用基于机器学习的现有训练，针对那些即将停止服务的设备我们将扩大自动升级到 Version 2004 的规模。我们将会在各个阶段训练我们的机器学习，从而更智能化的推广部署 Windows 10 系统，提供更流畅的更新体验。推荐服务状态是 Semi-Annual Channel。

这意味着那些依然运行旧版功能更新，但是符合升级条件的设备，微软将会自动更新到 Windows 10 Version 2004。虽然强制更新肯定会遭到一些用户的不满，但升级到新系统还是很有必要的。
