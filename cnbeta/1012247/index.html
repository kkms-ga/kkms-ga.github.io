[图]Windows 10 Build 20185发布：优化DNS设置 增强Your Phone应用

August 06, 2020

面向 Dev 频道，今天微软放出了 Windows 10 Build 20185 预览版更新。本次更新主要对设置应用中的 DNS 选项进行了优化，为 MDM 提供了新的 ADMX 支持策略，更新了 Your Phone 应用程序，允许用户直接访问 Android 应用程序。

Windows 10 Build 20185 中主要有以下新功能

改善设置应用中的 DNS 配置

● 让 DNS 设置更容易访问

让你进入网页属性页面的时候，DNS 服务器分配选项现在放在了顶层。

● 在设置应用中的加密 DNS 控制

加密DNS（DNS over HTTPS，或DoH）可以帮助提高浏览网页时的隐私和安全性。您现在可以直接在 “设置 “应用程序中配置DoH，以便更轻松地利用这一功能。

○ 以太网连接

进入设置>网络和互联网>状态。单击 “属性”，然后选择 “编辑IP分配 “或 “编辑DNS服务器分配”，它将在弹出式中可用

○ Wi-Fi 连接

进入设置 > 网络和互联网 > Wi-Fi。点击适配器属性链接，然后选择编辑IP分配或编辑DNS服务器分配，在弹出的对话框中就可以看到。目前，如果你进入各个网络的属性页面，将看不到加密选项。

你可以添加这里列出的任何IP地址来解锁DoH下拉菜单，并选择使用加密。启用加密后，你可以通过查看网络属性中应用的DNS服务器，看到它们被标注为”(加密) “服务器来确认是否已经生效。

为 MDM 提供全新的 ADMX 支持策略

从 Windows 10 Insider Preview Build 20175 开始，微软已经在 56 个 ADMX 文件中启用了 647 个新的 MDM 策略，使商业客户能够配置同样通过组策略支持的策略。这些基于 ADMX 的策略包括 App Compat, Event Forwarding, Servicing 和 Task Scheduler。这些新策略可以使用 Intune 自定义配置文件进行配置。Intune UX 计划在今年晚些时候推出。

Your Phone 应用改进

您的手机应用程序将继续发展。我们很高兴地宣布另一项激动人心的功能，这是我们的愿景和承诺的无缝延续，可以帮助您提高工作效率，让您的工作更加顺畅。

在部分三星设备上，通过与 Windows 的链接集成，您可以直接从 Windows 10 PC 上即时访问您手机的移动应用程序，从而利用您手机应用程序的强大功能。整个过程不需要在 PC 上安装、登录或者设置你的应用程序，从而为你节省时间和精力。

您可以方便地将您喜爱的移动应用程序钉在PC上的任务栏或开始菜单上，以便快速轻松地访问。当您启动一个应用程序时，它会在您的手机应用程序之外的一个单独窗口中打开，使您能够进行多任务处理。

今年晚些时候，三星Galaxy Note20用户将体验到并排运行多个应用程序的强大功能和便利性，我们将继续与三星合作，将这一功能带到更多设备上。应用程序将在单独的窗口中启动，使您能够同时与多个应用程序进行交互。

所以，无论你是在处理文档，还是在保持自己的工作状态，都可以在不同的窗口中进行操作，而我们也将继续与三星合作，为更多的设备带来这一功能。

面向开发者的更新

现在，Windows SDK 已经和 Dev 频道同步跟进，每当有新的操作系统版本到 Dev 频道时，相应的 SDK 也会被传送到开发频道。您可以随时从 aka.ms/InsiderSDK 安装最新的 Insider SDK。SDK 版本将和 OS 版本一起在Flight Hub存档。

常规修复

● 修复了有时任务栏中的 “关闭所有窗口 “操作并不能关闭所有已钉住的网站的打开标签。

● 修复了任务栏中的网络图标可能会在有活动连接的情况下显示没有网络。

● 修复了从 “开始 “的所有应用程序列表中拖放应用程序到磁贴网格以将其钉住，对于某些应用程序来说并不适用。

● 修复了当所有应用程序列表设置为隐藏时，“开始 “菜单的所有应用程序列表中的字母标题会出现不必要的左侧填充。

● 为了解决在使用全屏开始时，磁贴在某些背景下难以阅读的问题，微软正在调整背景，使其稍微不那么透明。

● 修复了当使用 MDM 应用的开始菜单布局时，磁贴组可能会在几天的运行时间后消失，而无需重启。

● 修复了如果通知中包含一个内嵌图片，应用图标会与它稍微不一致。

● 修复了对于某些桌面桥接应用程序来说，从其任务栏跳转列表启动应用程序不起作用，反而会导致应用程序从任务栏中消失。

● 修复了当使用 “打开文件 “对话框导航到共享的网络文件夹时，如果启用了以前的版本，会导致过多的网络流量。

● 修复了在卸载设置中的默认应用主页面上，卸载设置为默认的应用后，可能会导致显示空白图标而不是加号按钮。

● 修复了打开 “存储设置”，然后立即进入其中一个子页面，然后返回，可能会导致页面无法加载。

● 修复了导致新的Microsoft Edge没有被列在任务管理器的应用程序历史部分。

● 修复了一个影响触摸键盘启动可靠性的问题。

● 修复了TextInputHost.exe会进入无响应状态的问题，导致剪贴板历史记录、表情面板和听写在按下相应的键盘快捷键时无法显示。

● 修复了一个可能导致无法粘贴剪贴板历史记录中的第一个条目的问题。
