[图]B社宣布限免初代《雷神之锤》游戏

August 09, 2020

近日，B社（Bethesda）举行了第 25 届 QuakeCon 线上活动，进行了连续 60 个小时的直播节目。在本次活动中，B社宣布限免初代《雷神之锤》游戏，让骨灰级玩家可以重温一下这款1996年发布的经典射击游戏。

初代《雷神之锤》由 id Software 公司开发，也是《雷神之锤》系列的开山之作。这款游戏沿用了初代《毁灭战士》（DOOM）的元素，玩家需要控制主角 Ranger 来击败名为 Quake 的敌人。

想要免费玩到这款游戏，你必须要在 PC 上下载 Bethesda Launcher，然后登录 Bethesda 账号才可以。与其他游戏启动器一样，Bethesda Launcher 是该公司各种游戏的客户端，适用于 Windows 7/10。
