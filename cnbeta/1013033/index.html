因App Store政策 微软、Facebook接连谴责苹果

August 08, 2020

由于受到苹果App Store应用商店审核限制，微软今天宣布结束针对iOS设备的Xbox Project xCloud测试版本，并放弃在该平台上运行Project xCloud。Project xCloud是微软在2018年推出的首款云游戏平台。该服务可以让玩家在不使用Xbox One主机或Win10 PC设备的条件下，利用手机、平板电脑等移动设备，每月通过Xbox Game Pass平台花费15美元，通过云端服务器运行游玩100多款Xbox One游戏。

过去的几个月，微软Xbox部门一直在TestFlight上测试该软件的iOS版本，并计划在今年9月15日全面开放这一服务。

但很遗憾，由于苹果App Store应用商店审核规则，微软宣布放弃上架，并于9月11日结束xCloud在TestFlight的iOS测试版。

据The Verge报道，微软将继续支持Android平台的Xbox Game Pass及其附带的xCloud云游戏功能，并计划在下个月向公众推出。

微软发布声明：谴责App Store不公平待遇

北京时间8月7日上午，微软发布媒体声明，斥责苹果严格的开发者限制及其对云游戏应用的立场，称苹果以不同方式对待游戏应用程序，将更宽松的规则应用于非游戏App。

微软表示，“苹果公司是唯一一个拒绝消费者使用微软云游戏和Xbox Game Pass等游戏订阅服务的通用平台”。

微软在这份声明中指出，Xbox Game Pass目录中可用的所有游戏均由独立的行业评级机构（例如ESRB和地区等效机构）对内容进行评级。微软认为，可通过第三方内容评估等方式，找到一条途径让苹果上架。

“我们的iOS版Project xCloud预览应用的测试版已到期。不幸的是，我们没有办法通过苹果App Store应用商店将Xbox Game Pass Ultimate的云游戏愿景带给iOS上的游戏玩家。

苹果公司是唯一拒绝消费者拒绝使用云游戏和Xbox Game Pass等游戏订阅服务的通用平台。而且，它始终以不同方式对待游戏应用程序，将更宽松的规则应用于非游戏App，即使它们包含交互式内容也是如此。

Xbox Game Pass目录中可用的所有游戏均由独立的行业评级机构（例如ESRB和地区等效机构）对内容进行评级。我们致力于寻找一条途径，将Xbox Game Pass Ultimate的云游戏引入iOS平台。

我们认为，客户应该是游戏体验的核心，无论他们身在何处，他们都希望在任何地方玩，连接和共享游戏。”微软在声明中表示。

Facebook“跟风”谴责

继微软之后，Facebook也因Facebook Gaming内置功能无法上架而公开谴责苹果的App Store政策。

北京时间8月7日晚7点左右，Facebook首席运营官谢里尔·桑德伯格（Sheryl Sandberg）向《纽约时报》等媒体发表一份声明，谴责苹果公司数月内多次拒绝Facebook Gaming应用程序上线内置独立游戏功能，导致该公司被迫完全放弃，并从iOS版中删除该功能。

“为了获得苹果对独立的Facebook Gaming应用程序的认可，我们必须完全删除游戏功能，这意味着iOS用户的体验要逊于Android平台。”Facebook在声明中表示。

Facebook Gaming是一款游戏视频直播平台，类似Twitch。在Android移动端上，该应用程序内置Facebook Instant Games平台的许多mini游戏，通过类似微信小游戏方式独立运行，但苹果认为该功能是不允许上线的。

Facebook方面表示，苹果引用了《App Store审核指南》4.7来证明拒绝的理由，声称Facebook Gaming应用程序的主要目的是玩游戏，而非看视频。Facebook则引用平台用户使用量的数据回复苹果，称有95％的用户观看视频流，并非是一款游戏类型产品。但苹果并未回复，也没有让该独立应用程序上架。

（注：苹果《App Store审核指南》4.7是指，在app中分发的HTML5游戏不得提供对真实货币游戏、博彩或慈善捐助的访问，也不得支持数字商务。这项功能仅适用于嵌入在二进制文件，且使用苹果审核的相关代码。）

这不是Facebook第一次遇到App Store上架难题。包括在Facebook和Messenger独立软件上，iOS版也删除了内置在Android设备上的小游戏Instant Games功能。

Facebook游戏副总裁Vivek Sharma对The Verge表示，“这是整个游戏行业的共同痛苦，（苹果的做法）最终伤害了玩家和开发人员，并严重阻碍了其他类型格式（例如云游戏）在移动设备上的创新。”

苹果回应：旨在保护消费者

今天早些时候，苹果公司回应了此事，承认该公司不允许微软xCloud平台、谷歌Stadia平台等云游戏服务登陆App Store应用商店。

苹果还表示，应用商店无法通过Xbox Game Pass对每个游戏进行审核，因此相关服务在App Store中不可用。

“App Store的建立是一个让客户发现和下载应用程序的安全可靠的场所，并为所有开发人员提供了巨大的商机。

在进入我们的商店之前，所有应用程序都必须按照同一套指南进行审查，该指南旨在保护客户并为开发人员提供公平，公平的竞争环境。

我们的客户享受着数百万开发人员提供的出色应用和游戏，只要遵循适用于所有开发人员的同一套准则，游戏服务就可以在App Store应用商店上架。”苹果发言人对媒体表示。

苹果公司强调，这些限制旨在保护消费者，并为开发人员提供公平的竞争环境。

虽然微软、Facebook、苹果同为科技巨头，但苹果通过其强大App Store应用商店生态，控制着所有iOS、Mac软件的上架以及存留与否。这种行为让Facebook、微软等科技巨头感到头疼：一方面需要通过依赖苹果生态，来吸引更多用户；另一方面，对于苹果App Store审核规则中的各种“霸道条款”也要努力遵守。

值得注意的是，今年6月，欧盟委员会已经宣布将对苹果旗下的Apple Pay和App Store展开反垄断调查。若欧盟裁定苹果存在垄断行为，那么苹果将面临数十亿美元的罚款。
